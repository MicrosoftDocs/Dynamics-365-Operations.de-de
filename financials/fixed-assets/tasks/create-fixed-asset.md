[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d25d9-104">Für diese Aufgabenanleitung wird das Demo-Unternehmen USMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="d25d9-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="d25d9-105">Damit wird eine neue Anlage mithilfe der Listen Seite "Anlage" erstellt.</span><span class="sxs-lookup"><span data-stu-id="d25d9-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="d25d9-106">Wechseln Sie zu Anlagen > Anlagen > Anlagen.</span><span class="sxs-lookup"><span data-stu-id="d25d9-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>

    * <span data-ttu-id="d25d9-109">Die Nummer wird als Standardwert angegeben, wenn Sie die Funktion "Anlagen automatisch nummerieren" in den Anlageparametern und der Anlagengruppe aktiviert haben.</span><span class="sxs-lookup"><span data-stu-id="d25d9-109">The Number will default if you have enabled Autonumber fixed assets functionality in the Fixed assets parameters and the Fixed asset group.</span></span>  <span data-ttu-id="d25d9-110">Anderenfalls müssen Sie eine eindeutige Nummer zur Identifizierung der Anlage eingeben.</span><span class="sxs-lookup"><span data-stu-id="d25d9-110">If not, you must enter a unique number to identify the fixed asset.</span></span>
