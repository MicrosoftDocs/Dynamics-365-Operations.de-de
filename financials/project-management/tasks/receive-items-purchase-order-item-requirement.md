# <a name="receive-items-on-purchase-order-from-item-requirement"></a>Artikel zu Bestellung aus Artikelbedarf empfangen

[!include[task guide banner](../../includes/task-guide-banner.md)]

Diese Prozedur zeigt, wie Artikel in einer Bestellung aus einem Artikelbedarf empfangen werden.

Durch die Verwendung eines Artikelbedarfs anstelle einer Artikelbuchung kann die Lieferung direkt vor der tatsächlichen Verwendung der Artikel geplant werden. Sie können eine Bestellung erstellen, den Artikel in der Handelsvereinbarung berücksichtigen und den Artikelbedarf in die Produktionsplanung einbeziehen. 

Diese Aufgabe verwendet das USSI-Dataset.

1. Wechseln Sie zu "Projektverwaltung und -verrechnung" > "Projekte" > "Alle Projekte".
2. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
3. Klicken Sie im Aktivitätsbereich auf "Plan".
4. Klicken Sie auf „Artikelbedarf”.
5. Klicken Sie auf "Neu".
6. Markieren Sie in der Liste die ausgewählte Zeile.
7. Geben Sie im Feld "Artikelnummer" einen Wert ein oder wählen Sie einen Wert aus.
8. Geben Sie im Feld "Menge" eine Zahl ein.
9. Klicken Sie auf "Speichern".
10. Klicken Sie im Aktivitätsbereich auf Verwalten.
11. Klicken Sie auf Funktionen.
12. Klicken Sie auf „Bestellung erstellen”.
13. Aktivieren Sie das Kontrollkästchen „Einschließen”.
14. Geben Sie im Feld "Kreditorenkonto" einen Wert ein, oder wählen Sie einen Wert aus.
15. Klicken Sie auf "OK".
16. Wechseln Sie zu "Kreditoren" > "Bestellungen" > "Alle Bestellungen".
17. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
18. Klicken Sie im Aktivitätsbereich auf "Einkauf".
19. Klicken Sie auf "Bestätigen".
20. Klicken Sie im Aktivitätsbereich auf "Empfangen".
21. Klicken Sie auf "Produktzugang".
22. Markieren Sie in der Liste die ausgewählte Zeile.
23. Geben Sie im Feld "Produktzugang" einen Wert ein.
24. Klicken Sie auf "OK".
