### <a name="task-recordings-and-task-guides"></a>Aufgabenaufzeichnungen und Aufgabenhandbücher

Die Aufgabenaufzeichnung ist ein Microsoft Dynamics 365 for Operations-Tool, das Sie verwenden können, um Aktivitäten aufzuzeichnen, die Sie in der Produktbenutzeroberfläche (UI) ausgeführt haben. Wenn Sie eine *Aufgabenerfassung* erstellen, werden alle Ereignisse, die mit dem Server ausgeführt werden, erfasst, wenn Sie sie in der Benutzeroberfläche ausführen. Zu diesen Ereignissen zählen das Hinzufügen von Werten, das Ändern der Einstellungen und das Entfernen von Daten. Ein *Aufgabenleitfaden* ist eine kontrollierte, geführte, interaktive Erfahrung, die Sie durch die Schritte einer Aufgabe oder eines Geschäftsprozesses führt. Sie können einen Aufgabenleitfaden aus dem Hilfesystem öffnen (oder *wiedergeben*)
