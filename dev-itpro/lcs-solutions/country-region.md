# <a name="localization-and-regulatory-features"></a>Lokalisierung und behördlichen Funktionen

Microsoft Dynamics-356 für Vorgänge enthält Funktionen für einige länder-/regionsspezifische rechts- und Prozesse. Diese Funktionalität ist basierend auf die primäre Adresse der juristischen Person aktiv. Dieses Thema enthält Listen von Ressourcen für den Länder-/regionsspezifische Funktionen erfahren.
