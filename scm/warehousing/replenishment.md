# <a name="replenishment"></a><span data-ttu-id="7d8ec-103">Wiederbeschaffung</span><span class="sxs-lookup"><span data-stu-id="7d8ec-103">Replenishment</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="7d8ec-104">Dieser Artikel beschreibt die Beschaffungsstrategien, die für Lagerorte, die die Funktionen der Lagerortverwaltung verwenden, verfügbar sind.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="7d8ec-105">Dieser Artikel gilt nicht für die Lagerort-Lösung, die in der Lagerverwaltung verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="7d8ec-106">Die folgenden Wiederbeschaffungsstrategien sind verfügbar:</span><span class="sxs-lookup"><span data-stu-id="7d8ec-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="7d8ec-107">**Wellenbedarfsbeschaffung** – Diese Strategie erstellt Wiederbeschaffungsarbeit für ausgehende Aufträge oder Ladungen, sollte kein Bestand verfügbar sein, wenn Arbeit durch die Welle erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="7d8ec-108">Beispielsweise kann Wiederbeschaffungsarbeit erstellt werden, sollte die für einen Auftrag erforderliche Menge nicht verfügbar sein, wenn eine Welle verarbeitet wird.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
-   <span data-ttu-id="7d8ec-109">**Min./Max.-Auffüllung** – Diese Strategie verwendet minimale und maximale Lagergrenzen, um zu bestimmen, wann Lagerplätze aufgefüllt werden müssen.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="7d8ec-110">Die Artikel- und Lagerplatzkriterien definieren den Lagerbestand, der für die Auffüllung ausgewertet wird.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="7d8ec-111">Vorlagen für Min./Max.-Auffüllung sind der Hauptmechanismus zur Verwaltung optimaler Level in Entnahmeorten.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="7d8ec-112">Um sicherzustellen, dass genügend Komissionierlagerbestand für den Serienbedarf verfügbar ist, können Sie die Bedarfsauffüllung als Ergänzung zwischen Min./Max.-Auffüllungszyklen verwenden.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-112">To ensure that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
-   <span data-ttu-id="7d8ec-113">**Ladungsbedarfs-Wiederbeschaffung** – Diese Strategie summiert den Bedarf für verschiedene Ladungen und erstellt Wiederbeschaffungsarbeit, die erforderlich ist, um die relevanten Entnahmeorte aufzufüllen.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="7d8ec-114">Diese Strategie hilft sicherzustellen, dass die Ladungen, die erstellt werden, nach der Freigabe dem Lager entnommen werden können.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-114">This strategy helps to ensure that the loads that are created can be picked in the warehouse after they are released.</span></span>

<span data-ttu-id="7d8ec-115">Alle drei Strategien erstellen Wiederbeschaffungsarbeit, basierend auf einer Wiederbeschaffungsvorlage.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-115">All three strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="7d8ec-116">Wellenbedarfsbeschaffung</span><span class="sxs-lookup"><span data-stu-id="7d8ec-116">Wave demand replenishment</span></span>

<span data-ttu-id="7d8ec-117">Eine Wellenbedarfswiederbeschaffung erstellt Wiederbeschaffungsarbeit basierend auf dem Bedarf, sollte die notwendige Menge für ausgehende Aufträge oder Ladungen während der Arbeitserstellung einer Welle nicht verfügbar sein.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-117">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="7d8ec-118">Die Wiederbeschaffungsvorlage enthält Informationen zu den Kriterien von Artikel, der Maßeinheit, der Bedarfserhöhung und dem Lagerplatz.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-118">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="7d8ec-119">Lagerplatzrichtlinien dienen zum Bestimmen, welcher Lagerort wieder aufgefüllt werden soll.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-119">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="7d8ec-120">Verknüpfen Sie diese Lagerplatzrichtlinien mit der Wiederbeschaffungsvorlage mithilfe des Feldes **Richtliniencode**.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-120">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="7d8ec-121">Wenn das Feld **Richtliniencode** nicht festgelegt ist, werden Abfragen verwendet, um zu bestimmen, welche Lagerplatzrichtlinie verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-121">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="7d8ec-122">Beachten Sie, wenn eine Richtlinie nicht in der Wiederbeschaffungsvorlage angegeben ist und die Lagerplatzrichtlinie einen Richtliniencode besitzt, wird die Lagerplatzrichtlinie ignoriert, auch wenn die Abfrage in der Lagerplatzrichtlinie korrekt ist.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-122">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="7d8ec-123">Entnahmeortrichtlinien werden verwendet, um zu bestimmen, wo der Bestand für die Wiederbeschaffung entnommen werden soll.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-123">Pick location directives are used to determine where to get inventory for the replenishment.</span></span> 

<span data-ttu-id="7d8ec-124">Zusätzlich zur Erstellung einer Vorlage, müssen Sie einige Wiederbeschaffungseinstellungen in der Wellenvorlage angeben.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-124">In addition to creating a template, you must specify some replenishment settings in the wave template.</span></span> <span data-ttu-id="7d8ec-125">Die Wellenvorlage sollte einen Wellenschritt für Wiederbeschaffung enthalten, der nur ausgeführt wird, wenn die Zuordnung eines Artikels nicht erfolgreich ist.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-125">The wave template should contain a wave step for replenishment that is run only if allocation of an item isn't successful.</span></span> <span data-ttu-id="7d8ec-126">Dieser Wiederbeschaffungswellenschritt verwendet einen Wellenschrittcode, um zu bestimmen, welche Wiederbeschaffungsvorlage verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-126">This replenishment wave step uses a wave step code to determine which replenishment template should be used.</span></span> <span data-ttu-id="7d8ec-127">Wenn ein Wiederbeschaffungswellenschritt vorhanden ist, müssen Sie zusätzlich sicherstellen, dass **Wiederbeschaffen** im Abschnitt **Methoden** der Wellenvorlage ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-127">In addition to having a wave step for replenishment, you need to make sure that **Replenish** is selected in the **Methods** section of the wave template.</span></span> 

<span data-ttu-id="7d8ec-128">Die Seite **Wiederbeschaffungsvorlage** enthält ein Kontrollkästchen **Wellenbedarf die Nutzung nicht reservierter Mengen gestatten**.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-128">The **Replenishment template** page includes an **Allow wave demand to use unreserved quantities** check box.</span></span> <span data-ttu-id="7d8ec-129">Sie sollten dieses Kontrollkästchen aktivieren, wenn Sie der Bedarfswiederbeschaffung gestatten wollen, nicht reservierte Mengen aus Arbeit, die über die ausgewählte Wiederbeschaffungsvorlage generiert wurde, abzuziehen.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-129">You should select this check box if you want to allow demand replenishment to deduct unreserved quantities from work that is generated from the selected replenishment template.</span></span> <span data-ttu-id="7d8ec-130">Um die Bedarfswiederbeschaffungsvorlage für diese Logik zu aktivieren, muss das Kontrollkästchen für jede bestehende Wiederbeschaffungsvorlage festgelegt werden.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-130">To enable demand replenishment templates to use this logic, select this check box for every existing replenishment template.</span></span> <span data-ttu-id="7d8ec-131">Wenn die Bedarfswiederbeschaffung am Lagerort ausgelöst wurde, wird der Bedarf von den bestehenden Wiederbeschaffungsarbeiten mit nicht reservierten Mengen abgezogen, wenn die Arbeit auf einer Wiederbeschaffungvorlage mit aktivierten Kontrollkästchen **Wellenbedarf die Nutzung nicht reservierter Mengen gestatten** stammt.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-131">When demand replenishment is triggered in the warehouse, it will deduct the demand from existing replenishment work that has unreserved quantities, if the work originates from replenishment templates where the **Allow wave demand to use unreserved quantities** check box is selected.</span></span>


<span data-ttu-id="7d8ec-132">Bedarfswiederbeschaffung wird für Aufträge, Umlagerungsaufträge, Produktionsaufträge und Kanbans unterstützt.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-132">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="7d8ec-133">Min./Max.-Auffüllung</span><span class="sxs-lookup"><span data-stu-id="7d8ec-133">Min/Max replenishment</span></span>

<span data-ttu-id="7d8ec-136">Die Mindest- und Höchstbeträge sind in einer Wiederbeschaffungsvorlage festgelegt.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-136">The minimum and maximum amounts are set in a replenishment template.</span></span> <span data-ttu-id="7d8ec-137">Viele andere Einstellungen in der Vorlage ähneln den Einstellungen in Vorlagen, die in der Wellenbedarfsbeschaffung verwendeten werden.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-137">Many of the other settings in the template resemble the settings in templates that are used in Wave demand replenishment.</span></span> <span data-ttu-id="7d8ec-138">Die Vorlage sollte eine Position für jeden Artikel und Lagerplatz enthalten.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-138">The template should contain one line for each item and location.</span></span> <span data-ttu-id="7d8ec-139">Wenn Sie die Wiederbeschaffung über einen Stapelverarbeitungsauftrag ausführen, bewertet Finance and Operations, ob die Wiederbeschaffung in der Reihenfolge erforderlich ist, in der die Positionen zusammengefasst sind.</span><span class="sxs-lookup"><span data-stu-id="7d8ec-139">When you run replenishment by using the batch job, Finance and Operations evaluates whether replenishment is required, in the sequence that the lines are organized in.</span></span>
