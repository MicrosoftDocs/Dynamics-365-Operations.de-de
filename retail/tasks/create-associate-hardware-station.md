# <a name="create-and-associate-a-hardware-station"></a> Eine Hardwarestation erstellen und zuordnen

[!include[task guide banner](../includes/task-guide-banner.md)]

Diese Prozedur führt Sie Schritt für Schritt durch das Erstellen einer neuen Hardwarestation. Ein neues Hardwareprofil wird erstellt und verwendet, um neue Hardwarestationen eines vordefinierten Shops (Kanal) hinzuzufügen. Für diese Prozedur wird das Demo-Datenunternehmen USRT verwendet.

1. Wechseln Sie zu den Handels-Grundlagen > Kanälen >. > .. > .. > Hardwarestation-Profile.
2. Klicken Sie auf "Neu".
3. Geben Sie im Hardwarestation-Kennungsfeld "TestHWProfile" ein.
4. Geben Sie im Feld "Name" einen Wert ein.
5. Geben Sie im Feld "Portnummer" eine Zahl ein.
6. Klicken Sie im Feld "Hardwareprofil" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
7. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
8. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
9. Klicken Sie im Feld "Paketname" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
10. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
    * Dies ist das Standardpaket, das mit einer neuen Umgebung kommt. Die Versionsnummer kann variieren.  
11. Klicken Sie auf "Speichern".
12. Schließen Sie die Seite.
13. Navigieren Sie zu Einzelhandel und Handel > Kanäle > Alle Einzelhandelsgeschäfte.
14. Wählen Sie in der Liste die Zeile "17" aus.
    * Wenn Sie das USRT-Demodatenunternehmen verwenden, ist dies der Houston-Speicher.  
15. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
16. Schalten Sie die Erweiterung des Abschnitts "Hardwarestationen" ein/aus.
17. Klicken Sie auf Hinzufügen.
18. Markieren Sie in der Liste die ausgewählte Zeile.
19. Klicken Sie im Feld "Profilkennung" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
20. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
    * Dies muss das neue Hardwarestationsprofil sein, das in den vorherigen Schritten erstellt wurde.  
21. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
22. Geben Sie im Feld "Hostname" einen Wert ein.
23. Geben Sie einen Wert in das Feld "Terminalkennung für Überweisung (elektronisch)" ein.
24. Klicken Sie auf "Speichern".
