# <a name="create-record-templates"></a>Erstellen Sie Datensatzvorlagen

Dieser Artikel führt das Konzept der Datensatzvorlagen ein und erklärt, wie Sie verwendet werden können, um Datensätze zu erstellen, die Informationen freigeben.

Datensatzvorlagen können helfen, Datensätze in Microsoft Dynamics 365 für Arbeitsgänge schneller erstellen. Sie Datensatzvorlagen können nicht für alle Datensatztypen in Microsoft Dynamics 365 für Arbeitsgänge erstellen. So bieten z **** sich vor, Sie Mietinformationen für ein Automietgeschäft eingeben, das in San Francisco ist. Da die meisten Kunden vermutlich aus dem Bereich San Francisco kommen, wäre es schön, wenn die Werte für **Bundesland**, **Land** und **Ort** auf dem Mietvertragsformular automatisch gefüllt würden. ** Hinweis: ** Sie können Vorlagen nur auf die Bereiche von Microsoft Dynamics 365 für Arbeitsgänge anwenden, auf die Sie Zugriff haben. Wenn Sie einen neuen Datensatz erstellen, können Sie und andere Benutzer jedoch alle Vorlagentitel sehen, wenn Sie Vorlagen erstellen, die für alle Benutzer verfügbar sind. Berücksichtigen Sie dies beim Benennen von Vorlagen. Vermeiden Sie z. B. Namen, in denen Begriffe wie "Provision" vorkommen, wenn die Information, dass einige Mitarbeiter des Unternehmens auf Provisionen beruhende Gehälter haben, vertraulich ist. Wenn es für ein bestimmtes Formular eine oder mehrere Vorlagen gibt, auf die Sie zugreifen können, und Sie versuchen, einen neuen Datensatz im Formular zu erstellen, wird die Seite **Vorlage für … wählen** angezeigt. Durch Auswählen einer Vorlage aus der Liste wird der neue Datensatz mit den in der ausgewählten Vorlage festgelegten Standardinformationen versehen. Wenn Sie keine Vorlagen verwenden möchten, wenn Sie neue Datensätze erstellen, wählen Sie das ** bitten Sie nicht erneut ** Kontrollkästchen in der ** wählen Sie eine Vorlage für aus ** Seite aus. Soll das Auswahldialogfeld für Vorlagen wieder angezeigt werden, klicken Sie auf einen beliebigen Datensatz, um z ** ** und klicken Sie dann mit der rechten Maustaste ** zeigen Sie Vorlagen-Auswahl ** angezeigt.
