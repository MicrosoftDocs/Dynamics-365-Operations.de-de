# <a name="onboarding-home-page"></a>Startseite für das Onboarding

[!include[banner](../includes/banner.md)]


Wenn Sie mit Microsoft Dynamics 365 for Finance and Operations beginnen, können Sie eine Abonnementsstufe auswählen, die für Sie passt, einen Partner mit Erfahrung in Ihrer Region oder Branche suchen und Microsoft FastTrack-Programme verwenden, um rasch damit arbeiten zu können.  

Die folgenden Ressourcen geben einen Überblick über die Lifecycle Services und die Arbeit mit Ihrem Partner, damit Sie Finance and Operations nutzen können.

-   [Microsoft-FastTrack Überblick für Dynamics 365](fasttrack-dynamics-365-overview.md)
-   [Debitoren-Einführung zu den Lifecycle Services](/dynamics365/unified-operations/dev-itpro/lifecycle-services/lcs-works-lcs)
