# <a name="hire-an-existing-employee-through-recruiting"></a><span data-ttu-id="da09c-103">Einen vorhandenen Mitarbeiter mittels Personalbeschaffung einstellen</span><span class="sxs-lookup"><span data-stu-id="da09c-103">Hire an existing employee through recruiting</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="da09c-104">Manchmal können offene Stellen von den Kandidaten ausgefüllt werden, die bereits Mitarbeiter in der Organisation sind.</span><span class="sxs-lookup"><span data-stu-id="da09c-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="da09c-105">In diesem Verfahren werden Sie durch die Schritte zum Einstellen eines vorhandenen Mitarbeiters mittels Personalbeschaffungsprozess geführt.</span><span class="sxs-lookup"><span data-stu-id="da09c-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="da09c-106">In diesem Verfahren wurde bereits ein Personalbeschaffungsprojekt eingerichtet, und ein vorhandener Mitarbeiter hat bereits eine Bewerbung für das Personalbeschaffungsprojekt eingereicht.</span><span class="sxs-lookup"><span data-stu-id="da09c-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="da09c-107">Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.</span><span class="sxs-lookup"><span data-stu-id="da09c-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="da09c-108">Wechseln Sie zu Personalverwaltung > Personalbeschaffung > Bewerbungen >Bewerbungen.</span><span class="sxs-lookup"><span data-stu-id="da09c-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>
2. <span data-ttu-id="da09c-109">Suchen Sie in der Liste suchen nach der Bewerbung für den Mitarbeiter, den Sie einstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="da09c-109">In the list, find application for the employee that you would like to hire.</span></span> <span data-ttu-id="da09c-110">Beispiel: 00002 Jens Emory</span><span class="sxs-lookup"><span data-stu-id="da09c-110">Example:  00002  John Emory</span></span>
3. <span data-ttu-id="da09c-111">Klicken Sie auf "Bewerbungsstatus".</span><span class="sxs-lookup"><span data-stu-id="da09c-111">Click Application status.</span></span>
    * <span data-ttu-id="da09c-112">Der Bewerbungsstatus gibt an, in welcher Phase des Personalbeschaffungsprozesses sich die Bewerbung befindet.</span><span class="sxs-lookup"><span data-stu-id="da09c-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="da09c-113">Jeder dieser Schritte ist optional.</span><span class="sxs-lookup"><span data-stu-id="da09c-113">Each of these steps is optional.</span></span> <span data-ttu-id="da09c-114">In der Regel kann eine Bewerbung die Status in der folgenden Reihenfolge verschieben: empfangen, bestätigt und interviewt.</span><span class="sxs-lookup"><span data-stu-id="da09c-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="da09c-115">Nach dem Befragungsprozess wird eine Einstellungsentscheidung vorgenommen.</span><span class="sxs-lookup"><span data-stu-id="da09c-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="da09c-116">Klicken Sie auf "Position ändern".</span><span class="sxs-lookup"><span data-stu-id="da09c-116">Click Change position.</span></span>
5. <span data-ttu-id="da09c-117">Wählen Sie die Position aus, für die der Mitarbeiter eingestellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="da09c-117">Select the position that you are hiring the employee into.</span></span>
6. <span data-ttu-id="da09c-118">Geben Sie im Feld für das Startdatum der neuen Zuweisung das Datum ein, an dem der Mitarbeiter in der neuen Position beginnt.</span><span class="sxs-lookup"><span data-stu-id="da09c-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>  
7. <span data-ttu-id="da09c-119">Geben Sie im Feld für das Enddatum der neuen Zuweisung das Datum ein, an dem der Mitarbeiter aufhört, in der aktuellen Position zu arbeiten.</span><span class="sxs-lookup"><span data-stu-id="da09c-119">In the Assignment end date, enter the date that the employee will stop working in their current position.</span></span>
    * <span data-ttu-id="da09c-120">Das Startdatum für die neue Position und das Enddatum der alten Position können überlappen.</span><span class="sxs-lookup"><span data-stu-id="da09c-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="da09c-121">Dies kann beispielsweise der Fall sein, wenn eine Person Aufgaben für beide Positionen während einer Übergangsperiode ausführt.</span><span class="sxs-lookup"><span data-stu-id="da09c-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>  
8. <span data-ttu-id="da09c-122">Sie können optional einen Ursachencode auswählen.</span><span class="sxs-lookup"><span data-stu-id="da09c-122">Optionally, you can select a reason code.</span></span> <span data-ttu-id="da09c-123">Beispiel: Neuorganisation</span><span class="sxs-lookup"><span data-stu-id="da09c-123">Example: Reorganization</span></span>
9. <span data-ttu-id="da09c-124">Klicken Sie auf "Position ändern".</span><span class="sxs-lookup"><span data-stu-id="da09c-124">Click Change position.</span></span>
    * <span data-ttu-id="da09c-125">Sie werden zu diesem Zeitpunkt möglicherweise auch die Vergütung ändern.</span><span class="sxs-lookup"><span data-stu-id="da09c-125">You may also change compensation at this time.</span></span> <span data-ttu-id="da09c-126">Wenn Sie die Vergütung jetzt nicht zuweisen, können Sie sie ändern, indem Sie zum Arbeitskraftformular zurückkehren, die Registerkarte "Vergütung" auswählen und "Fester Plan" auswählen.</span><span class="sxs-lookup"><span data-stu-id="da09c-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="da09c-127">Nachdem Sie "Position ändern" auswählen, wird der Status in der Anwendung auf "Eingestellt" aktualisiert.</span><span class="sxs-lookup"><span data-stu-id="da09c-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
