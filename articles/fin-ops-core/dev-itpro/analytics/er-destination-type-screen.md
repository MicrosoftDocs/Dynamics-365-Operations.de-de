# <a name="ScreenDestinationType">Bildschirmziel</a>

[!include [banner](../includes/banner.md)]

Sie können ein Bildschirmziel für jede Ordner- oder Datei-Komponente eines ER-Formats (Electronic Reporting) konfigurieren, das zum Generieren ausgehender Dokumente konfiguriert ist. Basierend auf der Einstellung des Ziels wird die Vorschau eines generierten Dokuments auf einer separaten Browser-Registerkarte geöffnet.

Wenn Sie **Aktiviert** auf **Ja** setzen, wird eine Vorschau der Ausgabe erstellt werden. Sie können gewisse Dateitypen wie XML, TXT oder PDF direkt in einer Browserfenster anzeigen. Für andere Dateitypen wie Microsoft Excel oder Word wird der Microsoft Office Online-Service verwendet.
