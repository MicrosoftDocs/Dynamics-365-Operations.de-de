# <a name="list-of-er-functions-in-the-record-category"></a>Liste der EB-Funktionen in der Datensatzkategorie

[!include [banner](../includes/banner.md)]

Mithilfe der Datensatzfunktionen für die elektronische Berichterstellung (EB) können Informationen aus den Datenquellen des Datentyps *Container (Datensatz)* extrahiert werden. Dieses Thema enthält eine Zusammenfassung dieser Funktionen.

## <a name="list-of-supported-functions"></a>Liste der unterstützten Funktionen

| Funktion | Beschreibung |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Diese Funktion gibt für *Container (Datensatz)* den Wert null zurück, der dieselbe Struktur wie die angegebene Datensatzliste oder der angegebene Datensatz besitzt. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Diese Funktion gibt für *Container (Datensatz)* den Wert null zurück, der dieselbe Struktur wie die angegebene Datensatzliste oder der angegebene Datensatz besitzt. |
