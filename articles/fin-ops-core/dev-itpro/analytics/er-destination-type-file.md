# <a name="file-destination"></a>Dateizielort

[!include [banner](../includes/banner.md)]

Sie können ein Dateiziel für jede ORDNER- oder DATEI-Komponente eines ER-Formats (Electronic Reporting) konfigurieren, das zum Generieren ausgehender Dokumente konfiguriert ist. Basierend auf der Einstellung des Ziels steht ein generiertes Dokument im Webbrowser zum Download zur Verfügung.

Wenn Sie auf der Seite **Zieleinstellungen** **Aktiviert** auf **Ja** setzen, wird nach Ausführung der Konfiguration ein Öffnen- oder Speichern-Dialogfeld geöffnet.
