# <a name="power-bi-destination"></a>Power BI-Ziel

[!include [banner](../includes/banner.md)]

Sie können ein Microsoft Power BI-Ziel für jede Ordner- oder Datei-Komponente eines elektronischen Berichterstellungsformats (EB) konfigurieren, das zum Generieren ausgehender Dokumente konfiguriert ist. Basierend auf der Einstellung des Ziels wird ein generiertes Dokument in einem zuvor konfigurierten SharePoint-Ordner gespeichert.

Legen Sie **Aktiviert** auf **Ja** fest, um Ihre EB-Konfiguration zu verwenden, um die Übertragung von Daten aus Ihrer Dynamics 365 Finance-Instanz zu den Microsoft Power BI-Diensten zu veranlassen. Die übertragenen Dateien werden auf einem Microsoft SharePoint Server gespeichert, der für diesen Zweck konfiguriert wurden muss. Weitere Informationen finden Sie unter [Konfigurieren elektronischer Berichterstattung (EB), um Daten in Power BI einzubeziehen](general-electronic-reporting-report-configuration-get-data-powerbi.md).
