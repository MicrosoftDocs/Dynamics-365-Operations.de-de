# <a name="supported-web-browsers-for-lifecycle-services"></a>Unterstützte Webbrowser für Lifecycle Services 

[!include[banner](../includes/banner.md)]

Benutzer können auf Microsoft Dynamics Lifecyle Services (LCS) zugreifen, indem sie die neuesten Versionen dieser gängigen Browser verwenden: 

- Microsoft Edge (empfohlen: [Chromium-basierter Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Um die neueste Version für jeden Webbrowser zu suchen, wechseln Sie zur Website des Softwareherstellers. Für eine optimale Leistung und ein optimales Erlebnis empfehlen wir Ihnen, die neueste Version eines modernen Browsers zu verwenden, insbesondere Microsoft Edge.
