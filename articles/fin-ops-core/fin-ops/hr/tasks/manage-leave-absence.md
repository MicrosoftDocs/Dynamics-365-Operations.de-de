# <a name="manage-leave-of-absence"></a>Beurlaubung verwalten

[!include [banner](../../includes/banner.md)]

Diese Prozedur führt Sie Schritt für Schritt durch die Erstellung von Sonderurlaubs-Datensätzen für Mitarbeiter. Sie können Abwesenheitszeit nach Ursachen nachverfolgen. Dazu gehören medizinischen, pädagogischen oder elterliche Aktivitäten. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.

1. Wechseln Sie zu "Personalverwaltung" > "Arbeitskräfte" > "Mitarbeiter".
2. Wählen Sie in der Liste einen Mitarbeiter aus.
3. Zeigen Sie detaillierte Informationen für den ausgewählten Mitarbeiter an, indem Sie den Namen des Mitarbeiters auswählen.
4. Klicken Sie auf die Registerkarte "Anstellung".
5. Klicken Sie auf "Sonderurlaub".
6. Klicken Sie auf "Neu".
7. Klicken Sie im Feld "Sonderurlaubstyp" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
    * Sie können einen Sonderurlaubstyp einem Einkommenscode im Formular "Sonderurlaubstypen" zuordnen. Wenn ein Sonderurlaubstyp einem Einkommenscode zugeordnet ist, wird eine Einkommensposition mit dem zugeordneten Einkommenscode während der Sonderurlaubsperiode, die Sie eingeben, generiert.  
8. Wählen Sie in der Liste einen Sonderurlaubstyp aus. 
    * Beispiel: Annahme  
9. Geben Sie das Datum ein, an dem der Sonderurlaub beginnen soll. Beispiel: "26.10.2015"
    * Beispiel: 26.10.2015  
10. Geben Sie das Datum ein, an dem der Sonderurlaub beginnen soll. 
    * Beispiel: 20.11.2015  
11. Geben Sie im Hinweisfeld eine Beschreibung ein.
    * Beispiel: Sonderurlaub zur Annahme  
12. Klicken Sie auf "Speichern".
