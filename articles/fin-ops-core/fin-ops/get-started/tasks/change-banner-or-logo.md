# <a name="change-the-banner-or-logo"></a><span data-ttu-id="7e071-103">Das Banner oder das Logo ändern</span><span class="sxs-lookup"><span data-stu-id="7e071-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7e071-104">Im Folgenden werden die Schritte aufgeführt, die Systemadministratoren verwenden können, um das Banner- oder Logobild zu aktualisieren, das für eine juristische Person angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="7e071-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="7e071-105">Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.</span><span class="sxs-lookup"><span data-stu-id="7e071-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="7e071-106">Wechseln Sie zu **Navigationsseite > Module > Organisationsverwaltung > Organisationen > Juristische Personen**.</span><span class="sxs-lookup"><span data-stu-id="7e071-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="7e071-107">Wählen Sie in der Liste auf der linken Seite die juristische Person aus, für die Sie das Banner oder das Logo aktualisieren möchten.</span><span class="sxs-lookup"><span data-stu-id="7e071-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="7e071-108">Wenn sie es bereits ausgewählt haben, wechseln Sie zum nächsten Schritt.</span><span class="sxs-lookup"><span data-stu-id="7e071-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="7e071-109">Erweitern Sie die Registerkarte **Dashboardbild**.</span><span class="sxs-lookup"><span data-stu-id="7e071-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="7e071-110">Wählen Sie **Ändern** aus.</span><span class="sxs-lookup"><span data-stu-id="7e071-110">Select **Change**.</span></span>
    
    <span data-ttu-id="7e071-111">Die optimalen Auflösung für ein Bannerbild ist 1920 x 281 Pixel.</span><span class="sxs-lookup"><span data-stu-id="7e071-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="7e071-112">Die optimale Breite für ein Logobild ist 350 Pixel.</span><span class="sxs-lookup"><span data-stu-id="7e071-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="7e071-113">Wählen Sie **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="7e071-113">Select **Save**.</span></span>
6. <span data-ttu-id="7e071-114">Wechseln Sie zu **Navigationsbereich > Module > Allgemein > Allgemein > Standarddashboard**.</span><span class="sxs-lookup"><span data-stu-id="7e071-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="7e071-115">Sie sollten das neue Banner- oder Logobild im Dashboard finden.</span><span class="sxs-lookup"><span data-stu-id="7e071-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="7e071-116">Möglicher Weise müssen Sie Ihr Unternehmen mithilfe der Unternehmens-Auswahl in das Unternehmen ändern, für das Sie das Banner für hochgeladen haben.</span><span class="sxs-lookup"><span data-stu-id="7e071-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
