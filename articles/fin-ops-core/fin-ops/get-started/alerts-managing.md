<span data-ttu-id="915a8-120">Umgekehrt kann eine weniger häufig ausgeführte Stapelverarbeitung, die für Zeiten geplant ist, wenn die Systemlast gering ist, dazu beitragen, die Systemleistung zu verbessern.</span><span class="sxs-lookup"><span data-stu-id="915a8-120">On the other hand, a batch job that recurs less often, and that you schedule for times when the system load is low, might help improve system performance.</span></span> <span data-ttu-id="915a8-121">Eine geringere Häufigkeit der Stapelverarbeitung erfüllt jedoch möglicherweise nicht die Anforderungen der Benutzer für fristgerechte Warnungen.</span><span class="sxs-lookup"><span data-stu-id="915a8-121">However, a low frequency for batch processing might not meet the users' demands for timely alerts.</span></span>

<span data-ttu-id="915a8-122">Daher müssen Sie für die Einstellung der Stapelverarbeitungshäufigkeit von änderungsbasierten Ereignissen einen Kompromiss zwischen dem zeitgerechten Senden von Warnungen und der Leistung des Gesamtsystem finden.</span><span class="sxs-lookup"><span data-stu-id="915a8-122">Therefore, when you set up the frequency of batch processing for change-based events, consider the balance between the timeliness of alerts and the performance of the whole system.</span></span> <span data-ttu-id="915a8-123">Diese Überlegungen werden umso wichtiger, je mehr Benutzer es gibt, die Warnregeln erstellen.</span><span class="sxs-lookup"><span data-stu-id="915a8-123">These considerations become more relevant as the number of users who create alert rules increases.</span></span> <span data-ttu-id="915a8-124">Die Häufigkeit wirkt sich nicht auf die Anzahl der Ereignisse aus, die das System verarbeitet.</span><span class="sxs-lookup"><span data-stu-id="915a8-124">The frequency doesn't affect the number of events that the system processes.</span></span> <span data-ttu-id="915a8-125">Wenn jedoch mehr Benutzer Regeln erstellen, führt der Prozess mehrere Prüfungen aus.</span><span class="sxs-lookup"><span data-stu-id="915a8-125">However, if more users create rules, the process runs more checks.</span></span> <span data-ttu-id="915a8-126">Diese Art des Datenverkehrs kann sich auf die Systemleistung auswirken.</span><span class="sxs-lookup"><span data-stu-id="915a8-126">This type of data exchange might affect system performance.</span></span>

#### <a name="the-risks-of-low-batch-frequency"></a><span data-ttu-id="915a8-127">Die Risiken einer niedrigen Chargenfrequenz</span><span class="sxs-lookup"><span data-stu-id="915a8-127">The risks of low batch frequency</span></span>

<span data-ttu-id="915a8-130">Beispielsweise erstellen Sie eine Warnung, die ausgelöst wird, wenn das Ereignis **Debitorenkontakt ändert sich** ist und die Bedingung **Debitor = BB** ist.</span><span class="sxs-lookup"><span data-stu-id="915a8-130">For example, you create an alert to trigger when the event is **customer contact changes** and the condition is **customer = BB**.</span></span> <span data-ttu-id="915a8-131">Wenn sich also der Debitorenkontakt für Debitor BB ändert, protokolliert der Prozess das Ereignis.</span><span class="sxs-lookup"><span data-stu-id="915a8-131">In other words, when the customer contact for customer BB changes, the process logs the event.</span></span> <span data-ttu-id="915a8-132">Allerdings ist das Stapelverarbeitungssystem so eingerichtet, dass die Stapelverarbeitung weniger häufig als die Dateneingabe auftritt.</span><span class="sxs-lookup"><span data-stu-id="915a8-132">However, the batch processing system is set up so that batch processing occurs less often than data entry.</span></span> <span data-ttu-id="915a8-133">Wenn der Debitorenname sich von **BB** in **AA** ändert, bevor das Ereignis verarbeitet wird, entsprechen die Daten in der Datenbank nicht mehr der Bedingung in der Regel **Debitor = BB**.</span><span class="sxs-lookup"><span data-stu-id="915a8-133">If the customer name changes from **BB** to **AA** before the event is processed, the data in the database no longer matches the condition in the rule, **customer = BB**.</span></span> <span data-ttu-id="915a8-134">Wenn das Ereignis schließlich verarbeitet wird, wird daher keine Warnung generiert.</span><span class="sxs-lookup"><span data-stu-id="915a8-134">Therefore, when the event is finally processed, no alert is generated.</span></span>
