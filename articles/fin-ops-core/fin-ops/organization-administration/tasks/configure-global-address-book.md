<span data-ttu-id="375f3-104">Verwenden Sie diese Prozedur, um die Standardwerte und Sicherheitsrichtlinien für das globale Adressbuch festzulegen.</span><span class="sxs-lookup"><span data-stu-id="375f3-104">Use this procedure to set the default values and security policies for the global address book.</span></span>

10. <span data-ttu-id="375f3-116">Wählen Sie im Feld **Interessent** eine Option aus.</span><span class="sxs-lookup"><span data-stu-id="375f3-116">In the **Prospect** field, select an option.</span></span>
