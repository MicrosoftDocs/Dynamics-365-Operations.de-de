# <a name="configure-address-books"></a>Konfigurieren von Adressbüchern

[!include [banner](../../includes/banner.md)]

Verwenden Sie dieses Verfahren und die Entscheidungen, die Sie im Artikel „Konfiguration des globalen Adressbuchs und zusätzlicher Adressbücher planen“ getroffen haben, um zusätzliche Adressbücher für Ihre Organisation einzurichten.

Das Demodatenunternehmen, das verwendet wird, um diese Aufgabe zu erstellen, ist USMF.


## <a name="configure-address-books"></a>Adressbücher konfigurieren
1. Wechseln Sie im **Navigationsbereich** zu **Module > Organisationsverwaltung > Globales Adressbuch > Adressbücher**.
2. Klicken Sie auf **Neu**.
3. Geben Sie im Feld **Name** einen Wert ein.
4. Geben Sie im Feld **Beschreibung** einen Wert ein.
5. Klicken Sie auf **Speichern**.
6. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
7. Klicken Sie auf den Pfeil, um die ausgewählten verfügbaren Teams dem Adressbuch hinzuzufügen.
8. Klicken Sie auf **Speichern**.
