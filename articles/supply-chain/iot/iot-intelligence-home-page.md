# <a name="iot-intelligence-now-replaced-by-sensor-data-intelligence"></a>IoT-Intelligenz wird jetzt durch Sensor Data Intelligence ersetzt

[!include [banner](../../includes/banner.md)]

Die in diesem Artikel beschriebene IoT-Intelligenz-Funktion wurde auf Version 2.0 aktualisiert und gleichzeitig in *Sensor Data Intelligence* umbenannt. Die neue Version ist derzeit als Vorschaufunktion für Supply Chain Management Version 10.0.30 oder höher verfügbar.

Sensor Data Intelligence führt ein neues Bereitstellungsmodell ein, bei dem die erforderlichen Azure-Komponenten in Ihrem eigenen Azure-Abonnement bereitgestellt werden, anstatt von Microsoft verwaltet zu werden. Dieses neue Modell verbessert Ihre Möglichkeiten zur Anpassung der Funktion und zur Integration mit Fremdsystemen. Die neue Version bietet außerdem Unterstützung für zwei neue Internet-der-Dinge-(IoT-)Szenarien für die Anlagenverwaltung.

Aufgrund dieser bevorstehenden Änderungen empfiehlt Microsoft, keine neuen Projekte auf der Grundlage der bestehenden IoT Intelligence-Funktion zu starten. Wenn Sie daran interessiert sind, Ihrem System IoT-Funktionalitäten hinzuzufügen, empfehlen wir Ihnen, stattdessen mit der Vorschauversion von Sensor Data Intelligence zu arbeiten. Weitere Informationen über Sensor Data Intelligence und die Arbeit damit finden Sie auf der [Sensor Data Intelligence-Startseite](../sensor-data-intelligence/sdi-home-page.md).
