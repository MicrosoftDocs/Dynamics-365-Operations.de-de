# <a name="turn-on-the-landed-cost-module-for-your-system"></a>Modul für Gesamttransportkosten im System aktivieren

[!include [banner](../includes/banner.md)]

Bevor Sie das Gesamttransportkosten-Modul verwenden können, muss es für Ihr System aktiviert werden. Administratoren können mit den Einstellungen [Funktionsverwaltung](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) den Status der Funktion überprüfen und sie bei Bedarf aktivieren. Im Arbeitsbereich **Funktionsverwaltung** ist die Funktion wie folgt aufgeführt:

- **Modul:** *Transportverwaltung*
- **Funktionsname:** *Gesamttransportkosten*
