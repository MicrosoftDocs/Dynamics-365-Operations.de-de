# <a name="order-entry-deadlines"></a><span data-ttu-id="cce33-104">Fristen für Auftrag</span><span class="sxs-lookup"><span data-stu-id="cce33-104">Order entry deadlines</span></span>

<span data-ttu-id="cce33-105">Dieser Artikel enthält Informationen zu Auftragserfassungsfristen.</span><span class="sxs-lookup"><span data-stu-id="cce33-105">This article provides information about order entry deadlines.</span></span> <span data-ttu-id="cce33-106">Eine Auftragserfassungsfrist ist eine Sperrzeit, die bestimmt, ob ein Kundenauftrag als am aktuellen Tag oder dem nächsten Tag empfangen behandelt (und erfüllt) werden soll.</span><span class="sxs-lookup"><span data-stu-id="cce33-106">An order entry deadline is a cut-off time that determines whether a customer order is treated (and fulfilled) as if it was received on the current day or the next day.</span></span>

<span data-ttu-id="cce33-107">In vielen Unternehmen werden Aufträge, die bis zu einer bestimmten Uhrzeit eingegangen sind, so behandelt, als er wären sie am gleichen Tag erfasst worden.</span><span class="sxs-lookup"><span data-stu-id="cce33-107">In many companies, only sales orders that are received before a certain time of day are treated as if they were received on that day.</span></span> <span data-ttu-id="cce33-108">Alle Aufträge, die nach diesem Zeitpunkt eingehen, werden so behandelt, als ob sie am nächsten Werktag eingehen.</span><span class="sxs-lookup"><span data-stu-id="cce33-108">Any orders that are received after that time are treated as if they are received on the next business day.</span></span> <span data-ttu-id="cce33-109">Diese Sperrzeit für Aufträge wird Frist für die Auftragserfassung genannt.</span><span class="sxs-lookup"><span data-stu-id="cce33-109">This cut-off time for orders is known as the order entry deadline.</span></span>  

<span data-ttu-id="cce33-110">Fristen für die Auftragserfassung werden als Eingabe die Lieferterminzusage verwendet.</span><span class="sxs-lookup"><span data-stu-id="cce33-110">Order entry deadlines are used as input for order promising.</span></span> <span data-ttu-id="cce33-111">Dadurch helfen sie beim Verwalten der Debitorenerwartungen bezüglich der Auftragslieferungen.</span><span class="sxs-lookup"><span data-stu-id="cce33-111">Therefore, they help you manage customer expectations about order deliveries.</span></span> <span data-ttu-id="cce33-112">So können Debitoren sehen, dass Sie beim Bestellen eines Auftrags vor einem bestimmten Zeitpunkt die Waren am selben Tag versenden.</span><span class="sxs-lookup"><span data-stu-id="cce33-112">For example, customers can see that, if they place an order with you before a specific time, you will commit to shipping the goods on the same day.</span></span> <span data-ttu-id="cce33-113">Wenn sie diese Frist fehlen, können sie die Lieferung nur am folgenden Arbeitstag ein.</span><span class="sxs-lookup"><span data-stu-id="cce33-113">However, if they miss that deadline, they can expect the shipment only on the next business day.</span></span> <span data-ttu-id="cce33-114">Sie setzen Auftragsfristen auf Grundlage Ihrer Lagerortfunktionen und Spediteurzeitpläne.</span><span class="sxs-lookup"><span data-stu-id="cce33-114">You set order entry deadlines based on your warehouse capabilities and shipping carrier schedules.</span></span>
