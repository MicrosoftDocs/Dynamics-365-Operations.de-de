# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>Der Bestand kann nicht an einen Lagerplatz verschoben werden, der von Kennzeichen kontrolliert wird.

## <a name="symptoms"></a>Symptome

Sie können die entnommenen Mengen einer Ladung nicht reduzieren.

## <a name="resolution"></a>Lösung

In früheren Versionen konnten Sie die kommissionierten Mengen für eine Ladung nicht reduzieren. Jetzt können Sie jedoch die Kommissionierung rückgängig machen und an einen Lagerplatz verschieben, der von einem Kennzeichen kontrolliert wird. Sie müssen sowohl einen **Ort**-Wert als auch einen **Kennzeichen**-Wert für die Ladezeile auf der Seite **Kommissionierte Menge reduzieren** angeben.
