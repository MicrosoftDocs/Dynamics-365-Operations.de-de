# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Sendung zur Ladung wurde bestätigt, aber es wurden keine Positionen gebucht

## <a name="symptoms"></a>Symptome

Bei der Arbeit mit ausgehenden Lagerortvorgängen erhalten Sie möglicherweise die folgende Fehlermeldung:

> Die Lieferung für Ladung 1% wurde bestätigt.

Es erfolgte jedoch keine weitere Buchung.

## <a name="resolution"></a>Lösung

Dieses Verhalten ist beabsichtigt. Die Sendungsbestätigung hat keinen Einfluss auf die Buchung. Sie aktualisiert nur den Status der Sendung und der Ladung. Die Verbuchung muss in einem separaten Prozess erfolgen.
