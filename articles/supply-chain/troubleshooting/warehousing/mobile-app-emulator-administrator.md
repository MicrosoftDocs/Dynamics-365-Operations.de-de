# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Es kann nur über die Administratorrolle auf den Emulator der Mobilgeräte-App zugegriffen werden

## <a name="symptoms"></a>Symptome

Beim Einrichten von Lagern in Supply Chain Management können Sie außer der Administratorrolle keine Rolle verwenden, um auf den Emulator der mobilen Geräte-App zuzugreifen.

## <a name="resolution"></a>Lösung

Der App-Emulator für mobile Geräte ist so festgelegt, dass er nur mit dem Administratorkonto funktioniert. Für alle Test- und Live-Prozesszwecke empfehlen wir, die Warehouse Management Mobile App selbst zu verwenden.
