# <a name="packing-station-doesnt-show-product-notes"></a>In der Verpackungsstation werden keine Produkthinweise angezeigt

KB-Nummer: 4614615

## <a name="symptoms"></a>Symptome

Verpackungshinweise werden im Verpackungsformular nicht angezeigt, wenn die Verpackungsanweisungen als Anhang zu einem Produktmaster oder einer Produktvariante hinzugefügt werden.

## <a name="resolution"></a>Lösung

Das System verhält sich wie vorgesehen.

Die derzeitige Logik zum Anzeigen von Verpackungshinweisen im Verpackungsformular erfordert, dass die Hinweise den Lieferungen zugeordnet werden.
