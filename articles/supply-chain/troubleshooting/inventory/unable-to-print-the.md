# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Der Kommissionierlistenbericht zeigt keine neuen Hinweise an, die an eine Erfassung angehängt sind

KB-Nummer: 4611660

## <a name="symptoms"></a>Symptome

Der Kommissionierlistenbericht zeigt keine neuen Hinweise an, die an eine Kommissionierlistenerfassung angehängt sind.

## <a name="resolution"></a>Lösung

Der Kommissionierlistenbericht zeigt nur Hinweise an, die an die Auftragsposition und den Auftragskopf angehängt sind. Es werden keine neuen Hinweise angezeigt, die der Rechnungserfassung hinzugefügt wurden.
