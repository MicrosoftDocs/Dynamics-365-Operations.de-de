# <a name="you-cant-add-the-price-unit-field-to-the-purchase-agreement-page"></a>Sie können das Feld „Preiseinheit“ nicht zur Seite „Einkaufsvertrag“ hinzufügen

## <a name="symptoms"></a>Symptome

Wenn Sie die Seite **Kaufvertrag** in einem Positionsansichtsmodus öffnen, können Sie die Seite nicht personalisieren, indem Sie das Feld **Preiseinheit** in der Übersicht der Position hinzufügen.

## <a name="resolution"></a>Lösung

Einige gemeinsam genutzte Felder im Vereinbarungsframework können nicht in Personalisierungen einbezogen werden. Diese Einschränkung tritt aufgrund des implementierten Datenmodells auf. Daher können Sie das **Preiseinheit**-Feld nicht personalisieren.
