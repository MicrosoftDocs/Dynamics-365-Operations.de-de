# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>Sie können einer Bestellanforderung keine Position hinzufügen, nachdem Sie eine Änderung angefordert haben

KB-Nummer: 4611211

## <a name="symptoms"></a>Symptome

Das System erlaubt Ihnen nicht, einer Bestellanforderung eine Position hinzufügen, nachdem Sie eine Änderung angefordert haben.

## <a name="resolution"></a>Lösung

Sie müssen den Workflow zurückrufen. Nach der Bestellanforderung ist im Status *Entwurf* können Sie sie weiter bearbeiten oder eine Position hinzufügen.
