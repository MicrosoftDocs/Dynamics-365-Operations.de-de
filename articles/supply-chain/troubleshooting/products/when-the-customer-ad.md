# <a name="purchase-orders-dont-show-translated-product-text"></a>Bestellungen zeigen keinen übersetzten Produkttext

KB-Nummer: 4612098

## <a name="symptoms"></a>Symptome

Sie haben Textübersetzungen für Produktnamen festgelegt. Produktnamen werden jedoch immer in der Originalsprache angezeigt, unabhängig von der Spracheinstellung in der App.

## <a name="resolution"></a>Lösung

Produktübersetzungen werden nur auf nach außen gerichteten Dokumenten angezeigt. Sie werden internen Benutzern nicht angezeigt. Weitere Informationen finden Sie unter [Produktbezogene Übersetzungen – FAQ](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information).
