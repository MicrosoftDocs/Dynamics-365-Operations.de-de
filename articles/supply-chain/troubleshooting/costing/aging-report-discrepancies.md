# <a name="inventory-aging-report-discrepancies"></a>Diskrepanzen im Bestandsalterungsbericht

## <a name="symptoms"></a>Symptome

Der **Bestandsalterungsbericht** zeigt unterschiedliche Werte an, wenn er in verschiedenen Dimensionen der Lagerung (z.B. Standort oder Lagerort) betrachtet wird.

## <a name="resolution"></a>Lösung

Weitere Informationen über die Berichtslogik finden Sie unter [Beispiele und Logik des Berichts zur Bestandsalterung](../../cost-management/inventory-aging-report.md).
