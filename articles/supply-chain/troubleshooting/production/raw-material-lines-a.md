# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Benutzer können die Entnahme von Materialpositionen für Produkte mit einer Fertigmeldung rückgängig machen

KB-Nummer: 4614721

## <a name="symptoms"></a>Symptome

Das System erlaubt Benutzern, die Entnahme Rohmaterialpositionen für Produkte mit einer Fertigmeldung rückgängig zu machen.

## <a name="resolution"></a>Lösung

Das System verhält sich wie vorgesehen. Der Rohmaterialverbrauch kann angepasst werden, bis der Produktionsauftrag den Status *Beendet* erreicht.
