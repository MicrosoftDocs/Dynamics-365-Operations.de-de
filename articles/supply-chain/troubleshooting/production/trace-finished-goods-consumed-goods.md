# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Der Vergleich der Seriennummern von Fertigwaren mit denen von konsumierten Waren ist nicht möglich

## <a name="symptoms"></a>Symptome

Sie können die Seriennummern von fertiger Ware nicht gegen die Seriennummern von Material abgleichen, das ein Produktionsauftrag verbraucht, um diese fertige Ware herzustellen.

## <a name="workaround"></a>Problemumgehung

Die Abhilfe besteht darin, Produktionsaufträge für eine Menge von 1 zu erstellen.
