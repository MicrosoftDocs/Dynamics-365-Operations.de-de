# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Der Ressourcenname wird nach dem Ändern eines Arbeitskraftnamens im Adressbuch nicht aktualisiert

## <a name="symptoms"></a>Symptome

Wenn Sie einen Namen einer Arbeitskraft im globalen Adressbuch ändern, wird der passende Ressourcenname nicht im Ressourcengruppenstamm aktualisiert.

## <a name="resolution"></a>Lösung

Der übereinstimmende Ressourcenname wird derzeit nicht automatisch aktualisiert. Um das Problem zu beheben, müssen Sie den Ressourcennamen manuell aktualisieren.
