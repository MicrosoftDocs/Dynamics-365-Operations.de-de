# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Ein Produktionsauftrag kann aufgrund eines fehlerhaften Werts der Stückliste consumptionCost nicht beendet werden

## <a name="symptoms"></a>Symptome

Wenn Sie versuchen, einen Produktionsauftrag zu beenden, erhalten Sie die folgende Fehlermeldung:

> Der Wert für die Berechnung der Stückliste consumptionCost muss bei der Ausgabe aus dem Bestand negativ sein.

## <a name="resolution"></a>Lösung

Dieses Problem wurde in Version 10.0.15 behoben.
