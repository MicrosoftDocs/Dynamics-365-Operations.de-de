# <a name="cant-cancel-an-intercompany-purchase-order-thats-linked-to-a-sales-order"></a>Eine Intercompany-Bestellung, die mit einem Auftrag verknüpft ist, kann nicht storniert werden

## <a name="symptoms"></a>Symptome

Wenn Sie versuchen, eine Intercompany-Bestellung zu stornieren, die mit einem Auftrag verknüpft ist, erhalten Sie möglicherweise die folgende Fehlermeldung:

> Menge kann nicht verringert werden, da die verbleibende Aktualisierungsmenge das Vorzeichen ändert.

## <a name="resolution"></a>Lösung

Dieses Problem wurde in Microsoft Dynamics 365 Supply Chain Management Version 10.0.13 behoben. In dieser und höheren Versionen können Sie eine mit einem Auftrag verknüpfte Intercompany-Bestellung jetzt stornieren.
