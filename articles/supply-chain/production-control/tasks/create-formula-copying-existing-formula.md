# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Eine Formel durch Kopieren aus einer vorhandenen Formel erstellen

[!include [task guide banner](../../includes/task-guide-banner.md)]

Im folgenden Verfahren sehen Sie, wie schnell eine neue Formel durch das Kopieren aus einer vorhandenen Formel erstellt wird. Sie können die Demodatunternehmen USMF, USP2 oder USPI verwenden, um diese Prozedur zu erstellen.


## <a name="create-a-new-formula"></a>Erstellen einer neuen Formel
1. Wechseln Sie zu den Formeln.
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Formel" einen Wert ein.
4. Geben Sie im Feld "Name" einen Wert ein.
5. Geben Sie im Feld "Standort" einen Wert ein, oder wählen Sie einen Wert aus.
6. Geben Sie im Feld "Artikelgruppe" einen Wert ein oder wählen Sie einen Wert aus.
7. Klicken Sie auf "Speichern".

## <a name="copy-from-an-existing-formula"></a>Kopieren aus einer vorhandenen Formel
1. Klicken Sie auf Kopieren.
2. Geben Sie im Feld "Artikelnummer" einen Wert ein oder wählen Sie einen Wert aus.
3. Geben Sie im Feld "Formelversion" einen Wert ein, oder wählen Sie einen Wert aus.
4. Klicken Sie auf "OK".

## <a name="revise-the-copied-formula"></a>Überarbeiten der kopierten Formel
1. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
2. Klicken Sie auf Löschen.
3. Klicken Sie auf "Ja".

## <a name="approve-formula"></a>Formel genehmigen
1. Klicken Sie auf Formel genehmigen.
2. Geben Sie im Feld 'Genehmig von' einen Wert ein, oder wählen Sie einen Wert aus.
3. Klicken Sie auf "OK".
4. Schließen Sie die Seite.
