# <a name="report-progress-on-a-mobile-job-device"></a>Status auf einem mobilen Einzelvorgangsgerät melden

[!include [banner](../../includes/banner.md)]

Dieses Verfahren zeigt Ihnen an, wie Sie den Fortschritt zu einem Produktions-Einzelvorgang im Formular Einzelvorgangsgerätregistrierung" starten und anzeigen.



Damit Sie dieses Verfahren ausführen können, muss die Rolle Systemadministator oder Maschinist zugeordnet sein.

1. Wechseln Sie zu Produktionssteuerung > Fertigungssteuerung > Einzelvorgangslistengerät.
2. Wählen Sie im Feld "WorkerTextField" die Kennkarte einer Arbeitskraft aus. Im USMF-Demodattyp ist dies "123" für Christina Portra.
3. Klicken sie auf "Anmelden".
4. Klicken Sie auf die Schaltfläche "Filter".
5. Aktivieren oder deaktivieren Sie das Kontrollkästchen "Konfigurationsfilter anwenden". Wenn Sie einen Filter festgelegt, können Sie die Produktionseinheit 110 in USMF verwenden.
6. Wählen Sie im Feld "Produktionseinheit" die Ressourcengruppen aus, für deren Produktionseinzelvorgänge die Arbeitskraft arbeiten kann.
7. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
8. Klicken Sie auf "OK".
9. Klicken Sie auf die Schaltfläche "Einzelvorgang starten".
10. Klicken Sie auf "OK".
11. Klicken Sie auf die Schaltfläche "Fortschritt melden".
12. Klicken Sie auf "OK".
13. Klicken Sie auf die Schaltfläche "Nächster Einzelvorgang".
14. Klicken Sie auf die Schaltfläche "Zugewiesen", um eine Übersicht aller Produktions-Einzelvorgänge anzuzeigen.
15. Schließen Sie die Seite.
16. Klicken Sie auf die Schaltfläche "Pause".
17. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
18. Klicken Sie auf "OK".
19. Klicken Sie auf die Schaltfläche "Beenden".
20. Wählen Sie abmelden aus.
21. Klicken Sie auf "OK".
22. Melden Sie sich im Feld "WorkerTextField" erneut an. In den USMF-Demodaten können Sie Arbeitskraft "123" auswählen.
23. Klicken sie auf "Anmelden".
24. Klicken Sie auf "Pause stoppen".
25. Klicken Sie auf die Schaltfläche "Aktivität".
26. Klicken Sie auf "Abbrechen".
27. Klicken Sie auf die Schaltfläche "Beenden".
28. Wählen Sie "Ausstempeln" aus.
29. Klicken Sie auf "OK".
30. Wählen Sie eine Ursache für das frühzeitige Ausstempeln.
