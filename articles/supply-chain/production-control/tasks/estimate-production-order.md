# <a name="estimate-a-production-order"></a>Produktionsauftrag vorkalkulieren

[!include [task guide banner](../../includes/task-guide-banner.md)]

Sie können dieses Verfahren ausführen, indem Sie das USMF-Demodatunternehmen oder Ihren eigenen Datensatz verwenden. In beiden Fällen müssen Sie einen offenen Produktionsauftrag haben, der den Status "Erstellt" aufweist. Dies ist die zweite Prozedur von sieben, die den Produktionsauftrags-Lebenszyklus erklärt.


## <a name="estimate-a-production-order"></a>Produktionsauftrag vorkalkulieren
1. Wechseln Sie zu "Produktionssteuerung" > "Produktionsaufträge" > "Alle Produktionsaufträge".
2. Wählen Sie einen Auftrag aus, der den Status "Erstellt" im Raster hat.
3. Klicken Sie im Aktivitätsbereich auf "Produktionsauftrag".
4. Klicken Sie auf "Vorkalkulation".
    * In diesem Schritt werden die vorkalkulierten Gesamtkosten eines einzelnen Produktionsauftrags berechnet.   
5. Klicken Sie auf "OK".

## <a name="view-the-calculation-details"></a>Berechnungsdetails anzeigen
1. Klicken Sie im Aktivitätsbereich auf "Kosten verwalten".
2. Klicken Sie auf Berechnungsdetails anzeigen
    * Auf dieser Seite wird die Kostenaufschlüsselung angezeigt. So können Sie beispielsweise den gesamten Einstandspreis pro Einheit für das Fertigprodukt in der ersten Zeile anzeigen. Die folgenden Zeilen enthalten Kosten gemäß der Stückliste, den Produktionsarbeitsplan und die indirekten Kosten.
