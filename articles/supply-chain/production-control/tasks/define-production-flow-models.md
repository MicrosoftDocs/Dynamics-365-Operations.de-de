    * <span data-ttu-id="97021-128">Der Planungszeitraum beschreibt die Anzahl von Tagen, innerhalb derer Kapazitätsreservierungen für die zugehörigen Arbeitsgruppen vorgenommen werden können.</span><span class="sxs-lookup"><span data-stu-id="97021-128">The planning time fence describes the number of days where capacity reservations can be made for the related work cells.</span></span> <span data-ttu-id="97021-129">Geben Sie im Feld Planungszeitraum die Anzahl der Tage ein.</span><span class="sxs-lookup"><span data-stu-id="97021-129">In the Planning time fence, enter the number of days.</span></span>   <span data-ttu-id="97021-130">Kanban-Bearbeitungseinzelvorgänge, die außerhalb dieser Periode liegen, werden nicht mit der automatischen Planung geplant.</span><span class="sxs-lookup"><span data-stu-id="97021-130">Kanban process jobs that fall outside of this period are not planned with automatic planning.</span></span> <span data-ttu-id="97021-131">Der Planungszeitraum ist normalerweise das Zweifache der durchschnittlichen Lagerlieferzeit der in einem Produktionsfluss oder in einer Arbeitsgruppe produzierten Produkte.</span><span class="sxs-lookup"><span data-stu-id="97021-131">The planning time fence is typically two times the average inventory lead time of the products produced in a production flow or work cell.</span></span> <span data-ttu-id="97021-132">Der EPE-Zyklus sollte nicht mehr als Hälfte des Planungszeitraums betragen.</span><span class="sxs-lookup"><span data-stu-id="97021-132">The EPE Cycle should not be more than half the planning time fence.</span></span>
