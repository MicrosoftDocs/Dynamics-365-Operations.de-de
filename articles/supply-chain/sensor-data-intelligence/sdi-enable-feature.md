## <a name="procedure"></a>Prozedur

Bevor Sie Sensor Data Intelligence nutzen können, muss sie für Ihr System aktiviert werden.

1. Wechseln Sie zu **Systemverwaltung \> Arbeitsbereiche \> Funktionsverwaltung**. (Weitere Informationen finden Sie unter [Funktionsverwaltung – Übersicht](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).)
1. Auf der **Alle**-Registerkarte verwenden Sie das **Filter**-Feld, um nach der Funktion *Sensor Data Intelligence* zu suchen.
1. Wenn die *Sensor Data Intelligence*-Funktion auf Ihrem System aktiviert ist, wählen Sie sie in der Liste aus und wählen Sie dann **Deaktivieren**, um sie zu deaktivieren. Sie können diese ältere Version der Funktion nicht zusammen mit der neuen Vorschauversion verwenden, die Sie im nächsten Schritt aktivieren.
1. Verwenden Sie das **Filter**-Feld, um nach der Funktion *(Vorschau) Sensor Data Intelligence* zu suchen.
1. Wählen Sie die Funktion in der Liste aus, und wählen Sie dann **Jetzt aktivieren**, um sie zu aktivieren.
