# <a name="create-reason-codes"></a><span data-ttu-id="b5359-103">Erstellen von Ursachencodes</span><span class="sxs-lookup"><span data-stu-id="b5359-103">Create reason codes</span></span>    

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="b5359-104">Klicken auf **Serviceverwaltung** \> **Einrichtung** \> **Serviceaufträge** \> **Phasenursachencodes**.</span><span class="sxs-lookup"><span data-stu-id="b5359-104">Click **Service management** \> **Setup** \> **Service orders** \> **Stage reason codes**.</span></span>

<span data-ttu-id="b5359-107">Wenn Sie zur Angabe eines Ursachencodes aufgefordert werden, um zu erklären, warum Sie eine Phase des Serviceauftrags aktualisieren, können Sie unter allen Ursachencodes auswählen, die Sie hier erstellen.</span><span class="sxs-lookup"><span data-stu-id="b5359-107">When you are prompted for a reason code to explain why you are updating a stage of the service order, you can select any of the reason codes that you create here.</span></span>
