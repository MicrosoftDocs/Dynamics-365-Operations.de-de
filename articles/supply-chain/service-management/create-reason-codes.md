# <a name="create-reason-codes"></a><span data-ttu-id="22228-103">Erstellen von Ursachencodes</span><span class="sxs-lookup"><span data-stu-id="22228-103">Create reason codes</span></span>    

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="22228-104">Klicken auf **Serviceverwaltung** \> **Einrichtung** \> **Serviceaufträge** \> **Phasenursachencodes**.</span><span class="sxs-lookup"><span data-stu-id="22228-104">Click **Service management** \> **Setup** \> **Service orders** \> **Stage reason codes**.</span></span>

2.  <span data-ttu-id="22228-105">Erstellen Sie durch Drücken von STRG+N einen neuen Ursachencode.</span><span class="sxs-lookup"><span data-stu-id="22228-105">Press CTRL+N to create a new reason code.</span></span>

<span data-ttu-id="22228-107">Wenn Sie zur Angabe eines Ursachencodes aufgefordert werden, um zu erklären, warum Sie eine Phase des Serviceauftrags aktualisieren, können Sie unter allen Ursachencodes auswählen, die Sie hier erstellen.</span><span class="sxs-lookup"><span data-stu-id="22228-107">When you are prompted for a reason code to explain why you are updating a stage of the service order, you can select any of the reason codes that you create here.</span></span>
