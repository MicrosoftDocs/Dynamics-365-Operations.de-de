# <a name="create-subscription-fee-transactions"></a><span data-ttu-id="ad292-103">Dauerauftragsgebühren-Buchungen erstellen</span><span class="sxs-lookup"><span data-stu-id="ad292-103">Create subscription fee transactions</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="ad292-104">Klicken Sie auf **Servicemanagement** \> **Gemeinsam** \> **Daueraufträge** \> **Alle Daueraufträge**.</span><span class="sxs-lookup"><span data-stu-id="ad292-104">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="ad292-105">Wählen Sie auf der Listenseite **Alle Daueraufträge** den Dauerauftrag aus, für die eine Dauerauftragsgebührenbuchung erstellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="ad292-105">On the **All service subscriptions** list page, select the subscription for which you want to create a subscription fee transaction.</span></span>

5.  <span data-ttu-id="ad292-108">Klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="ad292-108">Click **OK**.</span></span>
