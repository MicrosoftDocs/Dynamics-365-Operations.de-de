# <a name="create-subscription-fee-transactions"></a>Dauerauftragsgebühren-Buchungen erstellen 

[!include [banner](../includes/banner.md)]


1.  Klicken Sie auf **Servicemanagement** \> **Gemeinsam** \> **Daueraufträge** \> **Alle Daueraufträge**.

2.  Wählen Sie auf der Listenseite **Alle Daueraufträge** den Dauerauftrag aus, für die eine Dauerauftragsgebührenbuchung erstellt werden soll.

3.  Klicken Sie im **Aktivitätsbereich** auf der Registerkarte **Dauerauftrag** auf **Dauerauftragsgebühren**.

4.  In der **Dauerauftragsgebühr erstellen** Formular geben Sie die entsprechenden Informationen ein.

5.  Klicken Sie auf **OK**.
