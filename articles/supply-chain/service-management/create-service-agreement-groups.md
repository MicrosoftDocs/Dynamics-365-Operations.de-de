# <a name="create-service-agreement-groups"></a><span data-ttu-id="8ea02-103">Erstellen von Servicevereinbarungsgruppen</span><span class="sxs-lookup"><span data-stu-id="8ea02-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8ea02-104">Servicevereinbarungsgruppen sind hilfreich beim Sortieren und Filtern von Servicevereinbarungen.</span><span class="sxs-lookup"><span data-stu-id="8ea02-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="8ea02-105">Klicken Sie auf **Serviceverwaltung** \> **Einstellungen** \> **Servicevereinbarungen** \> **Serviceverinbarungsgruppen**.</span><span class="sxs-lookup"><span data-stu-id="8ea02-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="8ea02-106">Erstellen Sie durch Drücken von **Alt+N** eine neue Servicevereinbarungsgruppe.</span><span class="sxs-lookup"><span data-stu-id="8ea02-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="8ea02-107">Weisen Sie im Feld **Servicevereinbarungsgruppe**  die Servicevereinbarung einer vorhandenen Servicevereinbarungsgruppe zu.</span><span class="sxs-lookup"><span data-stu-id="8ea02-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="8ea02-108">Geben Sie im Feld **Beschreibung** eine Beschreibung für die Servicevereinbarungsgruppe ein.</span><span class="sxs-lookup"><span data-stu-id="8ea02-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="8ea02-109">Schließen Sie das Formular, um Ihre Änderungen zu speichern.</span><span class="sxs-lookup"><span data-stu-id="8ea02-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="8ea02-110">Wenn im Feld **Servicevereinbarungsgruppe** eine Servicevereinbarungsgruppe definiert wurde auf de Seite **Servicemanagementparameter**, wird diese Gruppe für alle neu erstellten Servicevereinbarungen vorgeschlagen.</span><span class="sxs-lookup"><span data-stu-id="8ea02-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
