# <a name="indexed-sales-prices"></a><span data-ttu-id="6141d-103">Indizierte Verkaufspreise</span><span class="sxs-lookup"><span data-stu-id="6141d-103">Indexed sales prices</span></span>  

[!include [banner](../includes/banner.md)]


<span data-ttu-id="6141d-104">Die Erstellung des Index für einen Dauerauftragsverkaufspreis erfolgt beim Erstellen einer Dauerauftragsgebühr.</span><span class="sxs-lookup"><span data-stu-id="6141d-104">You set up the index for a subscription sales price when you create a subscription fee.</span></span>

<span data-ttu-id="6141d-105">Legen Sie im Formular **Dauerauftragsgebühr erstellen** das Feld **Preisgestaltung abrufen von** auf **Indizierter Basispreis** fest, und multiplizieren Sie anschließend den Basispreis mit dem im Feld **Prozentuale Preisänderung** angegebenen Prozentsatz, um den Verkaufspreis für die Dauerauftragsbuchung zu ermitteln.</span><span class="sxs-lookup"><span data-stu-id="6141d-105">In the **Create subscription fee** form, set the **Get pricing from** field to **Indexed base price**, and then multiply the base price by the percentage in the **Percent price change** field to get the sales price of the subscription transaction.</span></span>

<span data-ttu-id="6141d-106">Beispiel: Bei einem Basispreis von EUR 1000 und einem Indexwert von 110 ergibt sich ein Verkaufspreis von EUR 1100.</span><span class="sxs-lookup"><span data-stu-id="6141d-106">For example, if the base price is EUR 1,000, and the index is 110, the sales price is EUR 1,100.</span></span>
