# <a name="set-up-service-activity-types"></a>Einrichten von Serviceaktivitätstypen 

[!include [banner](../includes/banner.md)]


1.  Klicken Sie auf **Vertrieb und Marketing** \> **Einstellungen** \> **Aktivitäten** \> **Aktivitätstypen**.
    
    -oder-
    
    Klicken Sie auf **Organisationsverwaltung** \> **Einstellungen** \> **Aktivitäten** \> **Aktivitätstypen**.

2.  Drücken Sie im Formular **Aktivitätstypen** STRG+N, um eine neue Aktivitätstypposition hinzuzufügen.

3.  Geben Sie im Feld **Typ** einen Namen für den Aktivitätstyp ein.

4.  Geben Sie im Feld **Beschreibung** Text zur Beschreibung dieses Aktivitätstyps ein.

5.  Schließen Sie das Formular, um Ihre Änderungen zu speichern.
