# <a name="move-service-order-lines"></a><span data-ttu-id="16e87-103">Serviceauftragspositionen verschieben</span><span class="sxs-lookup"><span data-stu-id="16e87-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="16e87-104">Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="16e87-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="16e87-105">Doppelklicken Sie auf den Serviceauftrag, der die Positionen enthält, die in einen anderen Serviceauftrag verschoben werden sollen.</span><span class="sxs-lookup"><span data-stu-id="16e87-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="16e87-106">In der **Serviceaufträge** im Formular, und klicken Sie im Aktivitätsbereich auf die Schaltfläche **Positionen verschieben**.</span><span class="sxs-lookup"><span data-stu-id="16e87-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="16e87-107">In der **Serviceauftragspositionen verschieben** Formular auf das Feld **Serviceauftrag**, wählen Sie den Serviceauftrag aus, in den Sie die Positionen verschieben möchten.</span><span class="sxs-lookup"><span data-stu-id="16e87-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="16e87-108">Aktivieren Sie das Kontrollkästchen **Leeren Serviceauftrag löschen**, wenn der leere Serviceauftrag, aus dem die Serviceauftragspositionen verschoben wurden, gelöscht werden soll .</span><span class="sxs-lookup"><span data-stu-id="16e87-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="16e87-109">Klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="16e87-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="16e87-110">Serviceauftragspositionen können nur zwischen Serviceaufträgen verschoben werden, die der gleichen Servicevereinbarung zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="16e87-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
