# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="3ea41-103">Einrichten einer Vorlage für Objekte am Kundenstandort</span><span class="sxs-lookup"><span data-stu-id="3ea41-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="3ea41-104">Verwenden Sie dieses Verfahren zum Einrichten einer Vorlage, die Objekte an einem Kundenstandort darstellt.</span><span class="sxs-lookup"><span data-stu-id="3ea41-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="3ea41-105">Klicken Sie auf den Bereichsseitenknoten: **Serviceverwaltung** \> **Gemeinsam** \> **Servicevereinbarungen** \> **Servicevereinbarungen**.</span><span class="sxs-lookup"><span data-stu-id="3ea41-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="3ea41-106">Doppelklicken Sie auf einen Servicevertrag, oder klicken Sie auf **Servicevertrag**, um einen neuen Servicevertrag zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="3ea41-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="3ea41-107">Klicken Sie auf die Registerkarte **Einstellungen**.</span><span class="sxs-lookup"><span data-stu-id="3ea41-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="3ea41-108">Klicken Sie auf **Serviceobjekte**, um eine Vorlagenstückliste an den Servicevertrag anzufügen.</span><span class="sxs-lookup"><span data-stu-id="3ea41-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="3ea41-109">Im Formular **Serviceobjekte** wählen Sie das Objekt und klicken Sie anschließend **Funktionen** \> **Vorlagenstücklisten anfügen**.</span><span class="sxs-lookup"><span data-stu-id="3ea41-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="3ea41-110">Wählen Sie im Formular **Vorlagenstückliste auswählen** im Feld **Vorlagenstückliste** eine Vorlagenstückliste aus und klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="3ea41-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="3ea41-111">Klicken Sie im Formular **Serviceobjekte** auf **Designer**, um die Vorlagenstückliste so zu ändern, dass die Stückliste die gewünschten Objekte enthält.</span><span class="sxs-lookup"><span data-stu-id="3ea41-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="3ea41-112">Schließen Sie die Formulare, um Ihre Änderungen zu speichern.</span><span class="sxs-lookup"><span data-stu-id="3ea41-112">Close the forms to save your changes.</span></span>
