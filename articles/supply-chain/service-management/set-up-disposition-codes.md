# <a name="set-up-disposition-codes"></a><span data-ttu-id="80059-103">Dispositionscodes einrichten</span><span class="sxs-lookup"><span data-stu-id="80059-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="80059-104">Sie können Sie Dispositionscodes einrichten, um anzugeben, wie ein Artikel verarbeitet wird, der von einem Debitor zurückgegeben wird.</span><span class="sxs-lookup"><span data-stu-id="80059-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="80059-105">Sie erstellen beispielsweise einen Dispositionscode mit Namen **Reparatur und Rücklieferung**, um anzugeben, dass der zurückgelieferte Artikel repariert und dann an den Debitor zurückgegeben wird.</span><span class="sxs-lookup"><span data-stu-id="80059-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="80059-106">Weitere Beispiele von Dispositionscodes, die üblicherweise für Artikel verwendet werden, die von Debitoren zurückgegeben werden, finden Sie unter [Angaben zur Entsorgung zurückgelieferter Artikel](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="80059-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="80059-107">Sie können auch einen Ursachencode einrichten, der dabei hilft zu erläutern, warum ein Artikel zurückgegeben wurde.</span><span class="sxs-lookup"><span data-stu-id="80059-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="80059-108">Weitere Informationen zu den Differenzgründen finden Sie unter [Rückgabegrundcodes einrichten](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="80059-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="80059-109">Klicken Sie auf **Vertrieb und Marketing** \> **Einstellungen** \> **Aufträge** \> **Retouren** \> **Dispositionscodes**.</span><span class="sxs-lookup"><span data-stu-id="80059-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="80059-110">Klicken Sie **Neu** oder drücken Sie STRG+N, um einen neuen Dispositionscode zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="80059-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="80059-111">Geben Sie einen eindeutigen, aussagekräftigen Namen ein, wählen Sie eine Aktion aus, und geben Sie eine Beschreibung für den Dispositionscode ein.</span><span class="sxs-lookup"><span data-stu-id="80059-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="80059-112">Wenn Sie eine Debitorenbelastungen mit diesem Dispositionscode zuordnen möchten, klicken Sie auf die Schaltfläche **Belastungen**, um das Formular **Belastungen einrichten** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="80059-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="80059-113">Wenn Sie externe Codes auf die Dispositionscodes des Unternehmens abstimmen möchten, klicken Sie auf die Schaltfläche **Externe Codes**, um das Formular **Externe Codes** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="80059-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
