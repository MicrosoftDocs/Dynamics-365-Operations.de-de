# <a name="set-up-disposition-codes"></a><span data-ttu-id="0323f-103">Dispositionscodes einrichten</span><span class="sxs-lookup"><span data-stu-id="0323f-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="0323f-104">Sie können Sie Dispositionscodes einrichten, um anzugeben, wie ein Artikel verarbeitet wird, der von einem Debitor zurückgegeben wird.</span><span class="sxs-lookup"><span data-stu-id="0323f-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="0323f-105">Sie erstellen beispielsweise einen Dispositionscode mit Namen **Reparatur und Rücklieferung**, um anzugeben, dass der zurückgelieferte Artikel repariert und dann an den Debitor zurückgegeben wird.</span><span class="sxs-lookup"><span data-stu-id="0323f-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="0323f-106">Weitere Beispiele von Dispositionscodes, die üblicherweise für Artikel verwendet werden, die von Debitoren zurückgegeben werden, finden Sie unter [Angaben zur Entsorgung zurückgelieferter Artikel](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="0323f-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="0323f-107">Sie können auch einen Ursachencode einrichten, der dabei hilft zu erläutern, warum ein Artikel zurückgegeben wurde.</span><span class="sxs-lookup"><span data-stu-id="0323f-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="0323f-108">Weitere Informationen zu den Differenzgründen finden Sie unter [Rückgabegrundcodes einrichten](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="0323f-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="0323f-109">Gehen Sie zu **Vertrieb und Marketing** \> **Einrichten** \> **Verkaufsaufträge** \> **Rücklieferungen** \> **Dispositioncodes**.</span><span class="sxs-lookup"><span data-stu-id="0323f-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="0323f-110">Wählen Sie **Neu**, um einen neuen Dispositionscode zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="0323f-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="0323f-111">Geben Sie einen eindeutigen, aussagekräftigen Namen ein, wählen Sie eine Aktion aus, und geben Sie eine Beschreibung für den Dispositionscode ein.</span><span class="sxs-lookup"><span data-stu-id="0323f-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="0323f-112">Wenn Sie diesem Dispositionscode irgendwelche Kundengebühren zuordnen wollen, wählen Sie die Schaltfläche **Gebühren**, um das Formular **Gebühren festlegen** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="0323f-112">If you want to associate any customer charges with this disposition code, select the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="0323f-113">Wenn Sie irgendwelche externen Codes definieren wollen, die mit den firmeneigenen Dispositionscodes übereinstimmen sollen, wählen Sie die Schaltfläche **Externe Codes**, um das Formular **Externe Codes** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="0323f-113">If you want to define any external codes to match with the company's own disposition codes, select the **External codes** button to open the **External codes** form.</span></span>
