# <a name="delete-a-return-order"></a><span data-ttu-id="359e5-103">Löschen einer Rücklieferung</span><span class="sxs-lookup"><span data-stu-id="359e5-103">Delete a return order</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="359e5-104">Löschen Sie mithilfe des Formulars "**Rücklieferung löschen**" mehrere Rücklieferungen.</span><span class="sxs-lookup"><span data-stu-id="359e5-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="359e5-105">Klicken Sie auf **Vertrieb und Marketing** \> **Periodisch** \> **Bereinigen** \> **Rücklieferungen löschen**.</span><span class="sxs-lookup"><span data-stu-id="359e5-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>
