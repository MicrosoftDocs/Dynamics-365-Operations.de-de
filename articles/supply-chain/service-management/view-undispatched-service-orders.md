# <a name="view-undispatched-service-orders"></a><span data-ttu-id="10a96-103">Anzeigen von nicht versendeten Serviceaufträgen</span><span class="sxs-lookup"><span data-stu-id="10a96-103">View undispatched service orders</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="10a96-104">In diesem Thema wird beschrieben, wie Aktivitäten in einem Serviceauftrag angezeigt werden, denen keine Arbeitskraft von einem Versandteam zugewiesen ist.</span><span class="sxs-lookup"><span data-stu-id="10a96-104">This topic describes how to view activities in a service order that don’t have a worker assigned to them from a dispatch team.</span></span>

1.  <span data-ttu-id="10a96-105">Klicken Sie auf **Serviceverwaltung** \> **Periodisch** \> **Einsatzplanung**.</span><span class="sxs-lookup"><span data-stu-id="10a96-105">Click **Service management** \> **Periodic** \> **Dispatch board**.</span></span>

2.  <span data-ttu-id="10a96-106">Geben Sie im Fenster **Einsatzplanung** das Start- und Enddatum für den Zeitraum ein, für den die Versandaktivität angezeigt werden soll.</span><span class="sxs-lookup"><span data-stu-id="10a96-106">In the **Dispatch board** window, enter the start date and end date for the period to view dispatch activity for.</span></span>

3.  <span data-ttu-id="10a96-107">Optional: Wählen Sie aus, ob abgeschlossene Aktivitäten und zugehörige Teams in die Ergebnisse eingeschlossen werden sollen.</span><span class="sxs-lookup"><span data-stu-id="10a96-107">Optional: Select whether to include closed activities and related teams in the results.</span></span>

4.  <span data-ttu-id="10a96-108">Klicken Sie auf **OK**, um die Ergebnisse anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="10a96-108">Click **OK** to view the results.</span></span>
    

    > [!NOTE]
    > <P><span data-ttu-id="10a96-109">Im Formular <STRONG>Einsatzplanung</STRONG> wird die Ansicht <STRONG>Erweitert</STRONG> automatisch angezeigt.</span><span class="sxs-lookup"><span data-stu-id="10a96-109">In the <STRONG>Dispatch board</STRONG> form, the <STRONG>Advanced</STRONG> view is automatically displayed.</span></span> <span data-ttu-id="10a96-110">In dieser Ansicht wird das Gantt-Diagramm am oberen und die Registerkarten am unteren Seitenrand angezeigt.</span><span class="sxs-lookup"><span data-stu-id="10a96-110">In this view, the Gantt chart at the top of the page and the tabs at the bottom of the page are displayed.</span></span> <span data-ttu-id="10a96-111">Klicken Sie auf <STRONG>Einfach</STRONG>, um nur das Gantt-Diagramm anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="10a96-111">Click <STRONG>Simple</STRONG> to display only the Gantt chart.</span></span></P>



5.  <span data-ttu-id="10a96-112">Um eine Liste von Serviceaufträgen anzuzeigen, denen kein Versandteam zugewiesen ist, klicken Sie auf die Registerkarte **Nicht im Einsatz**.</span><span class="sxs-lookup"><span data-stu-id="10a96-112">To view a list of service orders that don’t have a dispatch team assigned, click the **Undispatched** tab.</span></span>
