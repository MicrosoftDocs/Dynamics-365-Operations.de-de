# <a name="create-a-service-template-group"></a>Erstellen von Servicevorlagengruppen  

[!include [banner](../includes/banner.md)]

Servicevorlagengruppen dienen zum Sortieren und Filtern von Servicevorlagen.

1. Klicken Sie auf **Serviceverwaltung** \> **Einstellungen** \> **Servicevereinbarungen** \> **Servicevorlagengruppen**.
2. Erstellen Sie eine neue Servicevorlagengruppe.
3. Geben Sie eine Gruppenkennung, eine Beschreibung sowie eine übergeordnete Servicevorlagengruppe ein (sofern vorhanden).
4. Speichern Sie die Servicevorlagengruppe.
5. Setzen Sie den Schieberegler **Struktursteuerung** auf **Ja** fest, um die Strukturansicht zu öffnen und die Servicevorlagengruppen entweder in einer hierarchischen Struktur oder als Übersicht anzuzeigen.
