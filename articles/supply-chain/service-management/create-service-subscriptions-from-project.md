# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="47866-103">Erstellen von Daueraufträgen auf der Grundlage eines Projekts</span><span class="sxs-lookup"><span data-stu-id="47866-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="47866-104">In "**Projektverwaltung und -buchhaltung**" können Sie Daueraufträge auf der Grundlage eines Projekts erstellen.</span><span class="sxs-lookup"><span data-stu-id="47866-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="47866-105">Klicken Sie auf **Projektverwaltung und -buchhaltung** \> **Allgemein** \> **Projekte** \> **Alle Projekte**.</span><span class="sxs-lookup"><span data-stu-id="47866-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="47866-106">Öffnen Sie das Projekt, für das Sie einen Dauerauftrag erstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="47866-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="47866-107">Klicken Sie im Formular **Projekte** auf den **Aktivitätsbereich**, klicken Sie auf die Registerkarte **Verwalten** klicken Sie auf **Service** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="47866-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="47866-110">Geben Sie die übrigen Informationen in das Formular ein.</span><span class="sxs-lookup"><span data-stu-id="47866-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="47866-111">Für den Dauerauftrag können nun Dauerauftragsbuchungen erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="47866-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="47866-112">Wurde für die Dauerauftragsgruppe ein Nummernkreis eingerichtet, wird für den Dauerauftrag beim Angeben der Dauerauftragsgruppe automatisch eine Dauerauftragskennung generiert.</span><span class="sxs-lookup"><span data-stu-id="47866-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
