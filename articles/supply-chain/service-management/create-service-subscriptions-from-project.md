<span data-ttu-id="a4d63-104">[!include [banner](../includes/banner.md)</span><span class="sxs-lookup"><span data-stu-id="a4d63-104">[!include [banner](../includes/banner.md)</span></span>


<span data-ttu-id="a4d63-105">In "**Projektverwaltung und -buchhaltung**" können Sie Daueraufträge auf der Grundlage eines Projekts erstellen.</span><span class="sxs-lookup"><span data-stu-id="a4d63-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

2.  <span data-ttu-id="a4d63-108">Klicken Sie im Formular **Projekte** auf den **Aktivitätsbereich**, klicken Sie auf die Registerkarte **Verwalten** klicken Sie auf **Service** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="a4d63-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

<span data-ttu-id="a4d63-112">Für den Dauerauftrag können nun Dauerauftragsbuchungen erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="a4d63-112">You can now create subscription transactions for the subscription.</span></span>

> <P><span data-ttu-id="a4d63-113">Wurde für die Dauerauftragsgruppe ein Nummernkreis eingerichtet, wird für den Dauerauftrag beim Angeben der Dauerauftragsgruppe automatisch eine Dauerauftragskennung generiert.</span><span class="sxs-lookup"><span data-stu-id="a4d63-113">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
