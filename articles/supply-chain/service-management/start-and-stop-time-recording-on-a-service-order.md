## <a name="start-time-recording"></a><span data-ttu-id="b4da7-105">Starten der Zeiterfassung</span><span class="sxs-lookup"><span data-stu-id="b4da7-105">Start time recording</span></span>

1.  <span data-ttu-id="b4da7-106">Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="b4da7-106">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="b4da7-107">Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Start**.</span><span class="sxs-lookup"><span data-stu-id="b4da7-107">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Start**.</span></span>

3.  <span data-ttu-id="b4da7-108">Geben Sie das Datum und die Uhrzeit für den Beginn der Zeiterfassung ein.</span><span class="sxs-lookup"><span data-stu-id="b4da7-108">Enter the date and time that the time recording should be started.</span></span>

## <a name="stop-time-recording"></a><span data-ttu-id="b4da7-109">Zeiterfassung beenden</span><span class="sxs-lookup"><span data-stu-id="b4da7-109">Stop time recording</span></span>

1.  <span data-ttu-id="b4da7-110">Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="b4da7-110">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="b4da7-111">Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Stop**.</span><span class="sxs-lookup"><span data-stu-id="b4da7-111">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Stop**.</span></span>

3.  <span data-ttu-id="b4da7-112">Geben Sie das Datum und die Uhrzeit für das Ende der Zeiterfassung ein.</span><span class="sxs-lookup"><span data-stu-id="b4da7-112">Enter the date and time that the time recording should be stopped.</span></span>

4.  <span data-ttu-id="b4da7-113">Klicken Sie auf **Grund für Sperrung hinzufügen**, und wählen Sie in der Liste **Phasenursachencode** einen Ursachencode aus, um einen Grund für die Beendigung der Zeiterfassung anzugeben.</span><span class="sxs-lookup"><span data-stu-id="b4da7-113">Select **Add a revocation reason**, and select a reason code in the **Stage reason code** list to provide a reason for stopping the time recording.</span></span>

> <P><span data-ttu-id="b4da7-114">Wenn im Formular <STRONG>Serviceverwaltungsparameter</STRONG> die Option <STRONG>Ursachencode bei Zeitüberschreitung</STRONG> ausgewählt wurde, muss zum Beenden der Zeiterfassung zunächst ein Ursachencode angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="b4da7-114">If <STRONG>Reason code on exceeding time</STRONG> is selected in the <STRONG>Service management parameters</STRONG> form, you must provide a reason code before you can stop the time recording.</span></span></P>
