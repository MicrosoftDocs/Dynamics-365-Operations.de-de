# <a name="start-and-stop-time-recording-on-a-service-order"></a>Starten und Beenden der Zeitaufzeichnung für einen Serviceauftrag 

[!include [banner](../includes/banner.md)]


Mithilfe dieses Verfahrens können Sie die Zeiterfassung für eine Servicevereinbarung starten und beenden, für die eine Vereinbarung zum Servicelevel definiert wurde.

## <a name="start-time-recording"></a>Starten der Zeiterfassung

1.  Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.

2.  Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Start**.

3.  Geben Sie das Datum und die Uhrzeit für den Beginn der Zeiterfassung ein.

## <a name="stop-time-recording"></a>Zeiterfassung beenden

1.  Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.

2.  Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Stop**.

3.  Geben Sie das Datum und die Uhrzeit für das Ende der Zeiterfassung ein.

4.  Klicken Sie auf **Grund für Sperrung hinzufügen**, und wählen Sie in der Liste **Phasenursachencode** einen Ursachencode aus, um einen Grund für die Beendigung der Zeiterfassung anzugeben.


> [!NOTE]
> <P>Wenn im Formular <STRONG>Serviceverwaltungsparameter</STRONG> die Option <STRONG>Ursachencode bei Zeitüberschreitung</STRONG> ausgewählt wurde, muss zum Beenden der Zeiterfassung zunächst ein Ursachencode angegeben werden.</P>
