# <a name="start-and-stop-time-recording-on-a-service-order"></a><span data-ttu-id="ec9eb-103">Starten und Beenden der Zeitaufzeichnung für einen Serviceauftrag</span><span class="sxs-lookup"><span data-stu-id="ec9eb-103">Start and stop time recording on a service order</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="ec9eb-104">Mithilfe dieses Verfahrens können Sie die Zeiterfassung für eine Servicevereinbarung starten und beenden, für die eine Vereinbarung zum Servicelevel definiert wurde.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-104">Use this procedure to start and stop time recording for a service order for which a service level agreement is defined.</span></span>

## <a name="start-time-recording"></a><span data-ttu-id="ec9eb-105">Starten der Zeiterfassung</span><span class="sxs-lookup"><span data-stu-id="ec9eb-105">Start time recording</span></span>

1.  <span data-ttu-id="ec9eb-106">Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-106">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="ec9eb-107">Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Start**.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-107">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Start**.</span></span>

3.  <span data-ttu-id="ec9eb-108">Geben Sie das Datum und die Uhrzeit für den Beginn der Zeiterfassung ein.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-108">Enter the date and time that the time recording should be started.</span></span>

## <a name="stop-time-recording"></a><span data-ttu-id="ec9eb-109">Zeiterfassung beenden</span><span class="sxs-lookup"><span data-stu-id="ec9eb-109">Stop time recording</span></span>

1.  <span data-ttu-id="ec9eb-110">Klicken auf **Serviceverwaltung** \> **Gemeinsam** \> **Serviceaufträge** \> **Serviceaufträge**.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-110">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="ec9eb-111">Klicken Sie auf der Registerkarte **Serviceauftrag**. Klicken Sie im **Aktivitätsbereich** in der Gruppe **Vereinbarung zum Servicelevel** auf **Stop**.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-111">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Stop**.</span></span>

3.  <span data-ttu-id="ec9eb-112">Geben Sie das Datum und die Uhrzeit für das Ende der Zeiterfassung ein.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-112">Enter the date and time that the time recording should be stopped.</span></span>

4.  <span data-ttu-id="ec9eb-113">Klicken Sie auf **Grund für Sperrung hinzufügen**, und wählen Sie in der Liste **Phasenursachencode** einen Ursachencode aus, um einen Grund für die Beendigung der Zeiterfassung anzugeben.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-113">Select **Add a revocation reason**, and select a reason code in the **Stage reason code** list to provide a reason for stopping the time recording.</span></span>


> [!NOTE]
> <P><span data-ttu-id="ec9eb-114">Wenn im Formular <STRONG>Serviceverwaltungsparameter</STRONG> die Option <STRONG>Ursachencode bei Zeitüberschreitung</STRONG> ausgewählt wurde, muss zum Beenden der Zeiterfassung zunächst ein Ursachencode angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="ec9eb-114">If <STRONG>Reason code on exceeding time</STRONG> is selected in the <STRONG>Service management parameters</STRONG> form, you must provide a reason code before you can stop the time recording.</span></span></P>
