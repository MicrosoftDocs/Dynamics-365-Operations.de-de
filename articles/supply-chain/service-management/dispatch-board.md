#  <a name="dispatch-board"></a>Einsatzplanung 

[!include [banner](../includes/banner.md)]

Eine Einsatzplanung bildet den zentralen Ort, von dem aus Sie den Status von Aktivitäten in einem Serviceauftrag anzeigen können. Sie können die folgenden Aufgaben in einer Einsatzplanung durchführen:

  - Filtern und Anzeigen von Serviceaktivitäten innerhalb eines bestimmten Datumsbereichs

  - Angeben der Priorität einer Serviceaktivität und Unterscheiden von Serviceprioritätsstufen anhand eines Farbschemas, das Sie einrichten

  - Überprüfen der Arbeitskräfte, die einem Versandteam zugewiesen sind

  - Ändern der Servicezeiten und zugewiesenen Techniker für eine Serviceaktivität

  - Überprüfen der Liste mit Serviceaktivitäten, für die noch kein Einsatz erfolgt ist

Informationen darüber, wie bestimmte Aufgaben in einer Einsatzplanung durchgeführt werden, finden Sie in den Artikeln im Abschnitt **Siehe auch**.
