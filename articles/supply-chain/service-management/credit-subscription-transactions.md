## <a name="credit-subscription-transactions"></a><span data-ttu-id="f36a4-103">Gutschreiben von Dauerauftragsbuchungen</span><span class="sxs-lookup"><span data-stu-id="f36a4-103">Credit subscription transactions</span></span>

1.  <span data-ttu-id="f36a4-104">Klicken Sie auf **Servicemanagement** \> **Gemeinsam** \> **Daueraufträge** \> **Alle Daueraufträge**.</span><span class="sxs-lookup"><span data-stu-id="f36a4-104">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="f36a4-105">Wählen Sie den Dauerauftrag aus, der der Dauerauftragsbuchung zugeordnet ist, für die eine Gutschrift erstellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="f36a4-105">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="f36a4-106">Wählen Sie die Registerkarte **Analysieren** aus, und klicken Sie im Aktivitätsbereich auf die Schaltfläche **Gebührenbuchungen**.</span><span class="sxs-lookup"><span data-stu-id="f36a4-106">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="f36a4-107">Wählen Sie im Formular **Gebührenbuchungen** die Buchung aus, für die die Gutschrift erstellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="f36a4-107">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="f36a4-108">Klicken Sie auf **Funktionen** \> **Für Gutschrift auswählen**.</span><span class="sxs-lookup"><span data-stu-id="f36a4-108">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="f36a4-109">Wählen Sie im Formular **Für Gutschrift auswählen** die Buchung aus, die gutgeschrieben werden soll, und klicken Sie anschließend auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="f36a4-109">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="f36a4-110">Achten Sie beim Erstellen der Gutschrift darauf, dass Sie <STRONG>Gutschriften</STRONG> auswählen.</span><span class="sxs-lookup"><span data-stu-id="f36a4-110">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="f36a4-111">Diese Option befindet sich im Dialogfeld <STRONG>Rechnung erstellen</STRONG> in der Liste <STRONG>Rechnungsstellungsmethode</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="f36a4-111">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="f36a4-112">Ist im Formular **Serviceverwaltungsparameter** das Feld **Abgrenzungen bei Gutschrift zurücksetzen** auf **Manuell** festgelegt, müssen vor dem Erstellen eines Gutschriftvorschlags für die Buchung alle Buchungen für den antizipierten Umsatzerlös einzeln storniert werden.</span><span class="sxs-lookup"><span data-stu-id="f36a4-112">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
