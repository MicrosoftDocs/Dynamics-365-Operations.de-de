# <a name="delete-service-orders"></a><span data-ttu-id="8906c-103">Serviceaufträge löschen</span><span class="sxs-lookup"><span data-stu-id="8906c-103">Delete service orders</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="8906c-104">Klicken Sie auf **Serviceverwaltung** \> **Periodisch** \> **Serviceaufträge** \> **Serviceaufträge löschen**.</span><span class="sxs-lookup"><span data-stu-id="8906c-104">Click **Service management** \> **Periodic** \> **Service orders** \> **Delete service orders**.</span></span>

2.  <span data-ttu-id="8906c-105">Klicken Sie im Formular **Serviceaufträge löschen** auf **Auswählen**, um die Kriterien für die Auswahl der zu löschenden Serviceaufträge anzugeben, und dann auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="8906c-105">On the **Delete service orders** form, click **Select** to specify the criteria to select the service orders to be deleted, and then click **OK**.</span></span>

3.  <span data-ttu-id="8906c-106">Aktivieren Sie zum Generieren eines Infologs mit den gelöschten Serviceaufträgen das Kontrollkästchen **Infolog anzeigen**.</span><span class="sxs-lookup"><span data-stu-id="8906c-106">Select the **Show Infolog** check box to generate an Infolog that displays the deleted service orders.</span></span>

4.  <span data-ttu-id="8906c-107">Klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="8906c-107">Click **OK**.</span></span>


> [!NOTE]

> <P><span data-ttu-id="8906c-110">Außerdem können nur Serviceaufträge mit einem Status gelöscht werden, für den Sie über die Berechtigung zum Löschen verfügen.</span><span class="sxs-lookup"><span data-stu-id="8906c-110">Also, you can only delete service orders with a stage that lets you delete them.</span></span></P>
