# <a name="set-up-subscription-groups"></a><span data-ttu-id="d02b7-103">Dauerauftragsgruppen einrichten</span><span class="sxs-lookup"><span data-stu-id="d02b7-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="d02b7-104">Klicken Sie auf **Servicemanagement** \> **Einrichtung** \> **Daueraufträge** \> **Dauerauftragsgruppen**.</span><span class="sxs-lookup"><span data-stu-id="d02b7-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="d02b7-105">Drücken Sie STRG+N, oder klicken Sie auf **Neu**, um eine neue Dauerauftragsgruppe zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="d02b7-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="d02b7-106">Wählen Sie im Feld **Dauerauftragsgruppe** einen Namen für die neue Dauerauftragsgruppe ein.</span><span class="sxs-lookup"><span data-stu-id="d02b7-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="d02b7-107">Wählen Sie im Feld **Rechnungsstellung** eine Rechnungsstellungshäufigkeit für den Dauerauftrag aus.</span><span class="sxs-lookup"><span data-stu-id="d02b7-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="d02b7-108">Die Rechnungsstellungshäufigkeit wird mithilfe eines Periodencodes angegeben, der im Formular **Periodentypen** eingerichtet wurde.</span><span class="sxs-lookup"><span data-stu-id="d02b7-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="d02b7-109">Aktivieren Sie das Kontrollkästchen **Umsatz antizipieren**, wenn Sie Umsatzerlöse für Dauerauftragsbuchungen antizipieren möchten, und wählen Sie anschließend im Feld **Antizipieren** eine Häufigkeit aus.</span><span class="sxs-lookup"><span data-stu-id="d02b7-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="d02b7-110">Die Antizipierungshäufigkeit basiert auf einem Periodencode, der im Formular **Perioden** eingerichtet wurde.</span><span class="sxs-lookup"><span data-stu-id="d02b7-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
