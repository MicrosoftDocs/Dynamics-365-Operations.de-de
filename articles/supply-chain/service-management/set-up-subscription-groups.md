# <a name="set-up-subscription-groups"></a>Dauerauftragsgruppen einrichten 

[!include [banner](../includes/banner.md)]


1.  Gehen Sie zu **Serviceverwaltung** \> **Einrichten** \> **Service-Abonnements** \> **Abonnementgruppen**.

2.  Wählen Sie **Neu**, um eine neue Abonnementgruppe zu erstellen.

3.  Wählen Sie im Feld **Dauerauftragsgruppe** einen Namen für die neue Dauerauftragsgruppe ein.

4.  Wählen Sie im Feld **Rechnungsstellung** eine Rechnungsstellungshäufigkeit für den Dauerauftrag aus. Die Rechnungsstellungshäufigkeit wird mithilfe eines Periodencodes angegeben, der im Formular **Periodentypen** eingerichtet wurde.

5.  Aktivieren Sie das Kontrollkästchen **Umsatz antizipieren**, wenn Sie Umsatzerlöse für Dauerauftragsbuchungen antizipieren möchten, und wählen Sie anschließend im Feld **Antizipieren** eine Häufigkeit aus. Die Antizipierungshäufigkeit basiert auf einem Periodencode, der im Formular **Perioden** eingerichtet wurde.
