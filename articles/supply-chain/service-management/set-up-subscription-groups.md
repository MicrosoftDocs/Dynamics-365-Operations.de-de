# <a name="set-up-subscription-groups"></a><span data-ttu-id="7f222-103">Dauerauftragsgruppen einrichten</span><span class="sxs-lookup"><span data-stu-id="7f222-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="7f222-104">Gehen Sie zu **Serviceverwaltung** \> **Einrichten** \> **Service-Abonnements** \> **Abonnementgruppen**.</span><span class="sxs-lookup"><span data-stu-id="7f222-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="7f222-105">Wählen Sie **Neu**, um eine neue Abonnementgruppe zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="7f222-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="7f222-106">Wählen Sie im Feld **Dauerauftragsgruppe** einen Namen für die neue Dauerauftragsgruppe ein.</span><span class="sxs-lookup"><span data-stu-id="7f222-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="7f222-107">Wählen Sie im Feld **Rechnungsstellung** eine Rechnungsstellungshäufigkeit für den Dauerauftrag aus.</span><span class="sxs-lookup"><span data-stu-id="7f222-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="7f222-108">Die Rechnungsstellungshäufigkeit wird mithilfe eines Periodencodes angegeben, der im Formular **Periodentypen** eingerichtet wurde.</span><span class="sxs-lookup"><span data-stu-id="7f222-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="7f222-109">Aktivieren Sie das Kontrollkästchen **Umsatz antizipieren**, wenn Sie Umsatzerlöse für Dauerauftragsbuchungen antizipieren möchten, und wählen Sie anschließend im Feld **Antizipieren** eine Häufigkeit aus.</span><span class="sxs-lookup"><span data-stu-id="7f222-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="7f222-110">Die Antizipierungshäufigkeit basiert auf einem Periodencode, der im Formular **Perioden** eingerichtet wurde.</span><span class="sxs-lookup"><span data-stu-id="7f222-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
