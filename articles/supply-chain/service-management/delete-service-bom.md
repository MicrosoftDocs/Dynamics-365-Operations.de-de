# <a name="delete-a-service-bom"></a>Löschen einer Servicestückliste

[!include [banner](../includes/banner.md)]

1. Gehen Sie zu **Serviceverwaltung** \> **Allgemein** \> **Servicevereinbarungen** \> **Servicevereinbarungen**. Wählen Sie die Servicevereinbarung aus, aus der Sie eine Dienstleistungsstückliste (BOM) löschen möchten.
1. Wählen Sie im **Aktionsbereich** **Serviceobjekte** aus.
1. Wählen Sie im Formular **Serviceobjekte** das Serviceobjekt aus, das von der Servicestückliste gelöscht werden soll. Wählen Sie **Funktionen** und anschließend **Servicestückliste löschen** aus.
1. Wählen Sie **Ja** aus.


> [!NOTE]
> Wenn keine Vorlagenstückliste angewendet wurde, kann die Servicestückliste auch auf der Seite **Vorlagenstücklisten** gelöscht werden. Wurde jedoch bereits eine Vorlagenstückliste angewendet, kann die Servicestückliste mit dieser Methode nicht mehr gelöscht werden.
