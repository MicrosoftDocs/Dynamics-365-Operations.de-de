# <a name="delete-a-service-bom"></a><span data-ttu-id="62310-103">Löschen einer Servicestückliste</span><span class="sxs-lookup"><span data-stu-id="62310-103">Delete a service BOM</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="62310-104">Klicken Sie auf den Bereichsseitenknoten: **Serviceverwaltung** \> **Gemeinsam** \> **Servicevereinbarungen** \> **Servicevereinbarungen**.</span><span class="sxs-lookup"><span data-stu-id="62310-104">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span> <span data-ttu-id="62310-105">Wählen Sie die Servicevereinbarung aus, aus der Sie eine Dienstleistungsstückliste (BOM) löschen möchten.</span><span class="sxs-lookup"><span data-stu-id="62310-105">Select the service agreement that you want to delete a service bill of materials (BOM) from.</span></span>

2.  <span data-ttu-id="62310-106">Klicken Sie im **Aktivitätsbereich** auf **Serviceobjekte**.</span><span class="sxs-lookup"><span data-stu-id="62310-106">On the **Action Pane**, click **Service objects**.</span></span>

3.  <span data-ttu-id="62310-107">Wählen Sie im Formular **Serviceobjekte** das Serviceobjekt aus, das von der Servicestückliste gelöscht werden soll.</span><span class="sxs-lookup"><span data-stu-id="62310-107">In the **Service objects** form, select the service object that you want to delete a service BOM from.</span></span> <span data-ttu-id="62310-108">Klicken Sie **Funktionen** und wählen Sie anschließend **Servicestückliste löschen** aus.</span><span class="sxs-lookup"><span data-stu-id="62310-108">Click **Functions**, and then select **Delete service BOM**.</span></span>

4.  <span data-ttu-id="62310-109">Klicken Sie auf **Ja**.</span><span class="sxs-lookup"><span data-stu-id="62310-109">Click **Yes**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="62310-110">Wenn keine Vorlagenstückliste angewendet wurde, kann die Servicestückliste auch im Formular <STRONG>Vorlagenstücklisten</STRONG> gelöscht werden.</span><span class="sxs-lookup"><span data-stu-id="62310-110">If you have not applied a template BOM, the service BOM can be deleted from the <STRONG>Template BOMs</STRONG> form.</span></span> <span data-ttu-id="62310-111">Wurde jedoch bereits eine Vorlagenstückliste angewendet, kann die Servicestückliste mit dieser Methode nicht mehr gelöscht werden.</span><span class="sxs-lookup"><span data-stu-id="62310-111">If you have applied a template BOM, the service BOM cannot be deleted by using this method.</span></span></P>
