# <a name="reverse-changes-in-a-dispatch-board"></a><span data-ttu-id="f384b-103">Rückgängigmachen von Änderungen in einer Einsatzplanung</span><span class="sxs-lookup"><span data-stu-id="f384b-103">Reverse changes in a dispatch board</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="f384b-104">In diesem Thema wird beschrieben, wie nicht gespeicherten Änderungen, die Sie in einer Einsatzplanung vornehmen, zurückgesetzt werden.</span><span class="sxs-lookup"><span data-stu-id="f384b-104">This topic describes how to reverse unsaved modifications that you make in a dispatch board.</span></span> <span data-ttu-id="f384b-105">Sie weisen beispielsweise eine Arbeitskraft einer Serviceaktivität zu, speichern die Informationen und entscheiden dann später, der Serviceaktivität eine andere Arbeitskraft zuzuweisen.</span><span class="sxs-lookup"><span data-stu-id="f384b-105">For example, you assign a worker to a service activity, save the information, and then later decide to assign a different worker to the service activity.</span></span> <span data-ttu-id="f384b-106">Sie ändern die Arbeitskraft in der Einsatzplanung und erfahren dann, bevor Sie die Änderung speichern, dass die Arbeitskraft, die gerade zugewiesen wurde, nicht verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="f384b-106">You modify the worker in the dispatch board, and then, before saving the change, learn that the worker just assigned is not available.</span></span> <span data-ttu-id="f384b-107">Sie können die nicht gespeicherten Änderung zurücksetzen, damit die ursprüngliche Arbeitskraft dem Serviceauftrag wieder zugeordnet wird.</span><span class="sxs-lookup"><span data-stu-id="f384b-107">You can reverse the unsaved modification so that the original worker is reassigned to the service order.</span></span>

<span data-ttu-id="f384b-108">Gehen Sie folgendermaßen vor, um nicht gespeicherten Änderungen in einer Einsatzplanung zurückzusetzen:</span><span class="sxs-lookup"><span data-stu-id="f384b-108">Use the following steps to reverse unsaved changes in a dispatch board:</span></span>

1.  <span data-ttu-id="f384b-109">Klicken Sie auf **Serviceverwaltung** \> **Periodisch** \> **Einsatzplanung**.</span><span class="sxs-lookup"><span data-stu-id="f384b-109">Click **Service management** \> **Periodic** \> **Dispatch board**.</span></span>

2.  <span data-ttu-id="f384b-110">Geben Sie im Formular **Einsatzplanung** die relevanten Informationen in die Felder ein, und klicken Sie dann auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="f384b-110">In the **Dispatch board** form, enter relevant information in the fields, and then click **OK**.</span></span> 

3.  <span data-ttu-id="f384b-111">Um die letzte Änderung, die nicht gespeichert wurde, zurückzusetzen, klicken Sie auf **Rückgängig**.</span><span class="sxs-lookup"><span data-stu-id="f384b-111">To reverse the most recent change that is not saved, click **Undo**.</span></span>

4.  <span data-ttu-id="f384b-112">Klicken Sie zum Rückgängigmachen mehrerer noch nicht gespeicherter Änderungen weiter auf **Rückgängig**, bis alle zu ignorierenden Änderungen rückgängig gemacht wurden.</span><span class="sxs-lookup"><span data-stu-id="f384b-112">To reverse a series of changes that are not saved, continue clicking **Undo** until each change that you want to discard is reversed.</span></span>
