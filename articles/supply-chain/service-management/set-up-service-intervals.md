# <a name="set-up-service-intervals"></a><span data-ttu-id="6a810-103">Einrichten von Serviceintervallen</span><span class="sxs-lookup"><span data-stu-id="6a810-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="6a810-104">Durch das Servicevereinbarungsintervall wird die Häufigkeit angegeben, mit der bei der automatischen Erstellung von Serviceaufträgen Serviceauftragspositionen für Servicevereinbarungspositionen erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="6a810-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="6a810-105">Klicken Sie auf **Serviceverwaltung** \> **Einstellungen** \> **Servicevereinbarungen** \> **Serviceintervalle**.</span><span class="sxs-lookup"><span data-stu-id="6a810-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="6a810-106">Erstellen Sie ein neues Serviceintervall.</span><span class="sxs-lookup"><span data-stu-id="6a810-106">Create a new service interval.</span></span>
3. <span data-ttu-id="6a810-107">Geben Sie die Kennung und eine Beschreibung des Serviceintervalls ein.</span><span class="sxs-lookup"><span data-stu-id="6a810-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="6a810-108">Wählen Sie im Feld **Bereich** den Bereich aus.</span><span class="sxs-lookup"><span data-stu-id="6a810-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="6a810-109">Geben Sie im Feld **Häufigkeit** die Häufigkeit ein.</span><span class="sxs-lookup"><span data-stu-id="6a810-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="6a810-110">Die Häufigkeit ist der Faktor, mit dem Sie den Bereich multiplizieren müssen, um das Intervall für eine Servicevereinbarung erhalten.</span><span class="sxs-lookup"><span data-stu-id="6a810-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="6a810-111">Drücken Sie **Alt+S**, um das Serviceintervall zu speichern.</span><span class="sxs-lookup"><span data-stu-id="6a810-111">Press **Alt+S** to save the service interval.</span></span>
