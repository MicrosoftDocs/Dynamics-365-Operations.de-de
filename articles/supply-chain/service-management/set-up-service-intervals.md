# <a name="set-up-service-intervals"></a><span data-ttu-id="e14a7-103">Einrichten von Serviceintervallen</span><span class="sxs-lookup"><span data-stu-id="e14a7-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e14a7-104">Durch das Servicevereinbarungsintervall wird die Häufigkeit angegeben, mit der bei der automatischen Erstellung von Serviceaufträgen Serviceauftragspositionen für Servicevereinbarungspositionen erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="e14a7-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="e14a7-105">Klicken Sie auf **Serviceverwaltung** \> **Einstellungen** \> **Servicevereinbarungen** \> **Serviceintervalle**.</span><span class="sxs-lookup"><span data-stu-id="e14a7-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="e14a7-106">Erstellen Sie ein neues Serviceintervall.</span><span class="sxs-lookup"><span data-stu-id="e14a7-106">Create a new service interval.</span></span>
3. <span data-ttu-id="e14a7-107">Geben Sie die Kennung und eine Beschreibung des Serviceintervalls ein.</span><span class="sxs-lookup"><span data-stu-id="e14a7-107">Enter the ID and description of the service interval.</span></span>
