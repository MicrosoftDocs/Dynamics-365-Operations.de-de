# <a name="receive-returned-items"></a>Zurückgegebene Artikel empfangen 

[!include [banner](../includes/banner.md)]


Wenn eine Rücklieferung eingeht, gleichen Sie sie mit einer Rücklieferung ab, die bereits erstellt wurde.

Je nach Unternehmensrichtlinie können die Artikel in der Rücklieferung in Quarantäne zur zusätzliches Prüfung versetzt werden, oder sie können in den Bestand zurückgegeben werden.
