# <a name="set-up-dispatch-teams"></a>Einsatzteams einrichten 

[!include [banner](../includes/banner.md)]


In diesem Abschnitt wird beschrieben, wie Sie ein Versandteam erstellen. Nachdem Sie ein Versandteam erstellt haben, können Sie Arbeitskräfte zum Team zuweisen. Wenn Sie einen Serviceauftrag erstellen, können Sie dann ein Mitglied eines Versandteams zu einer Aktivität im Serviceauftrag zuordnen.

1.  Klicken sie auf **Serviceverwaltung** \> **Einstellungen** \> **Einsatzteams**.

2.  Klicken Sie zum Erstellen eines neuen Teams im Formular **Einsatzteams** auf **Neu**.

3.  Geben Sie im Feld **Einsatzteam** einen eindeutigen Namen für das Versandteam ein.

4.  Wählen Sie im Feld **Besitzer** den Mitarbeiter aus, der für dieses Team verantwortlich ist.
    

    > [!NOTE]
    > <P>Sie können nur Mitarbeiter in diesem Feld auswählen. Sie können keine Kreditoren auswählen.</P>



5.  Optional: Geben Sie im Feld **Beschreibung** eine kurze Beschreibung dieses Versandteams ein.


> [!NOTE]
> <P>Sie können im Formular <STRONG>Arbeitskraft</STRONG> eine Arbeitskraft zu diesem Versandteam zuweisen. Weitere Informationen zum Zuweisen einer Arbeitskraft zu einem Versandteam finden Sie unter <A href="https://technet.microsoft.com/en-us/library/dn776288(v=ax.60)">Arbeitskräfte disponieren (Formular)</A>.</P>
