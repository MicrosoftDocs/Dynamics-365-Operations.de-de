# <a name="copy-service-templates-lines"></a><span data-ttu-id="0e228-103">Kopieren von Servicevorlagenpositionen</span><span class="sxs-lookup"><span data-stu-id="0e228-103">Copy service templates lines</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="0e228-104">Vorlageninformationen können entweder in eine Servicevereinbarung oder in einen Serviceauftrag kopiert werden.</span><span class="sxs-lookup"><span data-stu-id="0e228-104">Template information can be copied to either a service agreement or a service order.</span></span>

## <a name="copy-service-template-lines-into-a-service-agreement"></a><span data-ttu-id="0e228-105">Kopieren von Servicevorlagenpositionen in eine Servicevereinbarung</span><span class="sxs-lookup"><span data-stu-id="0e228-105">Copy service template lines into a service agreement</span></span>

1. <span data-ttu-id="0e228-106">**Servicevereinbarungen** anklicken.</span><span class="sxs-lookup"><span data-stu-id="0e228-106">Click **Service agreements**.</span></span>
2. <span data-ttu-id="0e228-107">Doppelklicken Sie auf eine Servicevereinbarung, um das Detailformular **Servicevereinbarung**zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="0e228-107">Double-click a service agreement to open the **Service agreements** page.</span></span>
3. <span data-ttu-id="0e228-108">Klicken Sie auf **Vorlagenpositionen**, um die Seite **Servicevorlagenpositionen kopieren** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="0e228-108">Click **Template lines** to open the **Copy service template lines** page.</span></span>
4. <span data-ttu-id="0e228-109">Erweitern Sie auf der Registerkarte  **Baumstruktur**die Struktur, suchen Sie die Vorlage mit den Positionen, die in die Servicevereinbarung kopiert werden sollen, und wählen Sie die Servicevorlage aus.</span><span class="sxs-lookup"><span data-stu-id="0e228-109">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service agreement, and then select the service template.</span></span>
5. <span data-ttu-id="0e228-110">Aktivieren Sie auf der Registerkarte **Zeile** und **wählen** Sie das Kontrollkästchen  für die zu kopierenden Servicevorlagenpositionen.</span><span class="sxs-lookup"><span data-stu-id="0e228-110">On the **Lines** tab, select the **Select** check box for the service template lines that you want to copy.</span></span>
6. <span data-ttu-id="0e228-111">Klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="0e228-111">Click **OK**.</span></span>
7. <span data-ttu-id="0e228-112">Schließen Sie die Seiten, um Ihre Änderungen zu speichern.</span><span class="sxs-lookup"><span data-stu-id="0e228-112">Close the pages to save your changes.</span></span>

<span data-ttu-id="0e228-113">Die Servicevorlagenpositionen werden zusammen mit den zugeordneten Serviceobjekt- und Serviceaufgabenbeziehungen in die Servicevereinbarung kopiert.</span><span class="sxs-lookup"><span data-stu-id="0e228-113">The service template lines are now copied to the service agreement, together with the attached service object and service task relations.</span></span>

4. <span data-ttu-id="0e228-118">Erweitern Sie auf der Registerkarte  **Baumstruktur**die Struktur, suchen Sie die Vorlage mit den Positionen, die in die Servicevereinbarung kopiert werden sollen, und wählen Sie die Servicevorlage aus.</span><span class="sxs-lookup"><span data-stu-id="0e228-118">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service order, and then select the service template.</span></span>

7. <span data-ttu-id="0e228-121">Schließen Sie die Seiten, um Ihre Änderungen zu speichern.</span><span class="sxs-lookup"><span data-stu-id="0e228-121">Close the pages to save your changes.</span></span>
