# <a name="attach-templates-to-service-objects"></a>Zuordnen von Vorlagen für Serviceobjekte    

[!include [banner](../includes/banner.md)]


Eine Vorlagenstückliste kann einer Serviceobjektbeziehung zugeordnet werden. Diese kann dann ihrerseits entweder Serviceaufträgen oder Servicevereinbarungen zugeordnet werden.

1.  Klicken Sie auf **Serviceverwaltung** \> **Einstellungen** \> **Serviceobjekte** \> **Serviceobjekte**.

2.  Wählen Sie ein Serviceobjekt aus.

3.  Klicken Sie zum Anzeigen der Servicevereinbarungen oder Serviceaufträge, denen das Serviceobjekt zugeordnet ist, auf **Serviceverträge** bzw. **Serviceaufträge**.

4.  Wählen Sie eine Servicevereinbarung oder einen Serviceauftrag aus, klicken Sie **Funktionen** und **Vorlagenstücklisten anfügen**.

5.  Wählen Sie die Vorlagenstückliste aus, die dem Objekt zugeordnet werden soll.


> [!TIP]
> <P>Soll eine zugeordnete Stückliste gelöscht bzw. in ein anderes Serviceobjekt verschoben werden, klicken Sie im Formular <STRONG>Serviceobjekte</STRONG> auf <STRONG>Funktionen</STRONG>, und klicken Sie anschließend auf <STRONG>Servicestückliste verschieben</STRONG> oder <STRONG>Servicestückliste löschen</STRONG>.</P>
