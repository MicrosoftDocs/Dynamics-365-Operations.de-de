# <a name="set-up-service-order-stages"></a>Serviceauftragsphasen einrichten 

[!include [banner](../includes/banner.md)]


1.  Klicken auf **Serviceverwaltung** \> **Einrichtung** \> **Serviceaufträge** \> **Servicephasen**.

2.  Drücken Sie STRG+N, um einen neuen Datensatz zu erstellen.

3.  Geben Sie in den Feldern **Servicephase** und **Beschreibung** eine Servicephasenkennung und eine Beschreibung an.

4.  Wählen Sie die geeigneten Parameter für die Phase aus.

5.  Wählen Sie die übergeordnete Phase für die aktuelle Phase aus, oder lassen Sie das Feld **Übergeordnet** leer, wenn es sich bei der Phase um die Anfangsphase handelt.


> [!NOTE]
> <P>Nach dem Speichern der Phase kann das Feld <STRONG>Übergeordnet</STRONG> nicht mehr geändert werden. Sie können den Datensatz stattdessen löschen und mit einer anderen Auswahl im Feld <STRONG>Übergeordnet</STRONG> neu erstellen.</P>
> <P>Eine Phase kann nur mit einem leeren Feld <STRONG>Übergeordnet</STRONG> erstellt werden. Anders ausgedrückt: Es ist immer nur eine Anfangsphase zulässig.</P>
