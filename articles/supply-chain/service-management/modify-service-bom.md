## <a name="update-a-service-bom-element"></a><span data-ttu-id="bac0f-107">Aktualisieren eines Servicestücklistenelements</span><span class="sxs-lookup"><span data-stu-id="bac0f-107">Update a service BOM element</span></span>

1.  <span data-ttu-id="bac0f-108">Klicken Sie auf den Bereichsseitenknoten: **Serviceverwaltung** \> **Gemeinsam** \> **Servicevereinbarungen** \> **Servicevereinbarungen**.</span><span class="sxs-lookup"><span data-stu-id="bac0f-108">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

3.  <span data-ttu-id="bac0f-110">Klicken Sie im **Aktivitätsbereich** auf **Serviceobjekte**, um das Formular **Serviceobjekte** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="bac0f-110">On the **Action Pane**, click **Service objects** to open the **Service objects** form.</span></span>

4.  <span data-ttu-id="bac0f-111">Wählen Sie das Objekt aus, für das Sie eine Stücklistenposition aktualisieren möchten, und klicken Sie anschließend auf **Designer**.</span><span class="sxs-lookup"><span data-stu-id="bac0f-111">Select the object to update a BOM line for, and then click **Designer**.</span></span>

5.  <span data-ttu-id="bac0f-112">Wählen Sie im Formular **Designer** die zu aktualisierende Stücklistenposition aus, und klicken Sie anschließend auf **Stücklistenposition bearbeiten**.</span><span class="sxs-lookup"><span data-stu-id="bac0f-112">In the **Designer** form, select the BOM line to update, and then click **Edit BOM line**.</span></span>
    
    > [!NOTE]
    > <P><span data-ttu-id="bac0f-113">Aktivieren Sie auf der Registerkarte <STRONG>Einrichtung</STRONG> das Kontrollkästchen <STRONG>Beim Hinzufügen bearbeiten</STRONG>, wenn das Formular <STRONG>Stücklistenposition bearbeiten</STRONG> geöffnet werden soll, wenn Sie eine Position in die Servicestückliste ziehen.</span><span class="sxs-lookup"><span data-stu-id="bac0f-113">On the <STRONG>Setup</STRONG> tab, select the <STRONG>Edit when adding</STRONG> check box if you want the <STRONG>Edit BOM line</STRONG> form to open when you drag a line into the service BOM.</span></span></P>

6.  <span data-ttu-id="bac0f-114">Geben Sie im Feld **Menge** die Menge ein.</span><span class="sxs-lookup"><span data-stu-id="bac0f-114">In the **Quantity** field, enter the quantity.</span></span>

7.  <span data-ttu-id="bac0f-115">Aktivieren Sie das Kontrollkästchen **Serviceauftragsposition erstellen**, wenn für den Ersetzungsartikel eine (fakturierbare) Serviceauftragsposition erstellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="bac0f-115">If you want to create a service order line for the replacement item, which can then be invoiced, select the **Create service order line** check box.</span></span>
