# <a name="view-outbound-planned-intercompany-demand"></a>Ausgehenden geplanten Intercompany-Bedarf anzeigen

[!include [banner](../../includes/banner.md)]

Dieses Verfahren zeigt, wie Sie alle Bestellvorschläge anzeigen, die von einem Intercompany-Kreditor erfüllt werden. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist DEMF.

1. Wählen Sie **Produktprogrammplanung**.
2. Geben Sie im Feld **Plan** einen Wert ein oder wählen Sie ihn aus.
    * Wählen Sie im Feld **Plan** die *10* aus.  
3. Wählen Sie *Ausführen* aus.
4. Geben Sie im Feld **Anzahl von Threads** eine Zahl ein.
    * Dies stellt die Anzahl von parallelen Threads für die Produktprogrammplanung dar.  
5. Wählen Sie **OK** aus.
    * Dies kann einige Zeit in Anspruch nehmen.  
6. Wählen Sie **Geplanter Intercompany-Bedarf** aus.
7. Wählen Sie **Ausgehender geplanter Intercompany-Bedarf** aus.
    * Diese Seite enthält eine Übersicht des gesamten geplanten Bedarfs, der durch einen internen Lieferkettenkreditor erfüllt wird.  
8. Erweitern Sie den Abschnitt **Bedarfsdetails hochstreamen**.
    * In diesem Abschnitt finden Sie Details zur Erfüllung des Bedarfs. Möglicherweise müssen Sie auf die im Lieferungsunternehmen auszuführende Produktprogrammplanung warten, bevor Sie hier zusätzliche Informationen sehen.
