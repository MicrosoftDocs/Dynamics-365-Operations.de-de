# <a name="planned-orders-simplified"></a>Geplante Aufträge vereinfacht

[!include [banner](../../includes/banner.md)]

Dieses Thema beschreibt, wie Sie über die Seite **Geplante Aufträge vereinfacht** geplante Aufträge anzeigen, verwalten und fixieren können.

Die Seite **Geplante Aufträge vereinfacht** ist eine vereinfachte Version der Seite **Geplante Aufträge**. Es bietet weniger Optionen, ist aber schneller zu handhaben, wenn Sie Ihre geplanter Aufträge lediglich überprüfen, genehmigen und festschreiben müssen.

## <a name="turn-on-this-feature-for-your-system"></a>Schalten Sie diese Funktion für Ihr System ein

Wenn Ihr System noch nicht über die in diesem Thema beschriebenen Funktionen verfügt, gehen Sie zu [Funktionsverwaltung](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) und aktivieren Sie die Funktion *Geplante Aufträge vereinfacht*.

## <a name="view-manage-and-firm-planned-orders"></a>Geplante Aufträge anzeigen, verwalten und umwandeln

Die Seite **Geplante Aufträge vereinfacht** ist in erster Linie für die Funktionalität der Umwandlung gedacht. Sie zeigt alle geplanter Aufträge an, die für den ausgewählten Plan erstellt wurden, genau wie die Standardseite **Geplante Aufträge**. Daher kann sie als optimierte Alternative zur Anzeige der Ergebnisse einer Produktprogrammplanung ausgeführt werden.

Um die Seite **Geplante Aufträge vereinfacht** zu öffnen, gehen Sie auf **Masterplanung \> Masterplanung \> Geplante Aufträge vereinfacht**. Auf dieser Seite können Sie die folgenden Aufgaben erledigen:

- Alle geplanten Aufträge für einen bestimmten Plan anzeigen.
- Erstellen Sie neue geplante Aufträge.
- Löschen Sie einzelne oder mehrere geplante Aufträge.
- Einzelne oder mehrere geplanter Aufträge umwandeln.
- Ändern Sie den Status des geplanten Auftrags für einzelne oder mehrere geplante Aufträge.
- Öffnen Sie nahtlos die Details des geplanten Auftrags für einen einzelnen geplanten Auftrag, um alle Details des Auftrags anzuzeigen.
- Erstellen Sie angepasste Ansichten mit Hilfe von gespeicherten Ansichten.

Zusätzliche Szenarien, wie z.B. Split, werden von der Standardseite **Geplante Aufträge** unterstützt.
