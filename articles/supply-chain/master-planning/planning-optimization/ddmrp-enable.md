# <a name="turn-on-the-ddmrp-feature-for-your-system"></a>Aktivieren Sie die DDMRP Funktion für Ihr System

[!include [banner](../../includes/banner.md)]
[!INCLUDE [preview-banner](../../includes/preview-banner.md)]

Bevor Sie die Funktion der bedarfsgesteuerten Materialbedarfsplanung (DDMRP) nutzen können, müssen diese und die zugehörige Funktion für Ihr System aktiviert werden. Admins können die [Funktionsverwaltung](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) verwenden, um den Status dieser Funktionen zu überprüfen und sie zu aktivieren. Schalten Sie die folgenden Funktionen in der folgenden Reihenfolge ein:

1. **Funktionsname:** *Prioritätsgesteuerte MRP-Unterstützung für die Planungsoptimierung*
1. **Funktionsname:** *(Vorschau) DDMRP für Planungsoptimierung*

Weitere Informationen finden Sie unter [Prioritätenbasierte Planung](priority-based-planning.md).
