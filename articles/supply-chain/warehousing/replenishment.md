# <a name="replenishment-overview"></a><span data-ttu-id="264ec-103">Nachschubübersicht</span><span class="sxs-lookup"><span data-stu-id="264ec-103">Replenishment overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="264ec-104">Dieser Artikel beschreibt die Wiederbeschaffungsstrategien, die für Lagerorte, die die Funktionen der Lagerortverwaltung verwenden, verfügbar sind.</span><span class="sxs-lookup"><span data-stu-id="264ec-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="264ec-105">Die Informationen in diesem Artikel gelten nicht für die Lagerort-Lösung, die in der Lagerverwaltung verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="264ec-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

<span data-ttu-id="264ec-106">Die folgenden Wiederbeschaffungsstrategien sind verfügbar:</span><span class="sxs-lookup"><span data-stu-id="264ec-106">The following replenishment strategies are available:</span></span>

- <span data-ttu-id="264ec-107">**Wellenbedarfsbeschaffung** – Diese Strategie erstellt Wiederbeschaffungsarbeit für ausgehende Aufträge oder Ladungen, sollte kein Bestand verfügbar sein, wenn Arbeit durch die Welle erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="264ec-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="264ec-108">Beispielsweise kann Wiederbeschaffungsarbeit erstellt werden, sollte die für einen Auftrag erforderliche Menge nicht verfügbar sein, wenn eine Welle verarbeitet wird.</span><span class="sxs-lookup"><span data-stu-id="264ec-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="264ec-109">**Min./Max.-Auffüllung** – Diese Strategie verwendet minimale und maximale Lagergrenzen, um zu bestimmen, wann Lagerplätze aufgefüllt werden müssen.</span><span class="sxs-lookup"><span data-stu-id="264ec-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="264ec-110">Die Artikel- und Lagerplatzkriterien definieren den Lagerbestand, der für die Auffüllung ausgewertet wird.</span><span class="sxs-lookup"><span data-stu-id="264ec-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="264ec-111">Vorlagen für Min./Max.-Auffüllung sind der Hauptmechanismus zur Verwaltung optimaler Level in Entnahmeorten.</span><span class="sxs-lookup"><span data-stu-id="264ec-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="264ec-112">Um sicherzustellen, dass genügend Komissionierlagerbestand für den Serienbedarf verfügbar ist, können Sie die Bedarfsauffüllung als Ergänzung zwischen Min./Max.-Auffüllungszyklen verwenden.</span><span class="sxs-lookup"><span data-stu-id="264ec-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="264ec-113">**Ladungsbedarfs-Wiederbeschaffung** – Diese Strategie summiert den Bedarf für verschiedene Ladungen und erstellt Wiederbeschaffungsarbeit, die erforderlich ist, um die relevanten Entnahmeorte aufzufüllen.</span><span class="sxs-lookup"><span data-stu-id="264ec-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="264ec-114">Diese Strategie hilft sicherzustellen, dass die Ladungen, die erstellt werden, nach der Freigabe dem Lager entnommen werden können.</span><span class="sxs-lookup"><span data-stu-id="264ec-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="264ec-115">**Sofortige Wiederbeschaffung** – Diese Strategie füllt Bestand auf, bevor eine Welle ausgeführt wird, wenn die Zuteilung für eine Richtlinienposition mit einer Wiederbeschaffungsvorlage fehlschlägt.</span><span class="sxs-lookup"><span data-stu-id="264ec-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="264ec-116">Alle vier Strategien erstellen Wiederbeschaffungsarbeit, basierend auf einer Wiederbeschaffungsvorlage.</span><span class="sxs-lookup"><span data-stu-id="264ec-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="264ec-117">Wellenbedarfsbeschaffung</span><span class="sxs-lookup"><span data-stu-id="264ec-117">Wave demand replenishment</span></span>
<span data-ttu-id="264ec-118">Eine Wellenbedarfswiederbeschaffung erstellt Wiederbeschaffungsarbeit basierend auf dem Bedarf, sollte die notwendige Menge für Produktionsaufträge, Kanbans, ausgehende Aufträge oder Ladungen während der Arbeitserstellung einer Welle nicht verfügbar sein.</span><span class="sxs-lookup"><span data-stu-id="264ec-118">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders, or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="264ec-119">Die Wiederbeschaffungsvorlage enthält Informationen zu den Kriterien von Artikel, der Maßeinheit, der Bedarfserhöhung und dem Lagerplatz.</span><span class="sxs-lookup"><span data-stu-id="264ec-119">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="264ec-120">Lagerplatzrichtlinien dienen zum Bestimmen, welcher Lagerort wieder aufgefüllt werden soll.</span><span class="sxs-lookup"><span data-stu-id="264ec-120">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="264ec-121">Verknüpfen Sie diese Lagerplatzrichtlinien mit der Wiederbeschaffungsvorlage mithilfe des Feldes **Richtliniencode**.</span><span class="sxs-lookup"><span data-stu-id="264ec-121">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="264ec-122">Wenn das Feld **Richtliniencode** nicht festgelegt ist, werden Abfragen verwendet, um zu bestimmen, welche Lagerplatzrichtlinie verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="264ec-122">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="264ec-123">Beachten Sie, wenn eine Richtlinie nicht in der Wiederbeschaffungsvorlage angegeben ist und die Lagerplatzrichtlinie einen Richtliniencode besitzt, wird die Lagerplatzrichtlinie ignoriert, auch wenn die Abfrage in der Lagerplatzrichtlinie korrekt ist.</span><span class="sxs-lookup"><span data-stu-id="264ec-123">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="264ec-124">Entnahmeortrichtlinien werden verwendet, um zu bestimmen, wo der Bestand für die Wiederbeschaffung entnommen werden soll.</span><span class="sxs-lookup"><span data-stu-id="264ec-124">Pick location directives are used to determine where to get inventory for the replenishment.</span></span>

<span data-ttu-id="264ec-133">**Nachschubeinheit** ist die minimale Einheit, die Sie auffüllen müssen.</span><span class="sxs-lookup"><span data-stu-id="264ec-133">**Replenishment unit** is the minimum unit to replenish.</span></span> <span data-ttu-id="264ec-134">Dies muss eine ganze Zahl sein, die ein Vielfaches der Einheit ist.</span><span class="sxs-lookup"><span data-stu-id="264ec-134">This must be a whole number that is a multiple of the unit.</span></span> <span data-ttu-id="264ec-135">Das System rundet beim Anlegen von Arbeiten auf die höchstmögliche Einheit auf.</span><span class="sxs-lookup"><span data-stu-id="264ec-135">The system will round up to the highest unit possible when creating work.</span></span>

<span data-ttu-id="264ec-136">Bedarfswiederbeschaffung wird für Aufträge, Umlagerungsaufträge, Produktionsaufträge und Kanbans unterstützt.</span><span class="sxs-lookup"><span data-stu-id="264ec-136">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="264ec-137">Min./Max.-Auffüllung</span><span class="sxs-lookup"><span data-stu-id="264ec-137">Min/Max replenishment</span></span>

<span data-ttu-id="264ec-140">Die Mindest- und Höchstbeträge sind in einer Wiederbeschaffungsvorlage festgelegt.</span><span class="sxs-lookup"><span data-stu-id="264ec-140">The minimum and maximum amounts are set in a replenishment template.</span></span> <span data-ttu-id="264ec-141">Viele andere Einstellungen in der Vorlage ähneln den Einstellungen in Vorlagen, die in der Wellenbedarfsbeschaffung verwendeten werden.</span><span class="sxs-lookup"><span data-stu-id="264ec-141">Many of the other settings in the template resemble the settings in templates that are used in Wave demand replenishment.</span></span> <span data-ttu-id="264ec-142">Die Vorlage sollte eine Position für jeden Artikel und Lagerplatz enthalten.</span><span class="sxs-lookup"><span data-stu-id="264ec-142">The template should contain one line for each item and location.</span></span> <span data-ttu-id="264ec-143">Wenn Sie die Wiederbeschaffung über einen Stapelverarbeitungsauftrag ausführen, bewertet das System, ob die Wiederbeschaffung in der Reihenfolge erforderlich ist, in der die Positionen zusammengefasst sind.</span><span class="sxs-lookup"><span data-stu-id="264ec-143">When you run replenishment by using the batch job, the system evaluates whether replenishment is required by using the sequence that the lines are organized in.</span></span>
