<span data-ttu-id="d5e9b-114">Sie können eine **Arbeitsvorlage** einem Zykluszählplan zuordnen, um zu definieren, wie die Zykluszählarbeit erstellt werden soll.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-114">You can associate a **work template** with a cycle count plan to define how the cycle count work should be created.</span></span> <span data-ttu-id="d5e9b-115">Auf die Arbeitsvorlage für Zählvorgänge wird direkt vom Zykluszählplan verwiesen.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-115">The work template for counting operations is directly referenced from the cycle count plan.</span></span> 

<span data-ttu-id="d5e9b-116">Wenn Sie die Arbeitsvorlagendetails definieren, können Sie die Option **Arbeitspositionsumbrüche** verwenden, um anzugeben, ob die Inventurarbeitspositionen nach Positionsnummer oder Produktvariantennummer gruppiert werden müssen.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-116">When you define the work template details, you can use the **Work line breaks** option to specify whether the counting work lines must be grouped by item number or product variant number.</span></span> <span data-ttu-id="d5e9b-117">Diese Einstellung ist erforderlich, wenn Sie verfügbare Lagerbestände nur für bestimmte Produkte an einem Lagerplatz zählen möchten.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-117">This setup is required if you want to count on-hand inventory only for specific products in a location.</span></span> <span data-ttu-id="d5e9b-118">Die Zykluszählungsarbeitspositionen, die erstellt werden, bekommen die Informationsebene, die Sie hier definieren. <span data-ttu-id="d5e9b-119">Wenn Sie Zykluszählpläne mit Arbeitsvorlagen verknüpfen, indem Sie die Option **Arbeitspositionsumbrüche** verwenden, ist das Feld **Teilweise Zykluszählung** für die Zykluszählungsarbeit ausgewählt, die erstellt wird, und mehrere Zykluszählungsarbeitspositionen werden auf Basis der Definition der Arbeitsvorlage erstellt.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-119">If you associate cycle count plans with work templates by using the **Work lines breaks** option, the **Partial cycle count** field is selected for the cycle counting work that is created, and multiple cycle counting work lines will be created based on the definition of the work template.</span></span> 

<span data-ttu-id="d5e9b-120">Bevor teilweise Zykluszählungsarbeit verarbeitet werden kann, müssen Sie zumindest **Artikelnummer anzeigen** für das Menüelement des mobilen Geräts als Teil des Zykluszählungssetups auswählen.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-120">Before partial cycle count work can be processed, you must, at a minimum, select **Display item number** for the mobile device menu item as part of the cycle counting setup.</span></span> <span data-ttu-id="d5e9b-121">Der Lagerortoperator wird aufgefordert, nur Zähldaten zu erfassen, die zu den Zählpositionen (Positionsnummern und Produktdimensionen) gehören.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-121">The warehouse operator will be asked to record only counting information that is related to the counting lines (item numbers and product dimensions).</span></span> <span data-ttu-id="d5e9b-122">Der gesamte andere verfügbare Bestand wird für diesen Zählungsprozess ignoriert.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-122">All other on-hand inventory will be ignored for this counting process.</span></span> 

<span data-ttu-id="d5e9b-123">Für den Teilzykluszählprozess werden das Datum und die Uhrzeit der **Zählung beim letzten Zyklus** für den Standort nicht aktualisiert, obwohl alle an einem bestimmten Standort vorhandenen Artikel gezählt werden.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-123">For the partial cycle count process, the **Last cycle count** date/time won’t be updated for the location, even though all the items on hand at a given location are counted.</span></span> <span data-ttu-id="d5e9b-124">Die Teilzykluszählung berücksichtigt den Parameter **Tage zwischen der Zykluszählung** auf der **Zykluszählpläne** -Seite nicht.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-124">The partial cycle count doesn't consider the parameter **Days between cycle counting** on  the **Cycle count plans** page.</span></span> <span data-ttu-id="d5e9b-125">Die Teilzykluszählung unterstützt nicht die gleichzeitige Zählung mehrerer Elemente am selben Lagerort.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-125">Partial cycle count doesn't support simultaneous counting of multiple items at the same location.</span></span> <span data-ttu-id="d5e9b-126">Die Teilzykluszählfunktion kann dazu führen, dass derselbe Lagerort für einen Artikel mehrmals gezählt wird, wenn der **Prozesszykluszählungsplan** ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-126">Partial cycle count functionality may result in the same location being counted multiple times for an item when **Process cycle counting plan** is run.</span></span> <span data-ttu-id="d5e9b-127">Um dieses Szenario zu vermeiden, geben Sie Filter im Feld **Lagerorte auswählen** an.</span><span class="sxs-lookup"><span data-stu-id="d5e9b-127">To avoid that scenario, specify filters in the **Select locations** field.</span></span>
