# <a name="troubleshoot-warehouse-setup"></a>Fehlerbehebung bei der Einrichtung von Lagerorten

[!include [banner](../includes/banner.md)]

Dieses Thema beschreibt, wie Sie allgemeine Probleme beheben können, die beim Festlegen von Lagerorten in Microsoft Dynamics 365 Supply Chain Management auftreten können.

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>Ich kann keine Rolle außer Administrator verwenden, um auf den App-Emulator für mobile Geräte zuzugreifen.

### <a name="issue-description"></a>Problembeschreibung

Sie können keine andere Rolle als die des Administrators verwenden, um auf den Emulator für Apps für mobile Geräte zuzugreifen.

### <a name="issue-resolution"></a>Problemlösung

Der App-Emulator für mobile Geräte ist so festgelegt, dass er nur mit dem Administratorkonto funktioniert. Für alle Test- und Live-Prozesszwecke empfehlen wir, die Lagerort App selbst zu verwenden.
