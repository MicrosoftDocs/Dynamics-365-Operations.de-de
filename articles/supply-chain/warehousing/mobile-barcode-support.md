# <a name="mobile-bar-code-support"></a><span data-ttu-id="a428e-103">Strichcodeunterstützung für Mobilgeräte</span><span class="sxs-lookup"><span data-stu-id="a428e-103">Mobile bar code support</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a428e-104">Da Android ein Open Source-Projekt ist, kann jeder Hersteller von Hardware für Lager-Strichcode-Scanner ein Gerät erstellen, um das Android-Betriebssystem auszuführen.</span><span class="sxs-lookup"><span data-stu-id="a428e-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="a428e-105">Ein Gerät ist nur Android-kompatibel, wenn es Apps ausführen kann, die für die Android-Ausführungsumgebung geschrieben wurden.</span><span class="sxs-lookup"><span data-stu-id="a428e-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="a428e-106">Allerdings kann ein Hardwareanbieter Overlays für die Android-Version ändern und erstellen, die auf der Hardware ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="a428e-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="a428e-107">Microsoft kann keine Verantwortung übernehmen, um sicherzustellen, dass eine mobile Strichcodescan-App für Android mit der Strichcode-Scan-Hardware des Herstellers und der Android-Version kompatibel ist, die auf dieser ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="a428e-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span> 

<span data-ttu-id="a428e-108">Die Lagerort-App für Microsoft Dynamics 365 for Finance and Operations ist mit einer Auswahl von Android-aktivierten Geräten für Scannen von Strichcodes getestet worden.</span><span class="sxs-lookup"><span data-stu-id="a428e-108">The Warehousing app for Microsoft Dynamics 365 for Finance and Operations has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="a428e-109">Diese Tests decken nur eine Auswahl der Geräte ab, die auf dem Markt verfügbar sind.</span><span class="sxs-lookup"><span data-stu-id="a428e-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="a428e-110">Ihnen als Kunden wird empfohlen, die mobile Strichcode-Scan-App auf ausgewählter Hardware zu testen, bevor Sie entscheiden, welche Hardware Sie kaufen möchten.</span><span class="sxs-lookup"><span data-stu-id="a428e-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
