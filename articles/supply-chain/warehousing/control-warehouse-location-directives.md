## <a name="work-templates"></a><span data-ttu-id="982ef-108">Arbeitsvorlagen</span><span class="sxs-lookup"><span data-stu-id="982ef-108">Work templates</span></span>

<span data-ttu-id="982ef-124">Sie können eine Abfrage einrichten, um zu steuern, wann eine bestimmte Arbeitsvorlage verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="982ef-124">You can <span data-ttu-id="982ef-130">Um einen Arbeitsprozess zu beenden oder anzuhalten, können Sie die Einstellung **Arbeit anhalten** für die Arbeitsposition verwenden.</span><span class="sxs-lookup"><span data-stu-id="982ef-130">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="982ef-131">In diesem Fall wird die Arbeitskraft, die die Arbeit ausführt, aufgefordert, den folgenden Arbeitspositionsschritt nicht auszuführen.</span><span class="sxs-lookup"><span data-stu-id="982ef-131">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="982ef-132">Um mit dem nächsten Schritt fortzufahren, muss diese Arbeitskraft oder eine andere Arbeitskraft für die Arbeit wieder ausgewählt werden.</span><span class="sxs-lookup"><span data-stu-id="982ef-132">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="982ef-133">Sie können die Aufgaben innerhalb einer Arbeit auch trennen, indem Sie ein andere *Arbeitsklassen-ID* für die Arbeitsvorlagenpositionen verwenden.</span><span class="sxs-lookup"><span data-stu-id="982ef-133">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="982ef-134">Lagerplatzrichtlinien</span><span class="sxs-lookup"><span data-stu-id="982ef-134">Location directives</span></span>

<span data-ttu-id="982ef-135">Lagerplatzrichtlinien sind Regeln, die dabei helfen, Entnahme- und Einlagerungslagerorte für die Lagerumlagerung zu identifizieren.</span><span class="sxs-lookup"><span data-stu-id="982ef-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="982ef-136">In einer Auftragsbuchung bestimmt eine Lagerplatzrichtlinie z. <span data-ttu-id="982ef-138">Im Kopf muss jede Lagerplatzrichtlinie einem *Arbeitsauftragstyp* zugeordnet werden, der den Typ der Lagerbuchung angibt, für den die Richtlinie verwendet wird, wie Aufträge, Auffüllung oder Rohmaterialentnahme.</span><span class="sxs-lookup"><span data-stu-id="982ef-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="982ef-139">Der *Arbeitstyp* gibt an, ob die Lagerplatzrichtlinie für die Entnahme oder die Lagerung der Arbeit verwendet wird oder für einen anderen Lagerortprozess, wie Inventur oder Bestandsstatusänderungen.</span><span class="sxs-lookup"><span data-stu-id="982ef-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="982ef-140">Sie müssen auch einen *Standort* und einen *Lagerort* angeben.</span><span class="sxs-lookup"><span data-stu-id="982ef-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="982ef-141">Ein *Richtliniencode*, den Sie im Kopf angeben, kann verwendet werden, um die Lagerplatzrichtlinie an eine oder mehrere Arbeitsvorlagen zu knüpfen.</span><span class="sxs-lookup"><span data-stu-id="982ef-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="982ef-145">Die richtungweisenden Positionen des Lagerplatzes setzen zusätzliche Einschränkungen für die Anwendung für die Regel für das Auffinden des Lagerplatzes fest.</span><span class="sxs-lookup"><span data-stu-id="982ef-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="982ef-146">Sie können eine minimale Menge und eine maximale Menge angeben, für die die Richtlinie gelten soll, und Sie können angeben, dass die Richtlinie für eine bestimmte Lagereinheit sein soll.</span><span class="sxs-lookup"><span data-stu-id="982ef-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="982ef-147">Wenn beispielsweise die Maßeinheit Paletten ist, können Artikel auf Paletten an einem bestimmten Lagerplatz eingelagert werden.</span><span class="sxs-lookup"><span data-stu-id="982ef-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="982ef-148">Sie können auch angeben, ob die Menge über mehrere Lagerplätze verteilt werden kann.</span><span class="sxs-lookup"><span data-stu-id="982ef-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="982ef-149">Wie der richtungweisende Kopf des Lagerplatzes hat jede richtungweisende Position des Lagerplatzes eine laufende Nummer, die die Reihenfolge bestimmt, in der die Positionen bewertet werden.</span><span class="sxs-lookup"><span data-stu-id="982ef-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>
