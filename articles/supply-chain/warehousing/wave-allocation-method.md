# <a name="wave-allocation"></a><span data-ttu-id="8bf9d-103">Wellenzuteilung</span><span class="sxs-lookup"><span data-stu-id="8bf9d-103">Wave allocation</span></span>

<span data-ttu-id="8bf9d-105">Diese Schritte können jetzt parallel ausgeführt werden, wodurch die Leistung der Wellenverarbeitung verbessert und ein größerer Wellendurchsatz im selben Lagerort ermöglicht wird.</span><span class="sxs-lookup"><span data-stu-id="8bf9d-105">It is now possible to run each of these steps in parallel, which can improve the performance of the wave processing, and allow for a larger throughput of waves in the same warehouse.</span></span> <span data-ttu-id="8bf9d-106">In diesem Thema wird erklärt, wie Sie die Wellenzuteilungsmethode für eine parallele Ausführung einrichten.</span><span class="sxs-lookup"><span data-stu-id="8bf9d-106">This topic explains how to set up the wave allocation method to run in parallel.</span></span> <span data-ttu-id="8bf9d-107">Weitere Informationen zum Einrichten der parallelen Ausführung der Arbeitserstellung finden Sie unter [Planen der Arbeitserstellung während der Welle](configure-wave-schedule-work-creation.md).</span><span class="sxs-lookup"><span data-stu-id="8bf9d-107">For more information about how to set up work creation to run in parallel, see [Schedule work creation during wave](configure-wave-schedule-work-creation.md).</span></span>

## <a name="performance-improvements"></a><span data-ttu-id="8bf9d-115">Leistungsverbesserungen</span><span class="sxs-lookup"><span data-stu-id="8bf9d-115">Performance improvements</span></span>

- <span data-ttu-id="8bf9d-117">**Verbesserter Durchsatz** – Der Durchsatz von Wellen verbessert sich normalerweise auch dann, wenn die Parallelverarbeitung nicht konfiguriert ist, insbesondere in Szenarien, in denen innerhalb der Wellen keine Artikel überlappen.</span><span class="sxs-lookup"><span data-stu-id="8bf9d-117">**Improved throughput** - The throughput of waves will typically improve even if parallel processing isn't configured, especially for scenarios where there is no overlap of items within the waves.</span></span>
