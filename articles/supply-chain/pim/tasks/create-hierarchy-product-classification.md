# <a name="create-a-hierarchy-of-product-classification"></a>Eine Hierarchie zur Produktklassifizierung erstellen

[!include [banner](../../includes/banner.md)]

Im folgenden Verfahren, wie eine neue Kategoriehierarchie erstellt und einen Warencodehierarchietyp zuweist. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF. Diese Prozedur ist für die Kategorie-Lagerverwaltung vorgesehen.


## <a name="create-the-new-category-hierarchy"></a>Neue Kategoriehierarchie erstellen
1. Wechseln Sie zu **Navigationsbereich > Module > Produktinformationsverwaltung > Einstellungen > Kategorien und Attribute > Kategoriehierarchien**.
2. Klicken Sie auf **Neu**.
3. Geben Sie im Feld **Name** einen Wert ein.
4. Geben Sie im Feld **Beschreibung** einen Wert ein.
5. Klicken Sie auf **Erstellen**.

## <a name="build-the-hierarchy"></a>Erstellen Sie die Hierarchie an
1. Klicken Sie auf **Neuer Kategorieknoten**.
2. Geben Sie im Feld **Name** einen Wert ein.
3. Geben Sie im Feld **Code** einen Wert ein.
4. Geben Sie im Feld **Anzeigename** einen Wert ein.
5. Klicken Sie auf **Neuer Kategorieknoten**.
6. Geben Sie im Feld **Name** einen Wert ein.
7. Geben Sie im Feld **Code** einen Wert ein.
8. Geben Sie im Feld **Anzeigename** einen Wert ein.
9. Klicken Sie auf **Neuer Kategorieknoten**.
10. Geben Sie im Feld **Name** einen Wert ein.
11. Geben Sie im Feld **Code** einen Wert ein.
12. Geben Sie im Feld **Anzeigename** einen Wert ein.
13. Klicken Sie auf **Neuer Kategorieknoten**.
14. Geben Sie im Feld **Name** einen Wert ein.
15. Geben Sie im Feld **Code** einen Wert ein.
16. Geben Sie im Feld **Anzeigename** einen Wert ein.
17. Schließen Sie die Seite.

## <a name="classify-the-hierarchy"></a>Klassifizieren Sie die Hierarchie
1. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
2. Klicken Sie im **Aktivitätsbereich** auf **Kategoriehierarchie**.
3. Klicken Sie auf **Hierarchietyp zuordnen**.
4. Klicken Sie auf **Neu**.
5. Wählen Sie im Feld **Kategoriehierarchietyp** eine Option aus. Wählen Sie den **Warencodekategoriehierarchietyp für Produktklassifizierung** aus.  
6. Klicken Sie im Feld **Kategoriehierarchie** auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
7. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
8. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
9. Schließen Sie die Seite.
