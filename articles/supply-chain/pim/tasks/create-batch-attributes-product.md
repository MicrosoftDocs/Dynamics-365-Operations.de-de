# <a name="create-batch-attributes-for-a-product"></a>Chargenattribute für ein Produkt erstellen

[!include [banner](../../includes/banner.md)]

Im folgenden Verfahren wird dargestellt, wie ein Stapelverarbeitungsattribut erstellt, Standardwertbereiche zugeordnet und das Attribut in eine Gruppe einbezogen wird. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist das Unternehmen USP2.

1. Wechseln Sie zu "Lagerverwaltung" > "Einrichten" > "Charge" > "Stapelverarbeitungsattribut".
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Attribut" einen Wert ein.
4. Geben Sie im Feld "Beschreibung" einen Wert ein.
5. Wählen Sie im Feld "Attribut" "Bruch" aus.
    * Bei diesem Verfahren wird der Bruchtyp verwendet, um Dezimalwerte zu aktivieren. Sie können andere Attributtypen auswählen. Wenn Sie den Enumerationstyp auswählen, müssen Sie Werte in der Enumerationsliste eingeben, bevor Sie einen Wert in das Feld "Ziel" eingeben können.  
6. Geben Sie im Feld "Minimum" eine Zahl ein.
7. Geben Sie im Feld "Maximum" eine Zahl ein.
8. Geben Sie im Feld "Inkrement" eine Zahl ein.
9. Geben Sie im Feld "Ziel" einen Wert ein.
10. Klicken Sie auf "Speichern".
11. Schließen Sie die Seite.
12. Wechseln Sie zu "Lagerverwaltung" > "Einrichten" > "Charge" > "Stapelverarbeitungsattributgruppen".
13. Klicken Sie auf "Neu".
14. Geben Sie im Feld "Attributgruppe" einen Wert ein.
15. Geben Sie im Feld "Beschreibung" einen Wert ein.
16. Klicken Sie auf "Speichern".
17. Klicken Sie auf Gruppenattribute.
18. Klicken Sie auf "Neu".
19. Klicken Sie im Feld "Attribut" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
20. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
21. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
    * Ein Attribut kann in eine der Gruppen einbezogen werden.  
22. Klicken Sie auf "Speichern".
23. Schließen Sie die Seite.
