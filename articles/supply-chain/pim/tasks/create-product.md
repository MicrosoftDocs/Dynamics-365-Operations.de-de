# <a name="create-a-product"></a>Ein Produkt erstellen

[!include [task guide banner](../../includes/task-guide-banner.md)]

Diese Prozedur führt Sie Schritt für Schritt durch das Erstellen eines neuen Produkts in der Produkthierarchie (Einzelhandel). Das Demodatenunternehmen, das verwendet wird, um diese Aufgabe erstellen, ist USRT, und die Aufgabe ist für den "Manager Verkaufsförderung" und "Produkt-Designer" bestimmt.

1. Gehen Sie zu Einzelhandel und Handel >... > Produkte nach Kategorie.
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Produktname" einen Namen für das Produkt ein.
4. Klicken Sie im Feld "Einzelhandelskategorie" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
5. Erweitern Sie in der Struktur "ALLES (ALLES)\Mode (Mode)".
6. Wählen Sie in der Struktur "ALLES (ALLES)\Mode (Mode)\Modeaccessoires (Modeaccessoires)" aus.
7. Klicken Sie auf "OK".
8. Geben Sie im Feld "Produktnummer" einen Wert ein.
9. Klicken Sie auf "OK".
