# <a name="create-a-bar-code-for-a-product"></a>Strichcode für ein Produkt erstellen

[!include [task guide banner](../../includes/task-guide-banner.md)]

Dieses Verfahren zeigt, wie Sie manuell einen Strichcode mithilfe der Artikelnummer M0001 als Beispiel erstellen. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.

1. Klicken Sie auf "Freigegebene Produktverwaltung".
2. Klicken Sie auf "Freigegebene Produkte".
3. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
4. Klicken Sie im Aktivitätsbereich auf "Lagerbestand verwalten".
5. Klicken Sie auf "Strichcodes".
6. Klicken Sie auf "Neu".
7. Markieren Sie in der Liste die ausgewählte Zeile.
8. Geben Sie im Feld 'Strichcodeeinstellung' einen Wert ein, oder wählen Sie einen Wert aus.
9. Geben Sie im Feld "Strichcode" einen Wert ein, oder wählen Sie einen Wert aus.
10. Geben Sie im Feld "Strichcode" einen Wert ein.
    * Drücken Sie die TAB-Taste.  
11. Schließen Sie die Seite.
12. Geben Sie im Feld "Menge" eine Zahl ein.
13. Klicken Sie auf "Speichern".
    * Wenn Sie im auf "Speichern" klicken, wird der Strichcodescheck ausgeführt, und in diesem Fall enthält er einen Fehler. Die erwartete Prüfziffer ist 8. Es wurde jedoch 3 gefunden. Aktualisieren Sie die Strichcodenummer manuell so, das am Ende 8 ist.  
14. Geben Sie im Feld "Strichcode" einen Wert ein, oder wählen Sie einen Wert aus.
15. Geben Sie im Feld "Strichcode" einen Wert ein.
    * Drücken Sie die TAB-Taste.  
16. Schließen Sie die Seite.
17. Klicken Sie auf "Speichern".
18. Schließen Sie die Seite.
