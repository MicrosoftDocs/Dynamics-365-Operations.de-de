[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c456a-104">In diesem Thema wird beschrieben, wie Sie manuell einen Strichcode mithilfe der Artikelnummer M0001 als Beispiel erstellen.</span><span class="sxs-lookup"><span data-stu-id="c456a-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="c456a-105">Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.</span><span class="sxs-lookup"><span data-stu-id="c456a-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="c456a-106">Wählen Sie **Freigegebene Produktverwaltung** auf der Startseite aus.</span><span class="sxs-lookup"><span data-stu-id="c456a-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="c456a-107">Wechseln Sie zu **Produkte > Freigegebene Produkte** unter dem Abschnitt **Links**.</span><span class="sxs-lookup"><span data-stu-id="c456a-107">Go to **Products > Released products** under the **Links** section.</span></span>

4. <span data-ttu-id="c456a-110">Wählen Sie im Aktivitätsbereich **Lagerbestand verwalten** aus.</span><span class="sxs-lookup"><span data-stu-id="c456a-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="c456a-111">Wählen Sie **Strichcodes** aus.</span><span class="sxs-lookup"><span data-stu-id="c456a-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="c456a-112">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="c456a-112">Select **New**.</span></span>
7. <span data-ttu-id="c456a-113">Markieren Sie die ausgewählte Zeile, die unten in der Liste erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="c456a-113">Mark the selected row that is created in the list below.</span></span>

9. <span data-ttu-id="c456a-115">Geben Sie im Feld **Strichcode** einen Wert ein, oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="c456a-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="c456a-116">Geben Sie im Feld **Strichcode** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="c456a-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="c456a-117">Geben Sie im Feld **Menge** eine Zahl ein.</span><span class="sxs-lookup"><span data-stu-id="c456a-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="c456a-118">Wählen Sie **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="c456a-118">Select **Save**.</span></span>
13. <span data-ttu-id="c456a-119">Schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="c456a-119">Close the page.</span></span>
