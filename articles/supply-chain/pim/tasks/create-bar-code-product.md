# <a name="create-a-bar-code-for-a-product"></a>Strichcode für ein Produkt erstellen

[!include [banner](../../includes/banner.md)]

In diesem Thema wird beschrieben, wie Sie manuell einen Strichcode mithilfe der Artikelnummer M0001 als Beispiel erstellen. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.

1. Wählen Sie **Freigegebene Produktverwaltung** auf der Startseite aus.
2. Wechseln Sie zu **Produkte > Freigegebene Produkte** unter dem Abschnitt **Links**.
3. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus. Wählen Sie in diesem Beispiel die Artikelnummer **M0001** aus.
4. Wählen Sie im Aktivitätsbereich **Lagerbestand verwalten** aus.
5. Wählen Sie **Strichcodes** aus.
6. Wählen Sie **Neu** aus.
7. Markieren Sie die ausgewählte Zeile, die unten in der Liste erstellt wird.
8. Geben Sie im Feld **Strichcodeeinstellung** einen Wert ein, oder wählen Sie einen Wert aus.
9. Geben Sie im Feld **Strichcode** einen Wert ein, oder wählen Sie einen Wert aus.
10. Geben Sie im Feld **Strichcode** einen Wert ein.  
11. Geben Sie im Feld **Menge** eine Zahl ein.
12. Wählen Sie **Speichern**.
13. Schließen Sie die Seite.
