# <a name="create-dimension-based-configurations"></a>Dimensionsbasierte Konfigurationen erstellen

[!include [banner](../../includes/banner.md)]

In der folgenden Prozedur wird gezeigt, wie eine Konfiguration für ein dimensionsbasiertes Produkt definiert wird. Dies ist die letzte Prozedur in der Reihe, die beschreibt, wie Kombinationen für eine dimensionsbasierte Konfiguration erstellt werden. Die Ausführung dieser Prozedur hängt von den Daten ab, die in den vorherigen sieben Aufzeichnungen erstellt wurden. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.

## <a name="find-the-dimension-based-product-master"></a>Suchen Sie den dimensionsbasierten Produktmaster

1. Wechseln Sie zu **Produktinformationsverwaltung \> Produkte \> Freigegebene Produkte**.
1. Markieren Sie in der Liste die ausgewählte Zeile.
    * Wählen Sie den dimensionsbasierten Produktmaster aus, den Sie in der ersten Aufzeichnung in dieser Reihenfolge von 8 Aufzeichnungen erstellt haben.  

## <a name="create-configurations"></a>Erstellen von Konfigurationen

1. Wählen Sie im Aktivitätsbereich **Technik** die Option **Konfigurationen beibehalten**.
1. Wählen Sie **Konfigurieren**.
1. Markieren Sie in der Liste die ausgewählte Zeile.
1. Geben Sie im Feld **Artikelnummer** einen Wert ein, oder wählen Sie einen Wert aus.
    * Wählen Sie beliebige der Artikel in der ersten Variantengruppe aus.  
1. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
1. Geben Sie im Feld **Artikelnummer** einen Wert ein, oder wählen Sie einen Wert aus.
    * Wählen Sie einen beliebigen Artikel aus der zweiten Variantengruppe aus.  
1. Wählen Sie **OK**.
1. Markieren Sie in der Liste die ausgewählte Zeile.
1. Geben Sie im Feld **Konfiguration** einen Wert ein.
    * Geben Sie einen Konfigurationsnamen ein, mit dem Sie einfach die Konfiguration identifizieren können.  
1. Geben Sie im Feld **Beschreibung** einen Wert ein.
    * Geben Sie eine Beschreibung für die Konfiguration ein, um zu erläutern, was sie enthält.  
1. Wählen Sie **OK**.
