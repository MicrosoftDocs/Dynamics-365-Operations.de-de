<span data-ttu-id="37779-104">In diesem Thema wird beschrieben, wie in Asset Management funktionalen Standorte erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="37779-104">This topic describes how to create functional location types in Asset Management.</span></span> <span data-ttu-id="37779-105">Funktionale Standorttypen werden verwendet, um die Anforderungen an funktionale Standorte zu verwalten. ## <a name="create-a-default-functional-location-type"></a><span data-ttu-id="37779-111">Erstellen eines standardmäßigen funktionalen Standorttyps</span><span class="sxs-lookup"><span data-stu-id="37779-111">Create a default functional location type</span></span>
