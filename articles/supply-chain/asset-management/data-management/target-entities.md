# <a name="target-entities"></a><span data-ttu-id="5557f-103">Zielentitäten</span><span class="sxs-lookup"><span data-stu-id="5557f-103">Target entities</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="5557f-104">In Arbeitsbereich **Datenverwaltung** können Sie sich einen Überblick über Zielentitäten, zugehörige Entitätstypen und Staging-Tabellen zum Modul **Anlagenmanagement** verschaffen.</span><span class="sxs-lookup"><span data-stu-id="5557f-104">In the **Data management** workspace, you can get an overview of target entities, related entity types and staging tables related to the **Asset management** module.</span></span>

2. <span data-ttu-id="5557f-106">Klicken Sie im Abschnitt **Import / Export** auf **Datenelemente**.</span><span class="sxs-lookup"><span data-stu-id="5557f-106">In the **Import / Export** section, click **Data entities**.</span></span> 

    ![Abbildung 1](media/01-data-management.png)

    <span data-ttu-id="5557f-109">Die folgende Abbildung zeigt einige der Anlagenmanagement-Einheiten.</span><span class="sxs-lookup"><span data-stu-id="5557f-109">The figure below shows some of the Asset management entities.</span></span>

   ![Abbildung 2](media/02-data-management.png)
