# <a name="active-work-order-maintenance-jobs-overview"></a>Aktive Wartungsaufträge für Arbeitsaufträge – Übersicht

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

Unter **Aktive Wartungsaufträge für Arbeitsaufträge** erhalten Sie einen Überblick über Arbeitsaufträge in Bezug auf wie viele Arbeitsaufträge für bestimmte Anlagen, Anlagentypen, Hersteller, Modelle, Wartungsauftragstypen usw. erstellt wurden. Wenn Sie einen Arbeitsauftragseinzelvorgang auswählen und auf **Bearbeiten** klicken, öffnen Sie den zugehörigen Arbeitsauftrag. Wenn Sie einen Einzelvorgang auswählen und dann auf eine der Schaltflächen auf der Registerkarte **Wartungsaufträge für Arbeitsaufträge** klicken, können Sie Daten für den Arbeitsauftrag anzeigen, dem der Arbeitsauftragseinzelvorgang zugeordnet ist.

Klicken Sie auf **Anlagenverwaltung** > **Allgemein** > **Arbeitsaufträge** > **Aktive Wartungsaufträge für Arbeitsaufträge**, um die Liste zu öffnen. Die Liste enthält alle aktiven Wartungsaufträge für Arbeitsaufträge und einige Informationen bezogen auf den Arbeitsauftrag oder Arbeitsauftragseinzelvorgang.

In der Spalte **%** gibt eine Zahl den Abschluss des Arbeitsauftrags in Prozent an. Der Abschluss basiert auf zwei Berechnungen: den gebuchten Stunden im Vergleich zu den geplanten Stunden und der Anzahl der abgeschlossenen Wartungsprüflisten.

![Abbildung 1](media/23-work-orders.png)

Eine Kurzbeschreibung der Schaltflächen in **Aktive Wartungsaufträge für Arbeitsaufträge**, finden Sie unter [Einführung in Arbeitsaufträge](../work-orders/introduction-to-work-orders.md), wo identische Schaltflächen beschrieben werden.
