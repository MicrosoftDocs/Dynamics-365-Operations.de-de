<span data-ttu-id="1cd7a-104">Unter **Aktive Wartungsaufträge für Arbeitsaufträge** erhalten Sie einen Überblick über Arbeitsaufträge in Bezug auf wie viele Arbeitsaufträge für bestimmte Anlagen, Anlagentypen, Hersteller, Modelle, Wartungsauftragstypen usw. erstellt wurden.</span><span class="sxs-lookup"><span data-stu-id="1cd7a-104">In **Active work order maintenance jobs**, you can get an overview of work orders in relation to how many work orders have been created on specific assets, asset types, manufacturers, models, maintenance job types, and so on.</span></span> <span data-ttu-id="1cd7a-105">Wenn Sie einen Arbeitsauftragseinzelvorgang auswählen und auf **Bearbeiten** klicken, öffnen Sie den zugehörigen Arbeitsauftrag.</span><span class="sxs-lookup"><span data-stu-id="1cd7a-105">If you select a work order job and click **Edit**, you open the related work order.</span></span> <span data-ttu-id="1cd7a-106">Wenn Sie einen Einzelvorgang auswählen und dann auf eine der Schaltflächen auf der Registerkarte **Wartungsaufträge für Arbeitsaufträge** klicken, können Sie Daten für den Arbeitsauftrag anzeigen, dem der Arbeitsauftragseinzelvorgang zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="1cd7a-106">If you select a job and click one of the buttons on the **Work order maintenance jobs** tab, you see data for the work order to which the work order job is related.</span></span>
