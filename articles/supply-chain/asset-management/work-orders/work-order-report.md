# <a name="work-order-report"></a>Arbeitsauftragsbericht


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


Sie können einen Arbeitsauftragsbericht erstellen, der detaillierte Informationen zu einem Arbeitsauftrag anzeigt. Es ist möglich, einen oder mehrere Arbeitsaufträge auszuwählen, die im Bericht angezeigt werden sollen.

1. Klicken Sie auf **Assetmanagement** > **Berichte** > **Arbeitsaufträge** > **Arbeitsauftragsbericht**.

2. Wählen Sie „Ja“/„Nein“ auf den Umschalttasten und füllen Sie bei Bedarf Felder aus, um die Details zu bestimmen, die in den Bericht aufgenommen werden sollen.  
  a. Im Abschnitt **Druckeinstellungen** können Sie auswählen, ob Sie Anlagen aus der zugehörigen Einrichtung des Wartungsauftragstyps in den Druck aufnehmen möchten.

3. Auf der Seite **Einschließende Datensätze** FastTab können Sie den Inhalt des Berichts nach **Arbeitsauftrag** filtern.

4. Bei Bedarf können Sie die Erstellung von Arbeitsauftragsberichten als Batch-Job einrichten, indem Sie die Felder auf dem Feld **Ausführen im Hintergrund** FastTab ausfüllen.

5. Klicken Sie auf **OK**, um den Bericht zu generieren.
