# <a name="work-order-report"></a>Arbeitsauftragsbericht

[!include [banner](../../includes/banner.md)]


Sie können einen Arbeitsauftragsbericht erstellen, der detaillierte Informationen zu mindestens einem ausgewählten Arbeitsauftrag anzeigt.

1. Wählen Sie **Anlagenverwaltung** > **Berichte** > **Arbeitsaufträge** > **Arbeitsauftragsbericht** aus.

2. Wählen Sie „Ja“/„Nein“ auf den Umschalttasten nach Bedarf, um die Details zu bestimmen, die in den Bericht aufgenommen werden sollen.  
    - Im Abschnitt **Druckeinstellungen** können Sie auswählen, ob Sie Anlagen aus der zugehörigen Einrichtung des Wartungsauftragstyps aufnehmen möchten.

3. Auf der Seite **Einschließende Datensätze** FastTab können Sie den Inhalt des Berichts nach **Arbeitsauftrag** filtern.

4. Im Inforegister **Im Hintergrund ausführen** können Sie die Berichterstellung des Arbeitsauftragsberichts als Batchauftrag einrichten.

5. Klicken Sie auf **OK**, um den Bericht zu generieren.
