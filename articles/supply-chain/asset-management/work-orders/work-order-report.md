[!include [banner](../../includes/banner.md)]


<span data-ttu-id="611f8-104">Sie können einen Arbeitsauftragsbericht erstellen, der detaillierte Informationen zu mindestens einem ausgewählten Arbeitsauftrag anzeigt.</span><span class="sxs-lookup"><span data-stu-id="611f8-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="611f8-105">Wählen Sie **Anlagenverwaltung** > **Berichte** > **Arbeitsaufträge** > **Arbeitsauftragsbericht** aus.</span><span class="sxs-lookup"><span data-stu-id="611f8-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="611f8-106">Wählen Sie „Ja“/„Nein“ auf den Umschalttasten nach Bedarf, um die Details zu bestimmen, die in den Bericht aufgenommen werden sollen.</span><span class="sxs-lookup"><span data-stu-id="611f8-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="611f8-107">Im Abschnitt **Druckeinstellungen** können Sie auswählen, ob Sie Anlagen aus der zugehörigen Einrichtung des Wartungsauftragstyps aufnehmen möchten.</span><span class="sxs-lookup"><span data-stu-id="611f8-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

5. <span data-ttu-id="611f8-110">Klicken Sie auf **OK**, um den Bericht zu generieren.</span><span class="sxs-lookup"><span data-stu-id="611f8-110">Select **OK** to generate the report.</span></span>
