# <a name="work-order-report"></a><span data-ttu-id="22e46-103">Arbeitsauftragsbericht</span><span class="sxs-lookup"><span data-stu-id="22e46-103">Work order report</span></span>

[!include [banner](../../includes/banner.md)]


<span data-ttu-id="22e46-104">Sie können einen Arbeitsauftragsbericht erstellen, der detaillierte Informationen zu mindestens einem ausgewählten Arbeitsauftrag anzeigt.</span><span class="sxs-lookup"><span data-stu-id="22e46-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="22e46-105">Wählen Sie **Anlagenverwaltung** > **Berichte** > **Arbeitsaufträge** > **Arbeitsauftragsbericht** aus.</span><span class="sxs-lookup"><span data-stu-id="22e46-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="22e46-106">Wählen Sie „Ja“/„Nein“ auf den Umschalttasten nach Bedarf, um die Details zu bestimmen, die in den Bericht aufgenommen werden sollen.</span><span class="sxs-lookup"><span data-stu-id="22e46-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="22e46-107">Im Abschnitt **Druckeinstellungen** können Sie auswählen, ob Sie Anlagen aus der zugehörigen Einrichtung des Wartungsauftragstyps aufnehmen möchten.</span><span class="sxs-lookup"><span data-stu-id="22e46-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="22e46-108">Auf der Seite **Einschließende Datensätze** FastTab können Sie den Inhalt des Berichts nach **Arbeitsauftrag** filtern.</span><span class="sxs-lookup"><span data-stu-id="22e46-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

4. <span data-ttu-id="22e46-109">Im Inforegister **Im Hintergrund ausführen** können Sie die Berichterstellung des Arbeitsauftragsberichts als Batchauftrag einrichten.</span><span class="sxs-lookup"><span data-stu-id="22e46-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>

5. <span data-ttu-id="22e46-110">Klicken Sie auf **OK**, um den Bericht zu generieren.</span><span class="sxs-lookup"><span data-stu-id="22e46-110">Select **OK** to generate the report.</span></span>
