# <a name="scheduled-work-order-maintenance-jobs"></a>Geplante Wartungsarbeitsaufträge

[!include [banner](../../includes/banner.md)]

 

Die Seite **Wartungsaufträge für planmäßige Arbeitsaufträge** dient dazu, sich einen Überblick über die einer Ressource zugeordneten Arbeitsaufträge zu verschaffen. Arbeitsaufträge mit den Ressourcentypen „Personal“, „Werkzeug“ und „Maschine“ werden angezeigt. Wenn sich z. B. ein Wartungsarbeiter krank gemeldet hat, können Sie diese Seite verwenden, um schnell Arbeitsaufträge zu finden, die dem Arbeiter zugeordnet sind, und dann einen anderen Wartungsmitarbeiter dem Auftrag zuzuordnen.

## <a name="view-scheduled-work-order-maintenance-jobs"></a>Geplante Wartungsarbeiten für Arbeitsaufträge anzeigen

1. Klicken Sie auf **Anlagenmanagement** > **Allgemein** > **Arbeitsaufträge** > **Planmäßige Wartungsarbeiten**. Sie sehen eine Liste aller Arbeitsaufträge, die sich im Lebenszykluszustand „Geplant“ oder „In Bearbeitung“ befinden.

2. Sie können die Liste z.B. nach Wartungspersonal sortieren. Sie können den Filter auch verwenden, um die Liste einzuschränken, um Arbeitsaufträge anzuzeigen, die einer bestimmten Ressource oder einem bestimmten Instandhalter zugeordnet sind.

3. Sie können Notizen zum Arbeitsauftrag sehen und bei Bedarf neue Notizen hinzufügen, indem Sie den Arbeitsauftrag auswählen und auf **Notizen** klicken.

4. Wenn Sie einen Wartungsarbeiter einem Arbeitsauftrag zuordnen möchten, markieren Sie den Arbeitsauftrag und klicken Sie auf **Arbeitsauftrag**.

5. Die Seite **Arbeitsauftrag** wird geöffnet. Klicken Sie auf **Versand**, um den Arbeitsauftrag für einen bestimmten Wartungsmitarbeiter zu planen.

>[!NOTE]
>Lesen Sie mehr über die Terminierung mehrerer Arbeitsaufträge oder eines Arbeitsauftrags unter [Arbeitsaufträge terminieren](../work-order-scheduling/schedule-work-orders.md) und [Arbeitsauftrag versenden](../work-order-scheduling/dispatch-work-order.md).
