<span data-ttu-id="f1ba3-104">Die Seite **Wartungsaufträge für planmäßige Arbeitsaufträge** dient dazu, sich einen Überblick über die einer Ressource zugeordneten Arbeitsaufträge zu verschaffen.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="f1ba3-105">Arbeitsaufträge mit den Ressourcentypen „Personal“, „Werkzeug“ und „Maschine“ werden angezeigt.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="f1ba3-106">Wenn sich z. B. ein Wartungsarbeiter krank gemeldet hat, können Sie diese Seite verwenden, um schnell Arbeitsaufträge zu finden, die dem Arbeiter zugeordnet sind, und dann einen anderen Wartungsmitarbeiter dem Auftrag zuzuordnen.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="f1ba3-107">Geplante Wartungsarbeiten für Arbeitsaufträge anzeigen</span><span class="sxs-lookup"><span data-stu-id="f1ba3-107">View scheduled work order maintenance jobs</span></span>

2. <span data-ttu-id="f1ba3-110">Sie können die Liste z.B. nach Wartungspersonal sortieren.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="f1ba3-111">Sie können den Filter auch verwenden, um die Liste einzuschränken, um Arbeitsaufträge anzuzeigen, die einer bestimmten Ressource oder einem bestimmten Instandhalter zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="f1ba3-112">Sie können Notizen zum Arbeitsauftrag sehen und bei Bedarf neue Notizen hinzufügen, indem Sie den Arbeitsauftrag auswählen und auf **Notizen** klicken.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="f1ba3-113">Wenn Sie einen Wartungsarbeiter einem Arbeitsauftrag zuordnen möchten, markieren Sie den Arbeitsauftrag und klicken Sie auf **Arbeitsauftrag**.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="f1ba3-114">Die Seite **Arbeitsauftrag** wird geöffnet.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-114">The **Work order** page opens.</span></span> <span data-ttu-id="f1ba3-115">Klicken Sie auf **Versand**, um den Arbeitsauftrag für einen bestimmten Wartungsmitarbeiter zu planen.</span><span class="sxs-lookup"><span data-stu-id="f1ba3-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
