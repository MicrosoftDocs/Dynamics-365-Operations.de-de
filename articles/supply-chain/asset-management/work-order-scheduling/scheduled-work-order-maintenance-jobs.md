[!include [banner](../../includes/preview-banner.md)]

<span data-ttu-id="274b5-104">Die Seite **Wartungsaufträge für planmäßige Arbeitsaufträge** dient dazu, sich einen Überblick über die einer Ressource zugeordneten Arbeitsaufträge zu verschaffen.</span><span class="sxs-lookup"><span data-stu-id="274b5-104">The **Scheduled work order maintenance jobs** page is used to get an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="274b5-105">Arbeitsaufträge mit den Ressourcentypen „Personal“, „Werkzeug“ und „Maschine“ werden in der Liste angezeigt.</span><span class="sxs-lookup"><span data-stu-id="274b5-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown in the list.</span></span> <span data-ttu-id="274b5-106">Die Liste kann verwendet werden, um sich einen Überblick über die Arbeitsaufträge zu verschaffen, die einer bestimmten Ressource zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="274b5-106">The list can be used to get an overview of work orders allocated to a specific resource.</span></span> <span data-ttu-id="274b5-107">Sie können damit auch einen Arbeitsauftrag schnell finden, der einem Instandhalter zugeordnet ist, der sich z.B. heute Morgen krank gemeldet hat, und dann schnell einen anderen Wartungsmitarbeiter dem Auftrag zuordnen.</span><span class="sxs-lookup"><span data-stu-id="274b5-107">You can also use it to quickly find a work order allocated to a maintenance worker who, for example, called in sick this morning, and then quickly allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="274b5-108">Geplante Wartungsarbeiten für Arbeitsaufträge anzeigen</span><span class="sxs-lookup"><span data-stu-id="274b5-108">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="274b5-109">Klicken Sie auf **Anlagenmanagement** > **Allgemein** > **Arbeitsaufträge** > **Planmäßige Wartungsarbeiten**.</span><span class="sxs-lookup"><span data-stu-id="274b5-109">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="274b5-110">Sie sehen eine Liste aller Arbeitsaufträge, die sich im Lebenszykluszustand „Geplant“ oder „In Bearbeitung“ befinden.</span><span class="sxs-lookup"><span data-stu-id="274b5-110">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="274b5-111">Sie können die Liste z.B. nach Wartungspersonal sortieren.</span><span class="sxs-lookup"><span data-stu-id="274b5-111">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="274b5-112">Sie können den Filter auch verwenden, um die Liste einzuschränken, um Arbeitsaufträge anzuzeigen, die einer bestimmten Ressource oder einem bestimmten Instandhalter zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="274b5-112">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="274b5-113">Sie können Notizen zum Arbeitsauftrag sehen und bei Bedarf neue Notizen hinzufügen, indem Sie den Arbeitsauftrag in der Liste auswählen und auf **Notizen** klicken.</span><span class="sxs-lookup"><span data-stu-id="274b5-113">You can see notes on the work order and, if required, add new notes by selecting the work order job in the list and clicking **Notes**.</span></span>

4. <span data-ttu-id="274b5-114">Wenn Sie einen Instandhalter einem Arbeitsauftrag zuordnen möchten, markieren Sie den Arbeitsauftrag in der Liste und klicken Sie auf **Auftrag**.</span><span class="sxs-lookup"><span data-stu-id="274b5-114">If you want to allocate one maintenance worker to a work order, select the work order in the list, and click **Work order**.</span></span>

5. <span data-ttu-id="274b5-115">Die Seite **Arbeitsauftrag** wird geöffnet.</span><span class="sxs-lookup"><span data-stu-id="274b5-115">The **Work order** page opens.</span></span> <span data-ttu-id="274b5-116">Klicken Sie auf **Versand**, um den Arbeitsauftrag für einen bestimmten Wartungsmitarbeiter zu planen.</span><span class="sxs-lookup"><span data-stu-id="274b5-116">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
