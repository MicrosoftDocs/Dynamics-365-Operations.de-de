## <a name="create-workers"></a><span data-ttu-id="1aa05-110">Arbeitskräfte erstellen</span><span class="sxs-lookup"><span data-stu-id="1aa05-110">Create workers</span></span>

1. <span data-ttu-id="1aa05-111">Wählen Sie **Anlagenverwaltung** \> **Einstellungen** \> **Arbeitskräfte** \> **Arbeitskräfte** aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-111">Select **Asset management** \> **Setup** \> **Workers** \> **Workers**.</span></span>
2. <span data-ttu-id="1aa05-112">Wählen Sie **Neu** aus, um der Liste eine Arbeitskraft hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="1aa05-112">Select **New** to add a worker to the list.</span></span>
3. <span data-ttu-id="1aa05-113">Wählen Sie im Feld **Arbeitskraft** die gewünschte Arbeitskraft aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-113">In the **Worker** field, select the worker.</span></span>
4. <span data-ttu-id="1aa05-114">Legen Sie die Option **Aktiv** auf **Ja** fest, um die Arbeitskraft für Arbeitsaufträge einzuplanen.</span><span class="sxs-lookup"><span data-stu-id="1aa05-114">Set the **Active** option to **Yes** to schedule the worker on work orders.</span></span>

    <span data-ttu-id="1aa05-115">Auf dem Inforegister **Allgemein** werden die Felder **Ressource** und **Beschreibung** automatisch ausgefüllt, wenn eine Ressource für die Arbeitskraft ausgewählt wurde.</span><span class="sxs-lookup"><span data-stu-id="1aa05-115">On the **General** FastTab, the **Resource** and **Description** fields are automatically filled in if a resource has been selected for the worker.</span></span> <span data-ttu-id="1aa05-116">Das Feld **Kalender** wird ebenfalls automatisch ausgefüllt, vorausgesetzt, dass Sie die Arbeitskraft als Ressource eingerichtet und dieser Ressource auf der Seite **Ressourcen** (**Organisationsverwaltung** \> **Ressourcen** \> **Ressourcen**) einen Kalender zugewiesen haben.</span><span class="sxs-lookup"><span data-stu-id="1aa05-116">The **Calendar** field is also automatically filled in, provided that you've set up the worker as a resource and allocated a calendar to that resource on the **Resources** page (**Organization administration** \> **Resources** \> **Resources**).</span></span>

5. <span data-ttu-id="1aa05-117">Wählen Sie auf dem Inforegister **Gruppen** die Option **Hinzufügen** aus, und wählen Sie dann eine Wartungsarbeitergruppe für die Arbeitskraft aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-117">On the **Groups** FastTab, select **Add**, and then select a maintenance worker group for the worker.</span></span> <span data-ttu-id="1aa05-118">Eine Arbeitskraft kann mehreren Gruppen angehören.</span><span class="sxs-lookup"><span data-stu-id="1aa05-118">A worker can be affiliated with more than one group.</span></span>
6. <span data-ttu-id="1aa05-119">In der Standardeinstellung ist die Zugehörigkeit einer Arbeitskraft zu einer Gruppe ab dem Datum wirksam, an dem Sie die Gruppe hinzuzufügen, und läuft nie ab.</span><span class="sxs-lookup"><span data-stu-id="1aa05-119">In the standard setup, a worker's affiliation with a group is effective from the date when you add the group, and it never expires.</span></span> <span data-ttu-id="1aa05-120">Dieses Datum wird im Feld **Gültig** angezeigt.</span><span class="sxs-lookup"><span data-stu-id="1aa05-120">This date is shown in the **Effective** field.</span></span> <span data-ttu-id="1aa05-121">Um das Feld **Gültig** anzuzeigen, wählen Sie **Anzeigen** \> **Alle** aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-121">To see the **Effective** field, select **View** \> **All**.</span></span> <span data-ttu-id="1aa05-122">Wenn die Zugehörigkeit der Arbeitskraft zu einer Gruppe auf einen bestimmten Zeitraum beschränkt sein soll, verwenden Sie die Felder **Gültig** und **Ablauf**, um den Zeitraum zu definieren.</span><span class="sxs-lookup"><span data-stu-id="1aa05-122">If the worker's affiliation with a group should be limited to a specific period, use the **Effective** and **Expiration** fields to define the period.</span></span>
7. <span data-ttu-id="1aa05-123">Wählen Sie auf dem Inforegister **Funktionale Standorte** die Option **Hinzufügen** aus, und wählen Sie dann einen funktionalen Standort für den Wartungsarbeiter aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-123">On the **Functional locations** FastTab, select **Add**, and then select a functional location for the maintenance worker.</span></span> <span data-ttu-id="1aa05-124">Geben Sie außerdem an, welcher Standort der primäre funktionale Standort für die Arbeitskraft ist.</span><span class="sxs-lookup"><span data-stu-id="1aa05-124">Also specify which location is the primary functional location for the worker.</span></span>

    > [!NOTE]
    > <span data-ttu-id="1aa05-125">Wenn Sie funktionale Standorte für eine Arbeitskraft hinzufügen, werden alle aktiven Anlagen, die diesen funktionalen Standorten zugeordnet sind, in verschiedenen Menüoptionen angezeigt, wie z. B. **Meine aktiven Anlagen** und **Meine aktiven funktionalen Standorte**.</span><span class="sxs-lookup"><span data-stu-id="1aa05-125">When you add functional locations to a worker, all active assets that are related to those functional locations appear on various menu items, such as **My active assets** and **My active functional locations**.</span></span> <span data-ttu-id="1aa05-126">Sie werden außerdem in den Anlagesuchen angezeigt, die angezeigt werden, wenn eine neue Anlage, eine Wartungsanfrage oder einen Arbeitsauftrag erstellen.</span><span class="sxs-lookup"><span data-stu-id="1aa05-126">They also appear in the asset lookups that are shown when you create a new asset, maintenance request, or work order.</span></span>

    <span data-ttu-id="1aa05-127">Die Felder auf dem Inforegister **Details** zeigen die Anzahl der Wartungsarbeitergruppen und funktionalen Standorte, denen der ausgewählte Wartungsarbeiter zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="1aa05-127">The fields on the **Details** FastTab show the number of maintenance worker groups and functional locations that the selected maintenance worker is related to.</span></span>

## <a name="create-worker-groups"></a><span data-ttu-id="1aa05-128">Arbeitskraftgruppen erstellen</span><span class="sxs-lookup"><span data-stu-id="1aa05-128">Create worker groups</span></span>

3. <span data-ttu-id="1aa05-131">Geben Sie im Feld **Wartungsarbeitergruppe** eine Gruppenkennung ein.</span><span class="sxs-lookup"><span data-stu-id="1aa05-131">In the **Maintenance worker group** field, enter a group ID.</span></span>

5. <span data-ttu-id="1aa05-133">Wählen Sie auf dem Inforegister **Arbeitskräfte** die Option **Hinzufügen** aus, und wählen Sie dann einen Wartungsarbeiter für die Arbeitskraftgruppe aus.</span><span class="sxs-lookup"><span data-stu-id="1aa05-133">On the **Workers** FastTab, select **Add**, and then select a maintenance worker for the worker group.</span></span> <span data-ttu-id="1aa05-134">Informationen über die Felder **Gültig** und **Ablauf** finden Sie Schritt 6 der vorherigen Prozedur.</span><span class="sxs-lookup"><span data-stu-id="1aa05-134">For information about the **Effective** and **Expiration** fields, see step 6 in the previous procedure.</span></span>

    <span data-ttu-id="1aa05-139">Das Feld auf dem Inforegister **Details** zeigt die Anzahl der Wartungsarbeiter, die in der ausgewählte Wartungsarbeitergruppe eingerichtet wurden.</span><span class="sxs-lookup"><span data-stu-id="1aa05-139">The field on the **Details** FastTab shows the number of maintenance workers that have been set up on the selected maintenance worker group.</span></span>
