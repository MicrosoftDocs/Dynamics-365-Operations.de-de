<span data-ttu-id="7f19a-104">Dieses Thema bietet einen Überblick über funktionale Standorte in Asset Management.</span><span class="sxs-lookup"><span data-stu-id="7f19a-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="7f19a-105">Funktionale Standorte sind Elementen einer technischen Struktur, wie die Funktionseinheiten in einem System.</span><span class="sxs-lookup"><span data-stu-id="7f19a-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="7f19a-106">Funktionale Standorte werden hierarchisch erstellt, und Sie installieren Anlagen für sie.</span><span class="sxs-lookup"><span data-stu-id="7f19a-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="7f19a-107">Das Einrichten von funktionalen Standorten in Ihrem Unternehmen hängt von den Anforderungen des Unternehmens ab.</span><span class="sxs-lookup"><span data-stu-id="7f19a-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="7f19a-108">Nachfolgend sind einige Beispiele für die Verwendung von funktionalen Standorten aufgeführt:</span><span class="sxs-lookup"><span data-stu-id="7f19a-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="7f19a-109">**Funktional** – Die funktionalen Standorte können benutzerorientiert sein und verwendet werden, um Anlagen zu verwalten, die ein ähnliches Verhalten haben.</span><span class="sxs-lookup"><span data-stu-id="7f19a-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>

<span data-ttu-id="7f19a-112">Jeder funktionale Standort wird in Asset Management eigenständig verwaltet.</span><span class="sxs-lookup"><span data-stu-id="7f19a-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="7f19a-113">Nachfolgend sind einige der wichtigsten Funktionen von funktionalen Standorten aufgeführt:</span><span class="sxs-lookup"><span data-stu-id="7f19a-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="7f19a-115">Einrichten von Anlagenspezifikationsanforderungen.</span><span class="sxs-lookup"><span data-stu-id="7f19a-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="7f19a-116">Einrichten von Wartungssequenzen für die vorbeugende und reagierende Wartung.</span><span class="sxs-lookup"><span data-stu-id="7f19a-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="7f19a-117">Verwalten von installierten Anlagen.</span><span class="sxs-lookup"><span data-stu-id="7f19a-117">Manage installed assets.</span></span>
- <span data-ttu-id="7f19a-118">Nachverfolgen von aktiven Anfragen und Arbeitsaufträge, die sich auf installierte Anlagen beziehen.</span><span class="sxs-lookup"><span data-stu-id="7f19a-118">Track active requests and work orders that are related to installed assets.</span></span>
- <span data-ttu-id="7f19a-119">Nachverfolgen von Fehlern, die für Anlagen erfasst wurden.</span><span class="sxs-lookup"><span data-stu-id="7f19a-119">Track faults that are registered on assets.</span></span>
- <span data-ttu-id="7f19a-120">Nachverfolgen von Wartungskosten für Anlagen, die zu einem funktionalen Standort gehören, zu jedem beliebigen Zeitpunkt.</span><span class="sxs-lookup"><span data-stu-id="7f19a-120">Track maintenance costs on the assets that are related to a functional location at any given time.</span></span>

<span data-ttu-id="7f19a-121">Funktionale Standorte ermöglichten das Nachverfolgen von Anlagen in Bezug auf Anfragen, Arbeitsaufträge, Fehlererfassungen, Bedingungsbewertungen, Produktionsstoperfassungen und Anlagenzählererfassungen.</span><span class="sxs-lookup"><span data-stu-id="7f19a-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>

<span data-ttu-id="7f19a-124">Funktionale Standorte sind **nicht** flexibel.</span><span class="sxs-lookup"><span data-stu-id="7f19a-124">Functional locations are **not** flexible.</span></span> <span data-ttu-id="7f19a-125">Daher können Sie nach dem Einrichten eine funktionalen Standorthierarchie die Standorte darin nicht mehr verschieben.</span><span class="sxs-lookup"><span data-stu-id="7f19a-125">Therefore, after you set up a functional location hierarchy, you can't move locations around in it.</span></span> 

<span data-ttu-id="7f19a-126">Nachdem Sie eine funktionale Standorthierarchie erstellt haben, besteht der nächste Schritt darin, Anlagen an ihm zu installieren.</span><span class="sxs-lookup"><span data-stu-id="7f19a-126">After you create a functional location hierarchy, the next step is to install assets on it.</span></span> <span data-ttu-id="7f19a-127">Weitere Informationen finden Sie unter [Anlagen an funktionalen Standorten installieren](../functional-locations/install-objects-on-functional-locations.md).</span><span class="sxs-lookup"><span data-stu-id="7f19a-127">For more information, see [Install assets on functional locations](../functional-locations/install-objects-on-functional-locations.md).</span></span>
