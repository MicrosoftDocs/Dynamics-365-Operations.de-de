# <a name="set-up-project-resource-information"></a>Projektressourceninformationen einrichten

[!include [task guide banner](../../includes/task-guide-banner.md)]

Diese Prozedur zeigt Ihnen an, wie eine Projektressource eingerichtet wird. Diese Aufgabe verwendet das USMF-Dataset.

1. Wechseln Sie zu „Projektverwaltung und -buchhaltung” > „Projektressourcen” Y „Ressourcenliste”.
2. Klicken Sie auf „Neue Ressource”.
3. Geben Sie im Feld 'Arbeitskraft' einen Wert ein, oder wählen Sie einen Wert aus.
4. Geben Sie im Feld "Kalender" einen Wert ein oder wählen Sie einen Wert aus.
5. Geben Sie im Feld „Periodencode” einen Wert ein oder wählen Sie einen Wert aus.
6. Klicken Sie auf "Erstellen".
7. Klicken Sie auf „Einstandspreise”.
8. Klicken Sie auf „Einstandspreis – Stunde”.
9. Klicken Sie auf "Neu".
10. Geben Sie im Feld "Gültigkeitsdatum" ein Datum ein.
11. Geben Sie im Feld "Kostenpreis" eine Zahl ein.
12. Klicken Sie auf "Speichern".
13. Schließen Sie die Seite.
14. Klicken Sie auf „Verkaufspreise”.
15. Klicken Sie auf „Verkaufspreis – Stunde”.
16. Klicken Sie auf "Neu".
17. Geben Sie im Feld "Gültigkeitsdatum" ein Datum ein.
18. Geben Sie im Feld „Preisgestaltung” eine Zahl ein.
19. Wählen Sie im Feld "Gültig für" eine Option aus.
20. Klicken Sie auf "Speichern".
21. Schließen Sie die Seite.
