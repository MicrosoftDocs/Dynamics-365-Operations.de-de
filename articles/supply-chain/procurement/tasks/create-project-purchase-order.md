2. <span data-ttu-id="43dd9-107">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="43dd9-107">In the list, click the link in the selected row.</span></span>

5. <span data-ttu-id="43dd9-110">Klicken Sie auf "Bestellung".</span><span class="sxs-lookup"><span data-stu-id="43dd9-110">Click Purchase order.</span></span>

9. <span data-ttu-id="43dd9-115">Klicken Sie auf "OK".</span><span class="sxs-lookup"><span data-stu-id="43dd9-115">Click OK.</span></span>
10. <span data-ttu-id="43dd9-116">Markieren Sie in der Liste die ausgewählte Zeile.</span><span class="sxs-lookup"><span data-stu-id="43dd9-116">In the list, mark the selected row.</span></span>
