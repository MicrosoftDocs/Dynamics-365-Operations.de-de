# <a name="create-project-purchase-order"></a><span data-ttu-id="7012f-103">Projektbestellung erstellen</span><span class="sxs-lookup"><span data-stu-id="7012f-103">Create project purchase order</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="7012f-104">Diese Prozedur zeigt Ihnen, wie Sie eine Projektbestellung manuell erstellen.</span><span class="sxs-lookup"><span data-stu-id="7012f-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="7012f-105">Diese Aufgabe verwendet das USSI-Dataset.</span><span class="sxs-lookup"><span data-stu-id="7012f-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="7012f-106">Wechseln Sie zu "Projektverwaltung und -verrechnung" > "Projekte" > "Alle Projekte".</span><span class="sxs-lookup"><span data-stu-id="7012f-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="7012f-107">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="7012f-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="7012f-108">Klicken Sie im Aktivitätsbereich auf Verwalten.</span><span class="sxs-lookup"><span data-stu-id="7012f-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="7012f-109">Klicken Sie auf "Artikelaufgabe".</span><span class="sxs-lookup"><span data-stu-id="7012f-109">Click Item task.</span></span>
5. <span data-ttu-id="7012f-110">Klicken Sie auf "Bestellung".</span><span class="sxs-lookup"><span data-stu-id="7012f-110">Click Purchase order.</span></span>
6. <span data-ttu-id="7012f-111">Geben Sie im Feld "Kreditorenkonto" einen Wert ein, oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="7012f-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="7012f-112">Geben Sie im Feld "Standort" einen Wert ein, oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="7012f-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="7012f-113">Diese Schritte sind nicht erforderlich, aber sie vereinfachen die Bestellung, indem ein Standardstandort und -lagerort für die Bestellpositionen eingerichtet wird.</span><span class="sxs-lookup"><span data-stu-id="7012f-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="7012f-114">Geben Sie im Feld 'Lagerort' einen Wert ein, oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="7012f-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="7012f-115">Klicken Sie auf "OK".</span><span class="sxs-lookup"><span data-stu-id="7012f-115">Click OK.</span></span>
10. <span data-ttu-id="7012f-116">Markieren Sie in der Liste die ausgewählte Zeile.</span><span class="sxs-lookup"><span data-stu-id="7012f-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="7012f-117">Geben Sie im Feld "Artikelnummer" einen Wert ein oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="7012f-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="7012f-118">Dies kann die Artikelnummer oder eine Beschaffungskategorie sein.</span><span class="sxs-lookup"><span data-stu-id="7012f-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="7012f-119">Erweitern Sie den Abschnitt "Positionsdetails".</span><span class="sxs-lookup"><span data-stu-id="7012f-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="7012f-120">Klicken Sie auf die Registerkarte „Projekt”.</span><span class="sxs-lookup"><span data-stu-id="7012f-120">Click the Project tab.</span></span>
    * <span data-ttu-id="7012f-121">Überprüfen Sie, dass die Verkaufs- und Einstandspreise verfügbar sind.</span><span class="sxs-lookup"><span data-stu-id="7012f-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="7012f-122">Wenn sie nicht verfügbar aber erforderlich sind, geben Sie die Informationen ein.</span><span class="sxs-lookup"><span data-stu-id="7012f-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="7012f-123">Klicken Sie auf "Speichern".</span><span class="sxs-lookup"><span data-stu-id="7012f-123">Click Save.</span></span>
