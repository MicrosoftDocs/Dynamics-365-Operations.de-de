# <a name="view-cost-entries-for-a-cost-object"></a>Kosteneinträge für ein Kostenobjekt anzeigen

[!include [banner](../../includes/banner.md)]

Im folgenden Verfahren sehen Sie, wie Sie Kosteneinträge für einen Kostenträger anzeigen. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF. Diese Prozedur ist für den Kostencontroller vorgesehen.

1. Klicken Sie auf "Kostenverwaltung".
2. Klicken Sie auf "Freigegebene Produkte".
3. Verwenden Sie den Schnellfilter, um Datensätze zu suchen. Filtern Sie beispielsweise im Feld "Artikelnummer" mit dem Wert "m0004".
4. Klicken Sie im Aktivitätsbereich auf "Kosten verwalten".
5. Klicken Sie auf "Kostenobjekte".
6. Klicken Sie auf "Kosteneinträge".
7. Verwenden Sie den Schnellfilter, um im Feld "Nummer" nach dem Wert "p000031" zu filtern.
    * Wenn Kosteneinträge leer sind, richten Sie das Von-Datum auf den 31. Januar 2012 und das Bis-Datum auf den 31. Dezember 2012 ein.
