# <a name="set-up-a-transportation-tender"></a>Transportausschreibung einrichten

[!include [banner](../../includes/banner.md)]

Dieses Verfahren zeigt Ihnen an, wie eine Transportausschreibung eingerichtet wird. Diese Einstellung wird normalerweise von einem Transportkoordinator vorgenommen. Sie können diese Prozedur im Demodatunternehmen USMF verwenden.


## <a name="select-a-route"></a>Arbeitsplan auswählen
1. Wechseln Sie zu Transportverwaltung > Planung > Ladungsplanungsworkbench.
2. Deaktivieren Sie das Kontrollkästchen "'Versendet' und 'Empfangen' ausblenden".
3. Wählen Sie die Position mit der Ladungskennung 00006.
4. Klicken Sie auf Bewertung und Weiterleitung.
5. Klicken Sie auf "Arbeitspläne".

## <a name="create-the-transportation-tender"></a>Erstellen der Transportausschreibung
1. Klicken Sie auf Transportausschreibungen.
2. Klicken Sie auf "Neu".
3. Erweitern Sie den Abschnitt "Allgemein".
4. Geben Sie im Feld 'Angeforderte Sätze' eine Zahl ein.
5. Klicken Sie auf "Speichern".
6. Klicken Sie auf Aktualisierungsstatus.
7. Klicken Sie auf Absenden.
8. Wählen Sie einen Arbeitsplan aus.
