# <a name="set-up-accessorial-assignments"></a>Zusatzleistungszuweisungen einrichten

[!include [banner](../../includes/banner.md)]

Dieses Verfahren zeigt Ihnen an, wie eine Zusatzleistungszuweisung eingerichtet wird. Diese Einstellung wird normalerweise von einem Transportkoordinator vorgenommen. Bevor Sie diese Anleitung verwenden, müssen Sie die Anweisungen unter "Hubzusatzgebühren und Zusatzleistungsmaster einrichten" ausführen.


## <a name="set-up-accessorial-assignment"></a>Einrichten von Zusatzleistungszuweisung
1. Wechseln Sie zu "Transportverwaltung" > "Einrichten" > "Bewertung" > "Zusatzleistungszuweisungen".
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Name" einen Wert ein.
4. Schalten Sie die Erweiterung des Abschnitts "Details" ein/aus.
5. Klicken Sie im Feld "Hub" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
6. Wählen Sie in der Liste den Hub aus, für den Sie einen Zusatzleistungsmaster erstellt haben, als Sie das Handbuch "Hubzusatzgebühren und Zusatzleistungsmaster einrichten" ausgeführt haben. 
7. Klicken Sie im Feld "Kennung für Hubzusatzgebühren" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
8. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
9. Schalten Sie die Erweiterung des Abschnitts "Kriterien" ein/aus.
    * Im Abschnitt "Kriterien" können Sie basierend auf den verschiedenen angebotenen Werten die genauen Kriterien dafür auswählen, wann die Belastung angewendet werden soll.  
10. Legen Sie die Option "Immer anwenden" auf "Ja" fest.
11. Wählen Sie im Feld "Zusatzleistungs-Zuweisungsebene" eine Option aus.
12. Schalten Sie die Erweiterung des Abschnitts "Berechnung" ein/aus.
13. Wählen Sie im Feld "Gebühr Zusatzleistung" "Pauschal" aus.
    * Der "Gebührentyp Zusatzleistung" bestimmt, wie die tatsächlichen Zuschläge berechnet werden. In diesem Beispiel ist es eine pauschale Belastung.  
14. Geben Sie im Feld "Gebühr Zusatzleistung" eine Zahl ein.
15. Klicken Sie auf "Speichern".
