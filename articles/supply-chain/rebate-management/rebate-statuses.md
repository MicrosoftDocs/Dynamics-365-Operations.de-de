# <a name="rebate-statuses"></a>Rückvergütungsstatus

[!include [banner](../includes/banner.md)]

Mit dem Rückvergütungsstatus können Sie den Status jedes Rückvergütungsverwaltungsdeals verfolgen. Sie können sie auch für Berichts- und Abfragezwecke verwenden. Sie können so viele Rückvergütungsstatuswerte einrichten und benennen, wie Sie zur Unterstützung Ihres Geschäftsprozesses benötigen. 

Um einen Rückvergütungsstatus einzurichten, gehen Sie zu **Rückvergütungsverwaltung \> Einstellungen \> Status \> Status**. Sie können dann die Schaltflächen im Aktivitätsbereich verwenden, um bei Bedarf Statuswerte hinzuzufügen und zu entfernen. Legen Sie für jeden Status die folgenden Felder fest:

- **Rückvergütungsstatus** – Geben Sie einen Namen für den Status ein (z. B. *Verhandlungen laufen*, *Vereinbart* oder *Erledigt*).
- **Beschreibung** – Geben Sie eine Beschreibung des Status ein, um weitere Informationen zur Verwendung bereitzustellen.

Auf der [Seite **Rückvergütungsverwaltungsparameter**](rebate-management-parameters.md) können Sie einen Standardstatus auswählen. Dieser Status wird dann allen neu erstellten Rückvergütungsverwaltungsdeals zugewiesen.
