# <a name="update-onboarding-guides"></a><span data-ttu-id="a675a-103">Onboarding Anleitungen aktualisieren</span><span class="sxs-lookup"><span data-stu-id="a675a-103">Update onboarding guides</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="a675a-104">Wenn Sie Änderungen an Onboarding Anleitungen vornehmen in Microsoft Dynamics 365 Talent: Onboard, können sie aktualisiert werden und die Änderungen vornehmen, selbst wenn die Anleitungen bereits übermittelt wurden.</span><span class="sxs-lookup"><span data-stu-id="a675a-104">If you must make changes to onboarding guides in Microsoft Dynamics 365 Talent: Onboard, you can update them and push the changes, even if you've already sent the guides.</span></span> <span data-ttu-id="a675a-105">Sie haben zwei Optionen zum Aktualisieren einer Onboarding Anleitung:</span><span class="sxs-lookup"><span data-stu-id="a675a-105">You have two options for updating an onboarding guide:</span></span>

- <span data-ttu-id="a675a-107">Bearbeiten Sie die Onboarding Vorlage, und aktualisieren Sie dann die Änderungen per Push auf allen Onboarding Anleitungen, die daraus erstellt wurden.</span><span class="sxs-lookup"><span data-stu-id="a675a-107">Edit the onboarding template, and then push the changes to all the onboarding guides that were created from it.</span></span>

## <a name="edit-an-onboarding-guide-directly"></a><span data-ttu-id="a675a-108">Bearbeiten eines Onboarding Handbuches direkt</span><span class="sxs-lookup"><span data-stu-id="a675a-108">Edit an onboarding guide directly</span></span>

1. <span data-ttu-id="a675a-109">Wählen Sie im linken Menü **Anleitungen** aus.</span><span class="sxs-lookup"><span data-stu-id="a675a-109">On the left menu, select **Guides**.</span></span>
2. <span data-ttu-id="a675a-110">Wählen Sie die Anleitung, die Sie bearbeiten möchten.</span><span class="sxs-lookup"><span data-stu-id="a675a-110">Select the guide that you want to edit.</span></span>
3. <span data-ttu-id="a675a-111">Nehmen Sie alle gewünschten Änderungen vor, und wählen Sie dann **Speichern** aus (das Diskettensymbol).</span><span class="sxs-lookup"><span data-stu-id="a675a-111">Make all the desired changes, and then select the **Save** button (the disk symbol).</span></span>

    ![[<span data-ttu-id="a675a-112">Änderung an einer Onboarding-Anleitung speichern</span><span class="sxs-lookup"><span data-stu-id="a675a-112">Saving changes to an onboarding guide</span></span>](./media/onboard-save.png)](./media/onboard-save.png)

<span data-ttu-id="a675a-113">Onboard wird automatisch den neuen Mitarbeiters eine E-Mail senden, die angibt, welche Änderungen vorgenommen wurden.</span><span class="sxs-lookup"><span data-stu-id="a675a-113">Onboard will automatically send the new hire an email that indicates what the changes are.</span></span> <span data-ttu-id="a675a-114">Zur einfacheren Kennung erscheint eine rote Markierung **Neu** neben jeder Änderung.</span><span class="sxs-lookup"><span data-stu-id="a675a-114">For easy identification, a red **New** tag will appear next to each change.</span></span>

## <a name="update-multiple-guides-by-editing-the-onboarding-template"></a><span data-ttu-id="a675a-115">Aktualisieren Sie mehrere Anleitungen, indem Sie die Onboarding Vorlage bearbeiten</span><span class="sxs-lookup"><span data-stu-id="a675a-115">Update multiple guides by editing the onboarding template</span></span>

1. <span data-ttu-id="a675a-116">Wählen Sie im linken Menü **Vorlagen** aus.</span><span class="sxs-lookup"><span data-stu-id="a675a-116">On the left menu, select **Templates**.</span></span>
2. <span data-ttu-id="a675a-117">Wählen Sie unter **Meine Vorlagen** die Vorlage aus, die Sie bearbeiten möchten.</span><span class="sxs-lookup"><span data-stu-id="a675a-117">Under **My templates**, select the template that you want to edit.</span></span>
3. <span data-ttu-id="a675a-118">Nehmen Sie alle gewünschten Änderungen vor, und wählen Sie dann **Speichern** aus (das Diskettensymbol).</span><span class="sxs-lookup"><span data-stu-id="a675a-118">Make all the desired changes, and then select the **Save** button (the disk symbol).</span></span>
4. <span data-ttu-id="a675a-119">Damit Ihre Änderungen an allen Anleitungen erfolgen, die auf der Vorlage basieren, wählen Sie **Änderungen per Push übertragen** aus.</span><span class="sxs-lookup"><span data-stu-id="a675a-119">To push your changes to all the guides that are based on the template, select **Push these changes**.</span></span>

    ![[<span data-ttu-id="a675a-120">Die Änderungen per Push auf allen Onboarding Anleitungen, die daraus erstellt wurden, speichern</span><span class="sxs-lookup"><span data-stu-id="a675a-120">Pushing the changes in an onboarding template to all guides that were created from it</span></span>](./media/onboard-push-changes.png)](./media/onboard-push-changes.png)
