# <a name="get-started-with-onboard"></a>Erste Schritte mit Onboard

[!include [banner](includes/banner.md)]

Durch Verwenden von Microsoft Dynamics 365 for Talent: Onboard können Sie den neuen Mitarbeiter unterstützen, dass er einen besseren Start hat und …ein Gefühl für seine neue Rolle erhält. Ein effektiver Onboarding-Prozess hift neuen Mitarbeitern, ihre neuen Rollen zu verstehen. Es gibt Ihnen einen Vorsprung beim Einrichten wertvoller Beziehungen, die dazu beitragen, den Erfolg sicherzustellen.

Mit Onboard  können Sie schnell ein Handbuch erstellen, das Ihre neuen Mitarbeiter begrüßt und sie mit Ihrer Organisation vertraut macht.
