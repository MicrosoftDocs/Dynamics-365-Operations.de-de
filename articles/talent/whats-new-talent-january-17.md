# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-january-17-2019"></a>Neuerungen oder Änderungen in Dynamics 365 for Talent Core HR (17. Januar 2019)

[!include [banner](includes/banner.md)]

**Build 8.1.2107**

In diesem Thema werden die Funktionen beschrieben, die in Core HR entweder neu oder geändert sind.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Geringfügige Änderungen und Fehlerkorrekturen in dieser Version

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>Neue Positionszuweisungsstartdatum/Zeitstandard korrigiert
Bei der Versetzung von Mitarbeitern schlägt das System das Datum/Uhrzeit auf das ausgewählte Datum und die Uhrzeit zu Beginn des Tages für die Zeitzone des Benutzers vor.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Alle Mitarbeiter haben die gleiche Planstellenbeschreibung in der Liste der ausgetretenen Mitarbeiteraufgabenverwaltung.
Im Arbeitsbereich **Aufgabenmanagement** wurde ein Update durchgeführt, um die korrekten Stellenbeschreibungen für alle Mitarbeiter anzuzeigen, die kürzlich das Unternehmen verlassen haben.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>Das Feld "Aktion angeforder von" wird nun auf der Mitarbeiteraktionenseite ausgefüllt.
Mit dieser Änderung ist der Standard für das Feld **Aktion angefordert von** nun der Benutzer, der die Änderung beantragt.

### <a name="ideas-portal-updated"></a>Ideenportal aktualisiert
In der App wurden Links zur Einreichung einer Idee für Talent auf das neue Ideenportal aktualisiert. Alle Ideen, die im alten Portal eingegeben wurden, wurden migriert, so dass alle deine großartigen Ideen noch vorhanden sind und von anderen bewertet werden können.
