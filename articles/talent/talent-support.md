# <a name="get-support-for-talent"></a>Erhalten Sie Support für Talent

[!include [banner](includes/banner.md)]

In diesem Thema wird erläutert, wie Hilfe von Microsoft Dynamics 365 for Talent Core HR, Dynamics 365 Talent: Attract, und Dynamics 365 Talent: Onboard erhalten.

## <a name="support-for-talent"></a>Support für Talent

Um Unterstützung für Talent zu erhalten, verwenden Sie die Ressourcen in der folgenden Tabelle. Diese Ressourcen werden in der Reihenfolge aufgelistet, die wir empfehlen, wenn Sie Fehler beheben möchten.

| Aufgabe | Weitere Informationen |
|------|------------------|
| Die Community fragen. | Wechseln Sie zu [Dynamics 365 for Talent Community](https://community.dynamics.com/365/talent), um Hilfe zu Ihren Fragen zu erhalten. Wenn Sie Antworten suchen, geben Sie eine Frage in das Suchfeld neben **Haben Sie eine Frage** ein. Wenn Sie eine Frage stellen möchten, wählen Sie **Eine Frage stellen** aus. |
| Nutzen Sie das in der Cloud bereitgestellte Supporttool. | Wechseln Sie in [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/) zu Ihrem Projekt, und wählen Sie die Kachel **Support** aus. Hier können Sie Probleme auf zwei Arten senden:<ul><li>Wählen Sie in der Registerkarte **Aktive Probleme** das Problem aus, das Sie senden möchten, und wählen Sie dann **An Microsoft senden** aus.</li><li>Wählen Sie auf die Registerkarte **An Microsoft übermittelt** die Option **Neues Problem übermitteln** aus, um ein neues Problem zu senden.</li></ul> |
| Erhalten Sie In-App-Support. | Wählen Sie in der rechten oberen Ecke der Seite die Schltfläche **Hilfe** (**?**) aus. In Core HR wählen Sie dann **Support** aus. In Attract oder Onboard wählen Sie **Ein Problem melden** aus. Probleme werden auf der Registerkarte **Aktive Probleme** in LCS gemeldet, wo ein Administrator bestimmen kann, ob interne Unterstützung zulässig ist oder die Probleme an Microsoft gesendet werden. (Wenn Sie die eigenständige Version von Onbord haben, werden Probleme direkt an Microsoft gesendet). |
| Erhalten Sie Hilfe bei Fragen zu Talentlizenzierung. | Kontaktieren Sie Ihren Partner oder Ihren Microsoft Vertreter. |
| Fordern Sie neue Funktionen und Funktionalitäten in Talent an. | Wählen Sie auf der Talent-Haupttalentseite die Schaltfläche **Hilfe** (**?**) aus und wählen Sie dann **Ideen** aus. Sie können auch [Dynamics 365-Anwendungsideen](https://experience.dynamics.com/ideas/) besuchen, nach vorhandenen Ideen suchen oder dafür abstimmen oder neue Ideen hinzuzufügen. |

## <a name="support-for-the-onboard-stand-alone-app"></a>Unterstützung für die eigenständige Onboard-App

Um Unterstützung für die eigenständige Onboard-App zu erhalten, verwenden Sie die Ressourcen in der folgenden Tabelle. Diese Ressourcen werden in der Reihenfolge aufgelistet, die wir empfehlen, wenn Sie Fehler beheben möchten. Sie können alle diese Supportoptionen finden, indem Sie die Schaltfläche **Hilfe** (**?**) in der oberen rechten Ecke der Seite auswählen.

| Aufgabe | Weitere Informationen |
|------|------------------|
| Die Community fragen. | Wechseln Sie zu [Dynamics 365 for Talent Community](https://community.dynamics.com/365/talent), um Hilfe zu erhalten. Wenn Sie Antworten suchen, geben Sie eine Frage in das Suchfeld neben **Haben Sie eine Frage** ein. Wenn Sie eine Frage stellen möchten, wählen Sie **Eine Frage stellen** aus. |
| Melden Sie ein Problem. | Wählen Sie die Schaltfläche **Hilfe** (**?**) rechts oben auf der Seite aus, und wählen Sie dann **Ein Problem melden** aus. Geben Sie die Beschreibung eines Problem ein, und fügen Sie eine E-Mail-Adresse hinzu, wenn Sie das gemeldete Problem nachverfolgen möchten. |
| Erhalten Sie Hilfe bei Fragen zu Talentlizenzierung. | Kontaktieren Sie Ihren Partner oder Ihren Microsoft Vertreter. |
| Fordern Sie neue Funktionen und Funktionalitäten in Onboard an. | Wählen Sie die Schaltfläche **Hilfe** (**?**) aus, und wählen Sie dann **Ideen** aus. Sie können auch [Dynamics 365 for Talent: Onboard-Ideen](https://experience.dynamics.com/ideas/categories/?forum=569a7fb2-8327-e911-a95a-000d3a4f3883&forumName=Dynamics%20365%20for%20Talent%3A%20Onboard) besuchen, nach vorhandenen Ideen suchen oder dafür abstimmen oder neue Ideen hinzuzufügen. |
