# <a name="close-a-job-in-attract"></a><span data-ttu-id="63e6d-103">Eine Stelle in Attract schließen</span><span class="sxs-lookup"><span data-stu-id="63e6d-103">Close a job in Attract</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="63e6d-104">In Microsoft Dynamics 365 Talent: Attract können Sie eine Stelle entweder als **Stelle storniert** oder **Position besetzt** schließen.</span><span class="sxs-lookup"><span data-stu-id="63e6d-104">In Microsoft Dynamics 365 Talent: Attract, you can close a job as either **Job cancelled** or **Position filled**.</span></span> <span data-ttu-id="63e6d-105">Sie können eine Stelle jederzeit löschen.</span><span class="sxs-lookup"><span data-stu-id="63e6d-105">You can cancel a job at any time.</span></span>

## <a name="cancel-a-job"></a><span data-ttu-id="63e6d-107">Stelle abbrechen</span><span class="sxs-lookup"><span data-stu-id="63e6d-107">Cancel a job</span></span>

<span data-ttu-id="63e6d-108">Sie können eine Stelle in Attract jederzeit löschen.</span><span class="sxs-lookup"><span data-stu-id="63e6d-108">You can cancel a job in Attract at any time.</span></span>

1. <span data-ttu-id="63e6d-109">Auf der Registerkarte **Stelle** in Attract wählen Sie die Stelle, die Sie stornieren möchten.</span><span class="sxs-lookup"><span data-stu-id="63e6d-109">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="63e6d-110">Wählen Sie **Stelle schließen** aus.</span><span class="sxs-lookup"><span data-stu-id="63e6d-110">Select **Close job**.</span></span>

   ![Stelle in Attract schließen](./media/attract-close-job.png)

3. <span data-ttu-id="63e6d-112">Im Dialogfeld **Diese Stelle beenden** wählen Sie **Stelle storniert** aus, geben Sie beliebige Kommentare ein, und wählen Sie dann **Stelle beenden** aus.</span><span class="sxs-lookup"><span data-stu-id="63e6d-112">In the **Close this job** dialog box, select **Job cancelled**, add any comments, and then select **Close job**.</span></span>

   ![Schließen Sie die Stelle als Stelle besetzt](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a><span data-ttu-id="63e6d-114">Schließen Sie die Stelle als Stelle besetzt</span><span class="sxs-lookup"><span data-stu-id="63e6d-114">Close a job as Position filled</span></span>

<span data-ttu-id="63e6d-115">Sie können eine Stelle schließen als **Position besetzt** nachdem mindestens ein Interessent die Angebotphase abgeschlossen hat.</span><span class="sxs-lookup"><span data-stu-id="63e6d-115">You can close a job as **Position filled** after at least one prospect has completed the Offer stage.</span></span>

1. <span data-ttu-id="63e6d-116">Auf der Registerkarte **Stelle** in Attract wählen Sie die Stelle, die Sie stornieren möchten.</span><span class="sxs-lookup"><span data-stu-id="63e6d-116">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="63e6d-117">Wählen Sie **Stelle schließen** aus.</span><span class="sxs-lookup"><span data-stu-id="63e6d-117">Select **Close job**.</span></span>

   ![Stelle in Attract schließen](./media/attract-close-job.png)

3. <span data-ttu-id="63e6d-119">Im Dialogfeld **diese Stelle beenden** wählen Sie **Position besetzt** und wählen Sie den Kandidaten aus, der das Angebot akzeptiert hat und wählen Sie **Stelle beenden** aus.</span><span class="sxs-lookup"><span data-stu-id="63e6d-119">In the **Close this job** dialog box, select **Position filled**, select the candidate who accepted the offer, add any comments, and then select **Close job**.</span></span>

   ![Schließen Sie die Stelle als Stelle besetzt](./media/attract-close-job-as-position-filled.png)

   > [!NOTE]
   > <span data-ttu-id="63e6d-121">Sie können nur eine Stelle schließen als **Position besetzt** nachdem mindestens ein Interessent die Angebotphase abgeschlossen hat.</span><span class="sxs-lookup"><span data-stu-id="63e6d-121">You can only close a job as **Position filled** when at least one prospect has completed the Offer stage.</span></span> <span data-ttu-id="63e6d-122">Wenn der Stellenbewerber nicht **Angebot akzeptiert** ausgewählt hat oder wenn die Anwendung nicht den Status **Angenommen** im Angebot anzeigt, wird die Option **Stelle besetzt** nicht im Dropdownmenü **Grund** angezeigt.</span><span class="sxs-lookup"><span data-stu-id="63e6d-122">If the job application doesn't have **Offer accepted** selected, or if the application doesn't display a status of **Accepted** in Offer, the **Position filled** option won't display in the **Reason** dropdown.</span></span>
