# <a name="configure-the-functionality-profile-for-a-sales-representative"></a> Funktionsprofil für einen Verkäufer konfigurieren

[!include[task guide banner](../includes/task-guide-banner.md)]

Dieses Verfahren zeigt, wie Funktionsprofileinstellungen eines Shops konfiguriert wird, das für Verkäufer gilt. Für diese Prozedur wird das Demo-Datenunternehmen USRT verwendet.

1. Navigieren Sie zu Einzelhandel und Handel > Kanaleinstellungen > POS-Einstellungen > POS-Profile Funktionsprofile.
2. Klicken Sie auf "Bearbeiten".
3. Erweitern Sie den Abschnitt "Funktionen".
    * Sie können die Funktionsprofileinstellungen verwenden, um den POS zu konfigurieren, um die Standardverkaufsgruppe des Kassierers automatisch hinzuzufügen, um nach Verkaufsgruppen zu fragen und um Verkaufsgruppen zu erfordern.  
4. Wählen Sie im Feld "Nachfrage nach Verkäufer" eine Option aus.
5. Wählen Sie "Ja" im Feld "Verkäufer erforderlich" aus.
