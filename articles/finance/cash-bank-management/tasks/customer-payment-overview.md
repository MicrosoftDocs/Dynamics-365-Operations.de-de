# <a name="customer-payment-overview"></a><span data-ttu-id="ec970-103">Übersicht Debitorenzahlung</span><span class="sxs-lookup"><span data-stu-id="ec970-103">Customer payment overview</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ec970-104">Dieser Aufgabenleitfaden führt durch verschiedene Methoden zum Eingeben von Debitorenzahlungen.</span><span class="sxs-lookup"><span data-stu-id="ec970-104">This task guide walks through various methods used to enter customer payments.</span></span> <span data-ttu-id="ec970-105">Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="ec970-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="ec970-106">Wechseln Sie zu **Navigationsbereich > Module > Debitorenkonten > Zahlungen > Zahlungserfassung**.</span><span class="sxs-lookup"><span data-stu-id="ec970-106">Go to **Navigation pane > Modules > Accounts receivable > Payments > Payment journal**.</span></span>
2. <span data-ttu-id="ec970-107">Klicken Sie auf **Neu**.</span><span class="sxs-lookup"><span data-stu-id="ec970-107">Click **New**.</span></span>
3. <span data-ttu-id="ec970-108">Wählen Sie die Zahlungserfassung aus, in der die Debitorenzahlungen gespeichert werden sollen.</span><span class="sxs-lookup"><span data-stu-id="ec970-108">Select the payment journal which the customer payments will be saved.</span></span>
4. <span data-ttu-id="ec970-109">Wählen Sie die Erfassung aus, oder geben Sie sie manuell ein.</span><span class="sxs-lookup"><span data-stu-id="ec970-109">Select or manually enter the journal.</span></span>
5. <span data-ttu-id="ec970-110">Klicken Sie im **Aktivitätsbereich** auf **Debitorenzahlungen eingeben**.</span><span class="sxs-lookup"><span data-stu-id="ec970-110">On **Action pane**, click **Enter customer payments**.</span></span> <span data-ttu-id="ec970-111">"Debitorenzahlungen eingeben" wird verwendet, um jeweils eine Debitorenzahlung zu erfassen.</span><span class="sxs-lookup"><span data-stu-id="ec970-111">Enter customer payments is used to record one customer payment at a time.</span></span> <span data-ttu-id="ec970-112">Sie geben die Zahlungsinformationen oben ein, anschließend können Sie auf derselben Seite die Rechnungen markieren, die anhand der Zahlung bezahlt wurden.</span><span class="sxs-lookup"><span data-stu-id="ec970-112">You enter the payment information at the top, and then you can mark the invoices that were paid by the payment, all from the same page.</span></span>  
6. <span data-ttu-id="ec970-113">Geben Sie den Debitor ein, von dem Sie die Zahlung erhalten haben.</span><span class="sxs-lookup"><span data-stu-id="ec970-113">Enter the customer from whom you received the payment.</span></span> <span data-ttu-id="ec970-114">Wenn Sie den Debitor nicht kennen, aber wissen, dass eine Buchung bezahlt wurde, können Sie das Buchungsfeld verwenden, um die Zahlung einzugeben.</span><span class="sxs-lookup"><span data-stu-id="ec970-114">If you don't know the customer but do know a transaction that was paid, you can use the Transaction field to enter the payment.</span></span> <span data-ttu-id="ec970-115">Geben Sie die Rechnung ins Transaktionsfeld ein oder wählen Sie sie aus.</span><span class="sxs-lookup"><span data-stu-id="ec970-115">Enter or select the invoice in the Transaction field.</span></span> <span data-ttu-id="ec970-116">Der Debitor wird automatisch auf den Standardwert gesetzt, wenn Sie die Buchung ausgewählt haben.</span><span class="sxs-lookup"><span data-stu-id="ec970-116">The customer will automatically default after you select the transaction.</span></span>
7. <span data-ttu-id="ec970-117">Geben Sie im Feld **Zahlungsreferenz** eine Zahlungsreferenz ein.</span><span class="sxs-lookup"><span data-stu-id="ec970-117">In the **Payment reference** field, enter a payment reference.</span></span> <span data-ttu-id="ec970-118">Die Zahlungsreferenz könnte die Schecknummer oder Referenz des Debitors von der elektronischen Zahlung des Debitors sein.</span><span class="sxs-lookup"><span data-stu-id="ec970-118">The payment reference could be the customer's check number or a reference from the customer's electronic payment.</span></span> <span data-ttu-id="ec970-119">Die Zahlungsreferenz ist nur erforderlich, wenn Sie angegeben haben, dass die Zahlung auf einem Einzahlungsbeleg enthalten sein soll.</span><span class="sxs-lookup"><span data-stu-id="ec970-119">The payment reference is only required if you mark to include the payment on a deposit slip.</span></span>  
8. <span data-ttu-id="ec970-120">Wählen Sie im Feld **Einzahlungsbeleg** aus, ob die Zahlung in einem Einzahlungsbeleg einbezogen ist.</span><span class="sxs-lookup"><span data-stu-id="ec970-120">In the **Deposit slip** field, select whether the payment will be included on a deposit slip.</span></span> 
9. <span data-ttu-id="ec970-121">Geben Sie im Feld **Betrag** den Betrag der Debitorenzahlung ein.</span><span class="sxs-lookup"><span data-stu-id="ec970-121">In the **Amount** field, enter the amount of the customer payment.</span></span> <span data-ttu-id="ec970-122">Der Zahlungsbetrag wird nicht standardmäßig festgelegt.</span><span class="sxs-lookup"><span data-stu-id="ec970-122">The payment amount will not default.</span></span> <span data-ttu-id="ec970-123">Er muss manuell eingegeben werden.</span><span class="sxs-lookup"><span data-stu-id="ec970-123">It must be manually entered.</span></span> 
10. <span data-ttu-id="ec970-124">Markieren Sie die Rechnungen, die vom Debitor bezahlt wurden.</span><span class="sxs-lookup"><span data-stu-id="ec970-124">Mark the invoices that were paid by the customer.</span></span> <span data-ttu-id="ec970-125">Wenn es sich bei der Zahlung um eine Vorauszahlung handelt, müssen Sie keine Rechnungen für den Ausgleich markieren.</span><span class="sxs-lookup"><span data-stu-id="ec970-125">If the payment is a prepayment, you are not required to mark any invoices for settlement.</span></span> <span data-ttu-id="ec970-126">Die Zahlung kann dennoch gespeichert und gebucht werden.</span><span class="sxs-lookup"><span data-stu-id="ec970-126">The payment can still be saved and posted.</span></span> <span data-ttu-id="ec970-127">Der Ausgleich einer Rechnung kann zu einem späteren Zeitpunkt erfolgen.</span><span class="sxs-lookup"><span data-stu-id="ec970-127">Settlement to an invoice can happen at a later time.</span></span>
11. <span data-ttu-id="ec970-128">Geben Sie den Betrag der Zahlung ein, der mit der ausgewählten Rechnunge ausgeglichen werden soll.</span><span class="sxs-lookup"><span data-stu-id="ec970-128">Enter the amount of the payment that should be settled to the marked invoice.</span></span> <span data-ttu-id="ec970-129">Dieses Feld kann verwendet werden, wenn die Zahlung für eine Teilzahlung ist.</span><span class="sxs-lookup"><span data-stu-id="ec970-129">This field can be used when the payment is for a partial payment.</span></span> <span data-ttu-id="ec970-130">Wenn Sie keinen Betrag eingeben, wird automatisch der auszugleichende Betrag für Sie bestimmt.</span><span class="sxs-lookup"><span data-stu-id="ec970-130">If you don't enter an amount, the amount to settle will automatically be determined for you.</span></span>
12. <span data-ttu-id="ec970-131">Klicken Sie auf **In Erfassung speichern**.</span><span class="sxs-lookup"><span data-stu-id="ec970-131">Click **Save in journal**.</span></span> <span data-ttu-id="ec970-132">Bevor Sie die Zahlung in der Erfassung speichern, vergewissern Sie sich, dass das Gegenkonto definiert ist.</span><span class="sxs-lookup"><span data-stu-id="ec970-132">Before you save the payment to the journal, make sure that the offset account is defined.</span></span> <span data-ttu-id="ec970-133">Bei **In Erfassung speichern** wird die Zahlung gespeichert und in die Erfassung verschoben.</span><span class="sxs-lookup"><span data-stu-id="ec970-133">Using **Save in journal** will save the payment and move it to the journal.</span></span> <span data-ttu-id="ec970-134">Sie können dann mit dem Eingeben der nächsten Zahlung fortfahren.</span><span class="sxs-lookup"><span data-stu-id="ec970-134">You can then continue entering the next payment.</span></span>
13. <span data-ttu-id="ec970-135">Schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="ec970-135">Close the page.</span></span>
14. <span data-ttu-id="ec970-136">Klicken Sie im **Aktivitätsbereich** auf „Zeilen“.</span><span class="sxs-lookup"><span data-stu-id="ec970-136">On **Action pane**, click Lines.</span></span> <span data-ttu-id="ec970-137">Beim Öffnen von Positionen sehen Sie alle Zahlungen, die Sie auf der Seite **Debitorenzahlungen eingeben** erfasst und in der Erfassung gespeichert haben.</span><span class="sxs-lookup"><span data-stu-id="ec970-137">When opening Lines, you will see any payments you recorded on the **Enter customer payments** page and saved into the journal.</span></span> <span data-ttu-id="ec970-138">Sie können diese Seite auch nutzen, um neue Debitorenzahlungen einzugeben oder bestehende Debitorenzahlungen zu bearbeiten, bevor sie gebucht werden.</span><span class="sxs-lookup"><span data-stu-id="ec970-138">You can also use this page to enter new customer payments, or edit existing customer payment before they are posted.</span></span>
15. <span data-ttu-id="ec970-139">Klicken Sie auf **Neu**, um eine weitere Zahlung zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="ec970-139">Click **New** to create another payment.</span></span> 
16. <span data-ttu-id="ec970-140">Wählen Sie den Debitor aus, von dem Sie die Zahlung erhalten haben.</span><span class="sxs-lookup"><span data-stu-id="ec970-140">Select the customer from whom you received the payment.</span></span> <span data-ttu-id="ec970-141">Wenn Sie den Debitor nicht kennen, jedoch wissen, dass durch die Zahlung eine Rechnung bezahlt wurde, verwenden Sie das Rechnungsfeld, um die Rechnung manuell einzugeben oder auszuwählen.</span><span class="sxs-lookup"><span data-stu-id="ec970-141">If you don't know the customer but know an invoice paid by the payment, use the Invoice field to manually enter or select the invoice.</span></span> <span data-ttu-id="ec970-142">Der Debitor wird standardmäßig festgelegt, nachdem die Rechnung ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="ec970-142">The customer will default after the invoice is selected.</span></span>  
17. <span data-ttu-id="ec970-143">Klicken Sie auf **Transaktionen ausgleichen**, um Rechnungen zu markieren, die bezahlt wurden.</span><span class="sxs-lookup"><span data-stu-id="ec970-143">Click **Settle transactions** to mark invoices that were paid.</span></span> <span data-ttu-id="ec970-144">Sie müssen die Zahlung nicht mit Rechnungen ausgleichen.</span><span class="sxs-lookup"><span data-stu-id="ec970-144">You are not required to settle the payment to any invoices.</span></span> <span data-ttu-id="ec970-145">Wenn es sich um eine Vorauszahlung handelt oder Sie nicht wissen, welche Rechnung bezahlt wurde, können Sie die Zahlung eingeben und buchen.</span><span class="sxs-lookup"><span data-stu-id="ec970-145">If this is a prepayment or if you don't know what invoice was paid, you can enter and post the payment.</span></span> <span data-ttu-id="ec970-146">Die Zahlung kann einer Rechnung zu einem späteren Zeitpunkt zugeordnet werden.</span><span class="sxs-lookup"><span data-stu-id="ec970-146">The payment can be settled to an invoice at a later point.</span></span>  
18. <span data-ttu-id="ec970-147">Markieren Sie bezahlte Rechnungen.</span><span class="sxs-lookup"><span data-stu-id="ec970-147">Mark the invoices paid by the payment.</span></span> 
19. <span data-ttu-id="ec970-148">Geben Sie im Feld **Betrag** den Betrag der Zahlung ein, der mit der ausgewählten Rechnung ausgeglichen werden soll.</span><span class="sxs-lookup"><span data-stu-id="ec970-148">In the **Amount** field, enter the amount of the payment that will be settled to the invoice.</span></span>
20. <span data-ttu-id="ec970-149">Klicken Sie auf **OK**.</span><span class="sxs-lookup"><span data-stu-id="ec970-149">Click **OK**.</span></span>
21. <span data-ttu-id="ec970-150">Geben Sie im Feld **Zahlungsreferenz** eine Zahlungsreferenz ein.</span><span class="sxs-lookup"><span data-stu-id="ec970-150">In the **Payment reference** field, enter a payment reference.</span></span> <span data-ttu-id="ec970-151">Die Zahlungsreferenz ist nur erforderlich, wenn Sie angegeben haben, dass die Zahlung auf einem Einzahlungsbeleg enthalten sein soll.</span><span class="sxs-lookup"><span data-stu-id="ec970-151">The payment reference is only required if you mark to include the payment on a deposit slip.</span></span>  
22. <span data-ttu-id="ec970-152">Klicken Sie im **Aktionsbereich** auf **Buchen**, um die Debitorenzahlungen zu buchen.</span><span class="sxs-lookup"><span data-stu-id="ec970-152">On **Action pane**, click **Post** to post the customer payments.</span></span>
