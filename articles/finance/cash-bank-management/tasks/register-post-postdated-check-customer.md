# <a name="register-and-post-a-postdated-check-for-a-customer"></a>Einen vordatierten Scheck für einen Debitor erfassen und buchen

[!include [banner](../../includes/banner.md)]

Sie können die Details eines vordatierten Schecks erfassen, den Sie von einem Debitor erhalten haben. Sie können den vordatierten Scheck buchen und Finanzbuchungen erstellen.   Führen Sie die folgenden Schritte aus, bevor Sie einen vordatierten Scheck erfasst und gebucht, der von einem Debitor erhalten haben:   * Einrichten von vordatierten Schecks in bar und in der Bankwesenseite * Einrichten einer Zahlungsmethode für vordatierte Schecks, für die der Rolle für diesen Schritt Schatzmeister ist. Für diese Prozedur wird das Demo-Unternehmen USMF verwendet.

1. Wechseln Sie zu "Debitoren" > "Zahlungen" > "Zahlungserfassung".
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Name" einen Wert ein.
4. Klicken Sie auf "Positionen".
5. Markieren Sie in der Liste die ausgewählte Zeile.
6. Geben Sie im Feld "Konto" die gewünschten Werte an.
7. Geben Sie im Feld "Kredit" eine Zahl ein.
    * Geben Sie den im vordatierten Scheck angegebene Betrag an.  
8. Klicken Sie auf die Registerkarte Zahlung.
9. Geben Sie im Feld "Zahlungsmethode" einen Wert ein.
    * Wählen Sie die Zahlungsmethode für den vordatierten Scheck aus.  
10. Klicken Sie auf die Registerkarte "Vordatierte Schecks".
11. Geben Sie im Feld "Fälligkeitsdatum" ein Datum ein.
    * Geben Sie das Datum an, an dem der vordatierte Scheck zur Zahlung fällig ist.  
12. Geben Sie im Feld "Ausstellende Bankfiliale" einen Wert ein.
    * Geben Sie die Details der Bank auf dem vordatierten Scheck ein.  
13. Geben Sie im Feld "Schecknummer" einen Wert ein.
14. Geben Sie im Feld "Ausstellender Bankname" einen Wert ein.
    * Geben Sie die Details der Bank auf dem vordatierten Scheck ein.  
15. Klicken Sie auf "Buchen".
16. Schließen Sie die Seite.
