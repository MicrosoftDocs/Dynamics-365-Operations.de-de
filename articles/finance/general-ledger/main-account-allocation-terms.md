<span data-ttu-id="1cf84-104">Dieses Thema enthält Informationen zur Verwendung von Zuweisungsbedingungen für ein Hauptkonto.</span><span class="sxs-lookup"><span data-stu-id="1cf84-104">This topic provides information about using allocation terms on a main account.</span></span> <span data-ttu-id="1cf84-105">Zuweisungsbedingungen werden verwendet, um Beträge auf mehrere Sachkontokombinationen zu verteilen.</span><span class="sxs-lookup"><span data-stu-id="1cf84-105">Allocation terms are used to distribute amounts across multiple ledger account combinations.</span></span> <span data-ttu-id="1cf84-106">Sie unterstützen, um sicherzustellen, dass Ausgaben oder Umsatzerlöse zum richtigen Objekt Buchhaltungsmodul in der Buchhaltung berechnet wird.</span><span class="sxs-lookup"><span data-stu-id="1cf84-106">They help ensure that expenses or revenues are charged to the correct object in accounting.</span></span>

<span data-ttu-id="1cf84-107">Jede Zuweisungsbedingung, die Sie auf einem Hauptkonto erstellen, legt den Prozentsatz eines Gutscheins, der von einem Hauptkonto aus einer einzigen Quelle zugewiesen werden soll, und eine Kombination aus Finanzdimensionen fest.</span><span class="sxs-lookup"><span data-stu-id="1cf84-107">Each allocation term that you create on a main account defines the percentage of a voucher to be allocated from a single-source main account and a combination of financial dimensions.</span></span> <span data-ttu-id="1cf84-108">Darüber hinaus definieren Sie das Zielhauptkonto und die Finanzdimensionen, denen der Betrag zugewiesen wird.</span><span class="sxs-lookup"><span data-stu-id="1cf84-108">In addition, you define the destination main account and financial dimensions where the amount will be allocated.</span></span>

<span data-ttu-id="1cf84-112">Wenn Sie das Zielsachkonto für eine Zuweisungsbedingung festlegen, müssen Sie das Hauptkonto angeben, dem Sie den Betrag zuordnen.</span><span class="sxs-lookup"><span
