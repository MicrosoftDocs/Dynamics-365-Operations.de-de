[!include [banner](../includes/banner.md)]

<span data-ttu-id="65883-104">In diesem Thema wird erläutert, wie Sie den Kalender ändern, der einem Sachkonto derzeit zugewiesen ist, und wie Sie diesem Sachkonto einen neuen Kalender zuweisen.</span><span class="sxs-lookup"><span data-stu-id="65883-104">This topic explains how to change the calendar that is currently assigned to a ledger, and how to assign a new calendar to the ledger.</span></span>

## <a name="issue"></a><span data-ttu-id="65883-105">Problem</span><span class="sxs-lookup"><span data-stu-id="65883-105">Issue</span></span>

<span data-ttu-id="65883-106">Unternehmen müssen gelegentlich entweder den vorhandenen Kalender ändern, der einem Sachkonto zugeordnet ist, oder einen neuen Kalender zuweisen.</span><span class="sxs-lookup"><span data-stu-id="65883-106">Sometime, companies must either change the existing calendar that is assigned to a ledger or assign a new calendar.</span></span>

## <a name="resolution"></a><span data-ttu-id="65883-107">Lösung</span><span class="sxs-lookup"><span data-stu-id="65883-107">Resolution</span></span>

<span data-ttu-id="65883-108">Es gibt zwei Szenarien zum Ändern oder Neuzuweisen eines Sachkontokalenders.</span><span class="sxs-lookup"><span data-stu-id="65883-108">There are two scenarios for changing or reassigning a ledger calendar.</span></span> <span data-ttu-id="65883-109">Das erste Szenario umfasst die Änderung eines vorhandenen Kalenders, der dem Sachkonto bereits zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="65883-109">The first scenario involves changing an existing calendar that is already assigned to the ledger.</span></span> <span data-ttu-id="65883-110">Das zweite Szenario besteht in der Erstellung eines neuen Kalenders und dessen Zuweisung zum Sachkonto.</span><span class="sxs-lookup"><span data-stu-id="65883-110">The second scenario involves creating a new calendar and assigning it to the ledger.</span></span> <span data-ttu-id="65883-111">Beide Änderungen können jederzeit vorgenommen werden, auch nachdem Buchungen im Sachkonto gebucht wurden.</span><span class="sxs-lookup"><span data-stu-id="65883-111">Both changes can be done at any time, even after transactions have been posted to the ledger.</span></span>

### <a name="change-an-existing-fiscal-calendar"></a><span data-ttu-id="65883-112">Einen vorhandenen Geschäftskalender ändern</span><span class="sxs-lookup"><span data-stu-id="65883-112">Change an existing fiscal calendar</span></span>

<span data-ttu-id="65883-113">Um einen vorhandenen Kalender zu ändern, der Ihrem Sachkonto zugewiesen ist, wechseln Sie zu **Hauptbuch \> Einrichten des Hauptbuchs \> Sachkonto** und wählen den Link für den Finanzkalender aus, um die Seite **Sachkontokalender** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="65883-113">To change an existing calendar that is assigned to your ledger, go to **General ledger \> Ledger setup \> Ledger**, and select the link for the fiscal calendar to open the **Ledger calendars** page.</span></span>

<span data-ttu-id="65883-114">Es bestehen Beschränkungen bei den möglichen Änderungen eines Kalenders.</span><span class="sxs-lookup"><span data-stu-id="65883-114">There are limits to the changes that can be made to a calendar.</span></span> <span data-ttu-id="65883-115">Beispielsweise können Sie das Start- und Enddatum der *Perioden* in einem Jahr ändern, Sie können jedoch das Start- und Enddatum eines *Jahres* im Kalender nicht ändern.</span><span class="sxs-lookup"><span data-stu-id="65883-115">For example, you can change the start and end dates of the *periods* in a year, but you can't change the start and end dates of a *year* in the calendar.</span></span>

<span data-ttu-id="65883-116">Um die Perioden in einem Jahr zu ändern, wählen Sie den entsprechenden Kalender und das entsprechende Jahr aus.</span><span class="sxs-lookup"><span data-stu-id="65883-116">To change the periods in a year, select the appropriate calendar and year.</span></span> <span data-ttu-id="65883-117">Verwenden Sie zunächst die Schaltfläche **Periode löschen** aus, um einige oder alle vorhandenen Betriebsperioden zu löschen.</span><span class="sxs-lookup"><span data-stu-id="65883-117">First, use the use the **Delete period** button to delete some or all of the existing operating periods.</span></span> <span data-ttu-id="65883-118">Alle Betriebsperioden außer der ersten können gelöscht werden.</span><span class="sxs-lookup"><span data-stu-id="65883-118">All operating periods except the first can be deleted.</span></span> <span data-ttu-id="65883-119">Verwenden Sie anschließend die Schaltfläche **Periode aufteilen**, um die verbleibende Periode oder Perioden in neue Perioden zu unterteilen, indem Sie ein geeignetes Startdatum für die nächste Periode eingeben.</span><span class="sxs-lookup"><span data-stu-id="65883-119">Then use the **Divide period** button to divide the remaining period or periods into new periods by entering an appropriate start date for the next period.</span></span>

<span data-ttu-id="65883-120">Nachdem Sie die Perioden in einem Kalender geändert haben, müssen Sie den Prozess **Sachkontoperioden neu berechnen** für jede juristische Person (Firma) ausführen, der der Kalender zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="65883-120">After you change the periods in a calendar, you must run the **Recalculate ledger periods** process in every legal entity (company) that the calendar is assigned to.</span></span> <span data-ttu-id="65883-121">Auch wenn Sie keinen Warnhinweis zum Abschließen des Schritts erhalten, ist der Neuberechnungsprozess erforderlich, um die Periode zu aktualisieren, die jeder gebuchten Buchung im Sachkonto zugewiesen ist.</span><span class="sxs-lookup"><span data-stu-id="65883-121">Although no warning message reminds you to complete this step, the recalculation process is required to update the period that is assigned to each posted transaction in General ledger.</span></span> <span data-ttu-id="65883-122">Wählen Sie **Sachkontoperioden neu berechnen** auf der Seite **Einrichten des Hauptbuchs** in jeder Firma aus, um den Neuberechnungsprozess auszuführen.</span><span class="sxs-lookup"><span data-stu-id="65883-122">To run the recalculation process, select **Recalculate ledger periods** on the **Ledger setup** page in each company.</span></span>

<span data-ttu-id="65883-123">Wenn der Neuberechnungsprozess nicht ausgeführt wird, werden Buchungssalden in einer Zwischenbilanz oder in Finanzabschlüssen möglicherweise fälschlicherweise in Perioden einbezogen.</span><span class="sxs-lookup"><span data-stu-id="65883-123">If the recalculation process isn't run, transaction balances on a trial balance or on financial statements might be incorrectly included in periods.</span></span> <span data-ttu-id="65883-124">In diesem Fall können Sie die Salden jederzeit korrigieren, indem Sie den Neuberechnungsprozess ausführen.</span><span class="sxs-lookup"><span data-stu-id="65883-124">In this case, you can correct the balances at any time by running the recalculation process.</span></span>

### <a name="assign-a-new-fiscal-calendar"></a><span data-ttu-id="65883-125">Einen neuen Geschäftskalender zuweisen</span><span class="sxs-lookup"><span data-stu-id="65883-125">Assign a new fiscal calendar</span></span>

<span data-ttu-id="65883-126">Um einen neuen Geschäftskalender zuzuweisen, gehen Sie zu **Hauptbuch \> Einrichten des Hauptbuchs \> Sachkonto** und wählen dort **Bearbeiten** aus, um die Seite **Sachkonto** in den Bearbeitungsmodus zu versetzen.</span><span class="sxs-lookup"><span data-stu-id="65883-126">To assign a new fiscal calendar, go to **General ledger \> Ledger setup \> Ledger**, and select **Edit** to put the **Ledger** page into edit mode.</span></span> <span data-ttu-id="65883-127">Wählen Sie im Feld **Geschäftskalender** anschließend einen neuen Geschäftskalender aus.</span><span class="sxs-lookup"><span data-stu-id="65883-127">Then, in the **Fiscal calendar** field, select a new fiscal calendar.</span></span>

<span data-ttu-id="65883-128">Nachdem Sie den Geschäftskalender für ein Sachkonto geändert haben, müssen Sie **Sachkontoperioden neu berechnen** ausführen.</span><span class="sxs-lookup"><span data-stu-id="65883-128">After you change the fiscal calendar for a ledger, you must run the **Recalculate ledger periods**.</span></span> <span data-ttu-id="65883-129">Wenn Sie einem Sachkonto einen neuen Geschäftskalender zuweisen, werden Sie in einer Nachricht daran erinnert, diesen Schritt auszuführen.</span><span class="sxs-lookup"><span data-stu-id="65883-129">When you assign a new fiscal calendar to a ledger, a message reminds you to complete this step.</span></span> <span data-ttu-id="65883-130">Obwohl die Meldung möglicherweise darauf hinweist, dass der Neuberechnungsprozess optional ist, ist dies nicht der Fall.</span><span class="sxs-lookup"><span data-stu-id="65883-130">Although the message might seem to indicate that the recalculation process is optional, it isn't.</span></span> <span data-ttu-id="65883-131">Es ist erforderlich, die Periode zu aktualisieren, die den einzelnen gebuchten Buchungen im Hauptbuch zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="65883-131">It's required to update the period that is assigned to each posted transaction in General ledger.</span></span> <span data-ttu-id="65883-132">Wählen Sie **Sachkontoperioden neu berechnen** auf der Seite **Einrichten des Hauptbuchs** aus, um den Neuberechnungsprozess auszuführen.</span><span class="sxs-lookup"><span data-stu-id="65883-132">To run the recalculation process, select **Recalculate ledger periods** on the **Ledger setup** page.</span></span>

<span data-ttu-id="65883-133">Wenn der Neuberechnungsprozess nicht ausgeführt wird, werden Buchungssalden in einer Zwischenbilanz oder in Finanzabschlüssen möglicherweise fälschlicherweise in Perioden einbezogen.</span><span class="sxs-lookup"><span data-stu-id="65883-133">If the recalculation process isn't run, transaction balances on a trial balance or on financial statements might be incorrectly included in periods.</span></span> <span data-ttu-id="65883-134">In diesem Fall können Sie die Salden jederzeit korrigieren, indem Sie den Neuberechnungsprozess ausführen.</span><span class="sxs-lookup"><span data-stu-id="65883-134">In this case, you can correct the balances at any time by running the recalculation process.</span></span>
