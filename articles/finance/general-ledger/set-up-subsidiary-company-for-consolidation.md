## <a name="map-subsidiary-main-accounts-to-consolidated-main-accounts"></a><span data-ttu-id="43aac-109">Hauptkonten der Tochtergesellschaft konsolidierten Hauptkonten zuordnen</span><span class="sxs-lookup"><span data-stu-id="43aac-109">Map subsidiary main accounts to consolidated main accounts</span></span>

<span data-ttu-id="43aac-110">Wenn der Kontenplan in der juristischen Personen der Tochtergesellschaft nicht dem Kontenplan in der konsolidierten juristischen Person entspricht, können Sie die Hauptkonten in der Tochtergesellschaft den Hauptkonten in der konsolidierten juristischen Person zuordnen.</span><span class="sxs-lookup"><span data-stu-id="43aac-110">If the chart of accounts in the subsidiary legal entity doesn't follow the chart of accounts in the consolidated legal entity, you can map the main accounts in the subsidiary to the main accounts in the consolidated legal entity.</span></span>

2. <span data-ttu-id="43aac-112">Wählen Sie einen Kontenplan aus.</span><span class="sxs-lookup"><span data-stu-id="43aac-112">Select a chart of accounts.</span></span> <span data-ttu-id="43aac-113">Wählen Sie im Inforegister **Hauptkonten** ein Hauptkonto und anschließend **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="43aac-113">On the **Main accounts** FastTab, select a main account, and then select **Edit**.</span></span>

    > [!NOTE]
    > <span data-ttu-id="43aac-117">Wenn sich die Hauptkontotypen der Tochtergesellschaftskonten, die zugeordnet werden, von den Hauptkontotypen der Konten in der konsolidierten juristischen Person unterscheiden, werden die Werte von Konten, die den Hauptkontotyp **Gesamt** haben, während der Konsolidierung überschrieben.</span><span class="sxs-lookup"><span data-stu-id="43aac-117">If the main account types of the subsidiary accounts that are mapped differ from the main account types of the accounts in the consolidated legal entity, the values of accounts that have a main account type of **Total** are overwritten during consolidation.</span></span>

    1. <span data-ttu-id="43aac-120">Rufen Sie in der *juristischen Person der Tochtergesellschaft* **Hauptbuch \> Einrichtung \> Finanzdimensionen \> Finanzdimensionen** auf. Wählen Sie eine Finanzdimension und anschließend **Finanzdimensionswerte** aus.</span><span class="sxs-lookup"><span data-stu-id="43aac-120">In the *subsidiary legal entity*, go to **General ledger \> Setup \> Financial dimensions \> Financial dimensions**, select a financial dimension, and then select **Financial dimension values**.</span></span>
    2. <span data-ttu-id="43aac-121">Wählen Sie den Finanzdimensionswert aus, der einem anderen Finanzdimensionswert in der konsolidierten juristischen Person zugeordnet werden soll.</span><span class="sxs-lookup"><span data-stu-id="43aac-121">Select the financial dimension value to map to a different financial dimension value in the consolidated legal entity.</span></span>

5. <span data-ttu-id="43aac-126">Wenn Sie eine Online-Konsolidierung durchführen, führen Sie die folgenden Schritte aus, um sicherzustellen, dass die Übertragungen wie beabsichtigt erfolgen:</span><span class="sxs-lookup"><span data-stu-id="43aac-126">If you're doing an online consolidation, follow these steps to ensure that the transfers occur as you intend:</span></span>

    1. <span data-ttu-id="43aac-127">Rufen Sie in der *konsolidierte juristischen Person* **Hauptbuch \> Periodisch \> Konsolidieren \> Konsolidieren \[Exportieren nach\]** auf, um die Seite **Konsolidieren \[Online\]** zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="43aac-127">In the *consolidated legal entity*, go to **General ledger \> Periodic \> Consolidate \> Consolidate \[Export to\]** to open the **Consolidate \[Online\]** page.</span></span>
    2. <span data-ttu-id="43aac-128">Aktivieren Sie auf der Registerkarte **Kriterien** das Kontrollkästchen **Konsolidierungskonto verwenden**.</span><span class="sxs-lookup"><span data-stu-id="43aac-128">On the **Criteria** tab, select the **Use consolidation account** check box.</span></span>
    3. <span data-ttu-id="43aac-129">Wählen Sie auf der Registerkarte **Finanzdimensionen** die entsprechenden Finanzdimensionen aus.</span><span class="sxs-lookup"><span data-stu-id="43aac-129">On the **Financial dimensions** tab, select the appropriate financial dimensions.</span></span>
    4. <span data-ttu-id="43aac-130">Geben Sie für jede Finanzdimension, die Sie auswählen, eine Zahl in das Feld **Segmentreihenfolge** ein, um die Reihenfolge festzulegen, in der die Dimensionen angezeigt werden sollen.</span><span class="sxs-lookup"><span data-stu-id="43aac-130">For each financial dimension that you select, enter a number in the **Segment order** field to indicate the order that the dimensions should appear in.</span></span>
