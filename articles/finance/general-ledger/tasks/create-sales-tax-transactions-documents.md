# <a name="create-sales-tax-transactions-on-documents"></a>Mehrwertsteuerbuchungen in Dokumenten erstellen

[!include [banner](../../includes/banner.md)]

Mehrwertsteuer zu Dokumenten wird berechnet, indem eine "Mehrwertsteuergruppe" und eine "Artikel-Mehrwertsteuergruppe" auf Dokumentpositionen bereitgestellt werden. Diese werden standardmäßig aus Masterdaten bereitgestellt, sie können aber bei Bedarf manuell geändert werden. Die berechnete Mehrwertsteuer kann auf einer Positions und einer Dokumentebene überprüft werden. Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.

1. Wechseln Sie zu "Debitoren" > "Aufträge" > "Alle Aufträge".
2. Klicken Sie auf "Neu".
3. Klicken Sie im Feld "Debitorenkonto" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
4. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
5. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
6. Klicken Sie auf "OK".
7. Markieren Sie in der Liste die ausgewählte Zeile.
8. Klicken Sie im Feld "Artikelnummer" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
9. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
10. Geben Sie im Feld "Einzelpreis" eine Zahl ein.
11. Erweitern oder reduzieren Sie den Abschnitt "Positionsdetails".
12. Klicken Sie auf die Registerkarte Einstellungen.
13. Klicken Sie im Feld "Mehrwertsteuergruppe" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
14. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
15. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
16. Klicken Sie auf "Finanzdaten".
17. Klicken Sie auf "Mehrwertsteuer".
18. Klicken Sie auf "OK".
19. Klicken Sie auf "Position hinzufügen".
20. Markieren Sie in der Liste die ausgewählte Zeile.
21. Klicken Sie im Feld "Artikelnummer" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
22. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
23. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
24. Geben Sie im Feld "Einzelpreis" eine Zahl ein.
25. Klicken Sie im Feld "Mehrwertsteuergruppe" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
26. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
27. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
28. Klicken Sie im Aktivitätsbereich auf "Verkaufen".
29. Klicken Sie auf "Mehrwertsteuer".
30. Klicken Sie auf "OK".
