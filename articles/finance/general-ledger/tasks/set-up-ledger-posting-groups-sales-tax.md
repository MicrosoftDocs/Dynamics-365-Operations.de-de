# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="098e1-103">Sachkontobuchungsgruppen für Mehrwertsteuer einrichten</span><span class="sxs-lookup"><span data-stu-id="098e1-103">Set up Ledger posting groups for sales tax</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="098e1-104">Die Mehrwertsteuer wird zu den Hauptkonten berechnet und gebucht, die in den "Sachkontobuchungsgruppen" angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="098e1-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="098e1-105">Sachkontobuchungsgruppen werden jedem einzelnen Mehrwertsteuercode zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="098e1-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="098e1-106">Sie können einzelne Sachkontobuchungsgruppen für jeden Mehrwertsteuercode einrichten, eine Sachkontobuchungsgruppe für alle Mehrwertsteuercodes verwenden oder mehrere Sachkontobuchungsgruppen den Mehrwertsteuercodes zuweisen.</span><span class="sxs-lookup"><span data-stu-id="098e1-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="098e1-107">Für diese Erfassung wird das Demo-Unternehmen DEMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="098e1-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="098e1-108">Gehen Sie zu **Navigationsbereich > Module > Steuer > Einrichtung > Umsatzsteuer > Kontenbuchungsgruppen**.</span><span class="sxs-lookup"><span data-stu-id="098e1-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="098e1-109">Klicken Sie auf **Neu**.</span><span class="sxs-lookup"><span data-stu-id="098e1-109">Click **New**.</span></span>
3. <span data-ttu-id="098e1-110">Geben Sie im Feld **Kontenbuchungsgruppe** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="098e1-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="098e1-111">Geben Sie im Feld **Beschreibung** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="098e1-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="098e1-112">Wählen Sie im Feld **Verkaufssteuerverbindlichkeiten** das Hauptkonto für ausgehende Umsatzsteuern, die an die Steuerbehörde zu zahlen sind.</span><span class="sxs-lookup"><span data-stu-id="098e1-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="098e1-113">Mehrwertsteuern werden im Namen der Steuerbehörde erhoben, wenn Sie steuerpflichtige Waren und Dienstleistungen verkaufen.</span><span class="sxs-lookup"><span data-stu-id="098e1-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="098e1-114">Wählen Sie im Feld **Verkaufsforderung** das Hauptkonto für eingehende Steuern, die von der Steuerbehörde erhalten werden.</span><span class="sxs-lookup"><span data-stu-id="098e1-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="098e1-115">Händler erheben Steuern im Auftrag der Steuerbehörde, wenn Sie steuerpflichtige Waren und Dienstleistungen kaufen.</span><span class="sxs-lookup"><span data-stu-id="098e1-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="098e1-116">Dieses Feld ist nicht verfügbar, wenn auf der Seite **Hauptbuchparameter** die Option Umsatzsteuerregeln anwenden markiert ist.</span><span class="sxs-lookup"><span data-stu-id="098e1-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="098e1-117">Stattdessen werden an Händler gezahlte Mehrwertsteuerbeträge vom selben Konto abgebucht, wie der Einkauf.</span><span class="sxs-lookup"><span data-stu-id="098e1-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="098e1-118">Wählen Sie im Feld **Steueraufwand verwenden** das Hauptkonto für die Buchung von abzugsfähigen Nutzungssteuern, die von den Verkäufern im Rahmen der EU-Reversion GST/HST nicht in Anspruch genommen oder an die Steuerbehörde gemeldet werden.</span><span class="sxs-lookup"><span data-stu-id="098e1-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="098e1-119">Die Option **Steuer verwenden** muss für das **Verkaufssteuerkennzeichen** im **Verkaufssteuerkennzeichen** ausgewählt werden, das in der Transaktion verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="098e1-119">The **Use tax** option needs to be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="098e1-120">Dieses Feld ist nicht verfügbar, wenn auf der Seite **Hauptbuchparameter** die Option **Umsatzsteuerregeln anwenden** markiert ist.</span><span class="sxs-lookup"><span data-stu-id="098e1-120">This field is not be available if the **Apply sales tax taxation rules** option is selected in the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="098e1-121">Wählen Sie im Feld **Verwendung der Steuerverbindlichkeiten** das Hauptkonto für die Buchung eingehender Verwendungssteuern, die an die Steuerbehörden zu zahlen sind.</span><span class="sxs-lookup"><span data-stu-id="098e1-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="098e1-122">Die Option **Steuer verwenden** muss in der Option **Umsatzsteuerkennzeichen** in der Option **Umsatzsteuerkennzeichen** ausgewählt sein, um **Steuer verwenden** zu buchen.</span><span class="sxs-lookup"><span data-stu-id="098e1-122">The **Use tax** option needs to be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="098e1-123">Wenn auf der Seite **Hauptbuchparameter** die Option **Verkaufssteuerregeln anwenden** ausgewählt ist, wird der Ausgleich auf das Aufwandskonto des Geschäfts gebucht.</span><span class="sxs-lookup"><span data-stu-id="098e1-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="098e1-124">Wählen Sie im Feld **Verrechnungskonto** das Hauptkonto aus, auf das der Nettosaldo der in den Feldern **Verwendungssteuerverbindlichkeiten** und **Verkaufssteuerforderungen** angegebenen Sachkonten gebucht werden soll.</span><span class="sxs-lookup"><span data-stu-id="098e1-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="098e1-125">Der Saldo wird bei der Umsatzsteuerabrechnung und dem Buchungsjob erstellt.</span><span class="sxs-lookup"><span data-stu-id="098e1-125">The balance will be created when the sales tax settle and post job is ran.</span></span>  <span data-ttu-id="098e1-126">Wenn die Steuerbehörde für den Abrechnungszeitraum einem Kreditorenkonto zugeordnet ist, wird der Saldo stattdessen auf das Kreditorenkonto gebucht.</span><span class="sxs-lookup"><span data-stu-id="098e1-126">If the tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>
10. <span data-ttu-id="098e1-127">Wählen Sie im Feld **Lieferant Skonto** das Hauptkonto zum Buchen von Skonto für Umsatzsteuerkennzeichen, die dieser Ledger-Buchungsgruppe zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="098e1-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="098e1-128">Dies ist optional und wenn kein Konto eingegeben wird, wird das Hauptkonto unter **Skontoabzugscodes** verwendet.</span><span class="sxs-lookup"><span data-stu-id="098e1-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="098e1-129">Es kann sinnvoll sein, unterschiedliche Konten pro **Ledger-Buchungsgruppe** zu verwenden, wenn Sie die Option Umsatzsteuer auf Skonto umgekehrt auf Umsatzsteuergruppen verwenden.</span><span class="sxs-lookup"><span data-stu-id="098e1-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="098e1-130">Wählen Sie im Feld **Kundenfallrabatt** das Hauptkonto zum Buchen von Skonto für **Verkaufssteuerkennzeichen**, das dieser **Sachkonto-Buchungsgruppe** zugeordnet ist.</span><span class="sxs-lookup"><span data-stu-id="098e1-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="098e1-131">Dies ist optional und wenn kein Konto eingegeben wird, wird das Hauptkonto auf den **Skontoabzugscodes** verwendet.</span><span class="sxs-lookup"><span data-stu-id="098e1-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="098e1-132">Es kann sinnvoll sein, unterschiedliche Konten pro **Sachkonto-Buchungsgruppe** zu verwenden, wenn Sie die Option Umsatzsteuer auf Skonto auf **Verkaufssteuergruppen** verwenden.</span><span class="sxs-lookup"><span data-stu-id="098e1-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="098e1-133">Klicken Sie auf **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="098e1-133">Click **Save**.</span></span>
