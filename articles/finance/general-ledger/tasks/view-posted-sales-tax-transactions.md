# <a name="view-posted-sales-tax-transactions"></a>Vorgenommene Mehrwertsteuerbuchungen anzeigen

[!include [task guide banner](../../includes/task-guide-banner.md)]

In diesem Thema werden einige Möglichkeiten beschrieben, gebuchte Umsatzsteuertransaktionen anzuzeigen. Sie können gebuchte Umsatzsteuertransaktionen von verschiedenen Seiten aus anzeigen, je nachdem, was Sie sehen möchten. Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.

# <a name="posted-sales-tax-page"></a>Veröffentlichte Umsatzsteuer-Seite

1. Gehen Sie im Navigationsbereich zu **Module > Steuern > Anfragen und Berichte > Umsatzsteueranfragen > Gebuchte Umsatzsteuer**.
2. Wählen Sie **Filter anzeigen**, um die Ansicht auf bestimmte Transaktionen einzuschränken.
3. Schließen Sie die Seite.

# <a name="sales-tax-settlement-periods-page"></a>Seite Abrechnungsperioden Umsatzsteuer

1. Gehen Sie im Navigationsbereich zu **Module > Steuer > Indirekte Steuern > Umsatzsteuer > Umsatzsteuer > Umsatzsteuerabrechnungsperioden**.
2. Erweitern Sie den Abschnitt **Periodenintervalle**.
3. Wählen Sie das gewünschte Intervall aus.
4. Sie können die Liste gebuchter Mehrwertsteuertransaktionen weiter filtern.
5. Schließen Sie die Seite.

# <a name="sales-tax-codes-page"></a>Seite mit den Umsatzsteuerkennzeichen

1. Gehen Sie im Navigationsbereich zu **Module > Steuern > Indirekte Steuern > Umsatzsteuer > Umsatzsteuer > Umsatzsteuerkennzeichen**.
2. Wählen Sie im Aktionsbereich **Verkaufssteuerkennzeichen**.
3. Wählen Sie **gebuchte Umsatzsteuer**.
