[!include [banner](../../includes/banner.md)]

2. <span data-ttu-id="8d165-106">Wählen Sie das Feld aus, für das Sie ein Filterkriterium definieren wollen.</span><span class="sxs-lookup"><span data-stu-id="8d165-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="8d165-107">Geben Sie die Filterkriterien für das ausgewählte Feld ein.</span><span class="sxs-lookup"><span data-stu-id="8d165-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="8d165-108">Sie können einen einzelnen Wert oder einen Bereich filtern.</span><span class="sxs-lookup"><span data-stu-id="8d165-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="8d165-109">Wenn Sie einen Bereich definiert, stellen Sie sicher, dass die richtige Syntax verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="8d165-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="8d165-110">Die Werte sollten durch einen doppelten Punkt getrennt sein (..).</span><span class="sxs-lookup"><span data-stu-id="8d165-110">The values should be separated by a double period (..).</span></span>

5. <span data-ttu-id="8d165-112">Wählen Sie im Baum **Tabellen/Allgemeine Journalbuchung**.</span><span class="sxs-lookup"><span data-stu-id="8d165-112">In the tree, select **Tables/General journal entry**.</span></span>

8. <span data-ttu-id="8d165-115">Klicken Sie auf die Registerkarte **Bereich**.</span><span class="sxs-lookup"><span data-stu-id="8d165-115">Click the **Range** tab.</span></span>
