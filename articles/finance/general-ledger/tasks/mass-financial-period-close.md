[!include [banner](../../includes/banner.md)]

<span data-ttu-id="58862-104">Dieses Thema zeigt, wie Sie einen Zeitraum zurückstellen oder dauerhaft schließen können, oder wie Sie mehr als eine juristische Person auf einmal.</span><span class="sxs-lookup"><span data-stu-id="58862-104">This topic shows how to place a period on hold or permanently close a period or more than one legal entity at a time.</span></span> <span data-ttu-id="58862-105">Darüber hinaus zeigt die Aufgabe, wie die Benutzergruppenbuchun auf bestimmte Module beschränkt wird.</span><span class="sxs-lookup"><span data-stu-id="58862-105">In addition, the task will show how to restrict user group posting to specific modules.</span></span>

2. <span data-ttu-id="58862-109">Wählen Sie **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="58862-109">Select **Edit**.</span></span>
3. <span data-ttu-id="58862-110">Wählen Sie den Zeitraum aus, für den Sie den Status ändern möchten.</span><span class="sxs-lookup"><span data-stu-id="58862-110">Select the period for which you want to modify the status.</span></span>

9. <span data-ttu-id="58862-124">Wählen Sie eine andere Periode aus, um den Status zu aktualisieren.</span><span class="sxs-lookup"><span data-stu-id="58862-124">Select another period to update the status.</span></span>
