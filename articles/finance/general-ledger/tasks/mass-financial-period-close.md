[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="6a2ef-104">Dieses Thema zeigt, wie Sie einen Zeitraum zurückstellen oder dauerhaft schließen können, oder wie Sie mehr als eine juristische Person auf einmal.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-104">This topic shows how to place a period on hold or permanently close a period or more than one legal entity at a time.</span></span> <span data-ttu-id="6a2ef-105">Darüber hinaus zeigt die Aufgabe, wie die Benutzergruppenbuchun auf bestimmte Module beschränkt wird.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-105">In addition, the task will show how to restrict user group posting to specific modules.</span></span>

2. <span data-ttu-id="6a2ef-109">Wählen Sie **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-109">Select **Edit**.</span></span>
3. <span data-ttu-id="6a2ef-110">Wählen Sie den Zeitraum aus, für den Sie den Status ändern möchten.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-110">Select the period for which you want to modify the status.</span></span>
4. <span data-ttu-id="6a2ef-111">Wählen Sie die juristischen Personen aus, für den Sie den Status aktualisieren möchten.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-111">Select the legal entities for which you want to update the status.</span></span> <span data-ttu-id="6a2ef-112">Sie können alle Rechtspersonen schnell auswählen, indem Sie das Häkchen oben links im Raster markieren.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-112">You can quickly select all legal entities by selecting the check mark on the upper left side of the grid.</span></span>

6. <span data-ttu-id="6a2ef-116">Wählen Sie im Modul **Anwendung** das Modul aus, für das Sie den Status aktualisieren möchten.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-116">In the **Application** module, select the module that you want to update the status.</span></span> <span data-ttu-id="6a2ef-117">Klicken Sie auf **Auswählen**.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-117">Click **Select**.</span></span>

8. <span data-ttu-id="6a2ef-123">Wählen Sie **Aktualisieren**.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-123">Select **Update**.</span></span>
9. <span data-ttu-id="6a2ef-124">Wählen Sie eine andere Periode aus, um den Status zu aktualisieren.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-124">Select another period to update the status.</span></span>
10. <span data-ttu-id="6a2ef-125">Wählen Sie die juristischen Personen aus, für den Sie die Periode aktualisieren möchten.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-125">Select the legal entites for which you want to update the period status.</span></span>

12. <span data-ttu-id="6a2ef-132">Wählen Sie **Aktualisieren**.</span><span class="sxs-lookup"><span data-stu-id="6a2ef-132">Select **Update**.</span></span>
