# <a name="set-up-a-ledger-account-alias"></a>Einrichten eines Sachkontoalias

[!include [banner](../../includes/banner.md)]

Das folgende Verfahren zeigt die Erstellung eines Kontoalias als Kürzel zur Eingabe einer Kontonummer. Für diese Prozedur wird das Demodatunternehmen USMF verwendet.

1. Wechseln Sie zu "Hauptbuch" > "Kontenplan" > "Konten" > "Sachkontoalias".
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Sachkontoalias" einen Wert ein.
4. Wählen Sie im Feld "Kontostruktur" die Struktur aus, zu der das Konto und Dimensionen gehören.
5. Klicken Sie im Feld Unternehmen auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
6. Wählen Sie in der Liste das Unternehmen aus, für die der Alias gilt.
7. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
8. Geben Sie im Feld "Sachkontoalias" das Konto und die Dimensionen an..
    * Das Konto und die Dimensionen werden ausgefüllt, wenn die Verknüpfung verwendet wird.  
9. Wählen Sie im Feld "Anfänglicher Fokus" die Dimension aus, die den Fokus hat, wenn der Alias verwendet wird.
    * Um nachdem Sie das Tastenkürzel verwendet haben, werden das Konto und die Dimensionen aufgefüllt. Der Cursor oder der Fokus wird zum Feld "Anfänglicher Fokus" verschoben.
