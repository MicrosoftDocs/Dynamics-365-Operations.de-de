# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="00364-103">Einrichten eines Sachkontoalias</span><span class="sxs-lookup"><span data-stu-id="00364-103">Set up a ledger account alias</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="00364-104">Das folgende Verfahren zeigt die Erstellung eines Kontoalias als Kürzel zur Eingabe einer Kontonummer.</span><span class="sxs-lookup"><span data-stu-id="00364-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="00364-105">Für diese Prozedur wird das Demodatunternehmen USMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="00364-105">This procedure users demo data company USMF.</span></span>

1. <span data-ttu-id="00364-106">Wechseln Sie zu "Hauptbuch" > "Kontenplan" > "Konten" > "Sachkontoalias".</span><span class="sxs-lookup"><span data-stu-id="00364-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="00364-107">Klicken Sie auf "Neu".</span><span class="sxs-lookup"><span data-stu-id="00364-107">Click New.</span></span>
3. <span data-ttu-id="00364-108">Geben Sie im Feld "Sachkontoalias" einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="00364-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="00364-109">Wählen Sie im Feld "Kontostruktur" die Struktur aus, zu der das Konto und Dimensionen gehören.</span><span class="sxs-lookup"><span data-stu-id="00364-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="00364-110">Klicken Sie im Feld Unternehmen auf die Dropdown-Schaltfläche, um die Suche zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="00364-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="00364-111">Wählen Sie in der Liste das Unternehmen aus, für die der Alias gilt.</span><span class="sxs-lookup"><span data-stu-id="00364-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="00364-112">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="00364-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="00364-113">Geben Sie im Feld "Sachkontoalias" das Konto und die Dimensionen an..</span><span class="sxs-lookup"><span data-stu-id="00364-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="00364-114">Das Konto und die Dimensionen werden ausgefüllt, wenn die Verknüpfung verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="00364-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="00364-115">Wählen Sie im Feld "Anfänglicher Fokus" die Dimension aus, die den Fokus hat, wenn der Alias verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="00364-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="00364-116">Um nachdem Sie das Tastenkürzel verwendet haben, werden das Konto und die Dimensionen aufgefüllt. Der Cursor oder der Fokus wird zum Feld "Anfänglicher Fokus" verschoben.</span><span class="sxs-lookup"><span data-stu-id="00364-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
