# <a name="set-up-a-ledger-account-alias"></a>Einrichten eines Sachkontoalias

[!include [banner](../../includes/banner.md)]

Das folgende Verfahren zeigt die Erstellung eines Kontoalias als Kürzel zur Eingabe einer Kontonummer. Für diese Prozedur wird das Demodatunternehmen USMF verwendet.

1. Wechseln Sie zu **Hauptbuch > Kontenplan > Konten > Sachkontoalias** und klicken Sie auf **Neu**.
2. Geben Sie im Feld **Sachkontoalias** einen Wert ein.
3. Wählen Sie im Feld **Kontostruktur** die Struktur aus, zu der das Konto und Dimensionen gehören.
4. Wählen Sie in der Liste im Feld **Unternehmen** das Unternehmen aus, für das der Alias gilt.
5. Geben Sie im Feld **Sachkontoalias** das Konto und die Dimensionen an.
    * Das Konto und die Dimensionen werden ausgefüllt, wenn die Verknüpfung verwendet wird.  
6. Wählen Sie im Feld **Anfänglicher Fokus** die Dimension aus, die den Fokus hat, wenn der Alias verwendet wird.
    * Nachdem Sie die Tastenkombination verwendet haben, werden das Konto und die Dimensionen aufgefüllt. Der Cursor oder der Fokus wird zum Feld **Anfänglicher Fokus** verschoben.
