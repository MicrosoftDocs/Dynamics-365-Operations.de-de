<span data-ttu-id="8ef95-104">Dieses Thema zeigt, wie Finanzdimensions-Standardvorlagen eingerichtet werden.</span><span class="sxs-lookup"><span data-stu-id="8ef95-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="8ef95-105">Die Vorlagen können verwendet werden, um Beträge in Quelldokumenten schnell zu verteilen.</span><span class="sxs-lookup"><span data-stu-id="8ef95-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="8ef95-106">Wechseln Sie im Navigationsbereich zu **Module > Hauptbuch > Kontenplan > Dimensionen > Standardvorlagen für Finanzdimensionen**.</span><span class="sxs-lookup"><span data-stu-id="8ef95-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>

6. <span data-ttu-id="8ef95-111">Wählen Sie **Hinzufügen** aus.</span><span class="sxs-lookup"><span data-stu-id="8ef95-111">Select **Add**.</span></span>
7. <span data-ttu-id="8ef95-112">Geben Sie einen weiteren Prozentsatz für die weiteren Dimensionswerte ein.</span><span class="sxs-lookup"><span data-stu-id="8ef95-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="8ef95-113">Im vorliegenden Beispiel geben Sie einen Prozentsatz ein. So müssen Sie weiterhin weniger als 100% zuordnen.</span><span class="sxs-lookup"><span data-stu-id="8ef95-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="8ef95-114">Geben Sie die Finanzdimensionswerte ein, die dem eingegebenen Prozent entsprechen.</span><span class="sxs-lookup"><span data-stu-id="8ef95-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="8ef95-115">Wählen Sie **Hinzufügen** aus.</span><span class="sxs-lookup"><span data-stu-id="8ef95-115">Select **Add**.</span></span>
10. <span data-ttu-id="8ef95-116">Geben Sie einen anderen Prozent ein, mit dem der Gesamtprozent wert 100% entspricht.</span><span class="sxs-lookup"><span data-stu-id="8ef95-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
