# <a name="reverse-posted-lease-transactions"></a><span data-ttu-id="f6ce5-104">Gebuchte Leasingtransaktionen stornieren</span><span class="sxs-lookup"><span data-stu-id="f6ce5-104">Reverse posted lease transactions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="f6ce5-105">Jede Transaktion, die durch Anlagenleasing erstellt wird, kann rückgängig gemacht werden.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-105">Any transaction that is created through Asset leasing can be reversed.</span></span> <span data-ttu-id="f6ce5-106">Transaktionen, die durch Anlagenleasing rückgängig gemacht werden, aktualisieren Ihre Leasingdaten.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-106">Transactions that are reversed through Asset leasing update your lease data.</span></span> <span data-ttu-id="f6ce5-107">Daher aktualisieren sie auch die Buchwerte der Leasingverbindlichkeit und des Nutzungsrechts am Leasingobjekt.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-107">Therefore, they also update the carrying values of the lease liability and right-of-use (ROU) asset.</span></span>

<span data-ttu-id="f6ce5-108">Führen Sie die folgenden Schritte aus, um eine Stornierungstransaktionsdaten für einen Mietvertrag zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-108">To create a reversing transaction for a lease, follow these steps.</span></span>

1. <span data-ttu-id="f6ce5-109">Wählen Sie entweder auf der **Anlagentransaktionen**-Seite oder der **Verbindlichkeitentransaktionen**-Seite die Transaktion und dann **Transaktion stornieren** aus.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-109">On either the **Asset transactions** page or the **Liability transactions** page, select the transaction, and then select **Reverse transaction**.</span></span>

<span data-ttu-id="f6ce5-116">Wenn Sie **Rückverfolgung** auswählen wird ein Dialogfeld angezeigt, in dem sowohl die ursprünglichen als auch die stornierten Transaktionen zusammen mit einer verknüpften Nummer angezeigt werden, die als *Tracenummer* bezeichnet wird.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-116">When you select **Reverse tracing**, a dialog box appears that shows both the original transactions and the reversed transactions, together with a linked number that is known as a *trace number*.</span></span> <span data-ttu-id="f6ce5-117">Um das Verständnis der Stornierungen zu erleichtern und die Sichtbarkeit zu verbessern, können Sie Stornierungen auch mithilfe der Mietpläne verfolgen.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-117">To make the reversals easier to understand and to improve visibility, you can also track reversals by using the lease schedules.</span></span>

<span data-ttu-id="f6ce5-118">Das **Neueste Erfassungsnummer**-Feld auf der **Zeitplan**-Seite zeigt die Erfassungsnummern der Transaktionen.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-118">The **Latest journal number** field on the **Schedule** page shows the journal numbers of transactions.</span></span> <span data-ttu-id="f6ce5-119">Wenn eine Transaktion storniert wird, wird dieses Feld mit der Erfassungsnummer einer stornierenden Transaktion aktualisiert.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-119">When a transaction is reversed, this field is updated with the journal number of a reversing transaction.</span></span> <span data-ttu-id="f6ce5-120">Darüber hinaus wird das Kontrollkästchen **Storniert** aktiviert, um anzuzeigen, dass die Transaktion storniert wurde, und das Feld **Gebucht** wird ausgewählt.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-120">Additionally, the **Reversed** check box is selected to indicate that the transaction is reversed, and the **Posted** field is selected.</span></span>

    - <span data-ttu-id="f6ce5-125">Wenn Sie die Transaktion auf der **Zeitplan**-Seite ausgewählt haben, befolgen Sie die Schritte zum Erstellen einer Erfassung in [Monatliche Erfassungseinträge in einem Batch erstellen](create-monthly-journals-batch.md).</span><span class="sxs-lookup"><span data-stu-id="f6ce5-125">If you selected the transaction on the **Schedule** page, follow the steps for creating a journal in [Create monthly journal entries in a batch](create-monthly-journals-batch.md).</span></span> <span data-ttu-id="f6ce5-126">Sie müssen die Erfassung manuell buchen.</span><span class="sxs-lookup"><span data-stu-id="f6ce5-126">You must manually post the journal.</span></span>
