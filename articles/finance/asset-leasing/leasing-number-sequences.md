[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="57d10-105">In diesem Thema wird erläutert, wie Sie Zahlenfolgen für Leasing-IDs erstellen.</span><span class="sxs-lookup"><span data-stu-id="57d10-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="57d10-106">Außerdem wird erläutert, wie Sie eindeutige IDs erstellen, die bei der Indexneubewertung verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="57d10-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>
