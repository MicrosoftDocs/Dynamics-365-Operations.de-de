# <a name="copy-a-billing-schedule"></a>Kopieren eines Abrechnungsplans

[!include [banner](../includes/banner.md)]

Sie können die Seite **Plan kopieren** verwenden, um einen neuen Abrechnungsplan aus einem bestehenden Abrechnungsplan zu erstellen. Der neue Fakturierungsplan hat die gleichen Einstellungen wie der ursprüngliche Fakturierungsplan. Sie können die Seite auch verwenden, um Elemente von einem Rechnungsplan in einen anderen zu kopieren.

## <a name="copy-a-billing-schedule"></a>Kopieren eines Abrechnungsplans

Um einen Abrechnungsplan zu kopieren, gehen Sie folgendermaßen vor.

1. Wählen Sie auf der Listenseite **Alle Abrechnungspläne** oder **Aktive Abrechnungspläne** einen vorhandenen Abrechnungsplan aus.
2. Wählen Sie auf der Registerkarte **Abrechnungsplan** die Option **Plan kopieren**.
3. Wählen Sie im Feld **Kundenkonto** das Kundenkonto aus.
4. Aktualisieren Sie das Start- und Enddatum der Fakturierung wie gewünscht.
5. Aktualisieren Sie die Werte der folgenden optionalen Felder nach Bedarf: **Endbenutzerkonto**, **Partnerkonto**, **Kundenreferenz**, **Kundenanforderung** und **Startdatum der Abgrenzung**.
6. Wählen Sie **Vorschau ansehen**, um die Rechnungseinteilungen, die Sie kopieren möchten, anzuzeigen und auszuwählen.
7. Wählen Sie **Kopieren**.

Nach Abschluss des Kopiervorgangs wird der neue Fakturierungsplan am unteren Ende der Listenseite **Alle Fakturierungspläne** oder **Aktive Fakturierungspläne** und auf der Seite **Fakturierungspläne** angezeigt.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Kopieren von Elementen aus einem Abrechnungsplan in einen anderen

1. Wählen Sie auf der Listenseite **Alle Abrechnungspläne** oder **Aktive Abrechnungspläne** einen vorhandenen Abrechnungsplan aus.
2. Wählen Sie auf der Registerkarte **Abrechnungsplan** die Option **Plan kopieren**.
3. Wählen Sie im Feld **Kundenkonto** das Kundenkonto aus.
4. Wählen Sie im Feld **Fakturierungseinteilungsnummer** eine Fakturierungseinteilungsnummer.
4. Aktualisieren Sie das Start- und Enddatum der Rechnungsstellung.
5. Aktualisieren Sie die Werte der folgenden Felder nach Bedarf: **Endbenutzerkonto**, **Partnerkonto**, **Kundenreferenz**, **Kundenanforderung** und **Startdatum der Abgrenzung**.
6. Wählen Sie **Vorschau ansehen**, um die Rechnungseinteilungen anzuzeigen und auszuwählen, die Sie in einen bestehenden Rechnungsplan kopieren möchten.
7. Wählen Sie **Kopieren**.

Nachdem der Kopiervorgang abgeschlossen ist, überprüfen Sie den Abrechnungsplan, in den Sie die Elemente kopiert haben, und stellen Sie sicher, dass die Elemente wie erwartet hinzugefügt wurden.
