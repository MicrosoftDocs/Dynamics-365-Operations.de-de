# <a name="establish-customer-payment-fees"></a><span data-ttu-id="f305a-103">Debitorenzahlungsgebühren einrichten</span><span class="sxs-lookup"><span data-stu-id="f305a-103">Establish customer payment fees</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f305a-104">Erstellen Sie Zahlungsgebühren für Debitorenzahlungen.</span><span class="sxs-lookup"><span data-stu-id="f305a-104">Create payment fees for customer payments.</span></span>

<span data-ttu-id="f305a-105">Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="f305a-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="f305a-106">Wechseln Sie im **Navigationsbereich** zu **Module > Debitoren > Zahlungseinstellungen > Zahlungsgebühr**.</span><span class="sxs-lookup"><span data-stu-id="f305a-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="f305a-107">Klicken Sie auf **Neu**.</span><span class="sxs-lookup"><span data-stu-id="f305a-107">Click **New**.</span></span>
3. <span data-ttu-id="f305a-108">Geben Sie im Feld **Gebührenkennung** eine Gebührenkennung ein.</span><span class="sxs-lookup"><span data-stu-id="f305a-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="f305a-109">Die "Gebührenkennung" wird auf Zahlungserfassungen angezeigt, damit sind sie beschreibend, um zu verstehen, welche Gebühr veranschlagt wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>  
4. <span data-ttu-id="f305a-110">Geben Sie im Feld **Name** einen Gebührennamen ein.</span><span class="sxs-lookup"><span data-stu-id="f305a-110">In the **Name** field, enter a fee name.</span></span>
5. <span data-ttu-id="f305a-111">Geben Sie im Feld **Gebührenbeschreibung** eine Beschreibung für die Gebühr ein.</span><span class="sxs-lookup"><span data-stu-id="f305a-111">In the **Fee description** field, enter a description for the fee.</span></span>
6. <span data-ttu-id="f305a-112">Wählen Sie im Feld **Belastung** aus, ob der Debitor oder ein Sachkonto mit der Gebühr belastet wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-112">In the **Charge** field, select whether the fee will be charged to the Customer or a Ledger account.</span></span> <span data-ttu-id="f305a-113">Wenn die Gebühr für den Debitor veranschlagt wird, wählen Sie "Debitor" aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-113">If the customer is assessed the fee, select Customer.</span></span> <span data-ttu-id="f305a-114">Wenn die Gebühr für Ihre Organisation als Ausgaben veranschlagt wird, wählen Sie "Sachkonto" aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-114">If the fee will be assess to your organization as an expense, select Ledger.</span></span> <span data-ttu-id="f305a-115">Wählen Sie für diese Aufgabe "Debitor" aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-115">For this task, select Customer.</span></span>  
7. <span data-ttu-id="f305a-116">Im Feld **Erfassungstyp** wählen Sie den Journaltyp aus, für den Sie diese Zahlungsgebühr verwenden können.</span><span class="sxs-lookup"><span data-stu-id="f305a-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="f305a-117">Wenn diese Gebühren für Debitorenzahlungen verwendet werden, wird der Erfassungstyp wahrscheinlich "Debitorenzahlung" sein.</span><span class="sxs-lookup"><span data-stu-id="f305a-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="f305a-118">Klicken Sie auf **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="f305a-118">Click **Save**.</span></span>
9. <span data-ttu-id="f305a-119">Klicken Sie auf **Zahlungsgebühreinstellungen**.</span><span class="sxs-lookup"><span data-stu-id="f305a-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="f305a-120">Die Zahlungsgebühreneinstellungen werden verwendet, um die Kriterien zu definieren, wann die Zahlungsgebühr veranschlagt wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="f305a-121">So können Sie beispielsweise definieren, dass die Gebühr berechnet wird, wenn das Bankkonto USMF-OPERATION ist und die Zahlungsmethode per Scheck ist.</span><span class="sxs-lookup"><span data-stu-id="f305a-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>  
10. <span data-ttu-id="f305a-122">Wählen Sie im Feld **Gruppierungen** entweder „Tabelle“, „Gruppe“ oder „Alle“ aus, um zu definieren, welche Bankkonten für diese Gebühr bewertet werden.</span><span class="sxs-lookup"><span data-stu-id="f305a-122">In the **Groupings** field, select either Table, Group or All to define which bank accounts will be assessed this fee.</span></span> <span data-ttu-id="f305a-123">Wenn Sie "Alle" auswählen, könnte für alle Bankkonten diese Gebühr veranschlagt werden.</span><span class="sxs-lookup"><span data-stu-id="f305a-123">If you select All, all bank accounts could be assessed this fee.</span></span>  <span data-ttu-id="f305a-124">Wenn Sie "Tabelle" auswählen, kann nur für das Bankkonto, das Sie auswählen, diese Gebühr veranschlagt werden.</span><span class="sxs-lookup"><span data-stu-id="f305a-124">If you select Table, only the bank account you select could be assessed this fee.</span></span> <span data-ttu-id="f305a-125">Wenn Sie "Gruppe" auswählen, könnte nur für Bankkonten in der ausgewählten Bankgruppe diese Gebühr veranschlagt bewertet werden.</span><span class="sxs-lookup"><span data-stu-id="f305a-125">If you select Group, only the bank accounts in the selected bank group could be assessed this fee.</span></span>  
11. <span data-ttu-id="f305a-126">Im Feld **Bankverbindung** wählen Sie entweder eine Bankgruppe oder ein Bankkonto aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="f305a-127">Wenn Sie "Tabelle" ausgewählt haben, zeigt die Suche Bankkonten an.</span><span class="sxs-lookup"><span data-stu-id="f305a-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="f305a-128">Wenn Sie "Gruppe" ausgewählt haben, zeigt die Suche Bankgruppen an.</span><span class="sxs-lookup"><span data-stu-id="f305a-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="f305a-129">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="f305a-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="f305a-130">Im Feld **Zahlungsmethode** wählen Sie die Zahlungsmethode aus, für die diese Gebühr veranschlagt wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-130">In the **Method of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="f305a-131">Beispielsweise veranschlagen Sie möglicherweise eine Gebühr für Ihre Debitoren, wenn sie Zahlungen als Scheck senden, anstatt als elektronische Zahlung.</span><span class="sxs-lookup"><span data-stu-id="f305a-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="f305a-132">Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="f305a-133">Falls relevant, geben Sie im Feld **Zahlungswährung** eine Zahlungswährung ein.</span><span class="sxs-lookup"><span data-stu-id="f305a-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="f305a-134">Die Zahlungswährung wird als zusätzliches Kriterien dafür verwendet, ob die Gebühr veranschlagt wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="f305a-135">Beispielsweise erhebt möglicherweise Ihrer Bank eine zusätzliche Gebühr für Zahlungseingänge in der USD-Währung, da sie normalerweise ihre Geschäfte nur in der EUR-Währung abwickelt.</span><span class="sxs-lookup"><span data-stu-id="f305a-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>  
16. <span data-ttu-id="f305a-136">Wählen Sie aus, ob die Gebühr ein Prozentsatz, ein Betrag oder ein Intervall ist.</span><span class="sxs-lookup"><span data-stu-id="f305a-136">Select whether the fee will be a percent, amount or interval.</span></span>
17. <span data-ttu-id="f305a-137">Geben Sie im Feld **Prozentsatz/Betrag** entweder einen Prozentsatz oder Betrag der Gebühr ein.</span><span class="sxs-lookup"><span data-stu-id="f305a-137">In the **Percentage/Amount field**, enter either percentage or amount of the fee.</span></span> <span data-ttu-id="f305a-138">Wenn das Feld "Prozentsatz/Betrag" die Bezeichnung "Prozent" hat, ist der Wert, den Sie hier eingeben, ein Prozentsatz.</span><span class="sxs-lookup"><span data-stu-id="f305a-138">If the Percentage/Amount field is Percent, then the value enter here will be a percentage.</span></span> <span data-ttu-id="f305a-139">Wenn das Feld "Prozentsatz/Betrag" die Bezeichnung "Betrag" hat, ist der Wert, den Sie hier eingeben, ein Betrag.</span><span class="sxs-lookup"><span data-stu-id="f305a-139">If the Percentage/Amount field is Amount, then the value you enter here will be an amount.</span></span> <span data-ttu-id="f305a-140">Wenn das Feld "Prozentsatz/Betrag" die Bezeichnung "Intervall" hat, verwenden Sie die Registerkarte "Intervall", um die Ebenen zu definieren.</span><span class="sxs-lookup"><span data-stu-id="f305a-140">If the Percentage/Amount field is Interval, use the Interval tab to define the tiers.</span></span>  
18. <span data-ttu-id="f305a-141">Wählen Sie im Feld **Gebührenwährung** die Währung der Gebühr aus.</span><span class="sxs-lookup"><span data-stu-id="f305a-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="f305a-142">Dies ist die Währung, in der die Gebühr erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="f305a-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="f305a-143">Klicken Sie auf **Speichern**.</span><span class="sxs-lookup"><span data-stu-id="f305a-143">Click **Save**.</span></span>
