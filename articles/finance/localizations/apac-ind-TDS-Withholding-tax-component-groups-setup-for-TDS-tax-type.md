# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="2833a-103">Quellensteuererklärungskomponentengruppen für den Steuertyp „TDS“ einrichten</span><span class="sxs-lookup"><span data-stu-id="2833a-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="2833a-104">In diesem Thema wird erklärt, wie Sie Quellensteuerkomponentengruppen, wie **Rente** und **Auftragnehmer** für die Quellenbesteuerung (TDS) einrichten.</span><span class="sxs-lookup"><span data-stu-id="2833a-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="2833a-105">Gehen Sie zu **Steuer \> Setup \> Quellensteuer \> Quellensteuerkomponentengruppen**.</span><span class="sxs-lookup"><span data-stu-id="2833a-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="2833a-106">[![Seite „Quellensteuerkomponentengruppen“](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="2833a-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="2833a-107">Wählen Sie im Feld **Steuertyp** **TDS** aus, um Quellensteuerkomponentengruppen für den Steuertyp „TDS“ einzurichten.</span><span class="sxs-lookup"><span data-stu-id="2833a-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="2833a-108">Wählen Sie im Aktivitätsbereich **Neu** aus, um eine Position zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="2833a-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="2833a-109">Geben Sie im Feld **Quellensteuerkomponentegruppe** einen Namen für die TDS-Komponentengruppe ein.</span><span class="sxs-lookup"><span data-stu-id="2833a-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="2833a-110">Geben Sie im Feld **Beschreibung** eine Beschreibung ein.</span><span class="sxs-lookup"><span data-stu-id="2833a-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="2833a-111">Wählen Sie im Inforegister **Allgemein** im Feld **Status** den Aufenthaltsstatus der TDS-Komponentengruppe aus.</span><span class="sxs-lookup"><span data-stu-id="2833a-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="2833a-112">Die Optionen sind **Ansässig** und **Nicht ansässig**.</span><span class="sxs-lookup"><span data-stu-id="2833a-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="2833a-113">Geben Sie im Feld **Abschnittscode** den Abschnittscode ein, der auf die TDS-Komponentengruppe angewendet wird.</span><span class="sxs-lookup"><span data-stu-id="2833a-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="2833a-114">Schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="2833a-114">Close the page.</span></span>
