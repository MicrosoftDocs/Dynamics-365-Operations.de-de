# <a name="de-00002-depreciation-adjustments-for-additional-acquisitions-in-the-second-year"></a>DE-00002 Abschreibungsregulierungen für zusätzliche Anschaffungen im zweiten Jahr

[!include [banner](../../includes/banner.md)]

Dieses Handbuch veranschaulicht, wie die Berechnung der Abschreibung für zusätzliche Anschaffungen eingerichtet. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist DEMF.


## <a name="allow-multiple-acquisitions"></a>Mehrere Anschaffungen zulassen
1. Wechseln Sie zu "Anlagen" > "Einstellungen" > "Anlagenparameter".
2. Prüfen Sie das mehrere Anschaffungskontrollkästchen des Zulässiger.
3. Klicken Sie auf "Speichern".

## <a name="set-up-depreciation-profile"></a>Einrichten von Abschreibungsprofilen
1. Wechseln Sie zu Anlagen > Einstellungen > Abschreibungsprofile.
2. Klicken Sie auf "Neu".
3. Geben Sie im Feld "Abschreibungsprofil" einen Wert ein.
4. Geben Sie im Feld "Name" einen Wert ein.
5. Abschreibungsmethode "Lineare verbleibende Nutzungsdauer"
6. Wählen Sie im Feld "Berichtszeitraum" "Monatlich" aus.
7. Ganzjährliche Abschreibung sonstiger Anschaffungen
8. Klicken Sie auf "Speichern".
