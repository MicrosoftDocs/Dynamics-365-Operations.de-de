# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a>Gebuchte TDS-Zahlungen und -Buchungen zu einem TDS-Abrechnungszeitraum anzeigen

[!include [banner](../includes/banner.md)]

In diesem Artikel wird erklärt, wie Sie die Zahlungen für die Quellenbesteuerung (TDS-Zahlungen) und entsprechenden Buchungen anzeigen, die für eine Ausgleichsperiode gebucht wurden.

1. Gehen Sie zu **Steuer \> Indirekte Steuern \> Quellensteuer \> Quellensteuerausgleichsperioden**.

    [![Seite „Quellensteuerausgleichsperioden“.](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)

2. Wählen Sie auf der Seite **Quellensteuerausgleichsperioden** **Quellensteuerzahlungen** aus, um die Seite **Quellensteuerzahlungen** zu öffnen, auf der Sie die TDS-Ausgleiche anzeigen können, die für eine bestimmte TDS-Ausgleichsperiode vorgenommen wurden.

    Die Registerkarte **Übersicht** zeigt die folgenden Informationen an:

    - **Datum**: Das Datum des TDS-Ausgleichs.
    - **Beleg**: Die Belegnummer der TDS-Ausgleichsbuchung.
    - **Steuertyp**: Der Steuertyp, für den der Ausgleichsprozess durchgeführt wurde.
    - **Steuerkontonummer (TAN)**: Die Steuerkontonummer (TAN) der ausgeglichenen TDS-Buchung.
    - **Ausgleichsperiode**: Die TDS-Ausgleichsperiode, für die der TDS-Ausgleichsprozess ausgeführt werden soll.
    - **Startdatum**: Das Startdatum der Ausgleichsperiode.
    - **Enddatum**: Das Enddatum der Ausgleichsperiode.
    - **Quellensteuerzahlungsversion**: Die Version der Quellensteuerzahlung: **Original**, **Neueste Korrekturen** oder **Gesamtliste**.

5. Wählen Sie **Belege**, um die Belegeinträge für die TDS-Buchung anzuzeigen.
6. Wählen Sie **Quellensteuerbuchungen**, um die Details der TDS-Buchungen anzuzeigen, die während einer Ausgleichsperiode für eine bestimmte Periode ausgeglichen wurden. Wählen Sie **Belege**, um die Belegeinträge für die TDS-Buchung anzuzeigen. Wählen Sie **Quellensteuerkomponenten** aus, um sich die TDS anzeigen zu lassen, die pro TDS-Steuerkomponente für einen bestimmten TDS-Steuercode berechnet wurde.
7. Schließen Sie die Seite **Quellensteuerzahlungen**, um zur Seite **Quellensteuerausgleichsperioden** zurückzukehren.
8. Wählen Sie **Quellensteuerbuchungen**, um die Details der TDS-Buchungen anzuzeigen, die während der Ausgleichsperiode ausgeglichen wurden.
