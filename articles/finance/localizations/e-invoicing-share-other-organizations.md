# <a name="share-globalization-features-with-other-organizations"></a>Globalisierungsfunktionen mit anderen Organisationen teilen

[!include [banner](../includes/banner.md)]

Befolgen Sie diese Schritte, um eine Globalisierungsfunktion für eine externe Organisation freizugeben.

1. Melden Sie sich bei Ihrem RCS-Konto (Regulatory Configuration Service) an.
2. Wählen Sie im Abschnitt **Funktionen** des Arbeitsbereichs **Globalisierungsfunktion** die Kachel **Elektronische Rechnungsstellung** aus.
3. Wählen Sie auf der Seite **Elektronische Rechnungsstellung** die Funktion und die Version, die Sie freigeben möchten. Sie können keine Versionen von Funktionen freigeben, die den Status **Entwurf** haben.
4. Wählen Sie auf der Registerkarte **Organisationen** die Option **Freigeben für** und geben Sie dann in das Dropdown-Dialogfeld den Domänennamen der Organisation ein, für die Sie die Funktion freigeben möchten.
5. Wählen Sie **Teilen**.

Die Funktion wird mit der ausgewählten Organisation geteilt und wird für diese Organisation im globalen Repository verfügbar. Von dort aus kann die Funktion in die RCS-Instanz des Unternehmens importiert und genutzt werden.
