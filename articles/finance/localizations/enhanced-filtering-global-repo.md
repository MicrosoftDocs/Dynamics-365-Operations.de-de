# <a name="enhanced-filtering-options-for-finding-configurations-in-the-global-repository"></a>Erweiterte Filteroptionen für das Auffinden von Konfigurationen im globalen Repository

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

In diesem Thema werden die erweiterten Filterfunktionen für das globale Repository der Regulatory Configuration Services (RCS) beschrieben, die um die folgenden Filter erweitert wurden: 
- **Land/Region** - basierend auf ISO-Ländercodes  
- **Tags** - für Funktions-/Merkmalsbereich; Industrie; Geschäftsdokumenttyp 

Sie können Filter entweder einzeln oder in Gruppen anwenden, um bestimmte oder verwandte Konfigurationen zu finden. Um z.B. alle konfigurierbaren Geschäftsdokumente zu finden, die mit Lieferantenrechnungen verbunden sind, können Sie den Filter **Geschäftsdokumenttyp** anwenden. 

Sie können eine Suche weiter verfeinern, indem Sie den Ländercode auswählen und **Filter anwenden** wählen.  

[![Filterabschnitt für globales Repository](media/rcs-enhanced-filter-section.JPG)](./media/rcs-enhanced-filter-section.JPG) 

Das folgende Beispiel zeigt die Ergebnisse beim Filtern auf **Geschäftsdokumenttyp**. 

[![Angewandter Filter und Import für Geschäftsdokumenttyp](media/rcs-enhanced-filtering-applied.JPG)](./media/rcs-enhanced-filtering-applied.JPG) 

Gefilterte Ergebnisse können in die RCS- oder Dynamics 365 Finance-Umgebung des Benutzers importiert werden, entweder einzeln oder als Set (durch Auswahl der Gruppe von Konfigurationen) und durch Klicken auf **Import**.
