# <a name="run-the-periodic-tds-settlement-process"></a><span data-ttu-id="a654b-103">Den periodischen TDS-Ausgleichsprozess ausführen</span><span class="sxs-lookup"><span data-stu-id="a654b-103">Run the periodic TDS settlement process</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a654b-104">In diesem Thema wird erklärt, wie die periodische Quellenbesteuerung (TDS) ausgeglichen wird.</span><span class="sxs-lookup"><span data-stu-id="a654b-104">This topic explains how to settle periodic Tax Deducted at Source (TDS).</span></span>

1. <span data-ttu-id="a654b-105">Gehen Sie zu **Steuer \> Erklärungen \> Quellensteuer \> Quellensteuerzahlung**.</span><span class="sxs-lookup"><span data-stu-id="a654b-105">Go to **Tax \> Declarations \> Withholding tax \> Withholding tax payment**.</span></span>

    <span data-ttu-id="a654b-106">[![Dialogfeld „Quellensteuerzahlung“](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span><span class="sxs-lookup"><span data-stu-id="a654b-106">[![Withholding tax payment dialog box](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span></span>

2. <span data-ttu-id="a654b-107">Wählen Sie im Dialogfeld **Quellensteuerzahlung** im Feld **Steuertyp** **TDS** aus.</span><span class="sxs-lookup"><span data-stu-id="a654b-107">In the **Withholding tax payment** dialog box, in the **Tax type** field, select **TDS**.</span></span>
3. <span data-ttu-id="a654b-108">Wählen Sie im Feld **Steuerkontonummer (TAN)** die Steuerkontonummer (TAN) aus, für die der Ausgleichsprozess ausgeführt werden soll.</span><span class="sxs-lookup"><span data-stu-id="a654b-108">In the **Tax Account Number (TAN)** field, select the Tax Account Number (TAN) to run the settlement process for.</span></span>
4. <span data-ttu-id="a654b-109">Wählen Sie im Feld **Quellensteuerkomponentengruppe** die TDS-Komponentengruppe aus, für die der Ausgleichsprozess ausgeführt werden soll.</span><span class="sxs-lookup"><span data-stu-id="a654b-109">In the **Withholding tax component group** field, select the TDS component group to run the settlement process for.</span></span>
5. <span data-ttu-id="a654b-110">Wählen Sie im Feld **Ausgleichsperiode** die TDS-Ausgleichsperiode aus, für die der Ausgleichsprozess ausgeführt werden soll.</span><span class="sxs-lookup"><span data-stu-id="a654b-110">In the **Settlement period** field, select the TDS settlement period to run the settlement process for.</span></span>

    > [!NOTE]
    > <span data-ttu-id="a654b-111">Der TDS-Ausgleichsprozess wird für alle Perioden ausgeführt, die für die TDS-Ausgleichsperiode in der Registerkarte **Perioden** der Seite **Quellensteuerausgleichsperioden** eingerichtet sind (**Steuer \> Indirekte Steuern \> Quellensteuern \> Quellensteuerausgleichsperioden**).</span><span class="sxs-lookup"><span data-stu-id="a654b-111">The TDS settlement process is run for all periods that are set up for the TDS settlement period on the **Periods** tab of the **Withholding tax settlement periods** page (**Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**).</span></span>

6. <span data-ttu-id="a654b-112">Wählen Sie im Feld **Startdatum** das Startdatum aus, ab dem aus der TDS-Ausgleichsprozess ausgeführt werden soll.</span><span class="sxs-lookup"><span data-stu-id="a654b-112">In the **From date** field, select the start date to run the TDS settlement process from.</span></span>

    <span data-ttu-id="a654b-113">Für eine bestimmten Periode innerhalb der Ausgleichsperiode wird das für die Periode festgelegte Startdatum als „Ab“-Datum verwendet.</span><span class="sxs-lookup"><span data-stu-id="a654b-113">For a specific period within the settlement period, the start date that is defined for the period is taken as the "from" date.</span></span> <span data-ttu-id="a654b-114">Die TDS-Ausgleichsperiode hat zum Beispiel zwei Perioden: 1. bis 30. April 2009 und 1. bis 31. Mai 2009.</span><span class="sxs-lookup"><span data-stu-id="a654b-114">For example, the TDS settlement period has two periods: April 1 through April 30, 2009, and May 1 through May 31, 2009.</span></span> <span data-ttu-id="a654b-115">Wenn Sie **06.04.2009** als Starttermin im Feld **Startdatum** auswählen, beginnt der Ausgleichsprozess trotzdem ab dem 1. April 2009.</span><span class="sxs-lookup"><span data-stu-id="a654b-115">If you select **04/06/2009** (April 6, 2009) as the start date in the **From date** field, the settlement process will still run from April 1, 2009.</span></span>

    <span data-ttu-id="a654b-116">Wenn Sie eine spätere Periode in das Feld **Startdatum** eingeben, aber keine vorherige Periode innerhalb der Ausgleichsperiode ausgleichen, erfolgt der Ausgleich nicht für frühere Perioden.</span><span class="sxs-lookup"><span data-stu-id="a654b-116">If you enter a later period in the **From date** field, but without settling a previous period within the settlement period, the settlement won't occur for any previous periods.</span></span> <span data-ttu-id="a654b-117">Zum Beispiel: Die TDS-Ausgleichsperiode umfasst drei Perioden: 1. bis 30. April 2009, 1. bis 31. Mai 2009 und 1. bis 30. Juni 2009.</span><span class="sxs-lookup"><span data-stu-id="a654b-117">For example, the TDS settlement period has three periods: April 1 through April 30, 2009, May 1 through May 31, 2009, and June 1 through June 30, 2009.</span></span> <span data-ttu-id="a654b-118">Wenn Sie **01.05.2009** als Startdatum im Feld **Startdatum** angeben, läuft der Ausgleichsprozess nur vom 1. bis 31. Mai 2009.</span><span class="sxs-lookup"><span data-stu-id="a654b-118">If you select **05/01/2009** (May 1, 2009) as the start date in the **From date** field, the settlement process will run only from May 1 through May 31, 2009.</span></span> <span data-ttu-id="a654b-119">Der Ausgleich erfolgt nicht vom 1. bis 30. April 2009.</span><span class="sxs-lookup"><span data-stu-id="a654b-119">The settlement won't occur for April 1 through April 30, 2009.</span></span>

7. <span data-ttu-id="a654b-120">Wählen Sie im Feld **Buchungsdatum** das Buchungsdatum für die TDS-Ausgleichsbuchung aus.</span><span class="sxs-lookup"><span data-stu-id="a654b-120">In the **Transaction date** field, select the date to post the TDS settlement transaction.</span></span>
8. <span data-ttu-id="a654b-121">Wählen Sie im Feld **Quellensteuerzahlungsversion** die TDS-Ausgleichsversion aus:</span><span class="sxs-lookup"><span data-stu-id="a654b-121">In the **Withholding tax payment version** field, select the TDS settlement version:</span></span>

     - <span data-ttu-id="a654b-122">**Original**: Verwenden Sie diese Option, um den TDS-Ausgleichsprozess zum ersten Mal auszuführen.</span><span class="sxs-lookup"><span data-stu-id="a654b-122">**Original** – Use this option to run the TDS settlement process for the first time.</span></span> <span data-ttu-id="a654b-123">Die Originalzahlungsversion wird nur einmal verwendet, um den TDS-Ausgleichsprozess für eine Kombination aus einer TAN, einer Quellensteuerkomponentengruppe und einer Quellensteuerausgleichsperiode auszuführen.</span><span class="sxs-lookup"><span data-stu-id="a654b-123">The original payment version is used only one time to run the TDS settlement process for a combination of a TAN, a withholding tax component group, and a withholding tax settlement period.</span></span>
    - <span data-ttu-id="a654b-124">**Letzte Versionen**: Verwenden Sie diese Option, wenn der TDS-Ausgleichsprozess bereits für die angegebene Periode ausgeführt wurde.</span><span class="sxs-lookup"><span data-stu-id="a654b-124">**Latest versions** – Use this option if the TDS settlement process has already been run for the specified period.</span></span> <span data-ttu-id="a654b-125">Nehmen Sie rückdatierte Buchungen mit auf, die gebucht wurden, nachdem der Ausgleichsprozess zuvor für die Periode ausgeführt wurde.</span><span class="sxs-lookup"><span data-stu-id="a654b-125">Include back-dated transactions that were posted after the settlement process was previously run for the period.</span></span> <span data-ttu-id="a654b-126">Mit dieser Option können Sie den Ausgleichsprozess beliebig oft ausführen.</span><span class="sxs-lookup"><span data-stu-id="a654b-126">You can use this option to run the settlement process any number of times.</span></span>

9. <span data-ttu-id="a654b-127">Aktivieren Sie das Kontrollkästchen **Aktualisieren**, um den TDS-Ausgleichsprozess auszuführen und die Beträge auf die Sachkonten zu buchen.</span><span class="sxs-lookup"><span data-stu-id="a654b-127">Select the **Update** check box to run the TDS settlement process and post the amounts to the ledger accounts.</span></span> <span data-ttu-id="a654b-128">Wenn dieses Kontrollkästchen deaktiviert ist, wird der Ausgleichsprozess nicht ausgeführt und die Finanzbuchungen werden nicht generiert.</span><span class="sxs-lookup"><span data-stu-id="a654b-128">If this check box is cleared, the settlement process won't be run, and the financial entries won't be generated.</span></span>
10. <span data-ttu-id="a654b-129">Wählen Sie **OK** aus, um den TDS-Ausgleichsprozess auszuführen und den Quellensteuerzahlungsbericht zu generieren.</span><span class="sxs-lookup"><span data-stu-id="a654b-129">Select **OK** to run the TDS settlement process and generate the withholding tax payment report.</span></span> <span data-ttu-id="a654b-130">Der Status von TDS-Buchungen, die im Ausgleichsprozess enthalten sind, wird als angezeigt **Ausgeglichen** auf der Seite **Ausgleich** angezeigt (gehen Sie zu **Kreditorenkonten \> Zahlungen \> Kreditorenzahlungserfassung**, wählen Sie **Positionen**, **Funktionen** und dann **Ausgleich**).</span><span class="sxs-lookup"><span data-stu-id="a654b-130">The status of TDS transactions that are included in the settlement process is shown as **Settled** on the **Settlement** page (go to **Accounts payable \> Payments \> Vendor payment journal**, select **Lines**, select **Functions**, and then select **Settlement**).</span></span>

### <a name="important-points"></a><span data-ttu-id="a654b-131">Wichtige Punkte</span><span class="sxs-lookup"><span data-stu-id="a654b-131">Important points</span></span>

- <span data-ttu-id="a654b-132">Wenn während des Ausgleichsprozesses keine Quellensteuerkomponentengruppe ausgewählt wird, erfolgt der Ausgleich für alle Quellensteuerkomponentengruppen für die ausgewählte TAN und Ausgleichsperiode.</span><span class="sxs-lookup"><span data-stu-id="a654b-132">If a withholding tax component group isn't selected during the settlement process, the settlement occurs for all the withholding tax component groups for the selected TAN and settlement period.</span></span> <span data-ttu-id="a654b-133">Für jede Quellensteuerkomponentengruppe wird auf der Seite **Bearbeiten offener Posten** eine eigene Position erstellt.</span><span class="sxs-lookup"><span data-stu-id="a654b-133">A separate line is created for each withholding tax component group on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="a654b-134">Der Ausgleichsprozess basiert auf der Steuerschuldnerkategorie für eine Ausgleichsperiode.</span><span class="sxs-lookup"><span data-stu-id="a654b-134">The settlement process is based on the nature of assessee category for a settlement period.</span></span> <span data-ttu-id="a654b-135">Buchungen, deren Steuerschuldnerkategorie **Unternehmen** lautet, sind ausgeglichen und werden auf der Seite **Bearbeiten offener Posten** als eine Position angezeigt.</span><span class="sxs-lookup"><span data-stu-id="a654b-135">Transactions where the nature of assessee category is **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span> <span data-ttu-id="a654b-136">Buchungen, deren Steuerschuldnerkategorie nicht **Unternehmen** lautet, sind ausgeglichen und werden auf der Seite **Bearbeiten offener Posten** als eine Position angezeigt.</span><span class="sxs-lookup"><span data-stu-id="a654b-136">Transactions where the nature of assessee is something other than **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="a654b-137">Das Fälligkeitsdatum für die ausgeglichenen TDS-Buchungspositionen auf der Seite **Bearbeiten offener Posten** basiert auf den Zahlungsbedingungen, die auf der Seite **Kreditoren** für den TDS-Behördenkreditor festgelegt sind.</span><span class="sxs-lookup"><span data-stu-id="a654b-137">The due date for the settled TDS transaction lines on the **Open transaction editing** page is based on the terms of payment that are defined for the TDS authority vendor on the **Vendors** page.</span></span> <span data-ttu-id="a654b-138">Wenn die Zahlungsbedingungen für den TDS-Behördenkreditor nicht definiert sind, wird der letzte Tag der Ausgleichsperiode als Fälligkeitsdatum angezeigt.</span><span class="sxs-lookup"><span data-stu-id="a654b-138">If the terms of payments aren't defined for the TDS authority vendor, the last day of the settlement period is shown as the due date.</span></span>
