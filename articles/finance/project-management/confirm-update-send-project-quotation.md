# <a name="confirm-update-and-send-a-project-quotation"></a>Bestätigen, aktualisieren und senden Sie ein Projektangebot

[!include [banner](../includes/banner.md)]

Nachdem Sie ein Projektangebot erstellt und an einen Kunden gesendet haben, müssen Sie eine Bestätigung vom Kunden erhalten, bevor Sie den Angebotsstatus auf **Geschickt** aktualisieren. Vom Kunden angeforderte Änderungen können im Angebot aktualisiert werden. Nachdem Sie den Angebotsstatus auf **Geschickt** aktualisiert haben, können Sie ihn nicht ändern. Das folgende Verfahren beschreibt die Optionen zum Senden eines Angebots zur Bestätigung, zum Aktualisieren basierend auf dem Feedback und zum Senden des Angebots.

## <a name="send-a-project-quotation-confirmation"></a>Senden Sie eine Projektangebotsbestätigung  

Sie können ein vorhandenes Projektangebot zur Bestätigung per E-Mail oder als gedruckte Kopie senden. 

1. Wechseln Sie zu **Projektverwaltung und -verrechnung** > **Angebote** > **Projektangebot.** 
2. Auf der Seite **Projektangebot** wählen Sie auf dieser Seite das Angebot aus, das Sie zur Bestätigung senden möchten. 
3. Auf der **Nachverfolgen** Registerkarte in der Gruppe **Erstellen** wählen Sie **Bestätigen**. 
4. Auf der **Angebot bestätigen** Seite stellen Sie die erforderlichen Parameter ein. Zum Beispiel, um die Bestätigung unter der Optionen **Drucken** zu drucken, aktivieren Sie die **Druckbestätigung** und wählen **OK**.
5. Auf der **Angebot senden** Seite wählen Sie **Optionen** und auf der Seite **Zieleinstellungen drucken** wählen Sie auf dieser Seite aus, ob das Angebot an **Bildschirm**, **E-Mail**, **Datei**, **Archiv drucken**, oder **Drucker** gesendet werden soll und nehmen Sie dann die gewünschten Anpassungen vor im Bereich **Spezifikation**, um das Angebot weiterzuleiten.
6. Wählen Sie auf der Seite **Zieleinstellungen drucken** die Option **OK** aus.  

## <a name="update-a-project-quotation"></a>Aktualisieren eines Projektangebots

Um ein vorhandenes Projektangebot zu ändern, muss der Angebotsstatus auf **Erstellt** sein. Führen Sie die folgenden Schritte aus, um ein vorhandenes Angebot zu aktualisieren. 

1. Wechseln Sie zu **Projektverwaltung und -verrechnung** > **Angebote** > **Projektangebot**.
2. Auf der **Projektzitate** Seite wählen Sie auf dieser Seite das Angebot aus, das Sie aktualisieren möchten, und wählen Sie im Aktionsbereich die Option **Bearbeiten**.
3. Aktualisieren Sie die erforderlichen Feldinformationen und wählen Sie im Aktionsbereich **speichern**.  

## <a name="send-a-project-quotation-to-a-customer"></a>Projekt- oder Angebotsdebitor einem Kunden senden 

1. Gehen Sie zu **Projektmanagement und Buchhaltung** > **Angebote** > **Projekangebote** und wählen Sie das Projektangebot aus, das Sie senden möchten.
2. Auf der **Projektangebot** Seite in der **Angebot**-Registerkarte in der Grupppe **Prozess** wählen Sie **Angebot senden**. Der Angebotsstatus wird auf **Gesendet** aktualisiert.

> [!NOTE]
> Sie können ein Projektangebot nicht mehr ändern, nachdem der Status in **Gesendet** geändert wurde.
