[!include [banner](../includes/banner.md)]

<span data-ttu-id="0b576-104">Wenn ein Projektangebot den Status hat **Bestätigt** können Sie einige oder alle Angebotszeilen in ein neues oder vorhandenes Projekt übertragen, indem Sie den Assistent **Auf Projekt übertragen** verwenden.</span><span class="sxs-lookup"><span data-stu-id="0b576-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="0b576-105">Führen Sie zum Verwenden des Assistenten die folgenden Schritte aus.</span><span class="sxs-lookup"><span data-stu-id="0b576-105">To use the wizard, follow these steps.</span></span>
