# <a name="transfer-a-quotation-to-a-project"></a>Angebot in Projekt übertragen

[!include [banner](../includes/banner.md)]

Wenn ein Projektangebot den Status hat **Bestätigt** können Sie einige oder alle Angebotszeilen in ein neues oder vorhandenes Projekt übertragen, indem Sie den Assistent **Auf Projekt übertragen** verwenden. 

Führen Sie zum Verwenden des Assistenten die folgenden Schritte aus.

1. Wechseln Sie zu **Projektverwaltung und -verrechnung** > **Angebote** > **Projektangebot**.
2. Wählen Sie das Angebot aus, das Sie übertragen möchten, und klicken Sie auf die **Nachverfolgen** Registerkarte und wählen Sie in der Gruppe **Ändern** aus und wählen Sie **Zum Projekt übertragen**.
3. Folgen Sie dem Assistenten **Angebot einem Projekt übertragen**, um ein Angebot einem Projekt zuzuordnen und das Angebot an eine Projektplanung zu übertragen.
