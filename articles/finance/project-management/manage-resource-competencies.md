## <a name="set-up-competencies-certificates"></a><span data-ttu-id="6aa02-112">Kompetenzen einrichten: Bescheinigungen</span><span class="sxs-lookup"><span data-stu-id="6aa02-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="6aa02-113">Wählen Sie auf der **Arbeitskräfte**-Listenseite die Position der Arbeitskraft aus, für die Sie Bescheinigungsinformationen hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="6aa02-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="6aa02-114">Klicken Sie im Aktivitätsbereich auf der Registerkarte **Arbeitskräfte** in der Gruppe **Kompetenzen** auf **Bescheinigungen**.</span><span class="sxs-lookup"><span data-stu-id="6aa02-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="6aa02-115">Wählen Sie **Neu** aus und anschließend im Feld **Bescheinigungstyp** **PMP**.</span><span class="sxs-lookup"><span data-stu-id="6aa02-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="6aa02-116">Wählen Sie im Feld **Eintrittstermin** **10/1/2015** und dann **Speichern** aus.</span><span class="sxs-lookup"><span data-stu-id="6aa02-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="6aa02-117">Kompetenzen einrichten: Qualifikationen</span><span class="sxs-lookup"><span data-stu-id="6aa02-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="6aa02-118">Stellen Sie auf der Listenseite **Arbeitskräfte** sicher, dass die Arbeitskraft, die Sie im vorherigen Verfahren verwendet haben, weiterhin ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="6aa02-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="6aa02-119">Klicken Sie dann im Aktivitätsbereich auf der Registerkarte **Arbeitskräfte** in der Gruppe **Kompetenzen** auf **Qualifikationen**.</span><span class="sxs-lookup"><span data-stu-id="6aa02-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="6aa02-120">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="6aa02-120">Select **New**.</span></span>
3. <span data-ttu-id="6aa02-121">Wählen Sie im Feld **Qualifikation** die Option **Projektmanagement**.</span><span class="sxs-lookup"><span data-stu-id="6aa02-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="6aa02-122">Wählen Sie im Feld **Stufe** **5 Experte** aus.</span><span class="sxs-lookup"><span data-stu-id="6aa02-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="6aa02-123">Wählen Sie im Feld **Stufendatum** das Datum **14.1.2015** aus.</span><span class="sxs-lookup"><span data-stu-id="6aa02-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="6aa02-124">Geben Sie in **Jahren Erfahrung** **10** ein.</span><span class="sxs-lookup"><span data-stu-id="6aa02-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="6aa02-125">Klicken Sie auf **Speichern** und schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="6aa02-125">Select **Save**, and then close the page.</span></span>
