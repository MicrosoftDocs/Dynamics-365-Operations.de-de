# <a name="enable-budget-proposal"></a>Budgetvorschlag aktivieren

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

In diesem Thema wird erläutert, wie Sie die Funktion „Budgetvorschlag“ in Finance Insights aktivieren.

1. Öffnen Sie den **Funktionsverwaltung**-Arbeitsbereich, und befolgen Sie diese Schritte:

    1. Wählen Sie **Nach Updates suchen**.
    2. Suchen Sie auf der Registerkarte **Alle** nach **Budgetvorschlag**. Wenn Sie diese Funktion nicht finden, suchen Sie nach **(Vorschau) Budgetvorschlag**. 
    3. Aktivieren Sie die Funktion.

2. Gehen Sie zu **Budgetierung \> Einrichten \> Grundlegende Budgetierung \> Budgetvorschlag** und wählen Sie **Funktion aktivieren**.
