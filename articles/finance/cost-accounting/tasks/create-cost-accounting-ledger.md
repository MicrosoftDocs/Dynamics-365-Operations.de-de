[!include [banner](../../includes/banner.md)]

<span data-ttu-id="08bd7-104">Ein Kostenrechnungssachkonto stellt die Gesamtberichtseinheit dar.</span><span class="sxs-lookup"><span data-stu-id="08bd7-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="08bd7-105">Es wird durch eine Kostenfaktordimension, statistische Dimension, Steuerkalenders und Währung definiert.</span><span class="sxs-lookup"><span data-stu-id="08bd7-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="08bd7-106">Es ist agnostisch zum Konzept der juristischen Personen.</span><span class="sxs-lookup"><span data-stu-id="08bd7-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="08bd7-107">Eine juristische Person und die Daten können vielen Kostenrechnungssachkonten zugeordnet werden.</span><span class="sxs-lookup"><span data-stu-id="08bd7-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="08bd7-108">Für diese Aufzeichnung wird das Demo-Datenunternehmen USP2 verwendet.</span><span class="sxs-lookup"><span data-stu-id="08bd7-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="08bd7-109">Wechseln Sie zu Kostenbuchhaltung > Sachkontoeinstellung > Kostenbuchhaltungs-Sachkonto .</span><span class="sxs-lookup"><span data-stu-id="08bd7-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="08bd7-110">Klicken Sie auf "Neu".</span><span class="sxs-lookup"><span data-stu-id="08bd7-110">Click New.</span></span>
3. <span data-ttu-id="08bd7-111">Geben Sie im Feld "Name" einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="08bd7-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="08bd7-112">Geben Sie im Feld "Kostenelementdimensionshierarchie" einen Wert ein oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="08bd7-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="08bd7-113">Geben Sie im Feld "Steuerkalender" einen Wert ein oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="08bd7-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="08bd7-114">Geben Sie im Feld "Buchhaltungswährung" einen Wert ein oder wählen Sie einen aus.</span><span class="sxs-lookup"><span data-stu-id="08bd7-114">In the Accounting currency field, enter or select a value.</span></span>

8. <span data-ttu-id="08bd7-116">Geben Sie im Feld "Statistische Dimension" einen Wert ein, oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="08bd7-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="08bd7-117">Klicken Sie auf "Speichern".</span><span class="sxs-lookup"><span data-stu-id="08bd7-117">Click Save.</span></span>
