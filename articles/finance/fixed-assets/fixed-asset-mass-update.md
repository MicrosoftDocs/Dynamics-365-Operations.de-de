# <a name="fixed-asset-mass-update"></a><span data-ttu-id="d3678-103">Anlagen-Massenaktualisierung</span><span class="sxs-lookup"><span data-stu-id="d3678-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d3678-104">Bei Verwendung von Büchern können die Abschreibungskonventionen für Gruppen von Anlagen geändert werden, die zum gleichen Buch gehören.</span><span class="sxs-lookup"><span data-stu-id="d3678-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="d3678-105">Wenn Sie z. B. in den USA 40 Prozent Ihrer Anlagen im vierten Quartal des Jahres in Betrieb genommen wurden, müssen Sie die Abschreibungskonvention für Quartalsmitte verwenden.</span><span class="sxs-lookup"><span data-stu-id="d3678-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="d3678-106">Mithilfe der Massenaktualisierung können alle Anlagen geändert werden, für die die neue Abschreibungskonvention erforderlich ist.</span><span class="sxs-lookup"><span data-stu-id="d3678-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="d3678-107">Wenn Sie die Abschreibungskonvention für Anlagen aktualisieren, löschen Sie alle Abschreibungsbuchungen, die für diese Anlagen vorhanden sind.</span><span class="sxs-lookup"><span data-stu-id="d3678-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="d3678-108">Sie löschen auch alle Buchungen für Abschreibungsregulierungen, Buchungen für vorzeitige Abschreibungen und Buchungen für außerordentliche Abschreibungen für diese Anlagen.</span><span class="sxs-lookup"><span data-stu-id="d3678-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="d3678-109">Um die Abschreibungskonvention für Anlagen zu aktualisieren, die bereits veräußert wurden, müssen Sie die vorhandenen Abgangsbuchungen erst löschen.</span><span class="sxs-lookup"><span data-stu-id="d3678-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="d3678-110">Sie müssen alle Buchungen löschen, die wegen der Veräußerung generiert wurden.</span><span class="sxs-lookup"><span data-stu-id="d3678-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="d3678-111">Nachdem Sie die Abschreibungskonvention für Anlagen aktualisiert haben, können Sie die Abschreibung und die außerordentliche Abschreibung für jede Anlage verarbeiten.</span><span class="sxs-lookup"><span data-stu-id="d3678-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="d3678-112">Sie können auch manuelle Abschreibungsregulierungen vornehmen, wenn Anpassungen erforderlich sind.</span><span class="sxs-lookup"><span data-stu-id="d3678-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
