# <a name="change-depreciation-conventions-for-multiple-fixed-assets"></a>Ändern der Abschreibungskonventionen für mehrere Anlagen

[!include [task guide banner](../../includes/task-guide-banner.md)]

Mithilfe dieser Aufgabe wird die Abschreibungskonvention für eine angegebene Anlagengruppe aktualisiert. Für diese Aufgabenanleitung wird das Demo-Unternehmen USMF verwendet.

1. Wechseln Sie zu "Anlagen" > "Periodische Aufgaben" > "Massenaktualisierung"
2. Klicken Sie im Feld "Abschreibungsbuch" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
3. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
4. Geben Sie im Feld "Startdatum für die Inbetriebnahme" ein Datum ein.
5. Geben Sie im Feld "Enddatum für die Inbetriebnahme" ein Datum ein.
    * Nur Anlagen, die dem ausgewählten Abschreibungsbuch zugeordnet sind und innerhalb dieses Zeitraums in Betrieb genommen wurden, werden aktualisiert.  
6. Wählen Sie im Feld "Aktuelle Abschreibungskonvention" eine Option aus.
    * Nur Anlagen, die die aktuelle Abschreibungskonvention haben, werden aktualisiert.  
7. Wählen Sie im Feld "Neue Abschreibungskonvention" eine Option aus.
    * Überprüfen Sie, dass der Bericht zum Drucken an das ausgewählte Ziel gesendet wird.  
8. Erweitern Sie den Abschnitt "Einzuschließende Datensätze".
9. Klicken Sie auf "Filter".
10. Wählen Sie in der Liste die Gruppe "Anlage" aus.
11. Klicken Sie im Feld "Kriterien" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
12. Wählen Sie die gewünschte Gruppe "Anlage" aus.
13. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
14. Klicken Sie auf "OK".
15. Klicken Sie auf "OK".
    *  Ergebnisse des Prozesses werden im "Massenaktualisierungsbericht" angezeigt.
