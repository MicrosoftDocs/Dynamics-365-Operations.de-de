# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="1b818-103">Anlage mithilfe einer Freitextrechnung veräußern</span><span class="sxs-lookup"><span data-stu-id="1b818-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="1b818-104">Dieses Thema erklärt, wie eine Anlage über Freitextrechnung veräußert wird.</span><span class="sxs-lookup"><span data-stu-id="1b818-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="1b818-105">Wechseln Sie im Navigationsbereich zu **Module > Debitoren > Rechnungen > Alle Freitextrechnungen**.</span><span class="sxs-lookup"><span data-stu-id="1b818-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="1b818-106">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="1b818-106">Select **New**.</span></span>
3. <span data-ttu-id="1b818-107">Geben Sie im Feld **Debitorenkonto** einen Wert ein oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="1b818-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="1b818-108">Überprüfen Sie das standardmäßige **Rechnungsdatum** und bearbeiten Sie es, wenn zutreffend.</span><span class="sxs-lookup"><span data-stu-id="1b818-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="1b818-109">Überprüfen Sie verbleibende Standardkopffelder wie **Währungen** und bearbeiten Sie diese, sofern zutreffend.</span><span class="sxs-lookup"><span data-stu-id="1b818-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="1b818-110">Im Abschnitt **Rechnungspositionen** prüfen Sie die Felder **Beschreibung** und **Hauptkonto**.</span><span class="sxs-lookup"><span data-stu-id="1b818-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="1b818-111">Prüfen Sie die Standardfelder **Mehrwertsteuergruppe** und **Artikel-Mehrwertsteuergruppe**.</span><span class="sxs-lookup"><span data-stu-id="1b818-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="1b818-112">Geben Sie den **Preis je Einheit** oder **Betrag** des Verkaufs der Anlage ein.</span><span class="sxs-lookup"><span data-stu-id="1b818-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="1b818-113">Wählen Sie den Abschnitt **Positionsdetails** aus.</span><span class="sxs-lookup"><span data-stu-id="1b818-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="1b818-114">Geben Sie im Feld **Anlage** einen Wert ein oder wählen Sie einen Wert aus.</span><span class="sxs-lookup"><span data-stu-id="1b818-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="1b818-115">Wählen Sie **Buchen** aus.</span><span class="sxs-lookup"><span data-stu-id="1b818-115">Select **Post**.</span></span>
