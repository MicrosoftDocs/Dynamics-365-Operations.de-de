# <a name="set-up-fixed-asset-groups"></a>Anlagengruppen einrichten

[!include [banner](../../includes/banner.md)]

In diesem Abschnitt erfahren Sie, wie Sie eine neue Anlagengruppe anlegen. Dabei werden die "Buchhalterrolle" und die Demodaten für die juristische Person USMF verwendet.

1. Gehen Sie im Navigationsbereich zu **Module > Anlagen > Einrichtung > Anlagengruppen**.
2. Wählen Sie **Neu** aus.
3. Geben Sie im Feld **Anlagegruppe** einen Wert ein.
4. Geben Sie im Feld **Name** einen Wert ein. Autonummerierung der Anlage und Nummernfolgecode auf der Gruppe **Anlage** übersteuern die Einstellungen zu den Parametern der Anlage. Sie können sie hier ändern, wenn die Anlagen in dieser Anlagengruppe eine andere Nummerierung als andere Gruppen haben.  
5. Wählen Sie **Bücher**.
6. Geben Sie im Feld **Buch** einen Wert ein, oder wählen Sie einen Wert aus. Das Feld **Abschreibung berechnen** ist auf **Ja** gesetzt, so dass das Anlagenbuch in die Abschreibungsvorschläge aufgenommen wird. Wenn **Abschreibung berechnen** auf **Nein** gesetzt ist, wird die Anlage nicht automatisch abgeschrieben.  
7. Legen Sie die Nutzungsdauer der Anlage in Jahren fest. Beachten Sie, dass der Feldwert **Abschreibungszeiträume** nach der Einstellung der Nutzungsdauer berechnet wird.  
8. Wählen Sie im Feld **Abschreibungskonvention** eine Option aus.
9. Schließen Sie die Seite.
