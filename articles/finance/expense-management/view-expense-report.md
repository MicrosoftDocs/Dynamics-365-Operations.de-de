# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a>Anzeigen einer Spesenabrechnung aus dem Hauptbuch an in Beschaffung

[!include [banner](../includes/banner.md)]

Wenn Sie eine Abfrage vornehmen, um Buchungen im Hauptbuch oder Beschaffung anzeigen, könnte die Abfrage zu vielen Informationen zurückgeben. Daher sollten Sie die Suche einschränken. Um die benötigten Informationen abzurufen, können Sie Detailinformationen zu einer Buchung und die dazugehörige Spesenabrechnung anzeigen.

Bei Spesenabrechnungen können Sie für eine Buchung die ursprüngliche Abrechnung anzeigen, die in der Ausgabenverwaltung gespeichert ist. Zusätzlich zu den Gesamtausgaben können Sie die nach Artikel aufgeschlüsselten Buchungen, Buchungen, für die eine Mehrwertsteuererstattung möglich ist, und die an die Ausgaben angefügten Belege anzeigen.

Wenn Sie Detailinformationen zur ursprünglichen Spesenabrechnung anzeigen möchten, für die eine Buchung angezeigt wird, wählen Sie die Buchung aus, und wählen Sie dann **Originaldokument** aus.

Beim Anzeigen der einzelnen Buchungen können Sie auch alle zusätzlichen Daten einsehen, die an eine Buchung angefügt sind. Für aufgeschlüsselte Hotelkosten können Sie beispielsweise eine ausführliche Liste der einzelnen Kosten anzeigen, wie Internetnutzung, Telefongespräche und Zimmerservice.
