> [!IMPORTANT]
> Die in diesem Artikel beschriebene IoT Intelligence-Funktion wird in Kürze auf Version 2.0 aktualisiert und gleichzeitig in *Sensor Data Intelligence* umbenannt. Die neue Version ist derzeit als Vorschaufunktion für Supply Chain Management Version 10.0.30 oder höher verfügbar.
> 
> Sensor Data Intelligence führt ein neues Bereitstellungsmodell ein, bei dem die erforderlichen Azure-Komponenten in Ihrem eigenen Azure-Abonnement bereitgestellt werden, anstatt von Microsoft verwaltet zu werden. Dieses neue Modell verbessert Ihre Möglichkeiten zur Anpassung der Funktion und zur Integration mit Fremdsystemen. Die neue Version bietet außerdem Unterstützung für zwei neue IoT-Szenarien für das Asset-Management.
>
> Aufgrund dieser bevorstehenden Änderungen empfiehlt Microsoft, keine neuen Projekte auf der Grundlage der bestehenden IoT Intelligence-Funktion zu starten. Wenn Sie daran interessiert sind, Ihrem System IoT-Funktionalitäten hinzuzufügen, empfehlen wir Ihnen, stattdessen mit der Vorschauversion von Sensor Data Intelligence zu arbeiten. Weitere Informationen über Sensor Data Intelligence und die Arbeit damit finden Sie auf der [Sensor Data Intelligence-Startseite](/dynamics365/supply-chain/sensor-data-intelligence/sdi-home-page).
