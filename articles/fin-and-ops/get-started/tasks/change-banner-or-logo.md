# <a name="change-the-banner-or-logo"></a><span data-ttu-id="37691-103">Das Banner oder das Logo ändern</span><span class="sxs-lookup"><span data-stu-id="37691-103">Change the banner or logo</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="37691-104">Im Folgenden werden die Schritte aufgeführt, die Systemadministratoren verwenden können, um das Banner- oder Logobild zu aktualisieren, das für eine juristische Person angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="37691-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="37691-105">Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.</span><span class="sxs-lookup"><span data-stu-id="37691-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="37691-106">Wechseln Sie zu Organisationsverwaltung > Organisationen > Juristische Personen.</span><span class="sxs-lookup"><span data-stu-id="37691-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="37691-107">Wählen Sie in der Liste auf der linken Seite die juristische Person aus, für die Sie das Banner oder das Logo aktualisieren möchten.</span><span class="sxs-lookup"><span data-stu-id="37691-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="37691-108">Wenn sie es bereits ausgewählt haben, wechseln Sie zum nächsten Schritt.</span><span class="sxs-lookup"><span data-stu-id="37691-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="37691-109">Erweitern Sie den Abschnitt "Bilder".</span><span class="sxs-lookup"><span data-stu-id="37691-109">Expand the Images section.</span></span> <span data-ttu-id="37691-110">Wenn Sie ihn bereits erweitert haben, wechseln Sie zum nächsten Schritt.</span><span class="sxs-lookup"><span data-stu-id="37691-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="37691-111">Klicken Sie auf Bearbeiten.</span><span class="sxs-lookup"><span data-stu-id="37691-111">Click Edit.</span></span>
5. <span data-ttu-id="37691-112">Wählen Sie im Feld "Dashboard-Unternehmensbildtyp" "Banner"aus.</span><span class="sxs-lookup"><span data-stu-id="37691-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="37691-113">Klicken Sie auf "Ändern".</span><span class="sxs-lookup"><span data-stu-id="37691-113">Click Change.</span></span>
    * <span data-ttu-id="37691-114">Tipp: Die optimalen Auflösung für ein Bannerbild ist 1920 x 281 Pixel.</span><span class="sxs-lookup"><span data-stu-id="37691-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="37691-115">Die optimale Breite für ein Logobild ist 350 Pixel.</span><span class="sxs-lookup"><span data-stu-id="37691-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="37691-116">Wenn der Upload abgeschlossen ist, sollten Sie Ihr neues Banner- oder Logobild im Dashboard-Bildausschnitt finden.</span><span class="sxs-lookup"><span data-stu-id="37691-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="37691-117">Klicken Sie auf "Speichern".</span><span class="sxs-lookup"><span data-stu-id="37691-117">Click Save.</span></span>
8. <span data-ttu-id="37691-118">Wählen Sie Allgemeines > Allgemeines > Standard-Dashboard.</span><span class="sxs-lookup"><span data-stu-id="37691-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="37691-119">Sie sollten das neue Banner- oder Logobild im Dashboard finden.</span><span class="sxs-lookup"><span data-stu-id="37691-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="37691-120">Möglicher Weise müssen Sie Ihr Unternehmen mithilfe der Unternehmens-Auswahl in das Unternehmen ändern, für das Sie das Banner für hochgeladen haben.</span><span class="sxs-lookup"><span data-stu-id="37691-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
