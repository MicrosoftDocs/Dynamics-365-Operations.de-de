# <a name="change-the-date-for-a-session"></a>Ändern des Datums für eine Sitzung

[!include [task guide banner](../../includes/task-guide-banner.md)]

In diesem Thema wird erläutert, wie das Datum für eine Sitzung in Dynamics 365 for Finance and Operations geändert wird. Standardmäßig wird das aktuelle Datum verwendet, wenn Journaleinträge oder Quelldokumente eingegeben und gebucht werden. Es besteht jedoch die Möglichkeit, das Datum zu ändern, das für die aktuelle verwenet wird. Verwenden Sie diese Funktion bei Bedarf für das Rückdatieren von Journaleinträgen oder Quelldokumenten.

1. Wechseln Sie im Navigationsbereich zu **Module > Allgemein > Allgemein > Sitzungsdatum und -uhrzeit**.
2. Geben Sie ein Datum in das Feld **Datum** ein.
3. Wählen Sie **OK**.
