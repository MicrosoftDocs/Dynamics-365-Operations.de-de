[!include [task guide banner](../../includes/task-guide-banner.md)]

    * <span data-ttu-id="5b409-112">Wählen Sie den zu erstellenden Bedingungstyp aus:</span><span class="sxs-lookup"><span data-stu-id="5b409-112">Select the type of operating unit you want to create.</span></span>

    * <span data-ttu-id="5b409-115">Erweitern Sie den allgemeinen Abschnitt, gegebenenfalls.</span><span class="sxs-lookup"><span data-stu-id="5b409-115">Expand the General section, if necessary.</span></span>
