# <a name="create-a-calendar-and-generate-working-times"></a>Kalender erstellen und Arbeitszeiten generieren

[!include[task guide banner](../../includes/task-guide-banner.md)]

Kalender beschreiben die Kapazität und die Arbeitszeiten von betrieblichen Ressourcen. Diese Prozedur zeigt Ihnen die Definition eines Arbeitskalenders auf Grundlage eines Schichtmodells. Sie können diese Prozedur Schritt für Schritt im Demodatenunternehmen USMF durchführen oder können Ihre eigenen Daten verwenden.

1. Wechseln Sie zu "Alle Arbeitsbereiche" > "Ressourcenlebenszyklusverwaltung".
2. Klicken Sei auf "Kalender".
3. Klicken Sie auf "Neu".
4. Geben Sie im Feld Kalender einen Wert ein.
    * Dies ist die Kennung des Kalenders, der als Referenz beim Zuweisen von Kalendern verwendet wird (z. B. zu einer betrieblichen Ressource oder Ressourcengruppe).  
5. Geben Sie im Feld "Name" einen Wert ein.
6. Geben Sie im Feld "Standardarbeitstag in Stunden" eine Zahl ein.
7. Markieren Sie in der Liste die ausgewählte Zeile.
8. Klicken Sie auf "Arbeitszeiten".
9. Klicken Sie auf "Arbeitszeiten einrichten".
    * Generieren Sie Arbeitsstunden für jeden Tag in der Periode, in der Sie Arbeit planen möchten. Im Laufe der Zeit können Sie Arbeitszeiten für zusätzliche Perioden generieren.  
10. Geben Sie in das Feld "Von Datum" ein Datum ein.
    * Dies ist der erste Tag, an dem dieser Kalender offen sein muss.  
11. Geben Sie in das Feld "Bis Datum" ein Datum ein.
    * Dies ist der erste Tag, an dem dieser Kalender offen ist.  
12. Geben Sie im Feld "Schichtmodell" einen Wert ein, oder wählen Sie einen Wert aus.
    * Die Schichtmodell definiert die Arbeitsstunden für jeden Wochentag.  
13. Klicken Sie auf "OK".
14. Schließen Sie die Seite.
