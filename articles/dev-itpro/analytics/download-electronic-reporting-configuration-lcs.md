    - <span data-ttu-id="a79ce-107">Entwickler für elektronische Berichterstellung</span><span class="sxs-lookup"><span data-stu-id="a79ce-107">Electronic reporting developer</span></span>
    - <span data-ttu-id="a79ce-108">Funktionaler Berater für elektronische Berichterstellung</span><span class="sxs-lookup"><span data-stu-id="a79ce-108">Electronic reporting functional consultant</span></span>
    - <span data-ttu-id="a79ce-109">Systemadministrator</span><span class="sxs-lookup"><span data-stu-id="a79ce-109">System administrator</span></span>

2. <span data-ttu-id="a79ce-110">Wechseln Sie zu **Organisationsverwaltung**&gt; **Elektronische Berichterstellung**..</span><span class="sxs-lookup"><span data-stu-id="a79ce-110">Go to **Organization administration** &gt; **Electronic reporting**.</span></span>
3. <span data-ttu-id="a79ce-111">Wählen Sie im Abschnitt **Konfigurationsanbieter** die Kachel **Microsoft** aus.</span><span class="sxs-lookup"><span data-stu-id="a79ce-111">In the **Configuration providers** section, select the **Microsoft** tile.</span></span>
4. <span data-ttu-id="a79ce-112">Klicken Sie auf der Kachel **Microsoft** auf **Repositorys**.</span><span class="sxs-lookup"><span data-stu-id="a79ce-112">On the **Microsoft** tile, click **Repositories**.</span></span>

    <span data-ttu-id="a79ce-113">[![Aktualisieren von ER über LCS für MS - MS Repositorys-Liste öffnen](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)</span><span class="sxs-lookup"><span data-stu-id="a79ce-113">[![update-er-from-lcs-for-ms-open-ms-repositories-list](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)</span></span>

    1. <span data-ttu-id="a79ce-116">Klicken Sie zum Hinzufügen neuer Repository auf **Hinzufügen**.</span><span class="sxs-lookup"><span data-stu-id="a79ce-116">Click **Add** to add a new repository.</span></span>
    2. <span data-ttu-id="a79ce-117">Aktivieren Sie **LCS** als Repository-Typ.</span><span class="sxs-lookup"><span data-stu-id="a79ce-117">Select **LCS** as the repository type.</span></span>
    3. <span data-ttu-id="a79ce-118">Klicken Sie auf **Repository erstellen**.</span><span class="sxs-lookup"><span data-stu-id="a79ce-118">Click **Create repository**.</span></span>
    4. <span data-ttu-id="a79ce-119">Wenn Sie aufgefordert werden, folgen Sie den Autorisierungsanweisungen.</span><span class="sxs-lookup"><span data-stu-id="a79ce-119">If prompted, follow the authorization instructions.</span></span>
    5. <span data-ttu-id="a79ce-120">Geben Sie einen Namen und eine Beschreibung für das Repository ein.</span><span class="sxs-lookup"><span data-stu-id="a79ce-120">Enter a name and description for the repository.</span></span>
    6. <span data-ttu-id="a79ce-121">Klicken Sie auf **OK**, um den neuen Repositoryeintrag zu bestätigen.</span><span class="sxs-lookup"><span data-stu-id="a79ce-121">Click **OK** to confirm the new repository entry.</span></span>
    7. <span data-ttu-id="a79ce-122">Wählen Sie im Raster das neue Repository vom Typ **LCS** aus.</span><span class="sxs-lookup"><span data-stu-id="a79ce-122">In the grid, select the new repository of the **LCS** type.</span></span>

6. <span data-ttu-id="a79ce-123">Klicken Sie auf **Öffnen**, um die Liste der ER-Konfigurationen für das ausgewählte Repository anzuzeigen.</span><span class="sxs-lookup"><span data-stu-id="a79ce-123">Click **Open** to view the list of ER configurations for the selected repository.</span></span>

    <span data-ttu-id="a79ce-124">[![update-er-from-lcs-for-ms-make-lcs-repository](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span><span class="sxs-lookup"><span data-stu-id="a79ce-124">[![update-er-from-lcs-for-ms-make-lcs-repository](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span></span>

7. <span data-ttu-id="a79ce-125">Wählen Sie in der Konfigurationsstruktur im linken Bereich die ER-Konfiguration aus, die Sie benötigen.</span><span class="sxs-lookup"><span data-stu-id="a79ce-125">In the configurations tree in the left pane, select the ER configuration that you require.</span></span>
8. <span data-ttu-id="a79ce-126">Wählen Sie im Inforegister **Versionen** die erforderliche Version der ausgewählten ER-Konfiguration aus.</span><span class="sxs-lookup"><span data-stu-id="a79ce-126">On the **Versions** FastTab, select the required version of the selected ER configuration.</span></span>
9. <span data-ttu-id="a79ce-127">Klicken Sie auf **Importieren**, um die ausgewählte Version vom LCS auf die aktuelle Finance and Operations Instanz herunterzuladen.</span><span class="sxs-lookup"><span data-stu-id="a79ce-127">Click **Import** to download the selected version from LCS to the current Finance and Operations instance.</span></span>

    > [!NOTE]
    > <span data-ttu-id="a79ce-128">Die Schaltfläche **Importieren** ist nicht für ER-Konfigurationsversionen verfügbar, die in der aktuellen Finance and Operations-Instanz bereits vorhanden sind.</span><span class="sxs-lookup"><span data-stu-id="a79ce-128">The **Import** button is unavailable for ER configuration versions that are already present in the current Finance and Operations instance.</span></span>

    <span data-ttu-id="a79ce-129">[![update-er-from-lcs-for-ms-download-configuration](./media/update-er-from-lcs-for-ms-download-configuration.png)](./media/update-er-from-lcs-for-ms-download-configuration.png)</span><span class="sxs-lookup"><span data-stu-id="a79ce-129">[![update-er-from-lcs-for-ms-download-configuration](./media/update-er-from-lcs-for-ms-download-configuration.png)](./media/update-er-from-lcs-for-ms-download-configuration.png)</span></span>
