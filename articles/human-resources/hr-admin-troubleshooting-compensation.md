# <a name="run-the-compensation-process"></a>Vergütungsprozesses ausführen

**Abgang**

Der Kunde versucht, den Kompensationsprozess in Microsoft Dynamics 365 Human Resources auszuführen, ist aber nicht erfolgreich.

**Ursache**

Der Kunde benötigt die Dokumentation.

**Auflösung**

Hier ist die Dokumentation für die Kompensation:

- "Einstellungen":

    - [Vergütungspläne](https://docs.microsoft.com/dynamics365/unified-operations/talent/compensation-plans)
    - [Erstellen fester Vergütungspläne](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-fixed-compensation-plans)
    - [Variable Vergütungspläne erstellen](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-variable-compensation-plans)

- "Bearbeiten":

    - [Vergütung verarbeiten](https://docs.microsoft.com/dynamics365/unified-operations/talent/process-compensation)
