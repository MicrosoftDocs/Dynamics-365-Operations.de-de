In diesem Thema wird der Optionssatz für Leer Ja Nein für Dynamics 365 Human Resources beschrieben.

Diese Aufzählung bietet einen Optionssatz für Ja/Nein-Eigenschaften, die auch Leerzeichen zulassen.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Leer | Es wurde kein Wert ausgewählt. |
| 200000001 | Ja | Ja. |
| 200000002 | Nr. | Nr. |
