# <a name="custom-fields"></a>Benutzerdefinierte Felder

Sie können zu Seiten in Dynamics 365 Human Resources benutzerdefinierte Elemente hinzufügen. Weitere Informationen zur Erstellung benutzerdefinierter Felder finden Sie unter [Erstellen von und arbeiten mit benutzerdefinierten Feldern](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Die folgenden Entitäten in Common Data Service unterstützen benutzerdefinierte Felder, die in Human Resources erstellt werden:

- Arbeitskraft
- Nationalität
- Veteranenstatus
- Sprachcode
- Auftrag
- Stellentyp
- Stellenfunktion
- Position
- Positionstyp

Stellen Sie sicher, dass alle erstellten benutzerdefinierten Felder aktiviert sind, damit Sie sie in Common Data Service verwenden können. Weitere Informationen finden Sie unter [Benutzerdefinierte Felder in Datenentitäten verfügbar machen](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
