# <a name="custom-fields"></a>Benutzerdefinierte Felder

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Sie können zu Seiten in Dynamics 365 Human Resources benutzerdefinierte Elemente hinzufügen. Weitere Informationen zum Erstellen benutzerdefinierter Felder finden Sie unter [Erstellen und arbeiten mit benutzerdefinierten Feldern](/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Die meisten Dataverse-Tabellen unterstützen benutzerdefinierte Felder. Für eine vollständige Liste der Dataverse-Tabellen finden Sie unter [Dataverse-Tabellen](./hr-developer-entities.md). 

Stellen Sie sicher, dass alle erstellten benutzerdefinierten Felder aktiviert sind, damit Sie sie in Dataverse verwenden können. Weitere Informationen finden Sie unter [Benutzerdefinierte Felder in Datenentitäten verfügbar machen](/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
