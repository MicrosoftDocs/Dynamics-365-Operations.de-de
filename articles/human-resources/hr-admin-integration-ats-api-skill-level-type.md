# <a name="skill-level-type"></a>Typ der Qualifikationsstufe

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Thema wird der Optionssatz für den Typ der Qualifikationsstufe für Dynamics 365 Human Resources beschrieben.

Physischer Name: mshr_hrmskillleveltype

Diese Aufzählung bietet den Optionssatz von Werten, die die Stufe kategorisieren, die der Qualifikation eines Kandidaten zugewiesen ist.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Tatsächlich | Tatsächliche Qualifikationsstufe. |
| 200000001 | Vorgabe | Zielwert für die Qualifikationsstufe. |
