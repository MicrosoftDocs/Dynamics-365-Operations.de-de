# <a name="completion-status"></a>Abschlussstatus


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Artikel wird der Optionssatz für den Abschlussstatus für Dynamics 365 Human Resources beschrieben.

Physischer Name: mshr_hcmcompletionstatus

Diese Aufzählung bietet den Optionssatz der Statuswerte für Kandidaten-Screenings. 

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Nicht vollständig | Der Kandidat hat das Screening noch nicht abgeschlossen. |
| 200000001 | Erfolgreich | Der Kandidat hat das Screening bestanden. |
| 200000002 | Fehlgeschlagen | Der Kandidat hat das Screening nicht bestanden. |
