# <a name="screening-frequency"></a>Screening-Häufigkeit

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Thema wird der Optionssatz der Screening-Häufigkeit für Dynamics 365 Human Resources beschrieben.

Physischer Name: mshr_hcmfrequencyunit

Diese Aufzählung bietet den Optionssatz der Werte für die Screening-Häufigkeit. 

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 Nur einmalig | Das Screening ist nur einmal erforderlich. |
| 200000001 Täglich | Die Screening-Häufigkeit wird in Tagen berechnet. |
| 200000002 Wöchentlich | Die Screening-Häufigkeit wird in Wochen berechnet. |
| 200000003 Monatlich | Die Screening-Häufigkeit wird in Monaten berechnet. |
| 200000004 Jährlich | Die Screening-Häufigkeit wird in Jahren berechnet. |
