# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="5cf5d-103">Namen des Mitarbeiter-Self-Service-Arbeitsbereichs ändern</span><span class="sxs-lookup"><span data-stu-id="5cf5d-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="5cf5d-104">Wenn Sie Freiwillige oder andere Nicht-Mitarbeiter haben, möchten Sie möglicherweise den Namen des **Mitarbeiter-Self-Service**-Arbeitsbereichs ändern.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="5cf5d-105">Sie können diesen Arbeitsbereich stattdessen in **Self-Service** ändern.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="5cf5d-106">Ändern des Namens des **Mitarbeiter-Self-Service**-Arbeitsbereichs ändert auch das Menüelement, das intern von Dynamics 365 Human Resources verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="5cf5d-107">Wenn Sie zuvor Sicherheitsanpassungen auf das Menüelement **HcmEmployeeSelfServiceWorkspace** angewendet haben, empfehlen wir, die gleichen Änderungen auf **HcmSelfServiceWorkspace** anzuwenden, um die Parität aufrecht zu erhalten.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="5cf5d-108">Wählen Sie in der Personalverwaltung **Personalverwaltung** aus, wählen Sie **Links** aus und dann **Personalverwaltungsparameter**.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="5cf5d-109">Wählen Sie die Registerkarte **Mitarbeiter-Self-Service** aus.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="5cf5d-110">Unter **Anzeigename** wählen Sie **Self-Service** aus.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-110">Under **Display name**, select **Self service**.</span></span>

   ![Namen des Mitarbeiter-Self-Service-Arbeitsbereichs zu Self-Service ändern](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="5cf5d-112">Wählen Sie **Speichern** aus.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-112">Select **Save**.</span></span>
