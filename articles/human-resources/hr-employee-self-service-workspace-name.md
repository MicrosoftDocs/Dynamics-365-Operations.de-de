# <a name="change-employee-self-service-workspace-name"></a>Namen des Mitarbeiter-Self-Service-Arbeitsbereichs ändern

Wenn Sie Freiwillige oder andere Nicht-Mitarbeiter haben, möchten Sie möglicherweise den Namen des **Mitarbeiter-Self-Service**-Arbeitsbereichs ändern. Sie können diesen Arbeitsbereich stattdessen in **Self-Service** ändern.

> [!NOTE]
> Ändern des Namens des **Mitarbeiter-Self-Service**-Arbeitsbereichs ändert auch das Menüelement, das intern von Dynamics 365 Human Resources verwendet wird. Wenn Sie zuvor Sicherheitsanpassungen auf das Menüelement **HcmEmployeeSelfServiceWorkspace** angewendet haben, empfehlen wir, die gleichen Änderungen auf **HcmSelfServiceWorkspace** anzuwenden, um die Parität aufrecht zu erhalten.

1. Wählen Sie in der Personalverwaltung **Personalverwaltung** aus, wählen Sie **Links** aus und dann **Personalverwaltungsparameter**.

2. Wählen Sie die Registerkarte **Mitarbeiter-Self-Service** aus.

3. Unter **Anzeigename** wählen Sie **Self-Service** aus.

   ![Namen des Mitarbeiter-Self-Service-Arbeitsbereichs zu Self-Service ändern](./media/hr-employee-self-service-workspace-name.png)

4. Wählen Sie **Speichern** aus.
