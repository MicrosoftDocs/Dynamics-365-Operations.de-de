# <a name="request-a-leave-of-absence"></a>Urlaubs- und Abwesenheitsanträge beantragen

>[!Important]
>Die in diesem Thema beschriebene Funktionalität ist derzeit für Kunden der eigenständigen Version von Dynamics 365 Human Resources verfügbar. Einige oder die gesamten Funktionen werden in einem zukünftigen Release der Finance-Infrastruktur nach Finance-Version 10.0.26 verfügbar sein.


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Sie können einen Antrag auf Urlaub stellen und den Status Ihrer Urlaubsanträge anzeigen in Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Urlaubs- und Abwesenheitsanträge beantragen

1. In dem **Mitarbeiter Selbstservice** Arbeitsbereich wählen Sie **Mehr** (...) in der Kachel **Freizeitguthaben**.

2. Um einen Antrag auf Beurlaubung einzureichen wählen Sie **Beurlaubung beantragen**.

3. Geben Sie Informationen ein für **Abwesenheitstyp**, **Anfangsdatum** und **Endtermin**.

4. Wenn Sie unterstützende Unterlagen einreichen müssen, wählen Sie **Hochladen** unter **Anhänge**.

5. Geben Sie Informationen ein unter **Kommentar**, sofern erforderlich.

6. Wählen **einreichen**, wenn Sie bereit sind, Ihre Anfrage einzureichen. Andernfalls wählen Sie **Entwurf speichern**.


## <a name="view-leave-of-absence-request-status"></a>Urlaubsantragsstatus anzeigen

1. In dem **Mitarbeiter Selbstservice** Arbeitsbereich wählen Sie **Mehr** (...) in der Kachel **Freizeitguthaben**.

2. Um einen Antrag auf Urlaubung anzuzeigen, wählen Sie **Urlaub anzeigen**.

## <a name="importing-leave-requests-from-other-systems-or-older-systems"></a>Import von Urlaubsanträgen aus anderen Systemen oder älteren Systemen

Um Urlaubsanträge aus einem anderen System zu importieren, müssen Sie den regulären Workflow durchlaufen, um die entsprechenden Urlaubsvorgänge zu erstellen. Alternativ können Sie die Abwesenheitsbanktransaktionen und die Abwesenheitsanträge im abgeschlossenen Zustand importieren. Beachten Sie, dass die Abwesenheitsbanktransaktionen nicht automatisch erstellt werden, wenn Sie nur die Abwesenheitsanträge importieren.
