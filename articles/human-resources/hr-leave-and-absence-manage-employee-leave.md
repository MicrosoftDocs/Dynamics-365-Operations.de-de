# <a name="manage-employee-leave"></a>Mitarbeiter-Sonderurlaub verwalten

>[!Important]
>Die in diesem Thema beschriebene Funktionalität ist derzeit für Kunden der eigenständigen Version von Dynamics 365 Human Resources verfügbar. Einige oder die gesamten Funktionen werden in einem zukünftigen Release der Finance-Infrastruktur nach Finance-Version 10.0.26 verfügbar sein.


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Sie können die Abwesenheit eines Mitarbeiters nach Abwesenheitstyp verwalten. Dies umfasst das Ablaufen der Abwesenheitsregistrierung und das Anpassen der Abwesenheitstypsguthaben. 

## <a name="adjust-leave-balances"></a>Abwesenheitsguthaben anpassen

1. Wählen Sie im Datensatz des Mitarbeiters **Abweseneheit** aus.

2. **Urlaubs- und Abwesenheitseinstellungen** auswählen.

3. Wählen Sie **Saldo anpassen** aus.

4. Wählen Sie **Abwesenheitstyp** aus.

5. Geben Sie einen **Regulierungsbetrag** ein. 

6. Sie können optional ein **Datum** eingeben. 

Sie können einen Ursachencode und einen Kommentar einfügen, wenn Sie das Abwesenheitsguthaben eines Mitarbeiters anpassen. 

Wenn Sie nun mit der Maus über einen beliebigen Urlaubssaldo fahren, werden die folgenden Informationen angezeigt:

- **Verfügbar** – Der **Gesamtwert dieses Jahr** abzüglich dem Wert **Dieses Jahr bezogen**.
- **Gesamtwert in diesem Jahr**: Alle aufgelaufenen, angepassten und übertragenen arbeitsfreien Zeiten des Jahres.
- **In diesem Jahr genommen** -  Alle genehmigten arbeitsfreien Zeiten.
