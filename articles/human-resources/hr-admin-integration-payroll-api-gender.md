# <a name="payroll---gender-option-set"></a>Gehaltsabrechnung – Optionssatz „Geschlecht“

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Thema wird der Optionssatz für das Geschlecht für Dynamics 365 Human Resources beschrieben.

## <a name="option-set-gender-option"></a>Optionssatz: Optionssatz „Geschlecht“

Diese Aufzählung bietet den Optionssatz des Geschlechts für den Mitarbeiter. Dies ist im Optionssatz mshr_hcmpersongender verfügbar.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Keines | Es wurde kein Geschlecht angegeben. |
| 200000001 | Männlich | Männlich. |
| 200000002 | Weiblich | Weiblich. |
| 200000003 | Nicht spezifisch | Auswahl für ein unspezifisches Geschlecht. |
