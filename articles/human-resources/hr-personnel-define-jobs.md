# <a name="define-new-jobs"></a>Neue Einzelvorgänge definieren

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



Diese Aufgabe konzentriert sich auf die Erstellung einer neuen Stelle. Sie wird anhand manueller Eingabe und der Kopie von Vorlagenfunktionen abgeschlossen. Das Demodatenunternehmen, das verwendet wird, um diese Aufgabe zu erstellen, ist USMF. Diese Aufgabe ist für den Personalverwaltungsspezialisten vorgesehen.

1. Wechseln Sie zu **Personalverwaltung \> Stellen \> Stellen**.
2. Wählen Sie **Neu** aus.
3. Geben Sie im Feld **Stelle** einen Wert ein.
4. Geben Sie im Feld **Beschreibung** einen Wert ein, oder wählen Sie einen Wert aus.
5. Wählen Sie einen Standardtitel aus, der für die Stelle verwendet wird.

    Dieser Titel wird als Standardwert für neue Positionen verwendet, die der Stelle zugewiesen werden.

6. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
7. Wählen Sie in der Liste den Link in der ausgewählten Zeile.
8. Wählen Sie in der Gruppe **Maximale Positionsanzahl** die Option **Maximale Positionsanzahl** aus.
9. Geben Sie die maximale Positionsanzahl ein. 

    Positionszuweisungen, die diese Zahl überschreiten, sind nicht zulässig.

10. Im Abschnitt **Beschreibung** im Feld **Hinweis** **Eine Standardbeschreibung für die Stelle eingeben**.

    Diese Beschreibung gilt für alle Positionen, die dem Stelle zugewiesenen werden, die Sie definieren.

11. Wählen Sie im Abschnitt **Stellenklassifizierung** eine Stellenfunktion aus, die in Standardberichten innerhalb der„Personalverwaltung“ verwendet wird.
12. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
13. Wählen Sie in der Liste den Link in der ausgewählten Zeile.
14. Wählen Sie im Feld **Stellentyp** einen Stellentyp aus, um Stellen in Ihrer Organisation weiter zu klassifizieren.

    "Stellenypen" können bei der Berichterstellung in der gesamten "Personalverwaltung" verwendet werden.

15. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
16. Wählen Sie in der Liste den Link in der ausgewählten Zeile.
17. Wählen Sie im Abschnitt **Vergütung** eine Vergütungsstufe aus.

    Die Vergütungsstufe definiert die minimale und maximale Vergütung, die ein Mitarbeiter für die Stelle erhalten kann. Die Vergütungsregeln bestimmen, ob die Vergütung der Mitarbeiter außerhalb des Bereichs liegen kann.

18. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
19. Wählen Sie in der Liste den Link in der ausgewählten Zeile.
20. Beachten Sie, dass im Abschnitt **Qualifikationen** die Felder in den Inforegistern **Qualifikationen**, **Ausbildung** und **Bescheinigungen** leer sind. Nachdem Sie die Informationen aus einer Stellenvorlage kopiert haben, werden die Informationen in diesen Feldern in einem Einzelschritt eingegeben.
21. Wählen Sie **Speichern** aus.
22. Wählen Sie **Kopieren von**.
23. Wählen Sie **Aus Vorlage kopieren** aus.
24. Wählen Sie im Feld **Stellenvorlage** den Dropdown-Pfeil aus, um die Suche zu öffnen.
25. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
26. Wählen Sie in der Liste den Link in der ausgewählten Zeile.
27. Wählen Sie die Option **Hinweis** aus.

    Nur die Artikel, die Sie auswählen, werden zur ausgewählten Stelle kopiert.

28. Wählen Sie **OK** aus.
29. Erweitern Sie den Abschnitt **Zertifikate**.
30. Erweitern Sie den Abschnitt **Ausbildung**.
31. Wählen Sie **Speichern** aus.
