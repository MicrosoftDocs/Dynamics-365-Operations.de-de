# <a name="get-support"></a>Support anfordern

Dieser Artikel erklärt, wie Sie Hilfe zu Microsoft Dynamics 365 Human Resources erhalten.

Um Unterstützung für Human Resources zu erhalten, verwenden Sie die Ressourcen in der folgenden Tabelle. Diese Ressourcen werden in der Reihenfolge aufgelistet, die wir empfehlen, wenn Sie Fehler beheben möchten.

| Aufgabe | Weitere Informationen |
|------|------------------|
| Die Community fragen. | Wechseln Sie zu [Dynamics 365 Human Resources Community](https://community.dynamics.com/365/talent), um Hilfe zu Ihren Fragen zu erhalten. Wenn Sie Antworten suchen, geben Sie eine Frage in das Suchfeld neben **Haben Sie eine Frage** ein. Wenn Sie eine Frage stellen möchten, wählen Sie **Eine Frage stellen** aus. |
| Nutzen Sie das in der Cloud bereitgestellte Supporttool. | Wechseln Sie in [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/) zu Ihrem Projekt, und wählen Sie die Kachel **Support** aus. Hier können Sie Probleme auf zwei Arten senden:<ul><li>Wählen Sie in der Registerkarte **Aktive Probleme** das Problem aus, das Sie senden möchten, und wählen Sie dann **An Microsoft senden** aus.</li><li>Wählen Sie auf die Registerkarte **An Microsoft übermittelt** die Option **Neues Problem übermitteln** aus, um ein neues Problem zu senden.</li></ul> |
| Erhalten Sie In-App-Support. | Wählen Sie in der rechten oberen Ecke der Seite die Schltfläche **Hilfe** (**?**) aus. Wählen Sie **Support** aus. Probleme werden auf der Registerkarte **Aktive Probleme** in LCS gemeldet, wo ein Administrator bestimmen kann, ob interne Unterstützung zulässig ist oder die Probleme an Microsoft gesendet werden. |
| Erhalten Sie Hilfe bei Fragen zu Human Resources-Lizenzierung. | Kontaktieren Sie Ihren Partner oder Ihren Microsoft Vertreter. |
| Fordern Sie neue Funktionen und Funktionalitäten in Human Resources an. | Wählen Sie in Human Resources die Schaltfläche **Hilfe** (**?**) aus, und wählen Sie dann **Ideen** aus. Sie können auch [Dynamics 365-Anwendungsideen](https://experience.dynamics.com/ideas/) besuchen, nach vorhandenen Ideen suchen oder dafür abstimmen oder neue Ideen hinzuzufügen. |
