# <a name="enroll-and-remove-benefits-from-workers"></a>Vorteile von Arbeitskräften registrieren und entfernen

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



Diese Verfahren zeigt, wie einzelne Arbeitskräfte für einen oder mehrere Vorteile registriert werden können und wie mehrere Arbeitskräfte für einen Vorteil registriert werden können. Das Demodatenunternehmen, das verwendet wird, um diese Prozedur zu erstellen, ist USMF.


## <a name="enroll-a-single-worker-in-benefits"></a>Registrieren einer Arbeitskraft für Leistungen
1. Wechseln Sie zu "Personalverwaltung" > "Arbeitskräfte" > "Mitarbeiter".
2. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
3. Klicken Sie auf "Vorteile".
4. Klicken Sie auf Neu.
5. Geben Sie im Feld "Vorteil" einen Wert ein, oder wählen Sie einen Wert aus.
6. Geben Sie im Feld "Startdatum der Disposition" ein Datum und eine Uhrzeit ein.
7. Geben Sie im Feld "Enddatum der Disposition" ein Datum und eine Uhrzeit ein.
8. Erweitern Sie den Abschnitt "Begünstigte", wenn Begünstigte für den Vorteil hinzugefügt werden müssen. Sie können über diese Seite auch Unterhaltsberechtigte hinzufügen, falls die Vorteile auch für diese zutreffen.
9. Sie können auch die Details einer Vorteilsregistrierung bearbeiten oder eine Registrierung auf dieser Seite löschen. Nachdem Sie die gewünschten Änderungen am Vorteil vorgenommen haben, schließen Sie die Seite.

## <a name="enroll-multiple-workers-in-a-benefit"></a>Mehrere Arbeitskräfte für eine Leistung registrieren
1. Schließen Sie die Seite.
2. Wechseln Sie zu "Personalverwaltung" > "Arbeitskräfte" > "Mitarbeiter".
3. Markieren Sie in der Liste die ausgewählte Zeile.
4. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
5. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
6. Klicken Sie auf "Für Vorteile registrieren".
7. Geben Sie im Feld "Vorteil" einen Wert ein, oder wählen Sie einen Wert aus.
8. Geben Sie im Feld "Startdatum der Disposition" ein Datum und eine Uhrzeit ein.
9. Geben Sie im Feld "Enddatum der Disposition" ein Datum und eine Uhrzeit ein.
10. Klicken Sie auf "Registriert".
11. Schließen Sie die Seite.
12. Wechseln Sie zu "Personalverwaltung" > "Registrierung" > "Berechtigung" > "Leistungsregistrierungsergebnisse".
13. Suchen Sie nach dem Vorteilsergebnisdatensatz, den Sie suchen.
14. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
15. Auf dieser Seite können Sie anzeigen, welche Mitarbeiter für die Leistung registriert wurden, sowie alle Mitarbeiter, die nicht registriert wurden.
