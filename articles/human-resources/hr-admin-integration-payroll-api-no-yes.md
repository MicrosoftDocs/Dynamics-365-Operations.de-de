# <a name="payroll---no-yes-option-set"></a>Gehaltsabrechnung – „Nein Ja“-Optionssatz


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Thema wird der Optionssatz für Nein Ja für Dynamics 365 Human Resources beschrieben.

## <a name="option-set-no-yes"></a>Optionssatz: Nein Ja

Physischer Name: mshr_noyes

Diese Aufzählung bietet den Optionssatz für die typische boolesche Eigenschaften in virtuellen Human Resources-Entitäten.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Nein | Nein. |
| 200000001 | Ja | Ja. |
