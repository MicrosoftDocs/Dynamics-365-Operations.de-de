# <a name="payroll---employment-type-option-set"></a>Gehaltsabrechnung – Optionssatz für den Beschäftigungstyp

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Artikel wird der Optionssatz für den Beschäftigungstyp für Dynamics 365 Human Resources beschrieben.

## <a name="option-set-employment-type"></a>Optionssatz: Beschäftigungstyp

Diese Enumeration stellt den Optionssatz für den Beschäftigungstyp für den Mitarbeiter bereit. Dies ist im Optionssatz mshr_hcmemploymenttype verfügbar.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Mitarbeiter | Mitarbeiter |
| 200000001 | Auftragnehmer | Auftragnehmer |
