# <a name="set-up-positions"></a>Positionen einrichten

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



Positionen ist ein wichtiges Element der untergeordneten Ebene einer Organisationshierarchie. Eine Position ist eine einzelne Instanz eines Einzelvorgangs. Beispielsweise ist die Position, „Verkaufsleiter (Ost)“ derzeit eine der Positionen, die der Stelle „Verkaufsleiter“ zugeordnet wird. Eine Position besteht in einer Abteilung und hat nur eine Arbeitskraft, die ihr zugeordnet ist. In dieser Aufgabe durchlaufen wir die Schritte, die zum Erstellen einer Position erforderlich sind. Diese Prozedur ist für Personalverwaltungsspezialisten vorgesehen.

1. Wählen Sie **Verwaltung des Personalbestands** aus.
2. Wählen Sie **Positionen öffnen** aus.
3. Wählen Sie **Neu** aus, um das Dropdown-Dialogfeld zu öffnen.
4. Geben Sie im Feld **Stelle** einen Wert ein, oder wählen Sie einen Wert aus.

    Die Felder **Stellenbeschreibung**, **Titel** und der **Beschäftigungsfaktor (entspricht Vollzeitbeschäftigung)** werden automatisch von der ausgewählten Stelle in die Position kopiert.

5. "ResolveChanges" die Stelle.
6. Wählen Sie **Position erstellen**.
7. Geben Sie im Feld **Abteilung** einen Wert ein, oder wählen Sie einen Wert aus.
8. Geben Sie im Feld **Positionstyp** einen Wert ein, oder wählen Sie einen Wert aus.
9. Geben Sie im Feld **Vergütungsregion** einen Wert ein oder wählen Sie einen aus.

    Das Feld **Vergütungsregion** bestimmt die Vergütungsberechtigungsregeln und die Budgets mit festen Erhöhungen, die für einen Mitarbeiter in dieser Position gelten.

10. Geben Sie im Feld **Für Zuweisung verfügbar** ein Datum und eine Uhrzeit ein.
11. Erweitern Sie den Abschnitt **Positionsdauer**.

    Die Positionsdauer wird standardmäßig auf Grundlage der Aktivierungs- und Ruhestandsdaten eingegeben, die zuvor eingegeben wurden.

12. Erweitern Sie den Abschnitt **Berichte an Position**.

    Wenn Sie eine Arbeitskraft einer Position zuweisen, die einer andere Position untergeben ist, erstellen Sie eine direkte Berichtsbeziehung zwischen den Arbeitskräften, die den beiden Positionen zugewiesen werden.

13. Wählen Sie **Neu** aus, um das Dropdown-Dialogfeld zu öffnen.
14. Geben Sie im Feld **Vorgesetzter** einen Wert ein, oder wählen Sie einen Wert aus.
15. Wählen Sie **Erstellen** aus.
16. Erweitern Sie den Abschnitt **Arbeitskraftzuweisung**.
17. Erweitern Sie den Abschnitt **Beziehungen**.

    Wenn Ihre Organisation eine Matrixhierarchie oder eine andere benutzerdefinierte Hierarchie verwendet, können Sie Positionshierarchietypen einrichten und dann Berichtsbeziehungen zu den Positionen für jede Hierarchie hinzufügen, Sie einrichten.

18. Wählen Sie **Hinzufügen** aus.
19. Markieren Sie in der Liste die ausgewählte Zeile.
20. Geben Sie im Feld **Hierarchiename** einen Wert ein, oder wählen Sie einen Wert aus.
21. Geben Sie im Feld **Positionsberichte** einen Wert ein, oder wählen Sie einen Wert aus.
22. Erweitern Sie den Abschnitt **Lohn**.
23. Geben Sie im Feld **Lohnzyklus** einen Wert ein, oder wählen Sie einen Wert aus.
24. Geben Sie im Feld **Bezahlt von** einen Wert ein, oder wählen Sie einen Wert aus.
25. Geben Sie im Feld **Regelmäßige Stunden pro Jahr** eine Zahl ein.

    Der Wert, den Sie eingeben, ist die Zahl der regelmäßig gezahlten Stunden, die die Arbeitskraft in dieser Position erwartungsgemäß jedes Jahr arbeitet.

26. Erweitern Sie den Abschnitt **Gewerkschaft**.
27. Reduzieren Sie den Abschnitt **Gewerkschaft**.
28. Erweitern Sie den Abschnitt **Finanzdimension**.
29. Geben Sie im Feld **Verteilungsvorlage** einen Wert ein, oder wählen Sie einen Wert aus.
30. Geben Sie im Feld **Abteilung** einen Wert ein, oder wählen Sie einen Wert aus.
31. Wählen Sie **Speichern** aus.
