# <a name="suspend-leave"></a><span data-ttu-id="94fac-103">Urlaub aussetzen</span><span class="sxs-lookup"><span data-stu-id="94fac-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="94fac-104">Sie können den Urlaub für einen Mitarbeiter aussetzen, um zu verhindern, dass Urlaubsrückstellungen für ausgewählte Urlaubstypen verarbeitet werden.</span><span class="sxs-lookup"><span data-stu-id="94fac-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="94fac-105">Abwesenheit und Urlaub für einen Mitarbeiter aussetzen</span><span class="sxs-lookup"><span data-stu-id="94fac-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="94fac-106">Wählen Sie im Datensatz des Mitarbeiters **Abweseneheit** aus.</span><span class="sxs-lookup"><span data-stu-id="94fac-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="94fac-107">Wählen Sie **Urlaub aussetzen**.</span><span class="sxs-lookup"><span data-stu-id="94fac-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="94fac-108">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="94fac-108">Select **New**.</span></span>

4. <span data-ttu-id="94fac-109">In dem Dialogfeld **Urlaubsrückstellung aussetzen** wählen Sie im Dialogfeld **Urlaub aussetzen** zusammen mit **Anfangsdatum** und **Endtermin**.</span><span class="sxs-lookup"><span data-stu-id="94fac-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="94fac-110">Optional können Sie einen **Kommentar** für die Aussetzung hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="94fac-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="94fac-111">Wenn Rückstellungen bearbeitet werden, während der Urlaub des Mitarbeiters ausgesetzt ist, werden keine Rückstellungen für die Arten des suspendierten Urlaubs gebildet.</span><span class="sxs-lookup"><span data-stu-id="94fac-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
