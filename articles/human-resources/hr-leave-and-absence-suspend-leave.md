# <a name="suspend-leave"></a><span data-ttu-id="7df8c-103">Urlaub aussetzen</span><span class="sxs-lookup"><span data-stu-id="7df8c-103">Suspend leave</span></span>

[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="7df8c-104">Sie können den Urlaub für einen Mitarbeiter aussetzen, um zu verhindern, dass Urlaubsrückstellungen für ausgewählte Urlaubstypen verarbeitet werden.</span><span class="sxs-lookup"><span data-stu-id="7df8c-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="7df8c-105">Abwesenheit und Urlaub für einen Mitarbeiter aussetzen</span><span class="sxs-lookup"><span data-stu-id="7df8c-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="7df8c-106">Wählen Sie im Datensatz des Mitarbeiters **Abweseneheit** aus.</span><span class="sxs-lookup"><span data-stu-id="7df8c-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="7df8c-107">Wählen Sie **Urlaub aussetzen**.</span><span class="sxs-lookup"><span data-stu-id="7df8c-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="7df8c-108">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="7df8c-108">Select **New**.</span></span>

4. <span data-ttu-id="7df8c-109">In dem Dialogfeld **Urlaubsrückstellung aussetzen** wählen Sie im Dialogfeld **Urlaub aussetzen** zusammen mit **Anfangsdatum** und **Endtermin**.</span><span class="sxs-lookup"><span data-stu-id="7df8c-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

<span data-ttu-id="7df8c-111">Wenn Rückstellungen bearbeitet werden, während der Urlaub des Mitarbeiters ausgesetzt ist, werden keine Rückstellungen für die Arten des suspendierten Urlaubs gebildet.</span><span class="sxs-lookup"><span data-stu-id="7df8c-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
