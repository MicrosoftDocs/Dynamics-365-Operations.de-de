# <a name="regulatory-job-category"></a>Kategorie für Behördenstellen


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Artikel wird der Optionssatz „Kategorie für Behördenstellen“ für Dynamics 365 Human Resources beschrieben.

Physischer Name: mshr_hcmregulatoryjobcategory

Diese Aufzählung gibt die Werte des Optionssatzes für die regulatorische Stellenkategorien an. Dies wird im vorhandenen Optionssatz mshr_hcmregulatoryjobcategory bereitstellt.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Keines | Keiner. |
| 200000001 | Führungskraft | Leitende Angestellte/Manager. |
| 200000002 | Verantwortlicher | Beamte und Manager der ersten/mittleren Ebene. |
| 200000003 | Beruflich | Experten. |
| 200000004 | Techniker | Techniker. |
| 200000005 | SalesWorker | Verkäufer. |
| 200000006 | Verwaltung | Administrative Support-Mitarbeiter. |
| 200000007 | CraftWorker | Handwerker. |
| 200000008 | Agent | Einsatzkräfte. |
| 200000009 | Arbeiter | Arbeiter/Helfer. |
| 200000010 | ServiceWorker | Dienstleister. |
