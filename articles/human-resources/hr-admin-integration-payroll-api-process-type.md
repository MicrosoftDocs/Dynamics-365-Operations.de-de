Diese Aufzählung bietet den Optionssatz für den Prozesstyp in virtuellen Human Resources-Entitäten.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Ohne | Keiner. |
| 200000001 | Jährlich | Jährlich. |
| 200000002 | Jahrestag | Jahrestag |
| 200000003 | Nicht zyklisch | Nicht zyklisch. |
| 200000004 | Halbes Jahr | Halbjährlich. |
| 200000005 | Quartalsweise | Quartalsweise. |
| 200000006 | Monatlich | Monatlich. |
