# <a name="process-life-event-changes"></a>Lebensereignisänderungen verarbeiten

Lebensereignisänderungen verarbeiten in Microsoft Dynamics 365 Human Resources für zwei Änderungen bei Lebensereignissen:

- Geburtstagsänderungen
- Ablaufänderungen Berechtigungsregelüberschreibung 

1. Wählen Sie im Arbeitsbereich **Vorteilsverwaltung** unter **Bearbeitung** die Option **Verarbeitung von Lebensereignisänderungen**.

2. Geben Sie im Dialogfeld **Verarbeitung von Lebensereignisänderungen ausführen** Werte für die folgenden Felder ein:

   | Feld | Beschreibung |
   | --- | --- |
   | Registrierungsperiode | Die Registrierungsperiode, für die die Lebensereignisänderung verarbeitet wird. |
   | Juristische Person | Die juristische Person, für die die Lebensereignisänderung verarbeitet wird. |

3. Wenn Sie den Prozess im Hintergrund ausführen möchten, wählen Sie **Im Hintergrund ausführen** und führen Sie die folgenden Aufgaben aus:

   1. Geben Sie Informationen für den Prozess ein.

   2. Um eine wiederkehrende Stelle einzurichten, wählen Sie **Wiederholung**, geben Sie die Wiederholungsinformationen ein und wählen Sie **OK**.

   3. Um eine Stellenbenachrichtigung einzurichten, wählen Sie **Benachrichtigungen**, wählen Sie die zu empfangenden Benachrichtigungen und wählen Sie dann **OK**.

   4. Wählen Sie **OK**. Der Prozess wird mit den von Ihnen festgelegten Parametern ausgeführt.

4. Wählen Sie **OK**.
