# <a name="create-employment-categories"></a>Mitarbeiterkategorien erstellen

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Beschäftigungskategorien einrichten in Dynamics 365 Human Resources, die zu Ihren Arbeitskräften passen.

1. Wählen Sie im Arbeitsbereich **Vorteilsverwaltung** unter **Einstellung** die Option **Beschäftigungskategorien**.

2. Wählen Sie **Neu** aus.

3. Geben Sie Werte für **Arbeitskrafttyp**, **Beschäftigungskategorie** und **Beschreibung** ein.

4. Wählen Sie **Speichern**.
