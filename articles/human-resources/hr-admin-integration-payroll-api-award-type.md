# <a name="award-type"></a>Prämientyp

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In diesem Thema wird der Optionssatz für den Prämientyp für Dynamics 365 Human Resources beschrieben.

## <a name="option-set-award-type"></a>Optionssatz: Prämientyp

Physischer Name: mshr_HrmCompVarAwardEmplType

Diese Aufzählung bietet den Optionssatz für die variable Vergütungsprämientyp in virtuellen Human Resources-Entitäten.

| Wert | Etikett | Beschreibung |
| --- | --- | --- |
| 200000000 | Allgemeine Prämie | Allgemeine Prämie. |
| 200000001 | Fester Lohn (Pauschalbetrag) | Fester Lohn (Pauschalbetrag). |
