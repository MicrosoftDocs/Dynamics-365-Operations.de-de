# <a name="add-fields-to-an-excel-workbook-to-edit-retail-transactions"></a>Einer Excel-Arbeitsmappe zum Bearbeiten von Einzelhandelstransaktionen Felder hinzufügen

[!include [banner](../includes/banner.md)]

In diesem Thema wird beschrieben, wie Sie einer Microsoft Excel-Arbeitsmappe Felder hinzufügen, damit Sie Einzelhandelstransaktionen in Microsoft Dynamics 365 Commerce bearbeiten können.

## <a name="overview"></a>Übersicht

Wenn Sie eine Excel-Datei generieren, damit Sie Einzelhandelstransaktionen bearbeiten können, wird die Datei mit einigen Standardfeldern gefüllt. Wenn ein Feld, das aktualisiert werden muss, in der generierten Excel-Datei nicht standardmäßig sichtbar ist, können Sie es hinzufügen.

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a>Einem Arbeitsblatt in einer Excel-Arbeitsmappe Felder hinzufügen

Führen Sie die folgenden Schritte aus, um einer Excel-Arbeitsmappe Felder hinzuzufügen, damit Sie Einzelhandelstransaktionen bearbeiten können.

1. Laden Sie die Excel-Datei von der Seite **Aufstellungen**, der Seite **Einzelhandelstransaktionen** oder der Kachel **Transaktionsüberprüfungsfehler** im Arbeitsbereich **Finanzdaten für Shop** herunter, und öffnen Sie sie.
1. Wählen Sie **Entwerfen** aus.
1. Wählen Sie das Stiftsymbol für die gewünschte Tabelle aus, und suchen Sie in der Liste der verfügbaren Felder das Feld, das Sie hinzufügen möchten.
1. Wählen Sie **Hinzufügen** und anschließend **Aktualisieren** aus. Sie können Felder neu anordnen.
1. Wählen Sie nach Abschluss des Updates **Aktualisieren** aus, um die Daten für die neue Spalte abzurufen.

Das neue Feld und die Daten dafür sollten jetzt zur Bearbeitung in Excel verfügbar sein.
