[!include [banner](../includes/banner.md)]

<span data-ttu-id="e29c1-104">In diesem Thema wird beschrieben, wie Sie einer Microsoft Excel-Arbeitsmappe Felder hinzufügen, sodass Sie Einzelhandelstransaktionen in Microsoft Dynamics 365 Commerce bearbeiten können.</span><span class="sxs-lookup"><span data-stu-id="e29c1-104">This topic describes how to add fields to a Microsoft Excel workbook so that you can edit retail transactions in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="e29c1-105">Übersicht</span><span class="sxs-lookup"><span data-stu-id="e29c1-105">Overview</span></span>

<span data-ttu-id="e29c1-106">Wenn Sie eine Excel-Datei generieren, um Einzelhandelstransaktionen zu bearbeiten, wird die Datei mit einigen Standardfeldern ausgefüllt.</span><span class="sxs-lookup"><span data-stu-id="e29c1-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="e29c1-107">Wenn ein Feld, das aktualisiert werden muss, in der generierten Excel-Datei nicht standardmäßig sichtbar ist, können Sie es hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="e29c1-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="e29c1-108">Einem Arbeitsblatt in einer Excel-Arbeitsmappe Felder hinzufügen</span><span class="sxs-lookup"><span data-stu-id="e29c1-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="e29c1-109">Führen Sie die folgenden Schritte aus, um einer Excel-Arbeitsmappe Felder hinzuzufügen, damit Sie Einzelhandelstransaktionen bearbeiten können.</span><span class="sxs-lookup"><span data-stu-id="e29c1-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="e29c1-110">Laden Sie die Excel-Datei von der Seite **Aufstellungen**, der Seite **Einzelhandelstransaktionen** oder der Kachel **Transaktionsüberprüfungsfehler** im Arbeitsbereich **Finanzdaten für Shop** herunter, und öffnen Sie sie.</span><span class="sxs-lookup"><span data-stu-id="e29c1-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="e29c1-111">Wählen Sie **Entwerfen** aus.</span><span class="sxs-lookup"><span data-stu-id="e29c1-111">Select **Design**.</span></span>
1. <span data-ttu-id="e29c1-112">Wählen Sie das Stiftsymbol für die gewünschte Tabelle aus, und suchen Sie in der Liste der verfügbaren Felder das Feld, das Sie hinzufügen möchten.</span><span class="sxs-lookup"><span data-stu-id="e29c1-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="e29c1-113">Wählen Sie **Hinzufügen** und anschließend **Aktualisieren** aus.</span><span class="sxs-lookup"><span data-stu-id="e29c1-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="e29c1-114">Sie können Felder neu anordnen.</span><span class="sxs-lookup"><span data-stu-id="e29c1-114">You can reorder fields.</span></span>
1. <span data-ttu-id="e29c1-115">Wählen Sie nach Abschluss des Updates **Aktualisieren** aus, um die Daten für die neue Spalte abzurufen.</span><span class="sxs-lookup"><span data-stu-id="e29c1-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="e29c1-116">Das neue Feld und die Daten dafür sollten jetzt zur Bearbeitung in Excel verfügbar sein.</span><span class="sxs-lookup"><span data-stu-id="e29c1-116">The new field and data for it should now be available for editing in Excel.</span></span>
