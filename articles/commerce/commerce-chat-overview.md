# <a name="commerce-chat-features-overview"></a>Übersicht über die E-Commerce-Chat-Funktionen

[!include [banner](includes/banner.md)]

Durch die Integration der Microsoft Dynamics 365 Omnichannel for Customer Service und Power Virtual Agents Anwendungen mit Dynamics 365 Commerce E-Commerce-Websites können Unternehmen ihre Online-Käufer durch Chat-Funktionen stärken. Dieser Artikel bietet einen Überblick über die Commerce Chat-Funktionen, die Online-Käufern zur Verfügung stehen, wenn Omnichannel for Customer Service und Power Virtual Agents mit Dynamics 365 Commerce E-Commerce-Websites integriert sind.

## <a name="omnichannel-for-customer-service"></a>Omnichannel for Customer Service

Omnichannel for Customer Service ist eine robuste Anwendung, die die Leistung von Dynamics 365 Customer Service erweitert. Es ermöglicht Unternehmen, sich über Kanäle wie Live-Chat, Sprache und Short Message Service (SMS) sofort mit ihren Kunden zu verbinden und mit ihnen in Kontakt zu treten. Weitere Informationen zu Omnichannel for Customer Service finden Sie unter [Einführung in Omnichannel for Customer Service](/dynamics365/customer-service/introduction-omnichannel). Informationen zur Integration von Omnichannel for Customer Service mit einer Dynamics 365 Commerce E-Commerce-Website, siehe [Commerce-Chat mit Omnichannel for Customer Service-Modul](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents können Sie leistungsstarke KI-gestützte Chatbots für eine Reihe von Anfragen erstellen. Weitere Informationen über Power Virtual Agents finden Sie unter [Power Virtual Agents Übersicht](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Informationen zur Integration von Power Virtual Agents mit einer Dynamics 365 Commerce E-Commerce-Website, siehe [Commerce Chat mit Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents ist bereits in Omnichannel for Customer Service enthalten. Wenn Sie sowohl virtuelle Agenten- (Chatbot) als auch Live-Agentenfunktionen benötigen, verwenden Sie Omnichannel for Customer Service.
