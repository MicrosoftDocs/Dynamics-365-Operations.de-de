## <a name="add-a-copyright-notice"></a><span data-ttu-id="9d79c-109">Hinzufügen eines Urheberrechtshinweises</span><span class="sxs-lookup"><span data-stu-id="9d79c-109">Add a copyright notice</span></span>

<span data-ttu-id="9d79c-110">Um einen Urheberrechtshinweis am Fuß jeder Seite hinzuzufügen die eine bestimmte Vorlage verwendet, führen Sie die folgenden Schritte aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="9d79c-111">Wechseln Sie zu **Fragmente**, und wählen Sie dann **Neues Seiten-Fragment** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="9d79c-112">Wählen Sie im Dialogfeld **Fußzeile** das Modul und geben Sie dem Fragment einen Namen.</span><span class="sxs-lookup"><span data-stu-id="9d79c-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="9d79c-113">Geben Sie zum Beispiel **Fußzeile-Urheberrecht** ein.</span><span class="sxs-lookup"><span data-stu-id="9d79c-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="9d79c-114">Wählen Sie **OK**.</span><span class="sxs-lookup"><span data-stu-id="9d79c-114">Select **OK**.</span></span>
1. <span data-ttu-id="9d79c-115">Im Navigationsbereich wählen Sie die Ellipsis-Schaltfläche (**...**) neben der **Fußzeile** aus, und wählen Sie dann **Fügen Sie Modul hinzu** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="9d79c-116">Wählen Sie im Dialogfeld **Fußzeilenkategorie** und **OK** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="9d79c-117">Im Navigationsbereich wählen Sie die Ellipsis-Schaltfläche (...) neben der **Fußzeilenkategorie** aus, und wählen Sie dann **Fügen Sie Modul hinzu** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="9d79c-118">Wählen Sie im Dialogfeld **Textblock** und dann **OK** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="9d79c-119">Wählen Sie im Navigationsbereich **Textblock** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="9d79c-120">Im Eigenschaftenbereich auf der rechten Seite im Feld **Absatz**, fügen Sie die Urheberrechtsnachricht hinzu.</span><span class="sxs-lookup"><span data-stu-id="9d79c-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="9d79c-121">Geben Sie beispielsweise **(C) Fabrikam 2019** ein.</span><span class="sxs-lookup"><span data-stu-id="9d79c-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="9d79c-122">Wählen Sie **Speichern** aus, wählen Sie **Beenden Sie die Bearbeitung** und **Veröffentlichen** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="9d79c-123">Navigieren Sie zu **Vorlagen**, wählen Sie die Vorlage und dann **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="9d79c-124">Erweitern Sie unter **Seiten-Gliederung**, **Text** und erweitern Sie dann **Standardseite**.</span><span class="sxs-lookup"><span data-stu-id="9d79c-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="9d79c-125">Aktivieren Sie die Schaltfläche neben **Fußzeilen-Slot**, und wählen Sie dann **Fügen Sie Fragment hinzu** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="9d79c-126">Wählen Sie das Fragment, das Sie eben erstellt haben, und wählen Sie dann **Auswählen** aus.</span><span class="sxs-lookup"><span data-stu-id="9d79c-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="9d79c-127">Wählen **Bearbeiten beenden**, um die Vorlage einzuchecken, und wählen Sie dann **Veröffentlichen**, um sie zu veröffentlichen.</span><span class="sxs-lookup"><span data-stu-id="9d79c-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="9d79c-128">Die Fußzeile, die den Urheberrechtshinweis automatisch enthält, wird im unteren Bereich alle Seiten hinzugefügt, die auf der ausgewählten Vorlage basieren.</span><span class="sxs-lookup"><span data-stu-id="9d79c-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
