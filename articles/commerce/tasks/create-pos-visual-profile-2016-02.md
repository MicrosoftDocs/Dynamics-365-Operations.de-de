# <a name="create-point-of-sale-pos-visual-profiles"></a>Visuelle Profile für Verkaufsstellen (POS) erstellen

[!include [banner](../includes/banner.md)]

Diese Prozedur führt Sie Schritt für Schritt durch das Erstellen eines neuen visuellen Verkaufsstellen (POS)-Profils. Ein visuelles Profil beinhaltet grundlegende Informationen zur Darstellung der POS-Register. Sie können mehrere visuelle Profile erstellen und bestimmte Profile zuweisen, die auf bestimmten Registern ausgeführt werden sollen. Für diese Prozedur wird das Demo-Datenunternehmen USRT verwendet.

1. Gehen Sie zu **Retail und Commerce > Kanaleinrichtung > POS-Einrichtung > POS-Profile > Visuelle Profile**.
2. Klicken Sie auf **Neu**.
3. Geben Sie in das Feld **Profilnummer** einen Wert ein.
4. Geben Sie im Feld **Beschreibung** einen Wert ein.
5. Klicken Sie im Feld **Anwendungstyp** auf die Dropdown-Schaltfläche, um das Nachschlagefeld zu öffnen.
6. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
7. Klicken Sie im Feld **Thema** auf die Dropdown-Schaltfläche, um das Nachschlagefeld zu öffnen.
8. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
9. Klicken Sie im Feld **Akzentfarbe** auf die Dropdown-Schaltfläche, um das Nachschlagefeld zu öffnen.
10. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
11. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
12. Schalten Sie die Erweiterung des Bereichs **Anmeldehintergrund** um.
13. Wählen Sie im Feld **Landschaftsbild-ID** eine Bild-ID aus oder geben Sie eine ein.
14. Wählen Sie im Feld** Porträtbild-ID** eine Bild-ID aus oder geben Sie eine ein.
15. Schalten Sie die Erweiterung des Abschnitts **Hintergrund** um.
16. Fordern Sie ein Popup für die Bildkennung an.
17. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
18. Klicken Sie auf **Speichern**.
