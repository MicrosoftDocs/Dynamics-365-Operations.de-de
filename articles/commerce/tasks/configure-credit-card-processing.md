# <a name="configure-credit-card-processing"></a> Verarbeitung der Kreditkartendaten konfigurieren

[!include [task guide banner](../includes/task-guide-banner.md)]

Diese Prozedur führt Sie Schritt für Schritt durch das Anzeigen der Liste der Zahlungsanbieter und Konfigurieren eines Zahlungskontos für Debitoren. Für diese Prozedur wird das Demo-Datenunternehmen USRT verwendet. Sie ist für Administratoren und IT-Experten vorgesehen.


## <a name="view-a-list-of-payment-providers"></a>Eine Liste der Zahlungsanbieter anzeigen
1. Wechseln Sie zu "Debitoren" > "Zahlungseinstellungen" > "Zahlungsdienste".
2. Klicken Sie auf "Verfügbare Anbieter".

## <a name="configure-payment-account"></a>Konfigurieren Sie ein Zahlungskonto
1. Klicken Sie auf "Neu".
2. Geben Sie im Feld "Zahlungsdienst" einen Wert ein.
3. Wählen Sie im Feld "Connector für Zahlungen" eine Option aus.
4. Schalten Sie die Erweiterung des Abschnitts "Zahlungsservicekonto" ein/aus.
5. Geben Sie "PROD" im Feld "Umgebung:" ein.
6. Klicken Sie auf "Kreditkartentypen".
7. Klicken Sie im Feld "Zahlungserfassung" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
8. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
9. Klicken Sie auf Hinzufügen.
10. Geben Sie im Feld "Währung" einen Wert ein.
11. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
12. Klicken Sie im Feld "Zahlungserfassung" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
13. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
14. Klicken Sie auf Hinzufügen.
15. Geben Sie im Feld "Währung" einen Wert ein.
16. Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.
    * Sie können diese Schritte für beliebig viele Kartentypen wiederholen.  
17. Klicken Sie im Feld "Zahlungserfassung" auf die Dropdown-Schaltfläche, um die Suche zu öffnen.
18. Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.
19. Klicken Sie auf Hinzufügen.
20. Geben Sie im Feld "Währung" einen Wert ein.
21. Klicken Sie auf "Speichern".
22. Schließen Sie die Seite.
23. Klicken Sie auf "Überprüfen".
24. Klicken Sie auf das Kontrollkästchen "Standardmäßige Verarbeitungsstelle für neue Kreditkarten".
25. Klicken Sie auf "Speichern".
