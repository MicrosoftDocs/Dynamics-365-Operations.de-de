## <a name="create-a-functionality-profile"></a><span data-ttu-id="2c94a-107">Ein Funktionsprofil erstellen</span><span class="sxs-lookup"><span data-stu-id="2c94a-107">Create a functionality profile</span></span>

<span data-ttu-id="2c94a-108">Führen Sie die folgenden Schritte aus, um ein Funktionsprofil zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="2c94a-108">To create a functionality profile, follow these steps.</span></span>

1. <span data-ttu-id="2c94a-109">Gehen Sie im Navigationsbereich zu **Module \> Kanaleinrichtung \> POS-Profile \> Funktionsprofile**.</span><span class="sxs-lookup"><span data-stu-id="2c94a-109">In the navigation pane, go to **Modules \> Channel setup \> POS profiles \> Functionality profiles**.</span></span>
1. <span data-ttu-id="2c94a-110">Wählen Sie im Aktivitätsbereich **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="2c94a-110">On the action pane, select **New**.</span></span>
1. <span data-ttu-id="2c94a-111">Geben Sie im Feld **Profil** eine Kennung für das Profil ein („FN006“ im folgenden Beispielbild).</span><span class="sxs-lookup"><span data-stu-id="2c94a-111">In the **Profile** field, enter an ID for the profile ("FN006" in the example image below).</span></span>
1. <span data-ttu-id="2c94a-112">Geben Sie im Feld **Beschreibung** einen Wert ein („Adventure Works Profile“ im folgenden Beispielbild).</span><span class="sxs-lookup"><span data-stu-id="2c94a-112">In the **Description** field, enter a value ("Adventure Works Profile" in the example image below).</span></span>
1. <span data-ttu-id="2c94a-113">Wählen Sie im Abschnitt **Allgemein** ein Land für das Gebietsschema **ISO** aus.</span><span class="sxs-lookup"><span data-stu-id="2c94a-113">In the **General** section, select a country for the **ISO** locale.</span></span>
1. <span data-ttu-id="2c94a-114">Ändern Sie im Abschnitt **Allgemein** weitere Einstellungen nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="2c94a-114">In the **General** section, modify other settings, as needed.</span></span>
1. <span data-ttu-id="2c94a-115">Wählen Sie im Abschnitt **Allgemein** eine **Belegprofilkennung** für E-Mail-Belege aus.</span><span class="sxs-lookup"><span data-stu-id="2c94a-115">In the **General** section, select a **Receipt profile ID** for email receipts.</span></span>
1. <span data-ttu-id="2c94a-116">Ändern Sie im Abschnitt **Funktionen** die Einstellungen nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="2c94a-116">In the **Functions** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="2c94a-117">Ändern Sie im Abschnitt **Betrag** die Einstellungen nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="2c94a-117">In the **Amount** section, modify settings as, needed.</span></span>
1. <span data-ttu-id="2c94a-118">Ändern Sie im Abschnitt **Infocodes** die Einstellungen nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="2c94a-118">In the **Info Codes** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="2c94a-119">Ändern Sie im Abschnitt **Belegnummerierung** die Einstellungen nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="2c94a-119">In the **Receipt numbering** section, modify settings, as needed.</span></span>
