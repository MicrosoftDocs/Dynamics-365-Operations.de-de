# <a name="set-up-retail-products"></a><span data-ttu-id="b8a58-103">Einzelhandelsprodukte einrichten</span><span class="sxs-lookup"><span data-stu-id="b8a58-103">Set up retail products</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="b8a58-104">In diesem Artikel wird beschrieben, wie Produkte in Dynamics 365 Commerce eingerichtet werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-104">This article describes how to set up products in Dynamics 365 Commerce.</span></span>

<span data-ttu-id="b8a58-105">Bevor Sie Produkte für den Verkauf in Ihren Commerce-Kanälen anbieten können, müssen Sie die Produkte erstellen und konfigurieren.</span><span class="sxs-lookup"><span data-stu-id="b8a58-105">Before you can offer products for resale in your commerce channels, you must create and configure the products.</span></span> <span data-ttu-id="b8a58-106">Commerce erstellt organisationsweit Produkte im Produktmaster.</span><span class="sxs-lookup"><span data-stu-id="b8a58-106">Commerce creates organization-wide products in the product master.</span></span> <span data-ttu-id="b8a58-107">Sie können die Produkte erstellen, die Produkteigenschaften und -attribute definieren sowie die Produkte den Commerce-Kategoriehierarchien zuweisen.</span><span class="sxs-lookup"><span data-stu-id="b8a58-107">You can create the products, define the product properties and attributes, and assign the products to commerce category hierarchies.</span></span> <span data-ttu-id="b8a58-108">Um die Produkte für Ihre Kanäle verfügbar zu machen und sie einem aktiven Sortiment hinzuzufügen, müssen Sie die Produkte für die juristischen Personen freigeben, in denen sie verfügbar sind.</span><span class="sxs-lookup"><span data-stu-id="b8a58-108">To make the products available to your channels and add them to an active assortment, you must release the products to the legal entities where they are available.</span></span> <span data-ttu-id="b8a58-109">Führen Sie zum Einrichten der Produkte, die Sie über Kanäle verkaufen, die folgenden Aufgaben aus.</span><span class="sxs-lookup"><span data-stu-id="b8a58-109">To set up the products that you sell by using channels, complete the following tasks.</span></span>

1. <span data-ttu-id="b8a58-110">**Definieren Sie eine Produkthierarchie.**</span><span class="sxs-lookup"><span data-stu-id="b8a58-110">**Define a product hierarchy.**</span></span> <span data-ttu-id="b8a58-111">Wenn Sie die Kategoriehierarchiefunktionen in Commerce verwenden, können Sie Kategoriehierarchien definieren, um die Produkte, die Sie an Ihre Kanäle vertreiben, zu gruppieren und kategorisieren.</span><span class="sxs-lookup"><span data-stu-id="b8a58-111">By using the category hierarchy features in Commerce, you can define category hierarchies to group and categorize the products that you distribute to your channels.</span></span> <span data-ttu-id="b8a58-112">Benutzerdefinierte und Systemattribute können auf der Kategorieebene definiert werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-112">User-defined and system attributes can be defined at the category level.</span></span> <span data-ttu-id="b8a58-113">Daraufhin erben alle Produkte, die der Kategorie zugewiesen sind, diese Attribute.</span><span class="sxs-lookup"><span data-stu-id="b8a58-113">Then, all products that are assigned to the category inherit those attributes.</span></span> <span data-ttu-id="b8a58-114">Mehrere Kategoriehierarchien können definiert werden, und jedes Produkt kann mehreren Hierarchien zugewiesen werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-114">Multiple category hierarchies can be defined, and each product can be assigned to multiple hierarchies.</span></span> <span data-ttu-id="b8a58-115">In einer einzelnen Hierarchie kann ein Produkt jedoch nur einer Kategorie zugewiesen werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-115">However, in a single hierarchy, each product can be assigned to only one category.</span></span>
2. <span data-ttu-id="b8a58-116">**Fügen Sie Produkte und Produktvarianten zum Produktmaster hinzu.**</span><span class="sxs-lookup"><span data-stu-id="b8a58-116">**Add products and product variants to the product master.**</span></span> <span data-ttu-id="b8a58-117">Produkte, die dem Produktmaster hinzugefügt werden, stellen eine globale Liste von Produkten dar.</span><span class="sxs-lookup"><span data-stu-id="b8a58-117">Products that are added to the product master represent a global list of products.</span></span> <span data-ttu-id="b8a58-118">Sie können Produkte manuell und einzeln hinzufügen, oder Sie können Produktdaten von Ihren Kreditoren importieren.</span><span class="sxs-lookup"><span data-stu-id="b8a58-118">You can add products manually, one at a time, or you can import product data from your vendors.</span></span>
3. <span data-ttu-id="b8a58-119">**Geben Sie die Produkte für juristische Personen frei.**</span><span class="sxs-lookup"><span data-stu-id="b8a58-119">**Release the products to legal entities.**</span></span> <span data-ttu-id="b8a58-120">Nur Produkte, die für juristische Personen freigegeben wurden, können in Ihren Kanälen verfügbar gemacht werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-120">Only products that have been released to legal entities can be made available to your channels.</span></span> <span data-ttu-id="b8a58-121">Wenn Sie ein Produkt erstmals definieren, definieren Sie das Produkt auf organisationsweiter Ebene.</span><span class="sxs-lookup"><span data-stu-id="b8a58-121">When you first define a product, you define it on an organization-wide level.</span></span> <span data-ttu-id="b8a58-122">Sie können mindestens eine juristische Person auswählen, um das Produkt freigeben.</span><span class="sxs-lookup"><span data-stu-id="b8a58-122">You can then select one or more legal entities to release the product to.</span></span> <span data-ttu-id="b8a58-123">Das Produkt wird dann in mehreren Kanälen in Ihrer Organisation verfügbar.</span><span class="sxs-lookup"><span data-stu-id="b8a58-123">The product then becomes available to multiple channels across your organization.</span></span> <span data-ttu-id="b8a58-124">Sie können diese Funktion verwenden, um ein Produkt einmal zu erstellen, Produktattribute und -eigenschaften an einer Stelle hinzuzufügen und zu aktualisieren, und das Produkt dann Organisationsweit an die Kanäle zu vertreiben, in denen es verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="b8a58-124">You can use this functionality to create a product one time, add and update product attributes and properties in one place, and then distribute the product across your organization, to the channels where it's available.</span></span>
4. <span data-ttu-id="b8a58-125">**Fügen Sie Produkte Sortimenten hinzu.**</span><span class="sxs-lookup"><span data-stu-id="b8a58-125">**Add products to assortments.**</span></span> <span data-ttu-id="b8a58-126">Ein Sortiment stellt eine Sammlung von Produkten dar, die Sie in Ihren Kanälen anbieten.</span><span class="sxs-lookup"><span data-stu-id="b8a58-126">An assortment represents a collection of products that you offer in your channels.</span></span> <span data-ttu-id="b8a58-127">Sie können ein oder mehrere Sortimente definieren, und jedes Produkt kann einem oder mehreren Sortimenten zugewiesen werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-127">You can define one or more assortments, and each product can be assigned to one or more assortments.</span></span> <span data-ttu-id="b8a58-128">Um Produkte den Kanälen zuzuweisen, weisen Sie die Sortimente diesen Kanälen zu.</span><span class="sxs-lookup"><span data-stu-id="b8a58-128">To assign products to channels, you assign the assortments to those channels.</span></span> <span data-ttu-id="b8a58-129">Wenn Sie ein Sortiment erstellen, können Sie Produkte hinzufügen, die noch nicht für eine juristische Person freigegeben wurden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-129">When you create an assortment, you can add products that haven't yet been released to a legal entity.</span></span> <span data-ttu-id="b8a58-130">Sie müssen die Produkte jedoch für eine juristische Person freigeben, bevor diese Produkte in den Kanälen verfügbar gemacht werden können.</span><span class="sxs-lookup"><span data-stu-id="b8a58-130">However, you must release the products to a legal entity before those products can be made available to the channels.</span></span>
5. <span data-ttu-id="b8a58-131">**Hinzufügen von Produkten Navigationshierarchien hinzu.**</span><span class="sxs-lookup"><span data-stu-id="b8a58-131">**Add products to navigation hierarchies.**</span></span> <span data-ttu-id="b8a58-132">Bevor Produkte online oder in Verkaufsstellen durchsucht werden können, müssen sie in einer Commerce-Navigationshierarchie kategorisiert werden.</span><span class="sxs-lookup"><span data-stu-id="b8a58-132">Before products can be browsed online or in point of sale (POS), they must be categorized in a Commerce navigation hierarchy.</span></span>
6. <span data-ttu-id="b8a58-133">**Produkte zum Katalog hinzufügen**</span><span class="sxs-lookup"><span data-stu-id="b8a58-133">**Add products to catalogs.**</span></span> <span data-ttu-id="b8a58-134">Obwohl dieser Schritt für Verkaufsstellen optional ist, erfordern Onlineshops, dass Produkte in mindestens einem Katalog enthalten sind.</span><span class="sxs-lookup"><span data-stu-id="b8a58-134">Although this step is optional for POS, online stores require that products be included in at least one catalog.</span></span>
