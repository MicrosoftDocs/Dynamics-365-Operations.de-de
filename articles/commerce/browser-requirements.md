# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Browseranforderungen von Dynamics 365 Commerce-Websites

[!include [banner](includes/banner.md)]

Dieses Thema behandelt die Browseranforderungen von Microsoft Dynamics 365 Commerce-Websites.

Dynamics 365 Commerce-Websites werden am besten von folgenden Webbrowsern unterstützt. Um die neueste Version für jeden Webbrowser zu suchen, wechseln Sie zur Website des Softwareherstellers.

- [Microsoft Edge](https://www.microsoft.com/edge) (bis zur aktuellen Version minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (bis zur aktuellen Version minus 1)
- [Google Chrome](https://www.google.com/chrome) (bis zur aktuellen Version minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (bis zur aktuellen Version minus 1)

> [!NOTE]
> Internet Explorer wird nicht unterstützt. Falls Sie den Internet Explorer benutzen, funktionieren möglicherweise einige Funktionen nicht, und Benutzer haben vielleicht eine minderwertige Erfahrung.
