# <a name="process-customer-order-pickups-in-pos"></a><span data-ttu-id="eed65-103">Kundenbestellungsabholung am POS bearbeiten</span><span class="sxs-lookup"><span data-stu-id="eed65-103">Process customer order pickups in POS</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="eed65-104">Wenn eine [Kundenbestellung](customer-orders-overview.md) für die Abholung im Geschäft erstellt wird, kann ein Ladenbenutzer mithilfe der POS-Anwendung (Point of Sale) die Abholung des Bestands starten.</span><span class="sxs-lookup"><span data-stu-id="eed65-104">When a [customer order](customer-orders-overview.md) is created for store pickup, a store user can use the point of sale (POS) application to start the pickup of inventory.</span></span> <span data-ttu-id="eed65-105">POS führt die endgültige Zahlungserfassung nach Bedarf durch.</span><span class="sxs-lookup"><span data-stu-id="eed65-105">POS will run the final payment capture as required.</span></span> <span data-ttu-id="eed65-106">Es werden auch der Bestand und die Finanzbuchung für die abgeholten Mengen abgeschlossen.</span><span class="sxs-lookup"><span data-stu-id="eed65-106">It will also complete the inventory and financial posting for the quantities that are picked up.</span></span>

<span data-ttu-id="eed65-107">Wenn Sie ein Ladenbenutzer sind, können Sie die Abholung entweder mit dem Vorgang **Bestellung zurückrufen** oder **Auftragsabwicklung** am POS durchführen.</span><span class="sxs-lookup"><span data-stu-id="eed65-107">If you're a store user, you can perform the pickup by using either the **Recall order** operation or the **Order fulfillment** operation in POS.</span></span> <span data-ttu-id="eed65-108">Um den **Abholen**-Vorgang verfügbar, müssen Sie zuerst einen der folgenden Schritte ausführen:</span><span class="sxs-lookup"><span data-stu-id="eed65-108">To make the **Pick up** operation available, you must first follow one of these steps:</span></span>

<span data-ttu-id="eed65-111">Wenn die ausgewählte Bestellung oder Bestellpositionen nicht für die Abholung in diesem bestimmten Geschäft konfiguriert sind oder wenn die Bestellung bereits vollständig abgeholt wurde, ist der **Abholen**-Vorgang ist nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="eed65-111">If the selected order or order lines aren't configured for pickup at that specific store, or if the order has already been fully picked up, the **Pick up** operation will be unavailable.</span></span>

<span data-ttu-id="eed65-113">In Microsoft Dynamics 365 Commerce Version 10.0.17 und höher kann die Funktion **Verbesserte Benutzererfahrung bei der Abwicklung der Bestellungsabholung am Point of Sale** über die Funktionsverwaltung in der Commerce-Zentralverwaltung aktiviert werden.zeigen.</span><span class="sxs-lookup"><span data-stu-id="eed65-113">In Microsoft Dynamics 365 Commerce version 10.0.17 and later, the **Improved user experience for pick up order processing in Point of Sale** feature can be turned on through Feature management in Commerce headquarters.</span></span> <span data-ttu-id="eed65-114">Wenn diese Funktion deaktiviert ist, können Benutzer keine Abholmengen auswählen.</span><span class="sxs-lookup"><span data-stu-id="eed65-114">If this feature is turned off, users can't select pickup quantities.</span></span> <span data-ttu-id="eed65-115">Standardmäßig ist die volle für die Position bestellte Menge auch die Menge, die abgeholt wird.</span><span class="sxs-lookup"><span data-stu-id="eed65-115">By default, the full quantity that was ordered for the line is the quantity that will be picked up.</span></span> <span data-ttu-id="eed65-116">Diese Erfahrung kann problematisch sein, da Benutzer möglicherweise vergessen, einige Artikel zur Abholung auszuwählen, wenn sie die Abholung durch Auftragserfüllung durchführen.</span><span class="sxs-lookup"><span data-stu-id="eed65-116">This experience can be problematic, because users might forget to select some items for pickup when they perform the pickup through order fulfillment.</span></span>

<span data-ttu-id="eed65-117">Die Funktion **Verbesserte Benutzererfahrung bei der Abwicklung der Bestellungsabholung am Point of Sale** bietet Benutzern mehr Kontrolle über die Auswahl der Produkte, die abgeholt werden sollen, und über die Menge der Produkte, die abgeholt werden.</span><span class="sxs-lookup"><span data-stu-id="eed65-117">The **Improved user experience for pick up order processing in Point of Sale** feature gives users more control over the selection of products that will be picked up and the quantity of those products that will be picked up.</span></span> <span data-ttu-id="eed65-118">Benutzer müssen nicht jede Position des Kundenauftrags auf der Seite zur Auftragserfüllung auswählen, bevor sie **Abholen** auswählen.</span><span class="sxs-lookup"><span data-stu-id="eed65-118">Users don't have to select every line of the sales order on the order fulfillment page before they select **Pick up**.</span></span> <span data-ttu-id="eed65-119">Alle Artikel, die abgeholt werden können, werden angezeigt.</span><span class="sxs-lookup"><span data-stu-id="eed65-119">All the items that can be picked up will be shown.</span></span> <span data-ttu-id="eed65-120">Benutzer können mehrere Positionen für die Abholung angeben, auch wenn nur eine Produktposition ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="eed65-120">Users can specify multiple lines for pickup even if only one product line is selected.</span></span>

<span data-ttu-id="eed65-121">Wenn die Funktion **Verbesserte Benutzererfahrung bei der Abwicklung der Bestellungsabholung am Point of Sale** aktiviert ist und Sie wählen den **Abholen**-Vorgang, wird das **Abholen**-Dialogfeld angezeigt.</span><span class="sxs-lookup"><span data-stu-id="eed65-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="eed65-122">Dort können Sie die Artikel und Mengen auswählen, die abgeholt werden sollen.</span><span class="sxs-lookup"><span data-stu-id="eed65-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="eed65-123">Standardmäßig gilt jede bestellte Menge mit einem Lagerbestand in einem kommissionierten oder verpackten Zustand als abholfähig.</span><span class="sxs-lookup"><span data-stu-id="eed65-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="eed65-124">Standardmäßig ist diese Menge als Abholmenge festgelegt.</span><span class="sxs-lookup"><span data-stu-id="eed65-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="eed65-125">Sie können die eingegebene Menge ändern, vorausgesetzt, die Menge ist nicht 0 (Null) und überschreitet nicht die gesamte offene (d. h. nicht in Rechnung gestellte) Menge für die ausgewählte Position.</span><span class="sxs-lookup"><span data-stu-id="eed65-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>

<span data-ttu-id="eed65-129">Auf der Transaktionsseite berechnet das System die fälligen Beträge, indem es den Gesamtbetrag berechnet, der für die ausgewählten Abholposten fällig ist, und anschließend alle zuvor angewendeten Einzahlungen oder autorisierten Kreditkartenzahlungen abzieht.</span><span class="sxs-lookup"><span data-stu-id="eed65-129">On the transaction page, the system calculates the amounts that are due by calculating the total that is due for the selected pickup items and then subtracting any previously applied deposits or authorized credit card payments.</span></span> <span data-ttu-id="eed65-130">Sie müssen die Zahlung verarbeiten, um die Abholung abzuschließen.</span><span class="sxs-lookup"><span data-stu-id="eed65-130">You must process payment to complete the pickup transaction.</span></span> <span data-ttu-id="eed65-131">Wenn die [Bildschirmgestaltung](pos-screen-layouts.md) der Transaktionsseite so konfiguriert ist, dass sie den Vorgang **Transaktion abschließen** enthält, wenn kein Betrag fällig ist, können Sie die Transaktion abschließen, ohne eine Zahlungsmethode auszuwählen.</span><span class="sxs-lookup"><span data-stu-id="eed65-131">If the [screen layout](pos-screen-layouts.md) of the transaction page is configured so that it includes the **Conclude transaction** operation, and no amount is due, you can complete the transaction without selecting a payment method.</span></span> <span data-ttu-id="eed65-132">Wenn der Vorgang **Transaktion abschließen** nicht verfügbar ist, können Sie den Link **0,00 USD Betrag fällig** im **Summe**-Bereich auswählen, um die Transaktion abzuschließen, ohne eine Zahlungsmethode auswählen zu müssen.</span><span class="sxs-lookup"><span data-stu-id="eed65-132">If the **Conclude transaction** operation isn't available, you can select the **$0.00 amount due** link in the **Totals** pane to conclude the transaction without having to select a payment method.</span></span>
