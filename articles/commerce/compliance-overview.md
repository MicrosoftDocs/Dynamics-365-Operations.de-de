# <a name="compliance-overview"></a>Konformität – Übersicht


[!include [banner](includes/banner.md)]

Dieses Thema enthält eine Übersicht über die Kompatibilitätsrichtlinien, die mit Microsoft Dynamics 365 Commerce bereitgestellt werden, um Ihnen dabei zu helfen, die Websites Ihres Unternehmens konform zu machen. Compliance ist ein wichtiger Faktor für alle Unternehmen, da sie der Benutzerbasis hilft, sich mit dem Inhalt ihrer Website zu verbinden. Indem Sie konform bleiben, können Sie auch dazu beitragen, Ihr Unternehmen vor teuren Rechtsstreitigkeiten oder hohen Geldstrafen zu schützen.

Die Compliance-Dokumentation enthält eine Überprüfung der Bereiche, die Sie bei der Verwendung von Dynamics 365 Commerce berücksichtigen sollten, um die Compliance-Anforderungen Ihres Unternehmens zu erfüllen.

Sie sind dafür verantwortlich, die Compliance-Anforderungen Ihres Unternehmens zu überprüfen und Websites zu erstellen und zu erstellen, die den Standards dieser Anforderungen entsprechen. In den folgenden Themen werden Beispiele aufgeführt, die zeigen, wie Sie die Einhaltung von Vorschriften bei der Verwendung der Commerce-Authoring-Tools berücksichtigen können.

Weitere Informationen zu den von Microsoft verwendeten Grundprinzipien finden Sie unter [Microsoft Trust Center](https://www.microsoft.com/trust-center). Auf dieser Site erhalten Sie auch weitere Informationen zu Compliance-Bereichen.
