# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>Hinzufügen von Produktempfehlungslisten zu E-Commerce-Seiten

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

In diesem Thema wird erläutert, wie eine Produktempfehlungsliste einer E-Commerce-Seite hinzugefügt wird.

1. Von einer vorhandenen E-Commerce-Erstellungsseite wählen Sie **Modul hinzufügen** aus.
2. Suchen Sie nach dem **Produktinkasso** Modul.
3. Im Modulassistenten wählen Sie die Liste aus, um sie der Seite hinzuzufügen.
4. Speichern Sie die Änderungen.

Weitere Informationen zu den Listentypen, die Sie einer Seite hinzufügen können, finden Sie unter [Produktempfehlungsüberblick](product-recommendations.md).
