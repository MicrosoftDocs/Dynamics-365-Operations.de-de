# <a name="add-a-favicon"></a><span data-ttu-id="4f717-103">Hinzufügen eines Favicons</span><span class="sxs-lookup"><span data-stu-id="4f717-103">Add a favicon</span></span>

## <a name="overview"></a><span data-ttu-id="4f717-105">Übersicht</span><span class="sxs-lookup"><span data-stu-id="4f717-105">Overview</span></span>

<span data-ttu-id="4f717-106">Ein favicon ist eine kleine Grafikdatei, die auf einer Webbrowserregisterkarte im Browserverlauf der Adressleiste sowie in Lesezeichen oder in den Favoriten angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="4f717-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="4f717-107">Es wird empfohlen, ein favicon dem Standort hinzufügen, da es Ihre Marke darstellt und verstärkt und hilft, den Standort von anderen Websites, die Ihre Kunden suchen, zu unterscheiden.</span><span class="sxs-lookup"><span data-stu-id="4f717-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="4f717-108">Zwar können Sie mehrere favicons von verschiedenen Größen und Dateitypen Ihrem Standort hinzufügen, dieses Thema behandelt lediglich das Hinzufügen von einem einzelnen favicon.</span><span class="sxs-lookup"><span data-stu-id="4f717-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="4f717-109">Allerdings werden der gleiche Prozess und Ort verwendet, um weitere favicons hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="4f717-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="4f717-110">Laden Sie ein favicon in die Anlagensammlung Ihrer Site hoch</span><span class="sxs-lookup"><span data-stu-id="4f717-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="4f717-111">Um ein favicon aus der Anlagensammlung zu entfernen, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="4f717-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="4f717-112">Wählen Sie im linken Navigationsbereich **Medienbibliothek**.</span><span class="sxs-lookup"><span data-stu-id="4f717-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="4f717-113">Wählen Sie in der Befehlsleiste **Hochladen \> Medienelemente hochladen**.</span><span class="sxs-lookup"><span data-stu-id="4f717-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="4f717-114">Navigieren Sie im Fenster „Datei-Explorer“ zu der Favicon-Bilddatei, die Sie hochladen möchten, wählen Sie sie aus und wählen Sie sie **Öffnen** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="4f717-115">Geben Sie im Dialogfenster **Medienelement hochladen** den gewünschten Titel und den Alt-Text ein.</span><span class="sxs-lookup"><span data-stu-id="4f717-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="4f717-116">Wenn Sie das Bild unmittelbar nach dem Hochladen veröffentlichen möchten, markieren Sie das Kontrollkästchen **Medienelemente nach dem Upload veröffentlichen**.</span><span class="sxs-lookup"><span data-stu-id="4f717-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="4f717-117">Wenn Sie das Kontrollkästchen **Medienelemente nach dem Hochladen veröffentlichen** aktivieren, müssen Sie auf die Seite **Medienelemente** zurückkehren und das Favicon später manuell veröffentlichen.</span><span class="sxs-lookup"><span data-stu-id="4f717-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="4f717-118">Wählen Sie **OK**.</span><span class="sxs-lookup"><span data-stu-id="4f717-118">Select **OK**.</span></span>
1. <span data-ttu-id="4f717-119">Im Eigenschaftenbereich auf der rechten Seite, kopieren Sie die öffentliche URL des favicon.</span><span class="sxs-lookup"><span data-stu-id="4f717-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="4f717-120">Sie werden diese URL später verwenden.</span><span class="sxs-lookup"><span data-stu-id="4f717-120">You will use this URL later.</span></span>

## <a name="create-a-page-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="4f717-124">Erstellen Sie ein Seitenfragment, das ein Metatag für Ihr Favicon enthält</span><span class="sxs-lookup"><span data-stu-id="4f717-124">Create a page fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="4f717-125">Um ein Seitenfragment zu erstellen, das ein Metatag für Ihr Favicon enthält, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="4f717-125">To create a page fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="4f717-126">Wechseln Sie zu **Fragmente** und wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="4f717-127">Wählen Sie im Dialogfeld **Neues Seitenfragment** die Option **Metatags** als das Modul aus, auf dem das Seitenfragment basiert.</span><span class="sxs-lookup"><span data-stu-id="4f717-127">In the **New page fragment** dialog box, select **Metatags** as the module that the page fragment is based on.</span></span>
1. <span data-ttu-id="4f717-128">Geben Sie einen Namen für das Seitenfragment ein, und wählen Sie dann **OK** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-128">Enter a name for the page fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="4f717-129">Wählen Sie in der Fragmenthierarchiestruktur das untergeordnete Element **Standard-Metatags** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="4f717-130">Im rechten Bereich unter **Metatags** wählen Sie **Hinzufügen** aus, und geben Sie dann die HTML-Zeichenfolge ein, die Sie zuvor für das Favicon erstellt haben.</span><span class="sxs-lookup"><span data-stu-id="4f717-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="4f717-131">Wählen Sie **Bearbeiten beenden** aus, und wählen Sie dann **Veröffentlichen** aus, um das Seitenfragment zu veröffentlichen.</span><span class="sxs-lookup"><span data-stu-id="4f717-131">Select **Finish editing**, and then select **Publish** to publish the page fragment.</span></span>

## <a name="add-the-metatag-page-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="4f717-132">Fügen Sie das Metatag-Seitenfragment zum HTML-Kopfbereich Ihrer Seiten hinzu</span><span class="sxs-lookup"><span data-stu-id="4f717-132">Add the metatag page fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="4f717-133">Um das Metatag-Seitenfragment zum HTML-**Kopf**-Bereich Ihrer Seiten hinzuzufügen, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="4f717-133">To add the metatag page fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="4f717-134">Gehen Sie zu **Vorlagen**, öffnen Sie die Vorlage für die Seiten, denen Sie Ihr Favicon hinzufügen möchten, und wählen Sie dann **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="4f717-135">Wählen Sie in der Vorlagenhierarchiestruktur die Auslassungspunkte (**...**)-Schaltfläche rechts neben dem **HTML-Kopf**-Container aus, und wählen Sie dann **Seitenfragment hinzufügen** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add page fragment**.</span></span>
1. <span data-ttu-id="4f717-136">Wählen Sie im Dialogfeld **Seitenfragment auswählen** das Metatag-Seitenfragment aus, das Sie zuvor erstellt haben, und wählen Sie dann **OK** aus.</span><span class="sxs-lookup"><span data-stu-id="4f717-136">In the **Select page fragment** dialog box, select the metatag page fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="4f717-137">Wählen Sie **Bearbeiten beenden** aus, und wählen Sie dann **Veröffentlichen** aus, um die Vorlage zu veröffentlichen.</span><span class="sxs-lookup"><span data-stu-id="4f717-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="4f717-138">Wenn Ihre Site mehr als eine Vorlage verwendet, müssen Sie allen das Metatags-Seitenfragment hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="4f717-138">If your site uses more than one template, you must add the metatags page fragment to all of them.</span></span>

<span data-ttu-id="4f717-139">Wenn Sie Seiten in der Vorschau anzeigen, die auf der Vorlage basieren, der Sie das Metatags-Seitenfragment hinzugefügt haben, sollten Sie jetzt das Favicon in der Browser-Registerkarte sehen.</span><span class="sxs-lookup"><span data-stu-id="4f717-139">When you preview pages that are based on the template that you added the metatags page fragment to, you should now see the favicon on the browser tab.</span></span>
