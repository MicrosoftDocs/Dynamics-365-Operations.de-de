## <a name="overview"></a><span data-ttu-id="43a24-105">Übersicht</span><span class="sxs-lookup"><span data-stu-id="43a24-105">Overview</span></span>

<span data-ttu-id="43a24-106">Die Medienbibliothek des Commerce-Site-Builders ermöglicht es Ihnen, Bilder zuzuschneiden, um sie für verschiedene Modultypen und Ansichtsfenster zu optimieren.</span><span class="sxs-lookup"><span data-stu-id="43a24-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="43a24-107">Bild zuschneiden</span><span class="sxs-lookup"><span data-stu-id="43a24-107">Crop an image</span></span>

<span data-ttu-id="43a24-108">Um ein Bild im Site Builder zuzuschneiden, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="43a24-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="43a24-109">Wählen Sie im linken Navigationsbereich des Commerce Site Builders **Medienbibliothek**.</span><span class="sxs-lookup"><span data-stu-id="43a24-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="43a24-110">Wählen Sie im Hauptfenster das Bild aus, das Sie ändern möchten.</span><span class="sxs-lookup"><span data-stu-id="43a24-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="43a24-111">Wählen Sie in der Befehlsleiste **Bearbeiten**, um die Datei auszuchecken.</span><span class="sxs-lookup"><span data-stu-id="43a24-111">On the command bar, select **Edit** to check out the file.</span></span>
1. <span data-ttu-id="43a24-112">Wählen Sie das Bild aus, um **Bearbeitungsmodus** aufzurufen.</span><span class="sxs-lookup"><span data-stu-id="43a24-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="43a24-113">Wählen Sie unter **Bearbeitungsmodus** **Ansicht nach Modul bearbeiten**.</span><span class="sxs-lookup"><span data-stu-id="43a24-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="43a24-114">Wählen Sie aus dem Dropdown-Menü **Modul** den Modultyp.</span><span class="sxs-lookup"><span data-stu-id="43a24-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="43a24-115">Wählen Sie aus dem Drop-Down-Menü **Ansichtstyp** den Ansichtstyp aus.</span><span class="sxs-lookup"><span data-stu-id="43a24-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="43a24-116">Wählen Sie aus dem Drop-Down-Menü **Platzierung** die Bildplatzierung aus.</span><span class="sxs-lookup"><span data-stu-id="43a24-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="43a24-117">Wählen Sie aus dem Drop-Down-Menü **Ansichtsfenster** die Größe des Ansichtsfensters aus.</span><span class="sxs-lookup"><span data-stu-id="43a24-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="43a24-118">Das Bild wird mit dem Bereich überlagert, der den Beschnittbereich darstellt.</span><span class="sxs-lookup"><span data-stu-id="43a24-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="43a24-119">Verschieben Sie den Beschnittbereich und ändern Sie die Größe nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="43a24-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="43a24-120">Das Seitenverhältnis wird automatisch beibehalten.</span><span class="sxs-lookup"><span data-stu-id="43a24-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="43a24-121">Wenn Sie fertig sind, wählen Sie in der Befehlsleiste **Speichern** und wählen Sie dann **Bearbeitung beenden**.</span><span class="sxs-lookup"><span data-stu-id="43a24-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="43a24-122">Nach Abschluss der benutzerdefinierten Beschneidung werden die Bildmodifikationen fast sofort wirksam.</span><span class="sxs-lookup"><span data-stu-id="43a24-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
