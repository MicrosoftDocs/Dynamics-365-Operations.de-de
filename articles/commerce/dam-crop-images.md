# <a name="crop-images"></a><span data-ttu-id="53142-103">Bilder zuschneiden</span><span class="sxs-lookup"><span data-stu-id="53142-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="53142-104">In diesem Thema wird beschrieben, wie man Bilder im Microsoft Dynamics 365 Commerce Site Builder zuschneidet.</span><span class="sxs-lookup"><span data-stu-id="53142-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="53142-105">Die Medienbibliothek des Commerce-Site-Builders ermöglicht es Ihnen, Bilder zuzuschneiden, um sie für verschiedene Modultypen und Ansichtsfenster zu optimieren.</span><span class="sxs-lookup"><span data-stu-id="53142-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="53142-106">Bild zuschneiden</span><span class="sxs-lookup"><span data-stu-id="53142-106">Crop an image</span></span>

<span data-ttu-id="53142-107">Um ein Bild im Site Builder zuzuschneiden, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="53142-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="53142-108">Wählen Sie im linken Navigationsbereich des Commerce Site Builders **Medienbibliothek**.</span><span class="sxs-lookup"><span data-stu-id="53142-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="53142-109">Wählen Sie im Hauptfenster das Bild aus, das Sie ändern möchten.</span><span class="sxs-lookup"><span data-stu-id="53142-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="53142-110">Wählen Sie in der Befehlsleiste **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="53142-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="53142-111">Wählen Sie das Bild aus, um **Bearbeitungsmodus** aufzurufen.</span><span class="sxs-lookup"><span data-stu-id="53142-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="53142-112">Wählen Sie unter **Bearbeitungsmodus** **Ansicht nach Modul bearbeiten**.</span><span class="sxs-lookup"><span data-stu-id="53142-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="53142-113">Wählen Sie aus dem Dropdown-Menü **Modul** den Modultyp.</span><span class="sxs-lookup"><span data-stu-id="53142-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="53142-114">Wählen Sie aus dem Drop-Down-Menü **Ansichtstyp** den Ansichtstyp aus.</span><span class="sxs-lookup"><span data-stu-id="53142-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="53142-115">Wählen Sie aus dem Drop-Down-Menü **Platzierung** die Bildplatzierung aus.</span><span class="sxs-lookup"><span data-stu-id="53142-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="53142-116">Wählen Sie aus dem Drop-Down-Menü **Ansichtsfenster** die Größe des Ansichtsfensters aus.</span><span class="sxs-lookup"><span data-stu-id="53142-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="53142-117">Das Bild wird mit dem Bereich überlagert, der den Beschnittbereich darstellt.</span><span class="sxs-lookup"><span data-stu-id="53142-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="53142-118">Verschieben Sie den Beschnittbereich und ändern Sie die Größe nach Bedarf.</span><span class="sxs-lookup"><span data-stu-id="53142-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="53142-119">Das Seitenverhältnis wird automatisch beibehalten.</span><span class="sxs-lookup"><span data-stu-id="53142-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="53142-120">Wenn Sie fertig sind, wählen Sie in der Befehlsleiste **Speichern** und wählen Sie dann **Bearbeitung beenden**.</span><span class="sxs-lookup"><span data-stu-id="53142-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="53142-121">Nach Abschluss der benutzerdefinierten Beschneidung werden die Bildmodifikationen fast sofort wirksam.</span><span class="sxs-lookup"><span data-stu-id="53142-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
