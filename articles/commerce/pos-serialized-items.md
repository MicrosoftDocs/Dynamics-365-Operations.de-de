<span data-ttu-id="2902e-113">Im Inforegister **Rückverfolgungsangaben**, wenn die Option **Leerer Zugang zulässig** aktiviert ist, ist die Seriennummer eine optionale Eingabe während des Bestandszugangsprozesses.</span><span class="sxs-lookup"><span data-stu-id="2902e-113">On the **Tracking dimensions** FastTab, if the **Blank receipt allowed** option is turned on, the serial number is an optional input during the inventory receipt process.</span></span> <span data-ttu-id="2902e-114">Wenn die Option deaktiviert ist, ist die Seriennummer erforderlich.</span><span class="sxs-lookup"><span data-stu-id="2902e-114">If the option is turned off, the serial number is required.</span></span>
