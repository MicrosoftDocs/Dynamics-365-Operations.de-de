# <a name="troubleshoot-store-commerce-performance-issues"></a>Beheben Sie Leistungsprobleme bei Store Commerce

[!include [banner](../includes/banner.md)]

In diesem Artikel wird erläutert, wie Sie Leistungsprobleme in der Microsoft Dynamics 365 Commerce Store Commerce-App beheben.

- Vermeiden Sie die Remote-Anmeldung, wenn Sie Leistungsprobleme für Store Commerce beheben. Melden Sie sich stattdessen direkt beim Store Commerce-Gerät an.
- Wenn die Store Commerce-App langsam ist, überprüfen Sie die CPU-, Arbeitsspeicher-, Netzwerk- und Festplattenauslastung mithilfe des Task-Managers oder des Ressourcenmonitors. Stellen Sie sicher, dass die Geräteressourcen nicht erheblich beansprucht werden.
- Überprüfen Sie Ihren Netzwerkkonnektivitätsstatus und vergewissern Sie sich, dass HTTPS-Anforderungen/Antworten nicht länger als ein paar Sekunden dauern.
