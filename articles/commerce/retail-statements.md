<span data-ttu-id="bd56a-112">Sie können eine Aufstellung manuell oder durch Stapelverarbeitungsvorgänge verwenden, die Sie so einrichten, dass sie in regelmäßigen Abständen während des ganzen Tages ausgeführt werden.</span><span class="sxs-lookup"><span data-stu-id="bd56a-112">You can create a statement manually or by using batch processes that you set up to run periodically throughout the day.</span></span> <span data-ttu-id="bd56a-113">In beiden Fällen werden die folgenden Schritte verwendet, um Auszüge zu erstellen und zu buchen.</span><span class="sxs-lookup"><span data-stu-id="bd56a-113">In both cases, the following steps are used to create and post statements.</span></span>

### <a name="create-the-statement"></a><span data-ttu-id="bd56a-114">Den Auszug erstellen</span><span class="sxs-lookup"><span data-stu-id="bd56a-114">Create the statement</span></span>

<span data-ttu-id="bd56a-134">Wenn das Kontrollkästchen **Nach Auszugsmethode aufteilen** auf der Seite **Filialen** aktiviert ist, wird ein separater Auszug basierend auf dem Wert erstellt, der im Feld **Auszugsmethode** ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="bd56a-134">If the **Split by Statement method** check box is selected on the **Stores** page, a separate statement is created based on the value that is selected in the **Statement method** field.</span></span>
