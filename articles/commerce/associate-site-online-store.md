<span data-ttu-id="84af8-105">Nachdem Sie E-Commerce bereitgestellt haben, indem Sie das Microsoft Dynamics Lifecycle Services (LCS) - Portal verwenden, sind Sie bereit, die erste E-Commerce-Website einzurichten.</span><span class="sxs-lookup"><span data-stu-id="84af8-105">After you've provisioned e-Commerce by using the Microsoft Dynamics Lifecycle Services (LCS) portal, you're ready to establish your first e-Commerce website.</span></span> <span data-ttu-id="84af8-106">Als Teil der ersten Websiteerstellung ordnen Sie die Site einem Onlineshop zu, der zuvor erstellt wurde.</span><span class="sxs-lookup"><span data-stu-id="84af8-106">As part of the initial site creation, you associate the site with an online store that was previously created.</span></span> <span data-ttu-id="84af8-107">Dieser Schritt bindet die Site an einen Online Kanal und die Site kann die Navigationshierarchie, Produkte, Kategorien, Preise, Versandoptionen und alles andere anzeigen, das Sie im Onlineshop definiert haben.</span><span class="sxs-lookup"><span data-stu-id="84af8-107">This step binds the site to an online channel and lets the site show the navigation hierarchy, products, categories, prices, shipping options, and everything else that you defined in the online store.</span></span>

<span data-ttu-id="84af8-108">Um eine neue Website einzurichten und einen Onlineshop zuzuordnen, wählen Sie in LCS den Link für die Standorterstellungsumgebung aus.</span><span class="sxs-lookup"><span data-stu-id="84af8-108">To establish a new site and associate an online store with it, in LCS, select the link for the site authoring environment.</span></span> <span data-ttu-id="84af8-109">Wählen Sie dann auf der Seite für die Standortserstellungsumgebung **Neue Site** aus.</span><span class="sxs-lookup"><span data-stu-id="84af8-109">Then, on the page for the site authoring environment, select **New site**.</span></span> <span data-ttu-id="84af8-110">Im Dialogfeld **Neue Site** müssen einige grundlegende Informationen über Ihre Site angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="84af8-110">In the **New site** dialog box, you must provide some basic information about your site.</span></span> <span data-ttu-id="84af8-111">Eine vollständige Erklärung der Informationen, die Sie bereitstellen müssen, finden Sie unter [Erstellen Sie eine neue E-Commerce-Site](create-ecommerce-site.md).</span><span class="sxs-lookup"><span data-stu-id="84af8-111">For a complete explanation of the information that you must provide, see [Create a new e-Commerce site](create-ecommerce-site.md).</span></span>

<span data-ttu-id="84af8-112">Nachdem Ihre Site erstellt wurde, können Sie prüfen, dass sie Ihrem Onlineshop zugeordnet ist, indem Sie die Registerkarte **Produkte** auswählen. Sie sollten das Sortiment von Produkten sehen, das dem Onlineshop zugewiesen wurde.</span><span class="sxs-lookup"><span data-stu-id="84af8-112">After your site is created, you can verify that it's associated with your online store by selecting the **Products** tab.
