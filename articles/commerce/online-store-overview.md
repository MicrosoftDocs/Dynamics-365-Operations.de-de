# <a name="online-store-overview"></a>Onlineshop – Überblick
[!include [banner](includes/banner.md)]


In diesem Thema wird das Konzept eines Onlineshops vorgestellt und die Verwendung von Onlineshops in Microsoft Dynamics 365 Commerce erläutert. Es enthält auch einen Link zu weiteren Informationen zu Onlineshops und Informationen zum Einrichten eines Onlineshops.

Bevor Sie eine Ihre Site in Dynamics 365 Commerce erstellen können, muss mindestens ein Onlineshop eingerichtet sein. In Dynamics 365 Commerce verwenden Sie einen Online-Shop, um die Produkte, Preise, Sprachen, Zahlungsmethoden, Versandarten, Fulfillment-Center und andere Aspekte des Online-Erlebnisses festzulegen, die Ihren Kunden zur Verfügung stehen sollten.

Es muss nur ein Online-Shop eingerichtet werden, bevor Sie mit Dynamics 365 Commerce starten können. Jedoch kann eine einzelne Dynamics 365 Commerce-Site die Onlineerfahrung für mehrere Onlineshops bereitstellen. Wenn zum Beispiel mehrere Online-Shops eingerichtet sind, um unterschiedliche geografische Regionen zu unterstützen, kann ein einzelner Satz von Seiten verwendet werden, um die von jedem Shop definierten einzigartigen Erlebnisse bereitzustellen. Weitere Informationen zum Konfigurieren einer Site für die Unterstützung mehrerer Onlineshops finden Sie unter [Verknüpfen Sie eine Online-Site mit einem Kanal](associate-site-online-store.md).

Nachdem ein Online-Shop eingerichtet wurde, kann er der Dynamics 365 Commerce-Site zugeordnet werden, die als Ihre Online-Storefront dient. Weitere Informationen zu Onlineshops und deren Einrichtung finden Sie unter [Richten Sie Online-Shops ein](https://docs.microsoft.com/dynamics365/unified-operations/retail/online-stores).
