# <a name="customize-image-focal-points"></a><span data-ttu-id="bf883-103">Bildfokuspunkte anpassen</span><span class="sxs-lookup"><span data-stu-id="bf883-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="bf883-104">In diesem Thema wird beschrieben, wie die Bildfokuspunkte in Microsoft Dynamics 365 Commerce Site Builder angepasst werden können.</span><span class="sxs-lookup"><span data-stu-id="bf883-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="bf883-105">Wenn ein Bild in die Medienbibliothek des Commerce-Site-Builders hochgeladen wird, versucht das System, den Brennpunkt des Bildes zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="bf883-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="bf883-106">Wenn sich beispielsweise eine Person auf dem Bild befindet, setzt das System den Fokus standardmäßig auf das Gesicht der Person.</span><span class="sxs-lookup"><span data-stu-id="bf883-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="bf883-107">In den meisten Fällen funktioniert der automatisch gesetzte Fokuspunkt für alle Ansichtsfenster gut, aber manchmal möchten Sie den Fokuspunkt vielleicht anpassen, um sicherzustellen, dass ein bestimmter Teil des Bildes immer sichtbar ist.</span><span class="sxs-lookup"><span data-stu-id="bf883-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="bf883-108">Definieren Sie einen benutzerdefinierten Brennpunkt für ein Bild</span><span class="sxs-lookup"><span data-stu-id="bf883-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="bf883-109">Um einen benutzerdefinierten Brennpunkt für ein Bild zu definieren, gehen Sie wie folgt vor.</span><span class="sxs-lookup"><span data-stu-id="bf883-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="bf883-110">Wählen Sie im linken Navigationsbereich des Commerce Site Builders **Medienbibliothek**.</span><span class="sxs-lookup"><span data-stu-id="bf883-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="bf883-111">Wählen Sie im Hauptfenster das Bild aus, das Sie ändern möchten.</span><span class="sxs-lookup"><span data-stu-id="bf883-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="bf883-112">Wählen Sie in der Befehlsleiste **Bearbeiten** aus.</span><span class="sxs-lookup"><span data-stu-id="bf883-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="bf883-113">Wählen Sie das Bild aus, um **Bearbeitungsmodus** aufzurufen.</span><span class="sxs-lookup"><span data-stu-id="bf883-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="bf883-114">Wählen Sie unter **Bearbeitungsmodus** **Fokuspunkt ändern**.</span><span class="sxs-lookup"><span data-stu-id="bf883-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="bf883-115">Ein kreisförmiger Brennpunktregler erscheint über dem Bild.</span><span class="sxs-lookup"><span data-stu-id="bf883-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="bf883-116">Wählen Sie das Fokussierpunkt-Steuerelement, um es über den gewünschten Fokuspunkt zu bewegen.</span><span class="sxs-lookup"><span data-stu-id="bf883-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="bf883-117">Wenn Sie fertig sind, wählen Sie in der Befehlsleiste **Speichern** und wählen Sie dann **Bearbeitung beenden**.</span><span class="sxs-lookup"><span data-stu-id="bf883-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
