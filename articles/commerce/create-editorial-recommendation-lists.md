## <a name="create-a-new-list"></a><span data-ttu-id="de954-106">Erstellen Sie eine neue Liste</span><span class="sxs-lookup"><span data-stu-id="de954-106">Create a new list</span></span>

<span data-ttu-id="de954-107">Um eine kuratierte Produktempfehlungsliste zu erstellen, folgen Sie diesen Schritten.</span><span class="sxs-lookup"><span data-stu-id="de954-107">To create a curated product recommendation list, follow these steps.</span></span>

1. <span data-ttu-id="de954-108">Gehen Sie zu **Einzelhandel** &gt; **Produktempfhelungen** &gt; **Empfehlungslisten**.</span><span class="sxs-lookup"><span data-stu-id="de954-108">Go to **Retail** &gt; **Product recommendations** &gt; **Recommendation lists**.</span></span>
1. <span data-ttu-id="de954-109">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="de954-109">Select **New**.</span></span>
1. <span data-ttu-id="de954-110">Geben Sie im Feld **Listen-ID** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="de954-110">In the **List Id** field, enter a value.</span></span>
1. <span data-ttu-id="de954-111">Geben Sie im Feld **Listennamen** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="de954-111">In the **List name** field, enter a value.</span></span>
    - <span data-ttu-id="de954-112">Der **Listenname** Ist der Titel der Liste, der in der kuratierten Listenauswahl des Moduls **Produktsammlung** angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="de954-112">The **List name** is the title of the list that will appear in the curated lists section of the **Product collection** module.</span></span>
1. <span data-ttu-id="de954-113">Um Produkte der Liste hinzuzufügen, wählen Sie **Produkte hinzufügen**.</span><span class="sxs-lookup"><span data-stu-id="de954-113">To add products to the list, select **Add products**.</span></span>
1. <span data-ttu-id="de954-114">Um die Reihenfolge der Produkte in der Liste zu ändern, geben Sie einen Wert in der Spalte **Anzeigereihenfolge** ein.</span><span class="sxs-lookup"><span data-stu-id="de954-114">To change the order of the products in the list, enter a value in the **Display order** column.</span></span>
    - <span data-ttu-id="de954-115">Wenn zwei Produkte den gleichen Wert aufweisen, dann kann der abschließende Auftrag dieser zwei Ergebnisse sich vom Back Office unterscheiden.</span><span class="sxs-lookup"><span data-stu-id="de954-115">If two products have the same display order value, then the final order of those two results may differ from the back office.</span></span>
1. <span data-ttu-id="de954-116">Wählen Sie **Speichern**, um die Liste zu speichern.</span><span class="sxs-lookup"><span data-stu-id="de954-116">Select **Save** to save the list.</span></span>
