# <a name="view-posted-sales-tax-transactions"></a>Vorgenommene Mehrwertsteuerbuchungen anzeigen

[!include [task guide banner](../../includes/task-guide-banner.md)]

Sie können gebuchte Mehrwertsteuertransaktionen aus verschiedenen Seiten anzeigen, abhängig davon, was Sie anzeigen möchten. Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.

1. Wechseln Sie zu "Steuer" > "Abfragen und Berichte" > "Mehrwertsteuerabfragen" > "Gebuchte Mehrwertsteuer".
2. Klicken Sie auf "Filter anzeigen".
3. Schließen Sie die Seite.
4. Wechseln Sie zu "Steuer" Wechseln Sie zu "Steuer" > "Indirekte Steuern" > "Mehrwertsteuer" > "Mehrwertsteuer-Abrechnungszeiträume".
5. Erweitern Sie den Abschnitt "Periodenintervalle".
6. Wählen Sie das gewünschte Intervall aus.
7. Zeigen Sie gebuchte Mehrwertsteuertransaktionen für das ausgewählte Abrechnungszeitraumintervall an.
8. Sie können die Liste gebuchter Mehrwertsteuertransaktionen weiter filtern.
9. Schließen Sie die Seite.
10. Wechseln Sie zu "Steuer" > "Indirekte Steuern" > "Mehrwertsteuer" > "Mehrwertsteuercodes".
11. Klicken Sie im Aktivitätsbereich auf "Mehrwertsteuercode".
12. Klicken Sie auf "Gebuchte Mehrwertsteuer"
