# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="b7a34-103">Einrichten von Sachkontobuchungsgruppen für Mehrwertsteuer</span><span class="sxs-lookup"><span data-stu-id="b7a34-103">Set up ledger posting groups for sales tax</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="b7a34-104">Die Mehrwertsteuer wird zu den Hauptkonten berechnet und gebucht, die in den "Sachkontobuchungsgruppen" angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="b7a34-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="b7a34-105">Sachkontobuchungsgruppen werden jedem einzelnen Mehrwertsteuercode zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="b7a34-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="b7a34-106">Sie können einzelne Sachkontobuchungsgruppen für jeden Mehrwertsteuercode einrichten, eine Sachkontobuchungsgruppe für alle Mehrwertsteuercodes verwenden oder mehrere Sachkontobuchungsgruppen den Mehrwertsteuercodes zuweisen.</span><span class="sxs-lookup"><span data-stu-id="b7a34-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="b7a34-107">Für diese Erfassung wird das Demo-Unternehmen DEMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="b7a34-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="b7a34-108">Wechseln Sie zu "Steuer" > "Einstellungen" > "Mehrwertsteuer" > "Sachkontenbuchungsgruppen".</span><span class="sxs-lookup"><span data-stu-id="b7a34-108">Go to Tax > Setup > Sales tax > Ledger posting groups.</span></span>
2. <span data-ttu-id="b7a34-109">Klicken Sie auf "Neu".</span><span class="sxs-lookup"><span data-stu-id="b7a34-109">Click New.</span></span>
3. <span data-ttu-id="b7a34-110">Geben Sie im Feld "Sachkontobuchungsgruppe" einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="b7a34-110">In the Ledger posting group field, type a value.</span></span>
4. <span data-ttu-id="b7a34-111">Geben Sie im Feld "Beschreibung" einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="b7a34-111">In the Description field, type a value.</span></span>
5. <span data-ttu-id="b7a34-112">Wählen Sie im Feld "Mehrwertsteuer" das Hauptkonto für ausgehende Mehrwertsteuern aus, die an die Steuerbehörde zu zahlen sind.</span><span class="sxs-lookup"><span data-stu-id="b7a34-112">In the Sales tax payable field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span>
    * <span data-ttu-id="b7a34-113">Mehrwertsteuern werden im Namen der Steuerbehörde erhoben, wenn Sie steuerpflichtige Waren und Dienstleistungen verkaufen.</span><span class="sxs-lookup"><span data-stu-id="b7a34-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="b7a34-114">Wählen Sie im Feld "Vorsteuer" das Hauptkonto für eingehende Steuern aus, die von der Steuerbehörde empfangen werden.</span><span class="sxs-lookup"><span data-stu-id="b7a34-114">In the Sales tax receivable filed, select the main account for incoming taxes that are received from the tax authority.</span></span>
    * <span data-ttu-id="b7a34-115">Händler erheben Steuern im Auftrag der Steuerbehörde, wenn Sie steuerpflichtige Waren und Dienstleistungen kaufen.</span><span class="sxs-lookup"><span data-stu-id="b7a34-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="b7a34-116">Dieses Feld ist nicht verfügbar, wenn die Option "Mehrwertsteuerregeln anwenden" auf der Seite "Hauptbuchparameter" ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="b7a34-116">This field is not available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span> <span data-ttu-id="b7a34-117">Stattdessen werden an Händler gezahlte Mehrwertsteuerbeträge vom selben Konto abgebucht, wie der Einkauf.</span><span class="sxs-lookup"><span data-stu-id="b7a34-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="b7a34-118">Wählen Sie im Feld "Verbrauchssteuerausgaben" das Hauptkonto für das Buchen abzugsfähiger "Verbrauchssteuern" aus, die nicht eingefordert oder durch Händler der Steuerbehörde gemeldet werden, als Teil der Verlagerung der Steuerschuld GST/HST in der EU.</span><span class="sxs-lookup"><span data-stu-id="b7a34-118">In the Use tax expense field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span>
    * <span data-ttu-id="b7a34-119">Die Option "Verbrauchssteuer" muss für den "Mehrwertsteuercode" in der Gruppe "Mehrwertsteuer" ausgewählt werden, die in der Transaktion verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="b7a34-119">The Use tax option needs to be selected for the Sales tax code in the Sales tax group that is used in the transaction.</span></span>  <span data-ttu-id="b7a34-120">Dieses Feld ist nicht verfügbar, wenn die Option "Mehrwertsteuerregeln anwenden" auf der Seite "Hauptbuchparameter" ausgewählt ist.</span><span class="sxs-lookup"><span data-stu-id="b7a34-120">This field is not be available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span>   
8. <span data-ttu-id="b7a34-121">Wählen Sie im Feld "Gegenkonto Verbrauchssteuer" das Hauptkonto für das Buchen eingehender Verbrauchssteuern aus, die an Steuerbehörden zu zahlen sind.</span><span class="sxs-lookup"><span data-stu-id="b7a34-121">In the Use tax payable field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span>
    * <span data-ttu-id="b7a34-122">Die Option "Verbrauchssteuer" muss im "Mehrwertsteuercode" in der Gruppe "Mehrwertsteuer" ausgewählt werden, um die "Verbrauchssteuer" zu buchen.</span><span class="sxs-lookup"><span data-stu-id="b7a34-122">The Use tax option needs to be selected in the Sales tax code in the Sales tax group to post Use tax.</span></span> <span data-ttu-id="b7a34-123">Wenn die Option "Mehrwertsteuerregeln anwenden" unter den "Hauptbuchparametern" ausgewählt ist, wird die Gegenbuchung im Ausgabenkonto der Transaktion gebucht.</span><span class="sxs-lookup"><span data-stu-id="b7a34-123">If Apply sales tax taxation rules option is selected in General ledger parameters, the offset is posted to the transaction’s expense account.</span></span>   
9. <span data-ttu-id="b7a34-124">Wählen Sie im Feld "Ausgleichskonto" das Hauptkonto aus, zu dem der Nettosaldo der Sachkonten gebucht wird, der in den Feldern "Verbrauchssteuer" und "Mehrwertsteuer" angegeben ist.</span><span class="sxs-lookup"><span data-stu-id="b7a34-124">In the Settlement account field, select the main account  that the net balance of the ledger accounts specified in the Use tax payable and Sales tax receivable fields will be posted.</span></span>
    * <span data-ttu-id="b7a34-125">Der Saldo wird erstellt, wenn der "Einzelvorgang für die Abrechnung und Buchung der Mehrwertsteuer" ausgeführt wurde.</span><span class="sxs-lookup"><span data-stu-id="b7a34-125">The balance will be created when the Sales tax settle and post job is ran.</span></span>  <span data-ttu-id="b7a34-126">Wenn die "Steuerbehörde für den Abrechnungszeitraum" einem Kreditorenkonto zugeordnet ist, wird der Saldo stattdessen zum Kreditorenkonto gebucht.</span><span class="sxs-lookup"><span data-stu-id="b7a34-126">If the Tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>   
10. <span data-ttu-id="b7a34-127">Wählen Sie im Feld "Kreditorenskonto" das Hauptkonto aus, um Skonto für Mehrwertsteuercodes zu buchen, die dieser "Sachkontobuchungsgruppe" zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="b7a34-127">In the Vendor cash discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="b7a34-128">Das ist optional und wenn kein Konto eingegeben wird, wird das Hauptkonto auf Skontobasis verwendet.</span><span class="sxs-lookup"><span data-stu-id="b7a34-128">This is optional and if no account is entered,  the main account on Cash discount codes will be used.</span></span> <span data-ttu-id="b7a34-129">Es kann hilfreich sein, verschiedene Konten pro Sachkontobuchungsgruppe zu verwenden, wenn Sie die Option zum Stornieren der Mehrwertsteuer auf Skonten bei "Mehrwertsteuergruppen" verwenden.</span><span class="sxs-lookup"><span data-stu-id="b7a34-129">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="b7a34-130">Wählen Sie im Feld "Debitorenskonto" das Hauptkonto aus, um Skonto für Mehrwertsteuercodes zu buchen, die dieser "Sachkontobuchungsgruppe" zugeordnet sind.</span><span class="sxs-lookup"><span data-stu-id="b7a34-130">In the Customer case discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="b7a34-131">Das ist optional und wenn kein Konto eingegeben wird, wird das Hauptkonto für die "Skontocodes" verwendet.</span><span class="sxs-lookup"><span data-stu-id="b7a34-131">This is optional and if no account is entered, the main account on the Cash discount codes will be used.</span></span> <span data-ttu-id="b7a34-132">Es kann hilfreich sein, verschiedene Konten pro Sachkontobuchungsgruppe zu verwenden, wenn Sie die Option zum Stornieren der Mehrwertsteuer auf Skonten bei "Mehrwertsteuergruppen" verwenden.</span><span class="sxs-lookup"><span data-stu-id="b7a34-132">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
12. <span data-ttu-id="b7a34-133">Klicken Sie auf "Speichern".</span><span class="sxs-lookup"><span data-stu-id="b7a34-133">Click Save.</span></span>
