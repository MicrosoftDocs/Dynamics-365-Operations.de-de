# <a name="set-up-main-account-categories"></a>Hauptkontokategorien einrichten

[!include [task guide banner](../../includes/task-guide-banner.md)]

Hauptkontokategorien werden für die Standardberichte in der Finanzberichterstellung und in Power BI verwendet. Hauptkontokategorien, die standardmäßig erstellt werden, können umbenannt jedoch nicht gelöscht werden. Zusätzliche Kontokategorien können den erstellt und für Berichts- und Analysezwecke verwendet werden. Für diese Aufgabe wird das Demo-Unternehmen USMF verwendet.


## <a name="create-a-main-account-category"></a>Erstellen einer Hauptkontokategorie
1. Wechseln Sie zu Hauptbuch > Kontenplan > Konten > Hauptkontokategorien.
2. Klicken Sie auf "Neu".
3. Geben Sie einen eindeutigen Namen für die Hauptkontokategorie ein.
4. Geben Sie im Feld "Beschreibung" eine Beschreibung für die Hauptkontokategorie ein.
5. Wählen Sie im Feld "Hauptkontotyp" den Hauptkontotyp aus, der mit der Kategorie verknüpft werden soll.

## <a name="link-main-accounts-to-account-category"></a>Verknüpfen von Hauptkonten mit Kontokategorien
1. Klicken Sie auf "Hauptkonten verknüpfen".
2. Wählen Sie in der Liste die Hauptkonten, die mit der Hauptkontokategorie verknüpft werden sollen.
    * Das Zuweisen von Hauptkonten zu einer Hauptkontokategorie fasst die Salden der Konten zusammen wenn diese Kategorie für die Finanzberichte und Analysen verwendet wird.  
3. Aktivieren oder deaktivieren Sie die Option "Verknüpft", um die Hauptkonten auszuwählen.
4. Klicken Sie auf "OK".
5. Klicken Sie auf "Ja".
