[!include [task guide banner](../../includes/task-guide-banner.md)]

2. <span data-ttu-id="228e6-106">Wählen Sie das Feld aus, für das Sie ein Filterkriterium definieren wollen.</span><span class="sxs-lookup"><span data-stu-id="228e6-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="228e6-107">Geben Sie die Filterkriterien für das ausgewählte Feld ein.</span><span class="sxs-lookup"><span data-stu-id="228e6-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="228e6-108">Sie können einen einzelnen Wert oder einen Bereich filtern.</span><span class="sxs-lookup"><span data-stu-id="228e6-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="228e6-109">Wenn Sie einen Bereich definiert, stellen Sie sicher, dass die richtige Syntax verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="228e6-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="228e6-110">Die Werte sollten durch einen doppelten Punkt getrennt sein (..).</span><span class="sxs-lookup"><span data-stu-id="228e6-110">The values should be separated by a double period (..).</span></span>

9. <span data-ttu-id="228e6-116">Klicken Sie zum Ausführen der Abfrage auf OK.</span><span class="sxs-lookup"><span data-stu-id="228e6-116">Click OK to run the query.</span></span>

    * <span data-ttu-id="228e6-118">Verschiedene Schaltflächen über dem Raster können verwendet werden, um zusätzliche Informationen zum Datensatz des ausgewählten Belegs zu ermitteln.</span><span class="sxs-lookup"><span data-stu-id="228e6-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="228e6-119">Einige Schaltflächen sind möglicherweise nicht verfügbar, je nach der Buchungsart und den Merkmalen der Buchung.</span><span class="sxs-lookup"><span data-stu-id="228e6-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>

13. <span data-ttu-id="228e6-122">Schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="228e6-122">Close the page.</span></span>
