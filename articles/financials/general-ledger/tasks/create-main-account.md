# <a name="create-a-main-account"></a><span data-ttu-id="474c3-103">Erstellen eines Hauptkontos</span><span class="sxs-lookup"><span data-stu-id="474c3-103">Create a main account</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="474c3-104">Diese Aufgabenleitfaden führt Sie durch das Hinzufügen eines Hauptkontos zu einem vorhandenen Kontenplan.</span><span class="sxs-lookup"><span data-stu-id="474c3-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="474c3-105">Für diese Erfassung wird das Demo-Unternehmen USMF verwendet.</span><span class="sxs-lookup"><span data-stu-id="474c3-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="474c3-106">Wechseln Sie zu **Navigationsbereich > Module > Hauptbuch > Kontenplan > Konten > Hauptkonten**.</span><span class="sxs-lookup"><span data-stu-id="474c3-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="474c3-107">Klicken Sie auf **Neu**.</span><span class="sxs-lookup"><span data-stu-id="474c3-107">Click **New**.</span></span>
3. <span data-ttu-id="474c3-108">Geben Sie im Feld **Hauptkonto** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="474c3-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="474c3-109">Geben Sie im Feld **Name** einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="474c3-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="474c3-110">Wählen Sie im Feld **Hauptkontotyp** den Typ aus, der den Kontensaldo und den Ort in Finanzaufstellungen am besten abbildet.</span><span class="sxs-lookup"><span data-stu-id="474c3-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="474c3-111">Wählen sie in der Liste die Kontokategorie, zu der das Hauptkonto gehört.</span><span class="sxs-lookup"><span data-stu-id="474c3-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="474c3-112">Kontokategorien werden für Standard-Finanzberichte und Power BI-Dashboardinhalt verwendet.</span><span class="sxs-lookup"><span data-stu-id="474c3-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="474c3-113">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="474c3-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="474c3-114">Ändern Sie den standardmäßigen Sollsaldo oder Habensaldo.</span><span class="sxs-lookup"><span data-stu-id="474c3-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="474c3-115">Wählen Sie im Feld **Standardwährung** einen Wert aus der Liste der Währungen aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="474c3-116">Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="474c3-117">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="474c3-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="474c3-118">Schalten Sie die Erweiterung des Abschnitts **Juristische Person überschreibt** ein/aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="474c3-119">Klicken Sie auf **Hinzufügen**, um eine juristische Person auszuwählen.</span><span class="sxs-lookup"><span data-stu-id="474c3-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="474c3-120">Wählen Sie die juristische Person in der Liste aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="474c3-121">Klicken Sie auf **Hinzufügen**.</span><span class="sxs-lookup"><span data-stu-id="474c3-121">Click **Add**.</span></span>
15. <span data-ttu-id="474c3-122">Markieren Sie in der Liste die ausgewählte Zeile.</span><span class="sxs-lookup"><span data-stu-id="474c3-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="474c3-123">Aktivieren oder deaktivieren Sie das Kontrollkästchen **Ausgesetzt**.</span><span class="sxs-lookup"><span data-stu-id="474c3-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="474c3-124">Erweitern Sie den Bereich **Finanzberichterstellung**.</span><span class="sxs-lookup"><span data-stu-id="474c3-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="474c3-125">Klicken Sie im Feld **Wechselkurstyp** auf die Dropdown-Schaltfläche, um die Suche zu öffnen.</span><span class="sxs-lookup"><span data-stu-id="474c3-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="474c3-126">Wählen Sie in der Liste den **Wechselkurstyp für das Konto** aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="474c3-127">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="474c3-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="474c3-128">Wählen Sie im Feld **Währungsumrechnungstyp** die Methode für die Berechnung der Wechselkurse für das Konto aus.</span><span class="sxs-lookup"><span data-stu-id="474c3-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="474c3-129">Schließen Sie die Seite.</span><span class="sxs-lookup"><span data-stu-id="474c3-129">Close the page.</span></span>
