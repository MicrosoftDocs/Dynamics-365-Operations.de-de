-----------------------------

<span data-ttu-id="a51cc-108">Mithilfe von Überwachungsrichtlinien werden Spesenabrechnungen, Bestellungen und Kreditorenrechnungen ermittelt, die den von Ihnen als Überwachungsrichtlinienregeln definierten und konfigurierten Geschäftsregeln widersprechen.</span><span class="sxs-lookup"><span data-stu-id="a51cc-108">Audit policies are used to identify expense reports, purchase orders, and vendor invoices that don't comply with business rules that you define and configure as audit policy rules.</span></span>

<span data-ttu-id="a51cc-111">Jede Richtlinienregel wertet eine Reihe von Dokumenten aus.</span><span class="sxs-lookup"><span data-stu-id="a51cc-111">Each policy rule evaluates a set of documents.</span></span> <span data-ttu-id="a51cc-112">Die Richtlinienregel wählt die Dokumente aus, die sich im Datumsbereich für die Dokumentauswahl befinden und den angegebenen Kriterien entsprechen.</span><span class="sxs-lookup"><span data-stu-id="a51cc-112">The policy rule selects documents that are in the document selection date range and that match the specified criteria.</span></span> <span data-ttu-id="a51cc-113">Mit einer Richtlinienregel können z. B. > [!NOTE]
> <span data-ttu-id="a51cc-122">Hinweis: Bei Überwachungsrichtlinienregeln, die auf dem **Duplizieren**-Abfragetyp basieren, werden Verstöße nicht nach Richtlinienregel oder nach den Kriterien gruppiert, die auf der Seite **Anfragegruppierungskriterien** angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="a51cc-122">For audit policy rules that are based on a **Duplicate** query type, violations aren't grouped by policy rule or by the criteria that are specified on the **Case grouping criteria** page.</span></span> <span data-ttu-id="a51cc-123">Die Gruppierung richtet sich stattdessen nach den in die Überwachungsrichtlinienregel integrierten Kriterien.</span><span class="sxs-lookup"><span data-stu-id="a51cc-123">Instead, they are grouped by the criteria that are built into the audit policy rule.</span></span> <span data-ttu-id="a51cc-124">Wenn beispielsweise eine Richtlinienregel Spesenrechnungen auf doppelte Ausgaben auswertet, bei denen Betrag, Händlerkennung und Datum übereinstimmen, stellen alle Ausgaben mit denselben Werten in diesen Feldern eine Anfrage dar.</span><span class="sxs-lookup"><span data-stu-id="a51cc-124">For example, if a policy rule evaluates expense reports for duplicate expenses of the same amount, merchant ID, and date, all expenses that have the same values in those fields will be one case.</span></span> <span data-ttu-id="a51cc-125">Alle Ausgaben, die verschiedene Werte besitzen, sind getrennte Anfragen.</span><span class="sxs-lookup"><span data-stu-id="a51cc-125">Any expenses that have different values will be a separate case.</span></span>

## <a name="document-selection-date-ranges"></a><span data-ttu-id="a51cc-127">Datumsbereich für die Dokumentauswahl</span><span class="sxs-lookup"><span data-stu-id="a51cc-127">Document selection date ranges</span></span>
<span data-ttu-id="a51cc-128">Beim Ausführen einer Überwachungsrichtlinie wählt jede Richtlinienregel Dokumente vom angegebenen Typ aus, die ein Datum aus dem Datumsbereich für die Dokumentauswahl aufweisen.</span><span class="sxs-lookup"><span data-stu-id="a51cc-128">When an audit policy is run, each policy rule selects documents of the specified type that have a date that is in the document selection date range.</span></span> <span data-ttu-id="a51cc-129">Der Datumsbereich für die Dokumentauswahl wird auf der Seite **Zusätzliche Optionen** festgelegt.</span><span class="sxs-lookup"><span data-stu-id="a51cc-129">The document selection date range is specified on the **Additional options** page.</span></span> <span data-ttu-id="a51cc-130">Vielen Dokumenten sind mehrere Datumsangaben zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="a51cc-130">Many documents have more than one date associated with them.</span></span> <span data-ttu-id="a51cc-131">Das Datumsfeld, das der Überwachungsrichtlinienregel verwendet, wird auf der Seite **Richtlinienregeltyp** angegeben.</span><span class="sxs-lookup"><span data-stu-id="a51cc-131">The date field that the audit policy rule uses is specified on the **Policy rule type** page.</span></span>
