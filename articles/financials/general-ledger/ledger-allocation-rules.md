# <a name="ledger-allocation-rules"></a><span data-ttu-id="78f96-104">Sachkonto-Zuordnungsregeln</span><span class="sxs-lookup"><span data-stu-id="78f96-104">Ledger allocation rules</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="78f96-105">Dieser Artikel enthält Informationen zu Sachkonto-Zuordnungsregeln.</span><span class="sxs-lookup"><span data-stu-id="78f96-105">This article provides information about ledger allocation rules.</span></span> <span data-ttu-id="78f96-106">Er beschreibt die verschiedenen Komponenten dieser Zuordnungsregeln und der Zuordnungsmethoden, die für sie verwendet werden können.</span><span class="sxs-lookup"><span data-stu-id="78f96-106">It describes the various components of these allocation rules and the allocation methods that can be used for them.</span></span>

<span data-ttu-id="78f96-107">Sachkonto-Zuordnungsregeln werden verwendet, um Zuordnungserfassungen und Kontoeinträge für die Zuordnung von Sachkontosalden oder -festen Beträgen automatisch zu berechnen und zu generieren.</span><span class="sxs-lookup"><span data-stu-id="78f96-107">Ledger allocation rules are used to automatically calculate and generate allocation journals and account entries for the allocation of ledger balances or fixed amounts.</span></span> <span data-ttu-id="78f96-108">Zuordnungsmethoden können variabel oder fest sein.</span><span class="sxs-lookup"><span data-stu-id="78f96-108">Allocation methods can be variable or fixed.</span></span> <span data-ttu-id="78f96-109">Folgende Zuordnungsmethoden können für Sachkontozuordnungsregeln verwendet werden:</span><span class="sxs-lookup"><span data-stu-id="78f96-109">The following allocation methods can be used for ledger allocation rules:</span></span>

-   <span data-ttu-id="78f96-110">**Basis** - Diese variable Methode wird verwendet, wenn die Verteilung vom tatsächlichen Sachkontensaldo abhängt (auf Grundlage von Filterkriterien).</span><span class="sxs-lookup"><span data-stu-id="78f96-110">**Basis** – This variable method is used when the allocation depends on the actual ledger balance, based on filter criteria.</span></span> <span data-ttu-id="78f96-111">Beispielsweise können Werbeausgaben eines Unternehmens zum Umsatz der einzelnen Abteilungen und dem Gesamtumsatz aller Abteilungen zugeordnet werden.</span><span class="sxs-lookup"><span data-stu-id="78f96-111">For example, advertising expenses can be allocated based on each department's sales in proportion to the total departmental sales.</span></span>
-   <span data-ttu-id="78f96-112">**Fester Prozentsatz** und **Festes Gewicht** – Für diese Methoden werden der Prozentsatz oder das Gewicht direkt für die Regel definiert.</span><span class="sxs-lookup"><span data-stu-id="78f96-112">**Fixed percentage** and **Fixed weight** – For these methods, the allocation percentage or weight is defined directly for the rule.</span></span> <span data-ttu-id="78f96-113">Beispielsweise können Werbekosten zugewiesen werden, damit Abteilung A 70 Prozent der Werbeausgaben zugeordnet werden und Abteilung B 30 Prozent zugeordnet werden.</span><span class="sxs-lookup"><span data-stu-id="78f96-113">For example, advertising expenses can be allocated so that Department A receives 70 percent of the advertising expense and Department B receives 30 percent.</span></span>
-   <span data-ttu-id="78f96-114">**Gleich** – Diese Methode verteilt den Betrag gleichmäßig jedem definierten Ziel.</span><span class="sxs-lookup"><span data-stu-id="78f96-114">**Equally** – This method distributes the amount equally to each defined destination.</span></span> <span data-ttu-id="78f96-115">Wenn beispielsweise Ziele für Abteilung A und Abteilung B definiert werden, können Werbekosten so zugewiesen werden, das Abteilung A und Abteilung B 50 Prozent der Werbeausgaben empfangen.</span><span class="sxs-lookup"><span data-stu-id="78f96-115">For example, if destinations are defined for Department A and Department B, advertising expenses can be allocated so that both Department A and Department B receive 50 percent of the advertising expense.</span></span>

<span data-ttu-id="78f96-116">Ist die Zuordnungsmethode Basis als Zuordnungsregel festgelegt ist, muss außerdem eine separate Basisregel für die Sachkontozuordnung erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="78f96-116">If Basis is used as the allocation method for an allocation rule, you must also define separate ledger allocation basis rules.</span></span> <span data-ttu-id="78f96-117">Der Prozess "Zuordnungsanforderung verarbeiten" bietet Benutzer die Möglichkeit Sachkontozuordnungsregel verarbeiten und die resultierenden Zuordnungserfassungseinträge in der Vorschau anzeigen, bevor sie entweder die berechneten Zuweisungen buchen oder löschen.</span><span class="sxs-lookup"><span data-stu-id="78f96-117">The "Process allocation request" process lets users process the ledger allocation rule and preview the resulting allocation journal entries before they either post or delete the calculated allocations.</span></span>

## <a name="components-of-ledger-allocation-rules"></a><span data-ttu-id="78f96-118">Komponenten von Zuweisungserfassungsregeln</span><span class="sxs-lookup"><span data-stu-id="78f96-118">Components of ledger allocation rules</span></span>
<span data-ttu-id="78f96-119">Jede Zuordnungsregel besitzt vier Komponenten: Allgemeines, Quelle, Ziel und Gegenkonto.</span><span class="sxs-lookup"><span data-stu-id="78f96-119">Each allocation rule has four components: general, source, destination, and offset.</span></span> <span data-ttu-id="78f96-120">Eine zusätzliche Komponente, die Sachkontozuordnungsbasisregel, ist erforderlich, wenn "Basis" als Zuordnungsmethode verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="78f96-120">An additional component, ledger allocation bases rules, is required if Basis is used as the allocation method.</span></span> <span data-ttu-id="78f96-121">Jede Komponente enthält eine entscheidende Information, die für die Verarbeitung der Zuordnungen erforderlich sind.</span><span class="sxs-lookup"><span data-stu-id="78f96-121">Each component provides a critical piece of the information that is required in order to process allocations.</span></span>

-   <span data-ttu-id="78f96-122">**Allgemein** – In dieser Komponente gibt der Benutzer Optionen wie die Zuordnungsmethode, Intercompany-Regeleinstellungen und den Bestimmungsbereich der Regel ein.</span><span class="sxs-lookup"><span data-stu-id="78f96-122">**General** – This component is where the user specifies options such as the allocation method, intercompany rule settings, and whether the rule is active.</span></span>
-   <span data-ttu-id="78f96-123">**Quelle** – Diese Komponente dient zur Angabe der Quelldaten für die Zuordnung durch den Benutzer.</span><span class="sxs-lookup"><span data-stu-id="78f96-123">**Source** – This component is where the user specifies the source data for the allocation.</span></span> <span data-ttu-id="78f96-124">Zuteilung können auf Sachkontosalden (**Datenquelle** = **Sachkonto**) oder feste Beträge basieren (**Datenquelle** = **Fester Wert**).</span><span class="sxs-lookup"><span data-stu-id="78f96-124">Allocation can be based on ledger balances (**Data source** = **Ledger**) or fixed amounts (**Data source** = **Fixed value**).</span></span> <span data-ttu-id="78f96-125">Wenn **Datenquelle** auf **Sachkonto** festgelegt ist, müssen Quellfilterkriterien für die Sachkontozuordnungsregel definiert werden (beispielsweise für die Werbekosten).</span><span class="sxs-lookup"><span data-stu-id="78f96-125">When **Data source** is set to **Ledger**, source filter criteria must be defined for the ledger allocation rule (for example, for the advertising expenses).</span></span>
-   <span data-ttu-id="78f96-126">**Ziel** – Mithilfe dieser wird definiert, wie das Ergebnis der Zuweisungsberechnung auf die verteilt und aufgeteilt werden soll.</span><span class="sxs-lookup"><span data-stu-id="78f96-126">**Destination** – This component defines how the result of the allocation calculation should be distributed and accounted for.</span></span> <span data-ttu-id="78f96-127">Beispielsweise kann es eine Zielposition für jede Abteilung geben.</span><span class="sxs-lookup"><span data-stu-id="78f96-127">For example, there can be one destination line for each department.</span></span>
-   <span data-ttu-id="78f96-128">**Gegenkonto** – Diese Komponente definiert, wie Hauptkonten und Dimensionen für die Gegenbuchungen bestimmt werden, die die Zieleinträge ausgleichen.</span><span class="sxs-lookup"><span data-stu-id="78f96-128">**Offset** – This component defines how main accounts and dimensions should be determined for the offset entries that balance the destination entries.</span></span> <span data-ttu-id="78f96-129">Benutzerdefinierte Optionen werden in der Regel anstelle von Konten und Dimensionen auf Quellbasis verwendet.</span><span class="sxs-lookup"><span data-stu-id="78f96-129">User-defined options are typically used instead of accounts and dimensions that are based on the source.</span></span> <span data-ttu-id="78f96-130">Wenn **Datenquelle** auf **Fester Wert** festgelegt ist, kann **Quelle** nicht als Option verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="78f96-130">When **Data source** is set to **Fixed value**, **Source** can't be used as an option.</span></span>
-   <span data-ttu-id="78f96-131">**Sachkontozuteilungsbasisregeln** – Diese Regeln verwenden ihre eigenen Quellfilterkriterien, um festzulegen, welche Sachkontosalden für die Zuweisung verwendet werden sollen (beispielsweise, Umsatzerlös pro Abteilung).</span><span class="sxs-lookup"><span data-stu-id="78f96-131">**Ledger allocation basis rules** – These rules use their own source filter criteria to determine which ledger balances should be used for allocation (for example, the revenue per department).</span></span> <span data-ttu-id="78f96-132">Jede Zuordnungsbasisregel kann für eine Vielzahl von Zuordnungsregeln verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="78f96-132">Each allocation basis rule can be used with multiple allocation rules.</span></span>
