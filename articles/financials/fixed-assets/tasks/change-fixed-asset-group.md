# <a name="change-a-fixed-asset-group"></a>Eine Anlagengruppe ändern

[!include [task guide banner](../../includes/task-guide-banner.md)]

Die Anlage muss der korrekten Anlagengruppe zugewiesen werden. Die feste Anlagegruppe wird verwendet, wenn Sie:

 - Erstellen von Abfragen und Berichten

 - Einrichten neuer Anlagen

 - Sachkonten integrieren und Anlagetransaktionen auf die entsprechenden Sachkonten buchen

Dieser Leitfaden verwendet das Demounternehmen USMF.

1. Wechseln Sie zu Anlagen > Anlagen > Anlagen.
2. Wählen Sie die Anlage, für die Sie die Anlagengruppe ändern möchten.
3. Klicken Sie auf "Anlagengruppe ändern"
4. Geben Sie im Feld "Neue Gruppe" einen Wert ein oder wählen Sie einen Wert aus.
5. Aktivieren Sie diese Option, um der ausgewählten Anlage eine Anlagennummer zuzuweisen.
    * Das Feld Anlagennummer ist verfügbar, wenn die Option "Neue Anlagennummer" festgelegt ist.   Wenn Sie die automatische Nummerierung von Anlagen eingerichtet haben, wird in diesem Feld die nächste verfügbare Anlagennummer angezeigt. Die Nummer kann geändert werden.   Wenn Sie die manuelle Nummerierung eingerichtet haben, ist dieses Feld leer, und Sie müssen die neue Anlagennummer eingeben.     
6. Klicken Sie auf "OK".
7. Klicken Sie auf "Ja".
