# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>Anlagenabschreibung über juristische Personen hinweg berechnen

[!include[task guide banner](../../includes/task-guide-banner.md)]

Verwenden Sie diese Prozedur, um die Anlagengruppe zu ändern, der die Anlage zugewiesen ist. Die Anlagen muss der richtigen Anlagengruppe zugewiesen werden. Die Anlagengruppe wird verwendet, wenn Sie Berichte und Abfragen erstellen, neue Anlagen einrichten und Sachkonten und Anlagenbuchungen für die entsprechenden Sachkonten integrieren.

Für diese Erfassung wird das Demo-Unternehmen USMF verwendet.

1. Wechseln Sie zu Anlagen > Anlagen > Anlagen.
2. Wählen Sie die Anlage, für die Sie die Anlagengruppe ändern möchten.
3. Klicken Sie auf "Anlagengruppe ändern"
4. Geben Sie im Feld "Neue Gruppe" einen Wert ein oder wählen Sie einen Wert aus.
5. Legen Sie die Option "Neue Anlagennummer" auf "Ja" fest, um der ausgewählten Anlage eine Anlagennummer zuzuweisen.
    * Das Feld Anlagennummer ist verfügbar, wenn die Option "Neue Anlagennummer" auf "Ja" festgelegt ist.   Wenn Sie die automatische Nummerierung von Anlagen eingerichtet haben, wird in diesem Feld die nächste verfügbare Anlagennummer angezeigt. Die Nummer kann geändert werden.   Wenn Sie die manuelle Nummerierung für Anlagen eingerichtet haben, ist dieses Feld leer, und Sie müssen die neue Anlagennummer eingeben.  
6. Klicken Sie auf "OK".
7. Klicken Sie auf "Ja".
