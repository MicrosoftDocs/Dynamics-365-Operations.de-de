# <a name="configure-base-entities"></a>Basisentitäten konfigurieren 

[!include[banner](../includes/banner.md)]


Die Kostenrechnung ist ein unabhängiges Modul, das erfordert, dass die folgenden Basisentitäten ordnungsgemäß in Dynamics 365 for Finance and Operations, Enterprise edition, konfiguriert werden.

-  Währung
-  Kurs
-  Steuerkalender  

Bevor Sie das Modul Kostenrechnung verwenden, muss zudem eine Währung, ein Wechselkurs und ein Steuerkalender für das Unternehmen festgelegt sein. 

Weitere Informationen über diese Entitäten finden Sie im e-Learning-Schulungskurs [Modul 2 Währung und Wechselkurs einrichten](https://mbspartner.microsoft.com/AX/CourseModules/1215) (erfordert CustomerSource-Konto).
