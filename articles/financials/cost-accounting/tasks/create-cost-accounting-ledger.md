[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="60497-104">Ein Kostenrechnungssachkonto stellt die Gesamtberichtseinheit dar.</span><span class="sxs-lookup"><span data-stu-id="60497-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="60497-105">Es wird durch eine Kostenfaktordimension, statistische Dimension, Steuerkalenders und Währung definiert.</span><span class="sxs-lookup"><span data-stu-id="60497-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="60497-106">Es ist agnostisch zum Konzept der juristischen Personen.</span><span class="sxs-lookup"><span data-stu-id="60497-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="60497-107">Eine juristische Person und die Daten können vielen Kostenrechnungssachkonten zugeordnet werden.</span><span class="sxs-lookup"><span data-stu-id="60497-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="60497-108">Für diese Aufzeichnung wird das Demo-Datenunternehmen USP2 verwendet.</span><span class="sxs-lookup"><span data-stu-id="60497-108">This recording uses the USP2 demo data company.</span></span>

3. <span data-ttu-id="60497-111">Geben Sie im Feld "Name" einen Wert ein.</span><span class="sxs-lookup"><span data-stu-id="60497-111">In the Name field, type a value.</span></span>
