[!include [banner](../includes/banner.md)]

<span data-ttu-id="6676e-105">Die EU-Direktive bietet Bestimmungen, die von allen Mitgliedsstaaten befolgt werden müssen, in Bezug auf Rechnungsregeln und Informationen, die bei der Rechnungserstellung beachtet werden müssen.</span><span class="sxs-lookup"><span data-stu-id="6676e-105">EU Directive provides regulations to be followed by all Member States regarding invoice rules and information that must be included when creating invoices.</span></span> <span data-ttu-id="6676e-106">Die EU-Direktive führt Informationen auf, die von Lieferanten für seine Kunden in der EU erforderlich sind.</span><span class="sxs-lookup"><span data-stu-id="6676e-106">EU Directive lists the information that is required by suppliers for their customers in the EU.</span></span> <span data-ttu-id="6676e-107">Beispielsweise könnte eine Rechnung eine Referenz enthalten, die den rechtlichen Gründe für eine Nullbewertung der Mehrwertsteuer oder eine Ausnahme beschreibt.</span><span class="sxs-lookup"><span data-stu-id="6676e-107">For example, an invoice could include a reference that describes the legal grounds for a VAT zero-rating or an exemption.</span></span> <span data-ttu-id="6676e-108">Wenn mit einer Rechnung eine zuvor ausgestellte Rechnung angepasst wird, beispielsweise eine Gutschrift, muss eine eindeutige Referenz zur ursprünglichen Rechnung vorgenommen werden.</span><span class="sxs-lookup"><span data-stu-id="6676e-108">If an invoice adjusts an earlier issued invoice, such as a credit note, an unambiguous reference should be made to the original invoice.</span></span>

- <span data-ttu-id="6676e-109">Um eine Beschreibung der Mehrwertsteuercodes auf einer Verkaufsrechnung, Freitextrechnung oder einem Projektrechnungsbericht zu drucken, richten Sie den Referenztext der Steuer-Direktive auf den Mehrwertsteuercodes ein.</span><span class="sxs-lookup"><span data-stu-id="6676e-109">To print a description of sales tax codes on a sales invoice, free text invoice, or project invoice report, set up the Tax Directive reference text on the sales tax codes.</span></span>
- <span data-ttu-id="6676e-110">Für Gutschriften werden die Nummer und das Datum der ursprünglichen Rechnung zusammen mit dem Grund für die Berichtigung gedruckt.</span><span class="sxs-lookup"><span data-stu-id="6676e-110">For credit notes, the number and date of the original invoice are printed along with the reason for correction.</span></span>
