# <a name="adjust-project-transactions"></a>Projektbuchungen regulieren

[!include [task guide banner](../../includes/task-guide-banner.md)]

In diesem Aufgabenleitfaden erfahren Sie, wie Sie eine Buchung an ein Projekt anpassen. Sie können Projektbuchungen jedoch einzeln regulieren oder eine Auswahl aus einer Liste aller Projektbuchungen treffen. Diese Aufgabe wird normalerweise vom Projekt-Supervisor durchgeführt. Dieser Aufgabenleitfaden verwendet den USSI-Datensatz.

1. Wechseln Sie zu "Projektverwaltung und -verrechnung" > "Projekte" > "Alle Projekte". 
2. Markieren Sie in der Liste die ausgewählte Zeile. 
3. Klicken Sie auf Buchungen regulieren. 
4. Klicken Sie auf Auswählen. 
5. Wählen Sie "Ja" im Feld "Stunde". 
6. Geben Sie in das Feld "Bis Datum" ein Datum ein. 
7. Klicken Sie auf "OK". 
8. Markieren Sie in der Liste die ausgewählte Zeile. 
9. Klicken Sie auf "Regulieren". 
10. Markieren Sie in der Liste die ausgewählte Zeile. 
11. Geben Sie im Feld "Positionseigenschaft" einen Wert ein, oder wählen Sie einen Wert aus. 
12. Klicken Sie auf "OK". 
13. Markieren Sie in der Liste die ausgewählte Zeile. 
14. Geben Sie im Feld "Kostenpreis" eine Zahl ein. 
15. Klicken Sie auf "Buchen".
