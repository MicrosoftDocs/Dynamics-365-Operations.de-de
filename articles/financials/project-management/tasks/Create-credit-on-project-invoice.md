# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="b834e-103">Erstellen einer Gutschrift auf Projektrechnungen</span><span class="sxs-lookup"><span data-stu-id="b834e-103">Create a credit note on project invoices</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="b834e-104">Wechseln Sie zu "Projektverwaltung und -verrechnung" > "Projekte" > "Alle Projekte".</span><span class="sxs-lookup"><span data-stu-id="b834e-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="b834e-105">Suchen Sie in der Liste den gewünschten Datensatz, und wählen Sie ihn aus.</span><span class="sxs-lookup"><span data-stu-id="b834e-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="b834e-106">Wählen Sie die Projektkennung aus, die für die Gutschrift verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="b834e-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="b834e-107">Klicken Sie in der Liste auf den Link in der ausgewählten Zeile.</span><span class="sxs-lookup"><span data-stu-id="b834e-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="b834e-108">Klicken Sie im Aktivitätsbereich auf Verwalten.</span><span class="sxs-lookup"><span data-stu-id="b834e-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="b834e-109">Klicken Sie auf "Rechnungserfassung".</span><span class="sxs-lookup"><span data-stu-id="b834e-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="b834e-110">Wählen Sie die Buchung für Gutschrift.</span><span class="sxs-lookup"><span data-stu-id="b834e-110">Select the transaction for credit note.</span></span> <span data-ttu-id="b834e-111">Markieren Sie in der Liste die ausgewählte Zeile.</span><span class="sxs-lookup"><span data-stu-id="b834e-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="b834e-112">Klicken Sie auf Funktionen.</span><span class="sxs-lookup"><span data-stu-id="b834e-112">Click Functions.</span></span> 
8. <span data-ttu-id="b834e-113">Klicken Sie auf "Für Gutschrift auswählen".</span><span class="sxs-lookup"><span data-stu-id="b834e-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="b834e-114">Wählen Sie die Buchung für Gutschrift durch Aktivieren des Kontrollkästchens aus.</span><span class="sxs-lookup"><span data-stu-id="b834e-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="b834e-115">Klicken Sie auf "OK".</span><span class="sxs-lookup"><span data-stu-id="b834e-115">Click OK.</span></span> 

<span data-ttu-id="b834e-116">Nach diesem Schritt ist die Gutschrift für Rechnungsvorschlag verfügbar.</span><span class="sxs-lookup"><span data-stu-id="b834e-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
