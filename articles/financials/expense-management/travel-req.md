# <a name="travel-requisitions"></a><span data-ttu-id="bdd07-104">Reiseanforderungen</span><span class="sxs-lookup"><span data-stu-id="bdd07-104">Travel requisitions</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="bdd07-105">Eine *Reiseanforderung* ist ein Quelldokument, das Ausgaben dokumentiert, die für eine Reise anfallen.</span><span class="sxs-lookup"><span data-stu-id="bdd07-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="bdd07-106">Eine Reiseanforderung wird zur Prüfung gesendet und kann danach zur Autorisierung von Ausgaben verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="bdd07-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="bdd07-107">Ihre Organisation verlangt u. U. das Senden einer Reiseanforderung, bevor ein Mitarbeiter eine Ausgabe tätigt, die der Organisation in Rechnung gestellt wird.</span><span class="sxs-lookup"><span data-stu-id="bdd07-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="bdd07-108">Dies ist unabhängig davon, ob der Mitarbeiter eine Kreditkarte des Unternehmens mit Ausgaben belastet, ob er Bargeld ausgibt, das er aus einem Barvorschuss erhalten hat, oder ob ihm Auslagen entstehen, die von der Organisation erstattet werden.</span><span class="sxs-lookup"><span data-stu-id="bdd07-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="bdd07-109">Zur Unterstützung der Budgetsteuerung können Reiseanforderungen und -richtlinien verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="bdd07-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="bdd07-110">Wenn die Organisation beispielsweise an einem Festpreisprojekt arbeitet, das Reisen erfordert, müssen die Reisekosten der Projektteammitglieder in das Budgets des Projekts passen.</span><span class="sxs-lookup"><span data-stu-id="bdd07-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="bdd07-111">Die Anforderung, dass Reisekosten genehmigt werden müssen, bevor sie entstehen, kann dazu beitragen, dass das Projekt die Budgetgrenzen nicht überschreitet.</span><span class="sxs-lookup"><span data-stu-id="bdd07-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="bdd07-112">Sie können Richtlinien für **Reiseanforderungen** auf der Listenseite  einrichten.</span><span class="sxs-lookup"><span data-stu-id="bdd07-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
