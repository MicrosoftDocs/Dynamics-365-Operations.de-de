# <a name="travel-requisitions"></a>Reiseanforderungen

[!include [banner](../includes/banner.md)]

Eine *Reiseanforderung* ist ein Quelldokument, das Ausgaben dokumentiert, die für eine Reise anfallen. Eine Reiseanforderung wird zur Prüfung gesendet und kann danach zur Autorisierung von Ausgaben verwendet werden.

Ihre Organisation verlangt u. U. das Senden einer Reiseanforderung, bevor ein Mitarbeiter eine Ausgabe tätigt, die der Organisation in Rechnung gestellt wird. Dies ist unabhängig davon, ob der Mitarbeiter eine Kreditkarte des Unternehmens mit Ausgaben belastet, ob er Bargeld ausgibt, das er aus einem Barvorschuss erhalten hat, oder ob ihm Auslagen entstehen, die von der Organisation erstattet werden.

Zur Unterstützung der Budgetsteuerung können Reiseanforderungen und -richtlinien verwendet werden. Wenn die Organisation beispielsweise an einem Festpreisprojekt arbeitet, das Reisen erfordert, müssen die Reisekosten der Projektteammitglieder in das Budgets des Projekts passen. Die Anforderung, dass Reisekosten genehmigt werden müssen, bevor sie entstehen, kann dazu beitragen, dass das Projekt die Budgetgrenzen nicht überschreitet.

Sie können Richtlinien für **Reiseanforderungen** auf der Listenseite  einrichten.
