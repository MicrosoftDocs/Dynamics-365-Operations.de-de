# <a name="view-an-expense-report"></a><span data-ttu-id="55aa6-103">Anzeigen einer Spesenabrechnung</span><span class="sxs-lookup"><span data-stu-id="55aa6-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="55aa6-104">Wenn Sie eine Abfrage vornehmen, um Buchungen im Hauptbuch oder Beschaffung anzeigen, könnte die Abfrage zu vielen Informationen zurückgeben.</span><span class="sxs-lookup"><span data-stu-id="55aa6-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="55aa6-105">Daher sollten Sie die Suche einschränken.</span><span class="sxs-lookup"><span data-stu-id="55aa6-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="55aa6-106">Um die benötigten Informationen abzurufen, können Sie Detailinformationen zu einer Buchung und die dazugehörige Spesenabrechnung anzeigen.</span><span class="sxs-lookup"><span data-stu-id="55aa6-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="55aa6-107">Bei Spesenabrechnungen können Sie für eine Buchung die ursprüngliche Abrechnung anzeigen, die in der Ausgabenverwaltung gespeichert ist.</span><span class="sxs-lookup"><span data-stu-id="55aa6-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="55aa6-108">Zusätzlich zu den Gesamtausgaben können Sie die nach Artikel aufgeschlüsselten Buchungen, Buchungen, für die eine Mehrwertsteuererstattung möglich ist, und die an die Ausgaben angefügten Belege anzeigen.</span><span class="sxs-lookup"><span data-stu-id="55aa6-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="55aa6-109">Wenn Sie Detailinformationen zur ursprünglichen Spesenabrechnung anzeigen möchten, für die eine Buchung angezeigt wird, wählen Sie die Buchung aus, und wählen Sie dann **Originaldokument** aus.</span><span class="sxs-lookup"><span data-stu-id="55aa6-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="55aa6-110">Beim Anzeigen der einzelnen Buchungen können Sie auch alle zusätzlichen Daten einsehen, die an eine Buchung angefügt sind.</span><span class="sxs-lookup"><span data-stu-id="55aa6-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="55aa6-111">Für aufgeschlüsselte Hotelkosten können Sie beispielsweise eine ausführliche Liste der einzelnen Kosten anzeigen, wie Internetnutzung, Telefongespräche und Zimmerservice.</span><span class="sxs-lookup"><span data-stu-id="55aa6-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
