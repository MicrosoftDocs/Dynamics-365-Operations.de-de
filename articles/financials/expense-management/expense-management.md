# <a name="expense-management"></a>Ausgabenverwaltung

[!include [banner](../includes/banner.md)]

Sie können die Ausgabenverwaltung nutzen, um einen integrierten Workflows zu erstellen, wo Sie Zahlungsmethoden speichern, Kreditkartentransaktionen importieren und Geld, das Mitarbeiter ausgeben, wenn Sie geschäftliche Ausgaben geltend machen, nachverfolgen können. Sie haben auch die Möglichkeit zum Erstellen von Ausgabenrichtlinien sowie zum Automatisieren der Reisekostenerstattung.

Weitere Informationen finden Sie unter [Konfigurieren der Ausgabenverwaltung](plan-expense-management.md).

#### <a name="expense-report-entry"></a>Eingabe von Ausgabenrechnungen

Ausgabenberichtseingabe wurde neu gestaltet, um die Erfahrung zu vereinfachen und die Zeit zu reduzieren, die benötigt wird, um Ausgabenabrechnungen abzuschließen. Sie können diese Funktionen in der Funktionsverwaltung aktivieren. Sie können eine neue Einrichtungsseite hinzufügen, um die Sichtbarkeit der Ausgabenenfeldern zu konfigurieren und zu definieren, welche Daten erforderlich, optional oder nicht verfügbar sind, wenn Ausgabenbereichte eingegeben werden. Wenn diese Funktion aktiviert ist, steht ein neuer Ausgabenenarbeitsbereich bereit. Der Arbeitsbereich ersetzt den vorherigen Ausgabenenarbeitsbereich und ist die Startseite für die verbesserte  Eintragserfahrung. verbesserte.

Weitere Informationen finden Sie unter [Ausgabenberichte neu gestaltet](ExpenseWorkspaceNew.md).
